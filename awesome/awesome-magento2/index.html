    Tools¶

    • n98-magerun2 -适用于Magento 2的CLI瑞士军刀.
    • Documentation Search for Alfred -将官方的Magento 2搜索轻松集成到您的Alfred工作流程中.
    • VSCode Magento Dev Search -更快地搜索Magento DevDocs和Magento StackExchange.
    • Tablerates Generator -使用在线工具生成Tablerate.
    • Pestle -Alan Storm的代码生成工具.
    • Mage2Gen -在线模块创建者.
    • Magento 2 Code Generator -Juan Alonso的代码生成器.
    • Magento 2 Docker -专注于开发过程，在Mac和Linux上更快.
    • Magento 2 Bitnami VM -专注于开发过程，效果很好.
    • Mage Chrome Toolbar -由Magento 2提供的必备Chrome扩展程序 MageSpecialist.
    • convertm1m2 -一个简单的脚本，将扩展从Magento1转换为Magento 2.
    • Magento 2 data migration tool -官方的Magento 1到Magento 2迁移工具
    • Quickorder -通过输入SKU和数量来启用批量订单创建.
    • Cron Scheduler -在管理面板中运行cron的概述.
    • QuickDevBar Magento2的开发人员工具栏
    • MageSpecialist DevTools for Magento 2 开发人员工具栏
    • mage2.docker -Nginx（Pagespeed），MariaDB，PHP，Redis，带有Magento 2安装程序的Elastic Search Configurator（更新：2020年4月19日）
    • markshust/docker-magento -Mark Shust的Magento Docker配置
    • Warden -CLI工具，用于通过docker-compose环境工作 David Alger
    • AmpersandHQ/ampersand-magento2-upgrade-patch-helper -帮助脚本，通过检测替代来帮助升级magento 2网站.
    • PhpStorm Magento2 Extension -官方的PhpStorm Magento2扩展程序.
    • PhpInsights -PHP质量检查（带有Magento2预设）
    • nuzil/magento-postman -Magento2的邮递员收藏
    • Tango -用于分析访问日志的CLI
    • Masquerade -Faker驱动，基于配置，与平台无关，与语言环境兼容的数据伪造工具
    • Magento 2 Composer patches helper -Chrome扩展程序，可为vaimo / composer-patches创建可复制复制的作曲家补丁定义.
    • extdn/installer-m2 -Magento 2的通用扩展安装程序，由 Magento Extension Developers Network
    • mage2click/m2c -具有mutagen.io同步的Magento Docker开发环境，用于macOS上的文件和Linux上的本机文件系统挂载 mage2.click
    • Migrate DB Magento 2 Commerce to Magento 2 Open-Source
