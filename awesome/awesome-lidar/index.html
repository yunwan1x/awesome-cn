    Awesome lidar

    Awesome LIDAR ¶

    > 精选的超棒激光雷达传感器及其应用列表.

    LIDAR是一种遥感传感器，它使用激光以~cm 的精度测量周围环境. 感官数据通常称为点云，这意味着 3D 或 2D 中的数据点集. 该列表包含硬件、数据集、点云处理算法、点云框架、模拟器等.

    欢迎投稿！ 请 check out 我们的指导方针.

    Conventions¶

    • 任何带有 OctoCat 的列表项 :octocat: 都有一个 GitHub 存储库或组织
    • 任何带有 RedCircle 的列表项：red_circle：有 YouTube 视频或频道
    • 任何带有论文的列表项：报纸：有科学论文或详细描述

    Datasets¶

    • Ford Dataset - 数据集带有时间戳，包含来自所有传感器的原始数据、校准值、姿态轨迹、地面真实姿态和 3D 地图. 数据与机器人操作系统 (ROS) 兼容.
    • Paper
    • GitHub repository :octocat:
    • Audi A2D2 Dataset - 该数据集具有 2D 语义分割、3D 点云、3D 边界框和车辆总线数据.
    • Paper
    • Waymo Open Dataset - 数据集包含独立生成的激光雷达和相机数据标签，而不仅仅是投影.
    • Oxford RobotCar - 牛津 RobotCar 数据集包含超过 100 次重复通过英国牛津的一致路线，这些路线在一年多的时间里被捕获.
    • YouTube channel
    • Paper
    • EU Long-term Dataset - 该数据集是使用我们的机器人汽车（当然在人类驾驶模式下）收集的，配备了多达 11 个异构传感器，位于法国蒙贝利亚尔的市中心（用于长期数据）和郊区（用于环形交叉路口数据）. 根据法国交通规则，车速限制在 50 公里/小时.
    • NuScenes - 用于自动驾驶的公共大规模数据集.
    • Paper
    • Lyft - 由配备激光雷达和摄像头的福特 Fusion 车队收集的公共数据集.
    • KITTI - Widespread public dataset, pirmarily focusing on computer vision applications, but also contains LIDAR point cloud.
    • Semantic KITTI - 用于语义和全景场景分割的数据集.
    • YouTube video
    • CADC - Canadian Adverse Driving Conditions Dataset - 在恶劣天气条件下（下雪天气）自动驾驶的公共大规模数据集.
    • Paper
    • UofTPed50 Dataset - 多伦多大学，aUToronto 的自动驾驶汽车数据集，其中包含 GPS/IMU、3D LIDAR 和单目相机数据. 它可用于 3D 行人检测.
    • Paper
    • PandaSet Open Dataset - Hesa​​i & Scale 提供的自动驾驶公共大规模数据集. 它使研究人员能够使用真正的自动驾驶汽车的完整传感器套件来研究具有挑战性的城市驾驶情况.
    • Cirrus dataset 来自 LIDAR 扫描模式非均匀分布的公共数据集，重点是远距离. 在这个数据集中使用了 Luminar Hydra LIDAR. 该数据集可在沃尔沃汽车创新门户网站上获得.
    • Paper
    • USyd Dataset- The Univerisity of Sydney Campus- Dataset - 在 1.5 年的时间里每周在悉尼大学校园及周边地区收集的长期、大规模数据集. 它包括多种传感器模式并涵盖各种环境条件. ROS兼容
    • Paper

    Algorithms¶

    Basic matching algorithms¶

    • Iterative closest point - 特征匹配应用程序（ICP）的必备算法.
    • GitHub repository :octocat: - simpleICP C++ /Julia / Matlab / Octave / Python 实现.
    • GitHub repository :octocat: - libpointmatcher，一个实现 ICP 算法的模块化库.
    • Paper - libpointmatcher：比较现实世界数据集上的 ICP 变体.
    • Normal distributions transform - 更新的大规模并行特征匹配 (NDT) 方法.

    • Paper
    • GitHub repository :octocat:
    • LIDAR-MOS - 3D LIDAR 数据中的移动对象分割
    • GitHub repository :octocat:
