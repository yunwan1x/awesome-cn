    Awesome game remakes

    Awesome Game Remakes ¶

    积极维护开源游戏重制版.

    如果您也对克隆和类似游戏（0 AD 等）感兴趣，请查看 Open Source Game Clones.

    Action¶

    • Carnage3D - 重新实现侠盗猎车手.
    • donut - 在现代 C++ 和现代 OpenGL 中重新实现《辛普森一家：Hit & Run》.
    • MafiaUnity - 开源黑手党游戏框架.
    • OpenJK - 社区努力维护和改进 Raven Software 的 Jedi Academy 和 Jedi Outcast.
    • OpenLara - 经典古墓丽影开源引擎.
    • OpenMB - Taleworlds 的骑马与砍杀系列的角色扮演游戏引擎.
    • OpenTomb - 具有现代更新、功能和附加功能的原始古墓丽影 1-5 体验.
    • re3 - GTA III 和 GTA: Vice City 的反向源代码.
    • remc2 - 来自 Assembler 的 Magic Carpet 2 的逆向工程.
    • Rigel Engine - 用 C++ 对经典 DOS 游戏 Duke Nukem II 进行现代重新实现.
    • San Andreas Unity - 在 Unity 中重新实现 GTA：圣安地列斯游戏引擎.
    • TOMB5 - 古墓丽影：编年史反汇编翻译成 C 源代码.

    Adventure¶

    • engge - Thimbleweed Park 的实验性游戏引擎.
    • raw(gl) - 另一个世界的重新实现.
    • ResidualVM - 游戏引擎重新实现，让您可以玩 3D 冒险游戏，例如 Grim Fandango 和 Escape from Monkey Island.
    • ScummVM - 允许您运行某些经典的图形点击式冒险游戏，例如 Maniac Mansion 和 Monkey Island.

    Arcade¶

    • Barbarian - 野蛮人的开源克隆：终极战士.
    • C-Dogs SDL - 经典的高架跑枪游戏.
    • Mr.Boom - 用于 RetroArch/Libretro 的 8 人炸弹人克隆.
    • OpenClonk - 多人动作游戏，您可以在其中控制小巧灵活的人形机器人.
    • Open Kick-Off - 改写邪教足球游戏 Kick Off 2 的尝试.
    • OpenTyrian - 从 Turbo Pascal 到 C 的 Tyrian 开源端口.
    • Taisei - 东方系列的粉丝制作的开源克隆.
    • The Great Escape in C - 经典 ZX Spectrum 游戏 The Great Escape 便携式 C.
    • Zelda Classic - 一种游戏和编辑工具，可让您创建自定义的塞尔达传说任务.

    FPS¶

    • Allegiance - 忠诚是 RTS 和 FPS 太空战斗游戏.
    • BetterSpades - 在旧机器上运行的黑桃王牌开源实现.
    • BStone - Blake Stone：Aliens Of Gold 和 Blake Stone：Planet Strike 的源端口.
    • CatacombGL - 地下墓穴 3D 和地下墓穴冒险系列的 Windows 源端口.
    • Chocolate Doom - 极简主义、历史准确的源端口，包括异端和妖魔的端口，以及 Strife.
    • Doom Legacy - 最受欢迎的 DOOM 源端口之一，它通过 TCP/IP 网络、OpenGL 渲染器等来增强它.
    • DoomRetro - 适用于 Windows PC 的经典、精致的 DOOM 源端口.
    • Doomsday Engine - id Software 的 DOOM I/II 和 Raven Software 的 Heretic 和 Hexen 的源端口.
    • DukeGDX - 毁灭公爵 3D 端口.
    • ECWolf - 基于 Wolf4SDL 的德军总部 3D 端口.
    • ezQuake - ezQuake 结合了所有现代 QuakeWorld 客户端的功能，使 QuakeWorld 更易于启动和播放.
    • FreeCS - 在 FTEQW 引擎上运行的 Counter-Strike 1.5 的免费软件重新实现.
    • Freedoom - 基于经典 DOOM 引擎的免费软件 FPS.
    • FTEQW - 适用于 Quake 1、QuakeWorld、Quake 2、Hexen II 和 Quake III Arena 的 FPS 引擎.
    • GZDoom - 基于 DOOM 引擎的 Modder 友好的 OpenGL 和 Vulkan 源端口.
    • Hexen II: Hammer of Thyrion - Hexen II 游戏的跨平台移植.
    • ioquake3 - 可用于玩 Quake 3 或制作您自己的游戏的免费软件 FPS 引擎.
    • NBlood - 经典 FPS Blood 的端口.
    • Odamex - 在线多人 DOOM 端口，重点关注原始游戏玩法，同时提供广泛的增强功能.
    • OpenSpades - 黑桃 A 的兼容客户端，体素 FPS.
    • QuakeSpasm - 现代、跨平台的 Quake 游戏引擎.
    • RBDOOM-3-BFG - Doom 3 BFG 版带有软阴影、清理过的源代码、Linux 和 64 位支持.
    • RedneckGDX - 乡下人狂暴港.
    • The Eternity Engine - 高级 DOOM 源端口.
    • The Force Engine - 适用于现代系统和使用该引擎的游戏的绝地引擎 - Dark Forces 和 Outlaws.
    • Tyrquake - Tyrquake 的 Libretro 港口.
    • Yamagi Quake II - id Software 的 Quake II 增强型客户端，专注于离线和合作游戏.
    • vkQuake - 使用 Vulkan 而不是 OpenGL 进行渲染的 Quake 端口.
    • Zandronum - 多人游戏端口，基于 Skulltag，用于 Doom 和 Doom II. 还经营异端、妖术和冲突.

    Online¶

    • 2006rebotted - 2006 年的 Runescape 仿真服务器.
    • FreeSO - 重新实施《模拟人生在线》，主要针对移动设备.
    • Meridian 59 - 第一款 3D MMORPG，1996 年发布，2012 年开源.Meridian 59 的原始代码库.
    • Opengate - 翻拍 Jumpgate.
    • RuneLite - 开源老式 RuneScape 客户端.
    • Ryzom Core - Ryzom Core 包含用于制作商业 MMORPG Ryzom 的全部代码（客户端、服务器、工具）.
    • Space Station 14 - Space Station 14 是对 Space Station 13 翻拍的复兴尝试.

    RPG¶

    • Arx Libertatis - 第一人称角色扮演游戏 Arx Fatalis 的跨平台移植.
    • Barony - Barony 的开源版本.
    • BtBuilder - Bard's Tale Construction Set 的开源实现.
    • Daggerfall Unity - 在 Unity 引擎中重新创建 Daggerfall.
    • Devilution - 暗黑破坏神原始源代码的重构形式.
    • DevilutionX - 为现代操作系统构建的暗黑破坏神.
    • DGEngine - 暗黑破坏神游戏引擎的实现.
    • Exult - 为现代操作系统重新创建 Ultima 7 的项目.
    • Falltergeist - 用 C++ 和 SDL 编写的跨平台辐射 2 游戏引擎.
    • Freeablo - 暗黑破坏神引擎的开源实现.
    • FreeFT - 受辐射策略启发的实时等距动作游戏引擎.
    • GemRB - Infinity Engine 的便携式开源重新实现（由 Baldur's Gate、Icewind Dale 等使用）.
    • OpenDiablo2 - 暗黑破坏神 2 的开源重新实现.
    • OpenGothic - 重新实现哥特式 2：乌鸦之夜.
    • OpenMW - 重新实现晨风游戏引擎.
    • OpenTESArena - Bethesda Softworks 对 The Elder Scrolls: Arena 的现代引擎重新实现.
    • OpenVIII - Final Fantasy VIII engine implementation in C#.
    • REGoth - zEngine 的开源重新实现，Gothic 和 Gothic II 背后的引擎.
    • Riiablo - 使用 Java 和 LibGDX 重新制作了《暗黑破坏神 II》.
    • sfall - 经典游戏《辐射 2》的一组引擎修改.
    • Shockolate - 同样很棒的系统冲击，新的很棒的味道.
    • UAlbion - 翻拍 1995 年的阿尔比恩.
    • WitchavenGDX - 女巫 Java 端口.
    • xoreos - BioWare 的 Aurora 引擎及其衍生产品的实现（被无冬之夜、巫师等使用）.
    • zorkClone - 重新实现由 Infocom 开发的原始 Zork 游戏.

    Platformer¶

    • Alive Reversing - 重新实现 Oddworld：Abe 的出埃及记.
    • Commander Genius - Commander Keen Games 的现代口译员.
    • Jazz² Resurrection - 重新实现 Jazz Jackrabbit 2.
    • Lionheart Remake - Lionheart Amiga 游戏的 Java 重制版.
    • NXEngine-evo - 重写了杰作跳跃和奔跑平台游戏 Doukutsu Monogatari（也称为洞穴物语）.
    • Omnispeak - 重新实现了《再见银河》中的指挥官基恩.
    • Prince-Monogame - 波斯王子 2 的 Monogame 实现.
    • SDLPoP - 波斯王子港，基于DOS版本的反汇编.
    • The Secret Chronicles of Dr. M - 改编的秘密马里奥编年史.
    • VVVVV - VVVVVV 的源代码.

    Racing¶

    • DreeRally - 死亡拉力赛引擎重新实现.
    • OpenNFS - 尝试重现经典的极品飞车游戏 (1-6).
    • Vangers - 赛车/RPG 混合体由其创建者开源，用 C++ 编写.

    Simulator¶

    • DXX-Rebirth - 适用于 Windows、macOS、Linux 的 Descent 和 Descent 2 引擎的源端口.
    • FS2Open - FreeSpace2 源代码项目.
    • Librelancer - 重新实现了 2003 年的太空游戏 Freelancer.

    Strategy¶

    • Antares - 重制战神，一个战术太空格斗游戏.
    • CivOne - 席德梅尔文明的实施.
    • CorsixTH - 主题医院的克隆，支持现代操作系统、高分辨率等等.
    • Cortex Command Community Project Source - 社区驱动的努力，以继续开发 Cortex Command.
    • fheroes2 - 免费实现英雄无敌 II 游戏引擎.
    • FreeCol - 基于旧游戏殖民化的回合制策略游戏.
    • Jagged Alliance Stracciatella - 使 Jagged Alliance 2 可在各种平台上使用.
    • Julius - 重新实施凯撒三世.
    • KaM Remake - 从头开始​​用 Delphi 编写的骑士与商人 RTS 游戏翻拍.
    • KeeperRL - 受矮人要塞启发的 Roguelike 地牢建造者.
    • openage - 帝国时代 II 引擎的克隆.
    • OpenApocalypse - 重建 X-COM：启示录引擎.
    • openblack - 重新实现黑白.
    • OpenDUNE - 流行游戏 Dune II 的重现.
    • Open Fodder - Cannon Fodder 的开源实现.
    • OpenHoW - Gremlin's Hogs of War 的再现.
    • OpenHV - 硬真空的开源实现.
    • OpenLoco - 重新实现克里斯索耶的运动.
    • Open Panzer - HTML5 Panzer General 2.
    • OpenRA - 使用 .NET/mono 和 OpenGL 实现红色警报引擎. 在 Windows、Linux 和 macOS 上运行.
    • OpenRCT2 - 过山车大亨 2 的娱乐.
    • OpenSAGE - C# reimplementation of SAGE, the game engine used by a bunch of EA Pacific games, including Command & Conquer: Generals and Battle for Middle Earth.
    • OpenSC2K - 由 Maxis 重制《模拟城市 2000》.
    • OpenTTD - OpenTTD 是一款基于流行游戏 Transport Tycoon Deluxe 的运输模拟游戏，由 Chris Sawyer 编写.
    • OpenXcom - UFO 克隆：未知敌人和 X-COM：深海恐怖.
    • OutpostHD - 重新制作 Sierra On-Line 的前哨站.
    • Race into Space - Interplay 的 Buzz Aldrin 的太空竞赛版本.
    • Return To The Roots - 用 C++ 编写的 Settlers II 翻拍.
    • Seven Kingdoms: Ancient Adversaries - 七大王国的 GPL 版本：古代对手.
    • Sourcehold - Firefly Studios 实施 Stronghold.
