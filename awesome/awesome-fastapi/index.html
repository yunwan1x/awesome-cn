    Awesome fastapi

    Awesome FastAPI | ¶

    > 与 FastAPI 相关的精彩内容的精选列表.

    FastAPI 是一个现代的、高性能的、包含电池的 Python Web 框架，非常适合构建 RESTful API.

    Third-Party Extensions¶

    Admin¶

    • FastAPI Admin - 功能性管理面板，提供用于对数据执行 CRUD 操作的用户界面.

    Auth¶

    • AuthX - FastAPI 的可定制身份验证和 Oauth2 管理.
    • FastAPI Auth - 支持 OAuth2 密码流和 JWT 访问和刷新令牌的可插入身份验证.
    • FastAPI Azure Auth - 具有单租户和多租户支持的 API 的 Azure AD 身份验证.
    • FastAPI Cloud Auth - FastAPI 和云身份验证服务（AWS Cognito、Auth0、Firebase 身份验证）之间的简单集成.
    • FastAPI Login - 账户管理和认证（基于 Flask-Login).
    • FastAPI JWT Auth - JWT 身份验证（基于 Flask-JWT-Extended).
    • FastAPI Permissions - 行级权限.
    • FastAPI Security - 在 FastAPI 中实现身份验证和授权作为依赖项.
    • FastAPI Simple Security - 可通过路径操作管理开箱即用的 API 密钥安全性.
    • FastAPI Users - 账户管理、认证、授权.

    Databases¶

    ORMs¶

    • FastAPI SQLAlchemy - FastAPI 和 SQLAlchemy.
    • FastAPIwee - 一种基于 REST API 创建的简单方法 PeeWee 楷模.
    • GINO - 构建在 SQLAlchemy 核心之上的轻量级异步 ORM，用于 Python asyncio.
    • FastAPI Example
    • ORM - 一个异步 ORM.
    • ormar - Ormar 是一个使用 Pydantic 验证的异步 ORM，可以直接在 FastAPI 请求和响应中使用，因此您只需要维护一组模型. 包括 Alembic 迁移.
    • FastAPI Example - 在 ormar 中使用 FastAPI.
    • Piccolo - 异步 ORM 和查询构建器，支持 Postgres 和 SQLite，带电池（迁移、安全等）.
    • FastAPI Examples - 将 FastAPI 与 Piccolo 一起使用.
    • Piccolo Admin - 强大且现代的管理 GUI，使用 Piccolo ORM.
    • Tortoise ORM - 受 Django 启发的易于使用的 asyncio ORM（对象关系映射器）.
    • FastAPI Example - Tortoise-ORM FastAPI 集成的一个例子.
    • Tutorial: Setting up Tortoise ORM with FastAPI
    • Aerich - Tortoise ORM 迁移工具.
    • SQLModel - SQLModel (which is powered by Pydantic and SQLAlchemy) is a library for interacting with SQL databases from Python code, with Python objects.

    Query Builders¶

    • asyncpgsa - 一个包装 asyncpg 用于 SQLAlchemy Core.
    • Databases - 异步 SQL 查询构建器，在 SQLAlchemy Core 表达语言.

    ODMs¶

    • Beanie - MongoDB 的异步 Python ODM，基于 Motor 和 Pydantic, which supports data 和 schema migrations out of the box.
    • MongoEngine - 用于从 Python 处理 MongoDB 的文档对象映射器（想想 ORM，但用于文档数据库）.
    • Motor - 用于 MongoDB 的异步 Python 驱动程序.
    • ODMantic - AsyncIO MongoDB ODM 集成 Pydantic.

    Other Tools¶

    • Pydantic-SQLAlchemy - 将 SQLAlchemy 模型转换为 Pydantic 楷模.
    • FastAPI-CamelCase - 对使用 FastAPI 的 CamelCase JSON 支持 Pydantic.
    • CamelCase Models with FastAPI and Pydantic - 扩展作者随附的博客文章.

    Developer Tools¶

    • FastAPI Code Generator - 从 OpenAPI 文件创建 FastAPI 应用程序，支持架构驱动开发.
    • FastAPI Client Generator - 从 OpenAPI 规范生成一个 mypy 和 IDE 友好的 API 客户端.
    • FastAPI Profiler - joerick/pyinstrument 的 FastAPI 中间件，用于检查您的服务性能.
    • FastAPI Versioning - API 版本控制.
    • Jupyter Notebook REST API - 将您的 Jupyter 笔记本作为 RESTful API 端点运行.
    • Manage FastAPI - 用于生成和管理 FastAPI 项目的 CLI 工具.
    • msgpack-asgi - 自动 MessagePack 内容协商.

    Email¶

    • FastAPI Mail - Lightweight mail system for sending emails and attachments (individual and bulk).

    Utils¶

    • ASGI Correlation ID - 请求 ID 日志记录中间件.
    • FastAPI Cache - 一个简单的轻量级缓存系统.
    • FastAPI Cache - 一种缓存 FastAPI 响应和函数结果的工具，支持 Redis、Memcached、DynamoDB 和内存后端.
    • FastAPI Chameleon - 将 Chameleon 模板语言集成到 FastAPI.
    • FastAPI Contrib - 自以为是的实用程序集：分页、身份验证中间件、权限、自定义异常处理程序、MongoDB 支持和 Opentracing 中间件.
    • FastAPI CRUDRouter - FastAPI 路由器，可自动为您的模型创建和记录 CRUD 路由.
    • FastAPI Jinja - 将 Jinja 模板语言添加到 FastAPI.
    • FastAPI Lazy - 使用 FastAPI 启动项目的惰性包.
    • FastAPI Limiter - FastAPI 的请求速率限制器.
    • FastAPI MQTT - MQTT 协议的扩展.
    • FastAPI Pagination - FastAPI 的分页.
    • FastAPI Plugins - Redis 和调度程序插件.
    • FastAPI ServiceUtils - 用于创建 API 服务的生成器.
    • FastAPI SocketIO - FastAPI 和 SocketIO 的轻松集成.
    • FastAPI Utilities - 可重用的实用程序：基于类的视图、响应推断路由器、周期性任务、计时中间件、SQLAlchemy 会话、OpenAPI 规范简化.
    • FastAPI Websocket Pub/Sub - 经典的发布/订阅模式可以通过网络和云实时轻松访问和扩展.
    • FastAPI Websocket RPC - Websockets 上的 RPC（双向 JSON RPC）变得简单、健壮和生产就绪.
    • OpenTelemetry FastAPI Instrumentation - 库提供 FastAPI Web 框架的自动和手动检测，检测由使用该框架的应用程序提供的 http 请求.
    • Prerender Python Starlette - Prerender 的 Starlette 中间件.
    • Prometheus FastAPI Instrumentator - 适用于 FastAPI 应用程序的可配置和模块化 Prometheus Instrumentator.
    • SlowApi - 速率限制器（基于 Flask-Limiter).
    • Starlette Context - 允许您在项目中的任何位置存储和访问请求数据，这对记录很有用.
    • Starlette Exporter - FastAPI 和 Starlette 的更多普罗米修斯集成.
    • Starlette OpenTracing - 对 Starlette 和 FastAPI 的 Opentracing 支持.
    • Starlette Prometheus - FastAPI 和 Starlette 的 Prometheus 集成.
    • Strawberry GraphQL - 基于数据类的 Python GraphQL 库.

    • FastAPI and Tortoise ORM - 强大而简单的 Web API 模板，带有 FastAPI（作为 Web 框架）和 Tortoise-ORM（用于通过数据库工作而不会头疼）.
    • FastAPI Model Server Skeleton - 为机器学习模型提供生产就绪的骨架应用程序.
    • cookiecutter-spacy-fastapi - 使用 FastAPI 快速部署 spaCy 模型.
    • cookiecutter-fastapi - FastAPI 项目的 Cookiecutter 模板，使用：机器学习、诗歌、Azure Pipelines 和 pytest.
    • openapi-python-client - 从 OpenAPI 生成现代 FastAPI Python 客户端（通过 FastAPI）.
    • Pywork - Yeoman 生成器来构建 FastAPI 应用程序.
    • uvicorn-gunicorn-fastapi-docker - Docker 镜像与 Uvicorn 由 Gunicorn 管理，用于 Python 3.7 和 3.6 中具有性能自动调整的高性能 FastAPI Web 应用程序.
    • fastapi-gino-arq-uvicorn - 使用 Python 提供高性能异步 REST API 的模板. FastAPI + GINO + Arq + Uvicorn（带 Redis 和 PostgreSQL）.
    • FastAPI and React Template - 使用 FastAPI、TypeScript、Docker、PostgreSQL 和 React 的全栈 cookiecutter 样板.
    • FastAPI Nano - 具有工厂模式架构的简单 FastAPI 模板.
    • inboard - Docker 映像为您的 FastAPI 应用程序提供支持并帮助您更快地交付.
    • FastAPI template - 灵活、轻量级的 FastAPI 项目生成器. 它包括对 SQLAlchemy、多个数据库、CI/CD、Docker 和 Kubernetes 的支持.
