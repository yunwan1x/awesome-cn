    Awesome Jupyter ¶

    一个很棒的精选清单 Jupyter 项目、图书馆和资源. Jupyter 是一个开源 Web 应用程序，允许您创建和共享包含实时代码、方程、可视化和叙述文本的文档.





    Runtimes/Frontends¶

    • Beaker - 具有从一种语言到另一种语言的无缝数据传输的开发环境.
    • docker-stacks - Docker 中可运行的 Jupyter 应用程序的分层堆栈.
    • Guild AI - 将笔记本作为实验来捕获和比较一段时间内的结果.
    • Hydrogen - 使用 Jupyter 内核在 Atom 中内联运行代码.
    • Jupyter Notebook - 主要的 Jupyter 笔记本运行时.
    • JupyterHub - Jupyter 的多用户服务器.
    • JupyterLab - JupyterLab 是 Jupyter 的下一代用户界面.
    • JupyterWith - 基于 Nix 的框架，用于定义声明性和可重现的 Jupyter 环境.
    • ShopRunner/jupyter-notify - 用于单元格完成的浏览器通知的单元格魔法.
    • kaggle/docker-python - 包含数据集和包的 Kaggle Python docker 镜像.
    • ML Workspace - Docker 镜像，包括 Jupyter(Lab) 和用于数据科学/机器学习的各种包.
    • nteract - 原生桌面笔记本前端.
    • Stencila - 原生桌面笔记本前端.
    • voila - 作为交互式独立 Web 应用程序的笔记本.
    • Visual Studio Code - 原生桌面笔记本前端.

    Collaboration/Education¶

    • callgraph - 显示函数调用图的魔法.
    • IllumiDesk - 基于 Docker 的 JupyterHub + LTI + nbgrader 教育发行版.
    • IPythonBlocks - 在 Jupyter 中使用彩色网格练习 Python.
    • jupyter-drive - 用于 Jupyter 的谷歌驱动器.
    • jupyter-viewer-xblock - 在 Open edX XBlock 中获取并显示部分或整个 Jupyter Notebook.
    • jupyter-edx-grader-xblock - 自动评分作为 Jupyter 笔记本创建的学生作业，并将分数写入 Open edX 成绩簿.
    • LTI Launch JupyterHub Authenticator - 通过 Edx 进行身份验证.
    • nbautoeval - 创建自动评估的练习.
    • nbgitpuller - 将 git 存储库单向同步到本地路径.
    • nbgrader - Jupyter 笔记本的分配和分级.
    • nbtutor - 可视化 Python 代码执行（逐行）.

    Visualization¶

    • Altair - 基于 Python 的声明式可视化库 Vega 和 Vega-Lite.
    • Bokeh - 针对现代 Web 浏览器进行演示的交互式可视化库.
    • bqplot - Jupyter 基于图形的交互式绘图框架的语法.
    • Evidently - 在验证或生产监控期间分析机器学习模型的交互式报告.
    • IPySigma - Jupyter 笔记本的原型网络可视化前端.
    • ipychart - Jupyter 中的交互式 Chart.js 绘图.
    • ipyleaflet - Jupyter 笔记本中 Leaflet.js 地图的交互式可视化库.
    • ipysheet - Jupyter 中的交互式电子表格.
    • ipytree - Jupyter 的树 UI 元素.
    • ipywebrtc - Jupyter 中的视频/音频流.
    • ipywidgets - Jupyter 的 UI 小部件.
    • ipyvolume - 基于小部件和 WebGL 在 Jupyter 中为 Python 进行 3D 绘图.
    • itk-jupyter-widgets - 以 2D 和 3D 形式可视化图像的交互式小部件.
    • jp_doodle - 用于构建 2D 和 3D 专用交互式图表的基础设施.
    • jupyter-manim - 展示 manim （数学动画引擎）Jupyter 笔记本中的视频或 GIF.
    • jupyter-gmaps - Jupyter 笔记本中 Google 地图的交互式可视化库.
    • lux - 每当在笔记本中打印数据框时，都会推荐一组可视化效果.
    • mpld3 - 结合 Matplotlib 和 D3js 进行交互式数据可视化.
    • pd-replicator - 一键将 Pandas DataFrame 复制到剪贴板.
    • pyecharts - Python 接口 ECharts 可视化库.
    • pythreejs - 利用 Jupyter 小部件基础结构的 Python / ThreeJS 桥接器.
    • Qgrid - 用于在 Jupyter 笔记本中排序、过滤和编辑 DataFrame 的交互式网格.
    • tributary - 支持 Jupyter 的 Python 数据流.
    • tqdm - 循环和迭代的快速、可扩展的进度条.
    • xleaflet - ipyleaflet 的 C++ 后端.
    • xwebrtc - ipywebrtc 的 C++ 后端.
    • xwidgets - 用于 ipywidgets 的 C++ 后端.

    Rendering/Publishing/Conversion¶

    • Binder - 将 GitHub 存储库转变为交互式笔记本的集合.
    • Bookbook - Bookbook 将目录中的一组笔记本转换为 HTML 或 PDF，保留笔记本内和笔记本之间的交叉引用.
    • ContainDS Dashboards - JupyterHub 扩展，可在任何框架（Voilà、Streamlit、Plotly Dash 等）中托管经过身份验证的脚本或笔记本.
    • Ganimede - 在沙箱中存储、版本、编辑和执行笔记本，并通过 REST 接口直接集成它们.
    • Jupyter Book - 从计算材料构建出版质量的书籍和文档.
    • Jupytext - 使用在版本控制下运行良好的文本格式（例如 Python 或 Markdown 文件）转换和同步笔记本.
    • jut - CLI 可以在终端中很好地显示笔记本.
    • Kapitsa - CLI to search local Jupyter notebooks.
    • nbconvert - 将笔记本转换为其他格式.
    • nbdev - 使用 Jupyter 作为开发、打包和分发 Python 包到 PyPI Literate Programing 环境.
    • nbflow - 使用 Jupyter 和 Scons 的一键式可重复工作流程.
    • nbinteract - 从 Jupyter 笔记本创建交互式网页.
    • nbscan - 搜索和打印 Jupyter 笔记本的单元格内容.
    • Nikola - 将笔记本转换为网站的静态站点生成器.
    • notedown - 将 Jupyter 笔记本转换为 Markdown（并返回）.
    • Papermill - 用于参数化、执行和分析 Jupyter 笔记本的工具.
    • Ploomber - 使用“pipeline.yaml”文件以可重现的方式运行一系列笔记本和脚本.
    • pynb - Jupyter Notebooks 作为带有嵌入 Markdown 文本的纯 Python 代码.
    • RISE - Reveal.js Jupyter/IPython 幻灯片.
    • rst2ipynb - 将独立的 reStructuredText 文件转换为 Jupyter 笔记本文件.
    • Voila - 使用交互式小部件呈现实时 Jupyter Notebooks，允许基于 Jupyter Notebooks 进行仪表板.

    Version Control¶

    • git - git 集成的扩展.
    • jupyter-nbrequirements - Jupyter Notebooks 中的依赖管理和优化.
    • nbdime - 用于区分和合并 Jupyter 笔记本的工具.
    • nbQA - 从命令行或通过预提交在 Jupyter Notebook 上运行任何标准 Python 代码质量工具.
    • ReviewNB - Jupyter Notebooks 的代码审查.

    JupyterLab Extensions¶

    • celltags - 使用单元格标签组织和执行笔记本的扩展.
    • code_formatter - 通用代码格式化程序.
    • debugger - Jupyter 笔记本、控制台和源文件的可视化调试器.
    • drawio - 显示 drawio/mxgraph 图表的扩展.
    • elyra - 用于在本地或远程创建和运行笔记本（或 Python 脚本）管道的可视化编辑器.
    • go-to-definition - 用于在 JupyterLab 中导航到变量或函数定义的扩展.
    • google-drive - Google Drive 集成扩展.
    • jupyterlab_email - 从 JupyterLab 中通过电子邮件发送笔记本及其内容.
    • jupyterlab-executor - 从 Jupyterlab 文件浏览器执行脚本的扩展.
    • jupyterlab-kyso - 将笔记本发布到 Kyso 来自 Jupyterlab 的平台.
    • jupyterlab_templates - JupyterLab 中的笔记本模板.
    • latex - 实时编辑 LaTeX 文档的扩展.
    • lsp - 类似 IDE 的功能（代码导航、悬停建议、linter、诊断、无内核自动完成等）
    • nb_black - 使用扩展保持 Python 代码自动格式化 black.
    • python-bytecode - 在 JupyterLab 中探索 CPython 字节码.
    • quickopen - 通过输入部分名称在 JupyterLab 中快速打开文件.
    • shortcutui - 用于管理键盘快捷键的扩展.
    • sidecar - JupyterLab 的边车输出小部件.
    • sql - JupyterLab 的 SQL GUI.
    • system-monitor - 显示系统指标的扩展.
    • templates - 支持 Jupyter Notebook 模板.
    • theme-darcula - Jupyterlab 的漂亮 Darcula 主题.
    • toc - 为笔记本提供目录的扩展.
    • topbar - JupyterLab 的顶部栏扩展.
    • variableinspector - 显示变量及其值的变量检查器扩展.
    • vim - Vim 笔记本单元格绑定.
    • voyager - Extension to view CSV and JSON data in Voyager.

    Testing¶

    • ipytest - 用于从笔记本中运行单元测试的测试运行器.
    • nbval - 用于验证 Jupyter 笔记本的 Py.test 插件.
    • sphinxcontrib-jupyter - 用于生成 Jupyter 笔记本的 Sphinx 扩展.
    • nosebook - 用于查找和运行 IPython 笔记本作为鼻子测试的鼻子插件.
    • treebeard - 用于测试/调度 Jupyter 笔记本的 GitHub 操作.
    • treon - 易于使用的 Jupyter Notebooks 测试框架.

    Domain-Specific Projects¶

    • ArcGIS - 用于处理地图和地理空间数据的库，由 Web GIS 提供支持.
    • GenePattern Notebook - 将基因组分析与交互式笔记本集成.
    • GeoNotebook - 探索性地理空间分析的扩展.
    • Jupylet - 在 Jupyter 笔记本中以交互方式创建 2D 和 3D 游戏、图形、现场音乐和声音.
    • keplergl - Jupyter 扩展，用于大规模地理定位数据集的可视化探索.
    • lolviz - 用于列表、列表、字典列表的数据结构可视化工具.
    • Quantopian Notebooks - 基于 Jupyter 的金融研究平台.
    • vpython-jupyter - 在 Jupyter 笔记本中运行的 VPython 3D 引擎.

    Hosted Notebook Solutions¶

    • Anaconda Enterprise - 多用户协作和一键部署模型、笔记本和仪表板.
    • Azure Notebooks - 在 Microsoft Azure 的云中运行的 Jupyter 笔记本.
    • CoCalc - 支持 17 种内核类型的笔记本、课程管理、LaTeX 文档创作、同步文档编辑以及与 SageMath 计算机代数系统的集成.
    • DataScience.com - 企业数据科学平台.
    • Deepnote - Jupyter 兼容的数据科学笔记本，具有实时协作、版本控制和易于部署的功能.
    • Domino Data Lab - 具有集成协作工具、环境管理和计算网格的数据科学平台.
    • Google Cloud AI Platform Notebooks - 在 Google Cloud Platform 上使用支持 GPU 的机器学习框架配置的托管 JupyterLab 笔记本实例.
