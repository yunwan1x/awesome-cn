    Awesome coq

    Awesome Coq ¶

    > 精选的 Coq 库、插件、工具和资源的精选列表.

    The Coq proof assistant 提供了一种形式语言来编写数学定义、可执行算法和定理，以及用于机器检查证明的半交互式开发的环境.

    欢迎投稿！ 阅读 contribution guidelines 第一的.

    Projects¶

    Frameworks¶

    • CoqEAL - 简化证明中数据表示更改的框架.
    • FCF - 密码学证明框架.
    • Fiat - 大部分是正确构建程序的自动合成.
    • FreeSpec - 使用效果和效果处理程序模块化验证程序的框架.
    • Iris - 高阶并发分离逻辑框架.
    • Q*cert - 用于实现和验证查询编译器的平台.
    • Verdi - 正式验证分布式系统实现的框架.
    • VST - 用于在高阶并发、不可预测的分离逻辑中验证 Coq 中的 C 代码的工具链，该逻辑与 CompCert 编译器的 Clight 语言相符.

    User Interfaces¶

    • CoqIDE - 用于与 Coq 交互的独立图形工具.
    • Coqtail - 基于 Vim 文本编辑器的 Coq 接口.
    • Proof General - 基于可扩展、可定制的文本编辑器 Emacs 的校样助手通用界面.
    • Company-Coq - Proof General 的 Coq 模式的 IDE 扩展.
    • jsCoq - Coq 到 JavaScript 的端口，它允许在浏览器中运行 Coq 项目.
    • Jupyter kernel for Coq - Coq 支持 Jupyter Notebook Web 环境.
    • VSCoq - Visual Studio Code 编辑器的扩展.

    Libraries¶

    • ALEA - 用于推理随机算法的库.
    • Bignums - 任意大数的库.
    • CoLoR - 关于重写理论、lambda 演算和终止的库，以及扩展 Coq 标准库的公共数据结构子库.
    • coq-haskell - 库为 Haskell 用户平滑过渡到 Coq.
    • Coq record update - 提供更新 Coq 记录字段的通用方法的库.
    • Coq-std++ - Coq 的扩展替代标准库.
    • ExtLib - 可能对其他 Coq 开发有用的理论和插件的集合.
    • FCSL-PCM - 用于验证指针操作程序的部分可交换幺半群的形式化.
    • Flocq - 浮点计算的形式化.
    • Formalised Undecidable Problems - 无法确定的问题和它们之间的减少库.
    • Hahn - 列表和二元关系推理库.
    • Metalib - 使用局部无名变量绑定表示的编程语言元理论库.
    • Monae - 一元效应和等式推理.
    • Paco - 参数化联合归纳库.
    • Regular Language Representations - 正则语言的不同定义之间的翻译，包括正则表达式和自动机.
    • Relation Algebra - 以异构二元关系为模型的代数的模块化形式化.
    • Simple IO - 具有用户可定义原始操作的输入/输出 monad.
    • TLC - Coq 标准库的非建设性替代方案.

    Package and Build Management¶

    • coq_makefile - 与 Coq 一起分发并基于生成 makefile 的构建工具.
    • Coq Nix Toolbox - Nix 辅助脚本，用于自动本地构建和 Coq 的持续集成.
    • Coq Package Index - 基于 OPAM 的 Coq 包集合.
    • Coq platform - 精选的软件包集合，以支持 Coq 在工业、教育和研究中的使用.
    • coq-community Templates - 用于为 Coq 项目生成配置文件的模板.
    • Docker-Coq - 多版本 Coq 的 Docker 镜像.
    • Docker-MathComp - 用于 Coq 和数学组件库版本的多种组合的 Docker 映像.
    • Docker-Coq GitHub Action - 可与 Docker-Coq 或 Docker-MathComp 一起使用的 GitHub 容器操作.
    • Dune - Coq 和 OCaml（前 jbuilder）的可组合和自以为是的构建系统.
    • Nix - Linux 和其他 Unix 系统的包管理器，支持原子升级和回滚.
    • Nix Coq packages - Nix 的 Coq 相关包的集合.
    • OPAM - 灵活且对 Git 友好的 OCaml 包管理器，具有多种编译器支持.

    Plugins¶

    • AAC Tactics - 重写通用量化方程、模结合性和某些运算符的交换性的策略.
    • Coq-Elpi - 基于 λProlog 的扩展框架提供了一个广泛的 API 来实现命令和策略.
    • CoqHammer - 通用自动推理锤工具，它将从以前的证明中学习与将问题转换为自动证明者以及重建找到的证明相结合.
    • Equations - Coq 的函数定义包.
    • Gappa - 实现有关浮点运算和舍入错误的目标的策略.
    • Hierarchy Builder - 基于打包类声明 Coq 层次结构的命令集合.
    • Ltac2 - 类似于 Coq 的经典 Ltac 语言的实验类型策略语言.
    • MetaCoq - 在 Coq 中对 Coq 进行形式化的项目，并提供用于操作 Coq 术语和开发认证插件的工具.
    • Mtac2 - 为反向推理添加类型化策略的插件.
    • Paramcoq - 用于生成 Coq 术语的参数化翻译的插件.
    • QuickChick - 用于随机属性测试的插件.
    • SMTCoq - 检查来自外部 SAT 和 SMT 求解器的证明证人的工具.
    • Unicoq - 用增强的算法替换现有的统一算法的插件.

    Tools¶

    • Alectryon - 用于编写混合 Coq 代码和散文的技术文档的工具集合.
    • Autosubst 2 - 生成 Coq 代码的工具，用于处理语法中的绑定器，例如重命名和替换.
    • CFML - Tool for proving properties of OCaml programs in separation logic.
    • coq2html - Coq 的替代 HTML 文档生成器.
    • coqdoc - 从 Coq 代码生成 LaTeX 或 HTML 文件的标准文档工具.
    • CoqOfOCaml - 从 OCaml 代码生成惯用 Coq 的工具.
    • coq-dpdgraph - Tool for building dependency graphs between Coq objects.
    • coq-scripts - 处理 Coq 文件的脚本，包括制表证明时间.
    • coq-tools - 帮助构建错误的小型复制示例、删除不需要的导入等的脚本.
    • Cosette - 用于推理 SQL 查询等价的自动求解器.
    • hs-to-coq - 从 Haskell 代码转换为等效的 Coq 代码.
    • lngen - 用于生成本地无名 Coq 定义和证明的工具.
    • Menhir - 可以为经过验证的解析器输出 Coq 代码的解析器生成器.
    • mCoq - Coq 项目的突变分析工具.
    • Ott - 用于编写可转换为 Coq 的编程语言和微积分定义的工具.
    • Roosterize - 在 Coq 项目中建议引理名称的工具.
    • SerAPI - 用于 Coq 代码与 JSON 和 S 表达式之间（反）序列化的工具和 OCaml 库.

    Type Theory and Mathematics¶

    • Analysis - 与数学组件兼容的经典实分析库.
    • Category Theory in Coq - 范畴论的无公理形式化.
    • Completeness and Decidability of Modal Logic Calculi - 逻辑 K、K*、CTL 和 PDL 的稳健性、完整性和可判定性.
    • CoqPrime - 使用 Pocklington 和 Elliptic Curve 证书来证明素性的库.
    • CoRN - 建设性实分析和代数库.
    • Coqtail Math - 从算术到实数和复数分析的数学结果库.
    • Coquelicot - 与标准库兼容并注重可用性的经典实分析的形式化.
    • Finmap - 使用有限映射、集合和多重集合扩展数学组件.
    • Four Color Theorem - 四色定理的正式证明，图论的一个里程碑式的结果.
    • Gaia - 实施布尔巴基的《数学原理》，包括集合论和数论.
    • GeoCoq - 基于塔斯基公理系统的几何形式化.
    • Goedel - Gödel-Rosser 不完备定理的构造性证明.
    • Graph Theory - 形式化的图论结果.
    • Homotopy Type Theory - 同伦理论思想的发展.
    • Infotheo - 信息论和线性纠错码的形式化.
    • Mathematical Components - 数学理论的形式化，特别关注群论.
    • Math Classes - 基于类型类的数学结构的抽象接口.
    • Odd Order Theorem - 奇数阶定理的形式证明，有限群论的一个里程碑式的结果.
    • Puiseuxth - Puiseux 定理的证明和 Puiseux 级数多项式根的计算.
    • UniMath - 旨在使用单价观点形式化大量数学的图书馆.

    Verified Software¶

    • CompCert - 几乎所有 C 语言 (ISO C99) 的高保证编译器，为 PowerPC、ARM、RISC-V 和 x86 处理器生成高效代码.
    • Ceramist - 经过验证的基于散列的近似成员结构，例如布隆过滤器.
    • Fiat-Crypto - 加密原始代码生成.
    • Incremental Cycles - 已验证的 OCaml 实现了一种用于图形中增量循环检测的算法.
    • JSCert - 带有经过验证的参考解释器的 ECMAScript 5 (JavaScript) 的 Coq 规范.
    • lambda-rust - Rust 核心语言和类型系统的正式模型、类型系统的逻辑关系以及一些 Rust 库的安全证明.
    • RISC-V Specification in Coq - RISC-V 处理器指令集架构和扩展的定义.
    • Vélus - 经过验证的编译器，用于类似 Lustre/Scade 的数据流同步语言.
    • Verdi Raft - Raft 分布式共识协议的实现，在 Coq 中使用 Verdi 框架进行验证.

    Books¶

    • Coq'Art - 致力于 Coq 的第一本书.
    • Software Foundations - 有关逻辑、函数式编程和编程语言基础的基于 Coq 的系列教科书，旨在让初学者易于使用.
    • Certified Programming with Dependent Types - 关于使用 Coq 进行实用工程的教科书，它教授高级实用技巧和非常具体的证明风格.
    • Program Logics for Certified Compilers - 解释如何使用分离逻辑构建程序逻辑的书，附有 Coq 中的形式模型，该模型应用于 Clight 编程语言和其他示例.
    • Formal Reasoning About Programs - 同时提供有关程序正确性的形式逻辑推理的一般介绍以及为此目的使用 Coq 的书.
    • Programs and Proofs - 这本书对 Coq 中的交互式证明进行了简短且实用的介绍，强调了通过来自 SSReflect 证明语言的一小组原语对可判定命题进行归纳推理的计算性质.
    • Computer Arithmetic and Formal Proofs - 描述如何使用 Flocq 库在 Coq 中正式指定和验证浮点算法的书籍.
    • The Mathematical Components book - 面向有数学倾向的用户的书籍，侧重于数学组件库和 SSReflect 证明语言.

    Course Material¶

    • Foundations of Separation Logic - 介绍使用分离逻辑来推理 Coq 中的顺序命令式程序.
    • Introduction to Computational Logic - 介绍基本逻辑原理、构造类型理论和使用 Coq 证明的交互式定理.

    Tutorials and Hints¶
