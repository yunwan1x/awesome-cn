    Awesome libgdx

    Awesome libGDX ¶

    libGDX 是一个相对较低级别的，免费的，开源的，用Java编写的跨平台游戏开发框架.

    >此列表是精选资源的精选集合，这些资源，工具，教程和项目均使用 libGDX 游戏框架，可帮助开发人员制作与_awesome__兼容的游戏 Awesome Manifesto.

    >捐款_非常欢迎_但先见 Contributing.

    Resources¶

    可用于libGDX代码中以增强框架功能的资源.

    Algorithms and AI¶

    • gdx-ai -具有转向行为，编队运动，寻路，行为树和有限状态机的人工智能框架.
    • Simple Graphs -用于图形数据结构和算法（包括快速寻路）的Java库.

    Controllers¶

    • Controllers -增加了对游戏手柄和操纵杆的支持.
    • gdx-controllerutils -添加了Scene2D按钮输入支持，用户可配置的按钮映射以及LWJGL2的热插拔功能.
    • sdl2gdx -使用SDL的强大游戏手柄实现. 在相同的映射，隆隆声和热插拔下支持数百个控制器.

    Editors¶

    • HyperLap2D -带有libGDX运行时的复杂2D世界和场景的可视编辑器.
    • Spine -基于骨骼的动画工具，专门针对游戏的2D动画.

    Entity Component System (ECS)¶

    • Artemis-odb -基于Java的高性能实体组件系统框架.
    • Ashley -受Ash＆Artemis启发的Java实体系统.

    Physics¶

    • Box2D -最受欢迎的2D游戏物理库之一.
    • Bullet -3D碰撞检测和刚体动力学库.
    • jbump -易于实施的AABB碰撞检测，对平台游戏和其他简单2D游戏很有用.

    Services¶

    • gdx-facebook -为Facebook Graph API提供跨平台支持.
    • gdx-fireapp -Firebase的跨平台API.
    • gdx-firebase -适用于Firebase的跨平台（仅限于台式机/ Android）API.
    • gdx-gameanalytics -libGDX的Game Analytics REST API客户端实现. 适用于所有后端.
    • gdx-gamesvcs -轻松集成游戏服务，例如Google Play游戏，Apple Game Center等.
    • gdx-pay -提供用于InApp购买的跨平台API.
    • steamworks4j -精简包装，允许Java应用程序访问Steamworks C ++ API.

    Setup and Deployment¶

    • gdx-liftoff -libGDX的现代安装工具，使用当前的Gradle 5.x系列.
    • Packr -打包您的JAR，资产和JVM，以在Windows，Linux和macOS上分发.

    User Interface¶

    • Freetype -从轻量级的.ttf字体文件中即时生成所需大小的BitmapFonts.
    • gdx-dialogs -为本地对话框提供跨平台支持.
    • gdx-skins -免费的Scene2D GUI外观.
    • InGameConsole -允许开发人员在其游戏中添加控制台（类似于源游戏中的功能）.
    • msdf-gdx -提供轻量级实用程序以在libGDX上绘制高质量的MSDF（多通道有符号距离字段）文本.
    • PieMenu -Scene2D的径向菜单非常灵活且易于自定义.
    • Ray3K Skins -具有示例代码，自定义可绘制对象和实验功能的免费Scene2D.UI外观.
    • Skin Composer -使用图形界面为libGDX scene2d.ui创建外观.
    • TenPatch -libGDX的9patch实现的替代方法，它实现了多个拉伸区域.
    • TypingLabel -一个libGDX标签，看起来就像是实时键入的一样.
    • VisUI -允许使用scene2d.ui在libGDX中创建漂亮的UI. 请注意，这不是UI编辑器.

    Visual Effects¶

    • Box2DLights -使用Box2D进行光线投射并使用OpenGL ES 2.0进行渲染的2D照明框架.
    • gdx-vfx -基于libgdx-contribs-postprocessing的灵活的后期处理着色器视觉效果.
    • libgdx-screenmanager -libGDX的屏幕管理器，支持各种过渡效果
    • Particle Park -带有实时预览的可下载粒子效果展示.
    • Shape Drawer -ShapeRenderer的高性能替代方案，可避免批量冲洗.
    • gdx-gltf -GLTF 3D文件格式支持（导入/导出），PBR着色器和其他高级渲染.
    • libgdx-graph -提供用于设计渲染管线和着色器的GUI，以及可轻松将其合并到游戏中的库. Guide here

    Others¶

    • gdx-dbgagent -用于调试常见问题的Java代理，例如未放置对象以及修改了诸如Color.WHITE之类的常量.
    • gdx-jnigen -小型库，允许C / C ++代码与Java源代码内联地编写.
    • gdxGifRecorder -记录GIF并自动保存的实用程序类.
    • KTX -libGDX的Kotlin扩展和实用程序.
    • noise4j -基于各种过程内容生成教程的简单地图生成器.
    • Texture Packer GUI -为libGDX游戏框架打包和管理纹理图集的简单方法.

    Tutorials¶

    _针对新手和经验丰富的开发人员的教程.

    Getting Started¶

    • Official libGDX Wiki -包含大量信息的官方libGDX Wiki.
    • Tann's Hello libGDX -对于初学者的优秀指南，介绍如何从头开始创建游戏.
    • Brandon Grasley's Space Shooter Game -有关从头开始制作完整的Android游戏的视频教程系列.
    • Splash Screens -有关创建初始屏幕以在游戏窗口加载之前显示的视频教程.
    • Creating a Launcher -有关使用libGDX制作游戏启动器的视频教程.
    • Deploying with JPackage -有关通过Gradle命令使用JPackage部署libGDX游戏的教程.
    • JSON in Game Dev -有关使用JSON存储数据的常规教程.
    • Progress Bar Design -通过示例讨论不同进度条设计技术的利弊.
    • libGDX External Tutorials -官方非官方教程的大清单.

    Advanced¶

    • Code Hotswapping - Video tutorial on enabling code hotswapping for libGDX projects to increase productivity.
    • Dynamic Textures with Pixmap -详细介绍了如何使用Pixmaps创建蒙版.
    • iOS Deployment Tutorial -使用RoboVM在2019年部署到iOS.
    • Sub-pixel Perfect Smooth Scrolling -像素完美的平滑滚动.

    Assets¶

    _收集免费和高质量的资产，使您的游戏更上一层楼.

    • Kenney Assets -从2D和3D艺术作品到音效，为您的游戏提供优质资产.
    • OpenGameArt.org -提供各种开放内容资产的存储库.
    • Game-Icons.net -包含大量酷炫游戏相关图形的存储库.
    • bfxr.net -只需按几个按钮即可快速创建独特的音效，非常适合制作原型.
    • freesound.org -巨大的音频片段，样本，录音，哔哔声协作数据库.
