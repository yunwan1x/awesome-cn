    AutoHotkey

    Awesome AutoHotkey ¶

    一个很棒的精选清单 AutoHotkey 库、库分发、脚本、工具和资源. 受到他人的启发 awesome lists . 请阅读 CONTRIBUTING.md 在贡献之前.

    已过时或已停产，但仍然可以找到与历史相关的项目 Historical.md

    开发状态：

    Libraries¶

    有用的 AutoHotkey 库列表. 库是具有一些可重用功能的代码，可以与您自己的代码结合以创建新功能.

    Clipboard¶

    • WinClip - 由 Deo - WinClip 是一个剪贴板操作类，扩展了 AutoHotkey 的剪贴板功能，包括对 RTF、HTML 和图像的支持. 论坛主题： link.

    Console¶

    • AHKonsole - 由 G33kdude - 用于控制台支持的基于类的 AutoHotkey 库. 该库使您能够创建一个表示要与之交互的控制台的对象，以及多个控制台缓冲区对象以促进双缓冲. 论坛主题： link.
    • LibCon - 由 joedf - 用于控制台支持的 AutoHotkey 库. 该库使您能够编写控制台应用程序并与其他控制台实例交互. 基本上，这个库促进了与编写和与控制台交互有关的任何事情. 论坛主题： link.

    Data format¶

    • AHK_ctable - 由 hoppfrosch - 以表格格式处理字符串的库 - 论坛主题： link.
    • AutoHotkey-JSON - 由 cocobelgica - AutoHotkey 的 JSON 库. 论坛主题： link.
    • CSV - 由 trueski/kdoske - 使用 CSV 文件和 Listview 函数的库. 论坛主题： link.
    • List manipulation functions - 由 Laszlo - 操作逗号分隔列表的函数库. 论坛主题： link.
    • ObjCSV - 由 JnLlnd - 将 CSV 文件加载/保存到对象和列表视图函数的库. 论坛主题： link.
    • ObjDump/ObjLoad - 通过 HotKeyIt - 将对象序列化/反序列化到/从变量/内存.
    • SerDes - by cocobelgica - 序列化/反序列化 AutoHotkey 对象结构. 论坛主题： link.
    • Table - 由 VxE - 以表格 (TSV) 格式和 Listview 函数操作字符串的库. 论坛主题： link.
    • XA - 由 trueski/hi5 - 序列化/反序列化数组到/从 XML. 论坛主题： link.

    Data Structures and Algorithms¶

    • Facade - 由 Shambles - 一组函数式编程库. - 论坛主题： link
    • HashTable - 由 Shambles - AutoHotkey 的哈希表实现.
    • LibCrypt - 由不同的作者 - 加密和编码功能的集合.
    • Type_Checking - 作者：Shambles - AutoHotkey 的类型检查 - 论坛主题： link

    Database¶

    • AHKDb - 由 AHKDb - 用于制表符分隔数据的数据库库.
    • ahkDBA - 由 IsNull - 一个 OOP-SQL 数据库访问框架. 论坛主题： link.
    • Class_SQLiteDB - 只有我 - AHK SQLite API 包装器类. 论坛主题： link.
    • Leya - MySQL API - by kevgk - 在 autohotkey 中使用 MySQL 数据库，而不会将服务器凭据暴露给客户端.

    Filesystem¶

    • FileGetProperties - by kon - 用于检索扩展文件属性的函数.

    Graphics¶

    • GDIp - by tic - 有助于与 Microsoft 的 gdiplus.dll 交互的全功能库 - 论坛主题： link.
    • AHKv2-GDIP - 更新与 AHK v1.1 和 AHK v2 兼容的上述 GDI+ 库 - 论坛主题： link.
    • GDIp_ImageSearch - by tic - 使用 gdiplus.dll 在屏幕上搜索图像实例的库. 有关 MasterFocus 的改进版本，请参见该线程的结尾，或查看他 GitHub repo here
    • Simple GDI class - GeekDude - 旨在使使用低级 GDI 函数变得简单的类.
    • Particle System - by tidbit - 一个使用 GDI+ 将粒子添加到 GUI 或屏幕的简单类. 论坛主题： link.

    Networking¶

    • FTP - 由 jNizM - FTP 会话（类）的 AutoHotkey 包装器 - 论坛 link
    • Socket Class (überarbeitet) - 由 Bentschi - 基于类的套接字库. 支持 TCP 和 UDP.
    • Socket.ahk - 作者 GeekDude - 基于 Bentschi 的套接字库 - 论坛 link
    • WebSocket.ahk - 作者 GeekDude - 基于类的 WebSocket 库 - 论坛 link
    • WinSCP.ahk - 由 Lipkau - Lib 允许在 AHK 中使用 WinSCP
