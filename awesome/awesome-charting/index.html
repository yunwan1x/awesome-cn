    Charting

    开发人员可能会有用的精选图表和dataviz资源列表. 专注于针对不同用例的相关且当前处于活动状态的JavaScript图表库. 在每个类别中按字母顺序排列.

    灵感来自真棒.

    带给你

    一个声明式，高效且简单的JavaScript库，用于构建响应式图表

    Commercial Libraries¶

    • amCharts -三个用于传统图表，股票和地图的库. 具有手绘样式主题选项.
    • AnyChart -用于绘制不同类型数据的产品集. 具有特殊的Oracle Apex集成选项.
    • CanvasJS -文件较小的库中一组紧密的图表类型. 与其他商业图书馆不同的默认主题.
    • FusionCharts -具有强大地图组件的制图库.
    • Highcharts -一系列的图表库，可用于多种用途. 可以兼容回IE6.
    • JSCharting - Integrated suite of charting libraries with traditional charts types plus maps, stock, Gantt, grid and org charts in one package.
    • Kendo UI -带有本机UI组件的高级图表库，可用于Angular，React和Vue.
    • ZingChart -用于呈现其他库无法处理的大型数据集. 超级可定制的样式和经常添加的新功能. 一切都在一个模块化库中.

    Free and Open Source Libraries¶

    • apexcharts.js -具有简单API的JavaScript图表库.
    • Billboard.js -跟踪D3的C3叉更加紧密.
    • C3.js -基于D3的可重用图表库.
    • Chart.js -包括核心图表类型的微型库（11kb！）.
    • Charts.css -CSS数据可视化框架.
    • Chartist -简单，轻量级的图表库，使用SVG呈现图表，并使用CSS设置样式.
    • Chartkick -带有一行Ruby的JavaScript图表.
    • D3.js -允许用户根据数据处理文档以在SVG中呈现图表.
    • dc.js -D3库 crossfilter 支持
    • dygraphs -交互式可缩放时间序列图.
    • ECharts -强大的浏览器图表和可视化库.
    • EJSChart -企业级图表库.
    • Graphosaurus -由WebGL支持的3D图形查看器（three.js）
    • lightweight-charts -使用HTML5画布构建的财务轻量级图表
    • Morris.js -呈现折线图，条形图，面积图和甜甜圈图的简单API
    • Plotly -建立在d3和stack.gl之上，允许用户创建基本图表和SVG地图.
    • Plottable -具有OOP样式语法的库以构建图表.
    • QuickChart -呈现静态图表图像的Web API.
    • rgraph -带有Google表格的2D / 3D JavaScript图表导入功能.
    • sigma.js -使用画布构建的图形/网络图库.
    • Smoothie Charts -实时流数据的JavaScript图表.
    • TauCharts -独特的语法，使开发人员可以使用DSL描述数据. 具有创建构面的能力.
    • μPlot -小型（<25 KB分钟）快速图表，用于显示时间序列，线，面积，ohlc和条形图.
    • uvCharts -使用d3.js构建的JavaScript图表库
    • vis.js -网络图，带有标签的描述性时间表以及3D图.
    • reaviz -用于React的模块化图表组件库.
    • reaflow -用于构建静态或交互式编辑器的模块化图引擎.

    Free Libraries¶

    • Google Charts -拨入地图（当然）和一些很酷的动画选项. 广泛的文档和使用社区（论坛）.

    Framework-Specific Libraries¶

    Angular¶

    • Angular-Chart-呈现折线图，条形图，面积图和甜甜圈图的简单API
    • n3-charts -用AngularJS编写的易于使用的库，用D3渲染图表.
    • Angular-scale -用于创建折线图的简单轻量库

    Ember¶

    • Ember Charts -准备在Ember.js项目中使用的五种基本图表类型.

    jQuery¶

    • Flot -jQuery的JavaScript图表库. 包含有用的图表类型，但看起来可能不再处于活动开发中.
    • jqPlot -用于绘制图表的开源jQuery插件. 包含许多常用功能，但这些项目可能使用不同的命名约定.
    • jqxChart -在SVG，Canvas和VML中渲染大量图表类型.
    • JQuery-linechart -用于创建折线图的简单轻量库

    Lit-html¶

    • lit-line -{快速，小型，交互式，响应迅速的}线形图Web组件建立在lit-html之上.

    React¶

    • nivo -在D3之上构建的React组件可轻松构建dataviz应用.
    • react-vis -Uber的可组合可视化系统
    • vx 在d3之上的可重用的低级可视化组件的集合
    • Potion -使用React＆D3构建动画可视化的低级构建块.
    • react-chart-js -Chart.js的React包装器
    • react-d3 -依靠React生成SVG标记的图表库，以及d3来计算路径值的图表库.
    • react-muze -反应包装 muze （免费的数据可视化库，用于使用WebAssembly在浏览器中创建探索性数据可视化）
    • react-vis -一组React组件以呈现通用数据可视化图表
    • recharts -使用React和D3构建的重新定义的图表库
    • Victory -用于构建交互式数据可视化的可组合React组件的集合

    Data Visualization Resources¶

    • Dataviz Catalogue -想知道哪种图表类型最好的地方. Severino Ribecca的一个项目.
    • Evergreen Data -有关更有效地呈现数据的信息和意见.
    • Flowing Data -由前《纽约时报》的内森·丘（Nathan Yau）提供的有关地图，可视化，统计数据和设计的资源.
    • Junk Charts -Dataviz评论家（兼哥伦比亚大学教授）Kaiser Fung分解了数据可视化，以找出问题所在，并提出了使它们变得更好的方法.
    • The Functional Art -著名的数据可视化作者和Aberto Cairo教授关于dataviz趋势和最佳实践的博客.
    • Visualizing Data -Dataviz顾问Andy Kirk汇编了有关图表和可视化的示例，资源和参考资料.
    • Well Formed Data -之前世界经济论坛的莫里茨·斯特凡纳（Moritz Stefaner）关于信息可视化和设计的思想性文章.
