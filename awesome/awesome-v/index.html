    Awesome v

    Awesome V ¶

    > 精选的 V 框架、库、软件和资源的精选列表.

    V 是一种用于开发可维护软件的简单、快速、安全的编译语言.

    Applications¶

    Interpreters/Compilers¶

    • v - V 本身. 用于开发可维护软件的简单、快速、安全的编译语言.
    • vbf - 一个脑残的解释器/编译器.
    • monkey_v - 实施 Thorsten Ball's Monkey Language 并在.
    • vcc - 用 V 编写的 AC 编译器.
    • Vork - 用 Python 编写的替代 V 编译器/解释器.
    • Foxil - 一种旨在促进编译器代码生成的中间语言.

    Operating systems/Kernels¶

    • Vinix - V 中的小而简单的操作系统.运行 bash.

    Package managers¶

    • vpm - 用V编写的V语言包管理工具.

    Editors¶

    • ved - 1 MB 用 V 编写的文本编辑器，带有硬件加速文本渲染. 在 <1 秒内编译.

    Web¶

    • vorum - 用 V 编写的开源博客/论坛软件.
    • vblog - 一个简单、快速且反应灵敏的博客系统.
    • Heroku Buildpack for V - 在 Heroku 上部署 V 应用程序.

    Graphics¶

    • vRayTracer - 用 V 编写的简单光线追踪器.

    Games¶

    • Boundstone - 高性能/快速编译/轻量级 Minecraft：基岩版服务器.
    • v-pong - 经典的桨游戏通过 V 的力量重获生机.
    • flappylearning-v - v. 中的一个简单的学习演示（已存档）

    Command-line¶

    • HN-top - 一个简单的命令来列出来自黑客新闻的最新消息.
    • vast - 一个简单的 vlang 工具，将 v 源文件生成为 AST json 文件.
    • symlinker - 用于管理符号链接的小型 Linux 工具.
    • runner - 一种自动运行/编译用各种编程语言编写的代码的工具.
    • vcredits - 从依赖项的许可证文件创建信用的工具.
    • vspect - 检查 vlang 源文件的工具.
    • vinit - 生成 v 项目的工具.

    Project management¶

    • vset - V 项目的项目设置和配置工具.

    Serialization¶

    • vproto - V 中的 Protobuf 编译器和运行时.
    • v-toxml - V 的 XML 序列化库.

    Libraries¶

    Command line interface (CLI) / Terminal / Shell¶

    • boxx - 创建高度可定制的接线盒，看起来也很棒！
    • lol - lolcat 的 V 版本（文本/字符彩虹器）.
    • progressbar - 一个易于使用的 V 库，用于在 cli 中创建进度条.
    • termtable - V Terminal Tables：简单且高度可定制的库，用于在终端中显示表格.
    • vargs - 用于从类似 argv 的数组中解析参数的 V 库. （存档）

    Text processing¶

    • v-regex - 一个简单的 V 正则表达式库.
    • chalk - 为终端中的字符串着色.
    • crayon - 像毕加索一样绘制您的终端输出. ️
    • cjson - 为 vlang 包装 cJSON.
    • ascii_robot - 用 V 编写的 ASCII 机器人生成器.
    • iconv - 为 vlang 包装 iconv.

    Graphics¶

    • vgl - 低级图形 API 访问.
    • viup - 基于 C 的跨平台 UI 库 IUP 的 V 包装器.
    • vsdl - 基于 C 的 SDL 库的 V 包装器.
    • vsdl2 - 一个 libSDL2 包装器.
    • V Earcut - 基于的快速（实时）多边形三角剖分库 mapbox/Earcut 处理孔洞、扭曲多边形、退化和自相交.

    Game development¶

    • vraylib - V 包装器（绑定）用于 raylib，C游戏开发框架.

    Web¶

    • vex - 受 Express 和 Sinatra 启发，在 V 上编写的 Web 框架.
    • vweb - V 的内置网络框架. 由 Vorum 使用.
    • validate - 一个简单的库来验证 V 中的字符串.
    • valval - 用 V 编写的 Web 框架，由 vweb 改进.
    • v-jsonrpc - 在 V 上编写的基本 JSON-RPC 2.0 兼容服务器.
    • pico.v - 基于 picoev 和 picohttpparser 的 V 中的 Web 服务器.
    • vxbloauth - 用于 vweb 的简约 Xbox Live 身份验证器.
    • vcurrency - API 包装器（用 V 编写）用于 https://api.exchangeratesapi.io.

    Database clients¶

    • redis - V 的 Redis 客户端，用 V 编写.
    • vsql - V 的 sql 查询构建器.
    • vmemcached - V 的 Memcached 客户端，用 V 编写.

    Audio¶

    • vave - 一个疯狂的简单库，用于在 V 中读/写 WAV 文件.
    • vspeech - Mozilla 的基于 DeepSpeech TensorFlow 的 Speech-to-Text 库的完整 V 绑定.
    • v-miniaudio - 优秀的 miniaudio C 音频库的绑定.

    Operating system¶

    • clipboard - 用于与操作系统剪贴板交互的 V 模块. 完全跨平台.
    • vlipboard - 一个易于使用的剪贴板包装器，支持 Wayland 和 Termux.

    Automation¶

    • vrobot - V 的桌面自动化.仅支持 Windows.

    Telegram¶

    • vgram - 电报机器人库.

    Discord¶

    • viscord - 用于连接到 Discord 网关的非常基本的库.
    • discord.v - 用户友好的 Discord 机器人库.

    IRC¶

    • vitric - 一个透明的 IRC 库.

    Files¶

    • v-mime - V 的 MIME 检测库.

    Eventing¶

    • eventbus - 一个简单的 V 事件总线系统.

    User Interface toolkits¶

    • V UI - 适用于 Windows、macOS、Linux、Android、iOS 和 Web 的集成跨平台 UI 工具包.
    • vgtk3 - V 中 GTK3 的包装器.
    • vig - 绑定 Dear ImGui GUI toolkit.
    • vnk - 绑定 Nuklear GUI toolkit.

    Scientific computing¶

    • vsl - 具有多种功能的纯 V 科学库.
    • vtl - 支持 n 维数据结构的数值计算库，由 LAPACKE 和 OpenBLAS 支持.

    Utility¶

    • range - V 中 Python 的 range() 的功能.
    • random - 一个用 V 编写的通用随机库.
    • vdotenv - 支持加载环境变量的 .env 文件.

    Other¶

    Syntax highlighting¶

    • v-vim - 支持 Vim 中的 V 语法高亮显示.
    • vim-v - V 编程语言的高质量语法突出显示.
    • vscode-vlang - Visual Studio Code 的 V 语言扩展.
    • kate-syntax-highlight-v - V 语法高亮显示 Kate.
    • scite-v-support - V 语法高亮显示 SciTE.
    • language-v - 对 Atom 的 V 语言支持（vscode-vlang 的端口）.
    • vlang-mode.el - V 编程语言的 Emacs 主要模式.
    • v-mode - V 编程语言的 Emacs 主要模式.
    • vlang-sublime - Sublime text 3 支持 Vlang 编程语言.

    Editor plugins¶

    • vim-vtools - Vim 的 V 工具，包括自动格式化.
    • sublime-v - 用于 V 编程语言的功能齐全的 Sublime Text 3 包.

    GitHub actions¶

    • setup-v-action - 在您的工作流程中使用 V 的 GitHub 操作自动化.
    • action-create-v-docs - 为 V 模块创建文档的 GitHub 操作.
