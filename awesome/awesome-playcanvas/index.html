    Awesome playcanvas

    c# Awesome PlayCanvas

    PlayCanvas 是一个用于构建交互式 Web 内容的开源平台. 它是用 JavaScript、HTML5、CSS3 和 WebGL 编写的.

    > 此列表是使用 PlayCanvas 引擎帮助开发人员制作 awesome 游戏和可视化效果的精选资源、工具、教程和项目的精选集合.

    > 贡献_非常欢迎_但先看看 Contributing.

    PlayCanvas¶

    • Editor - 基于浏览器的实时协作 WebGL 编辑器.
    • Engine - 获得 MIT 许可的 PlayCanvas 运行时的完整源代码.
    • glTF Viewer - 符合 glTF 2.0 的模型查看器.
    • NPM Package - PlayCanvas 引擎运行时的 Node.js 包.
    • PCUI - PlayCanvas 工具集使用的开源前端框架.
    • Reference - 引擎运行时的 API 参考手册.
    • User Manual - PlayCanvas 用户指南.

    Extensions and Utilities¶

    • 8th Wall - 无标记 AR 运行时.

    • playcanvas-node - Node.js 的 REST API 包装器.
    • playcanvas-p2.js - 与 p2.js 2D 物理引擎的集成.
    • playcanvas-spine - 用于 2D 动画的 PlayCanvas 的 Spine 插件.
    • playcanvas-tween - 官方支持的补间动画插件.
    • playcanvas-vue - PlayCanvas 与 Vue.js 的集成.
    • tween.js - 流行的 JavaScript 补间库，可轻松与 PlayCanvas 集成.
    • Pirron One - 实现后期效果、地形创建等的编辑器扩展.
    • Sublime Completions - PlayCanvas API 的 Sublime Text 自动完成功能.
