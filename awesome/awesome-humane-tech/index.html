    Humane Technology




    About this list¶

    科技和社交媒体正在对我们的社会产生重大影响. 虽然许多创新技术发明正在改善我们的生活，但人们越来越意识到这些趋势带来的负面影响，例如大规模隐私侵犯、监视资本主义和技术垄断. 它们导致社交媒体成瘾、心理健康问题，甚至侵蚀我们社会的结构.

    此列表汇总了与人文技术主题直接相关的开源项目. 请阅读 guidelines for contributing 在发送您的 PR 之前.

    > 重要：以下资源由此列表的贡献者真诚收集，希望它们对您有用. 但根据 License 我们不能保证所引用软件的质量或适用性. 因此，在安装任何东西之前请正确告知自己，如果您遇到不合适或错误的东西，请在我们 Issue Tracker 所以我们可以采取适当的行动. 谢谢！

    如果您的项目在此列表中，那么您有权自豪地佩戴我们的人文科技徽章. 看 instructions.

    Community¶

    我们的使命是： To Help Improve Wellbeing, Freedom and Society!

    我们很高兴邀请您参加我们 Humane Tech Community Forum 阅读更多有趣的人文科技主题，参与我们的讨论，并成为一个真正的 Humane Tech Activist!

    您也可以在 Fediverse 上关注我们 @humanetech@mastodon.social.

    Humane design¶

    人性化设计是人性化技术的一个子集，是信息技术领域的一个新兴领域. 此空间中的资源在一个策划列表中单独跟踪，该列表是 delightful project 在 Codeberg.

    • delightful-humane-design - 为 UX 设计师和开发人员精心策划的人性化设计资源列表.
    • delightful-small-ai - 开放的人文人工智能资源的精选列表.

    • awesome security - 软件、图书馆、文档、书籍、资源和关于安全的很酷的东西.
