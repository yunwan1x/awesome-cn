    Raspberry Pi

    Awesome Raspberry Pi¶

    > Raspberry Pi 是 Raspberry Pi 基金会在英国开发的一系列信用卡大小的单板计算机，用于促进学校和发展中国家的基础计算机科学教学. 官方链接： Raspberry Pi Homepage

    此列表是符合标准的工具、项目、图像和资源的集合 Awesome Manifesto

    Models¶

    （消费者）Raspberry Pi 模型的完整集合包括：

    • Raspberry Pi 1，A 型和 B 型（2012 年，停产）
    • Raspberry Pi 1, Model B+ （2014 年 7 月）
    • Raspberry Pi 1, Model A+ （2014 年 11 月）
    • Raspberry Pi 2, Model B （2015 年 2 月）
    • Raspberry Pi Zero （2015 年 11 月）
    • Raspberry Pi 3, Model B （2016 年 3 月）
    • Raspberry Pi Zero W （2017 年 2 月）
    • Raspberry Pi 3, Model B+ （2018 年 3 月）
    • Raspberry Pi 3, Model A+ （2018 年 11 月）
    • Raspberry Pi 4, Model B （2019 年 6 月）
    • Raspberry Pi 400 （2020 年 11 月）
    • Raspberry Pi Pico （2021 年 1 月）

    OS Images¶

    • Alpine Linux - 基于 musl libc 和 busybox 的面向安全的轻量级 Linux 发行版.
    • Android Things - 为各种消费者、零售和工业应用构建连接设备.
    • Arch Linux ARM - 试图保持简单的轻量级和灵活的 Linux 发行版.
    • balenaOS - 用于在嵌入式设备上运行 Docker 容器的开源操作系统，该操作系统专为可靠性而设计并在生产中得到验证.
    • BerryBoot - 引导加载程序/通用操作系统安装程序，支持 VNC 和 HDMI-CEC.
    • CentOS - 树莓派上的 CentOS.
    • chilipie-kiosk - 直接启动到全屏 Chrome 的图像，非常适合仪表板和构建监视器.
    • Channels DVR Server - 频道 DVR 服务器的自定义图像，提供整个家庭 DVR 系统.
    • DietPi - 设计为适合 2GB SD 卡的最小图像，具有大量可配置的设置和脚本.
    • DroneBridge - 一个 WifiBroadcast 扩展，可真正替代 DJI Lightbridge 和其他类似系统.
    • EZ-WifiBroadcast - 经济实惠的无线数字高清视频传输变得容易.
    • FabScanPi - FabScanPi 是使用 Raspberry Pi 相机模块的开源 3D 激光扫描仪.
    • Fedora - 为 Pi 构建的 Linux Fedora 发行版.
    • FreeBSD - FreeBSD 是一种先进的计算机操作系统，用于支持现代服务器、台式机和嵌入式平台.
    • Gentoo - 适用于 Raspberry Pi 的 Gentoo Stage 3 tarball.
    • Gladys Assistant - Gladys，隐私至上的开源家庭助理.
    • Hass.io - 嵌入式设备的家庭自动化操作系统/应用程序，也可独立使用.
    • HypriotOS - 最小的基于 Debian 的操作系统，优化运行 Docker.
    • Kali Linux - 用于 ARM 设备的渗透测试和道德黑客 Linux 发行版.
    • Kano OS - 用于探索、创建和游戏的开源操作系统 - Raspberry Pi 和新的 Pi 3 免费.
    • Lakka - 完全基于 RetroArch 构建的 Raspberry Pi 上的复古游戏.
    • LibreELEC - 刚好够 Kodi 使用的操作系统
    • Manjaro - 适用于 Raspberry Pi 的友好开源 Linux 发行版.
    • Minibian - 最小的 Raspbian（比 Jessie Lite 轻）.
    • MoodleBox - MoodleBox 在树莓派上提供 Moodle 学习管理系统.
    • motionEyeOS - 将单板计算机变成视频监控系统的 Linux 发行版.
    • NetBSD - NetBSD 是一个免费、快速、安全且高度便携的类 Unix 开源操作系统.
    • NextCloudPi - 基于 Raspbian 的 Nextcloud 就绪图像. 特性 Nextcloud 11 在 Raspbian 8 上运行，具有 PHP 7 和支持 HTTP2 的 Apache 服务器.
    • NOOBS - 新的开箱即用软件，适合初学者的简单操作系统安装程序.
    • OctoPi - 3d 打印机的分发.
    • OpenELEC - 打开嵌入式 Linux 娱乐中心，这是一个非常轻量级的 Kodi 构建.
    • OpenHABian - OpenHAB 家庭自动化软件的预配置版本.
    • OpenMediaVault - OpenMediaVault 是基于 Debian Linux 的下一代网络附加存储 (NAS) 解决方案，包含 SSH、(S)FTP、SMB/CIFS、DAAP 媒体服务器、RSync、BitTorrent 客户端等服务.
    • openSUSE - openSUSE 是一个主要的 Linux 发行版，它的图像基于其稳定版 Leap 和滚动版 Tumbleweed. 可以找到许多 Pis 的图像，包括 Raspberry Pi 3 和 Raspberry Pi 4.
    • OpenWRT - OpenWrt 被描述为用于网络管理的嵌入式设备的 Linux 发行版.
    • OSMC - 基于 Kodi 的开源媒体中心.
    • PiDeck - 小型 DVS 系统，允许您使用时间码乙烯基控制数字音乐文件.
    • PiNet - 管理树莓派教室的系统.
    • PirateBox - 匿名离线移动文件共享和通信系统.
    • RasComm RaspberryPi MorseCode Translator - RasComm 是一种通信设备，可将纯文本以视觉或听觉方式转换为莫尔斯电码 (IMC).
    • Raspberry Pi OS - 官方支持的 Raspberry Pi OS（以前称为“Raspbian”），基于 Debian，可作为 lite 版本使用.
    • Raspbian - Raspbian 是基于 Debian 的免费操作系统，针对 Raspberry Pi 硬件进行了优化.
    • RasPlex - Raspberry Pi 的 Plex 客户端.
    • Recalbox - 将轻量级复古游戏和媒体中心拖放到 Raspberry Pi 上.
    • RetroPie - 在 Raspberry Pi 上进行复古游戏.
    • Risc OS - Non-Linux OS 起源于开发 ARM 微处理器的团队.
    • Rocket Show - 在舞台上现场播放同步的背景音乐、视频和 DMX 灯光序列.
    • RuneAudio - 将嵌入式硬件变成 Hi-Fi 音乐播放器的免费开源操作系统.
    • SamplerBox - Drop'n'play 采样器：将 .WAV 样本放到 SD 卡上，然后播放！
    • SARPi - 代表 Raspberry Pi 上的 Slackware ARM.
    • Snappy Ubuntu Core - 用于物联网的官方（最小）Ubuntu 发行版.
    • Twister OS - 树莓派操作系统的皮肤版本，预装了 box86 和 Retropie 等应用程序，将新手引入 Linux 和树莓派.
    • Ubuntu MATE - 基于 MATE 桌面的 Raspberry Pi Ubuntu 发行版.
    • Ubuntu Server - 用于 Raspberry Pi 的 Ubuntu 服务器发行版.
    • Volumio - 无头发烧友音乐播放器，旨在以尽可能高的保真度播放音乐.
    • Windows 10 ARM - 社区在 Raspberry Pi 上维护了 Windows 10.
    • Windows 10 IoT Core - 适用于物联网的 Windows 10 分发版.

    Tools¶

    • Alpha - 通过此系统级 GDB 服务器使用 GDB 远程加载、调试和测试裸机程序.
    • ApplePi Bakery - macOS 应用程序可轻松将图像安装/备份/恢复到 SD 卡上.
    • Atlas toolkit - 非常轻量且易于安装的工具包，可以用 Python 编写单页 Web 应用程序来试用您的 RPi，而无需安装 Web 服务器.
    • CusDeb - 用于创建安装了特定软件包的自定义图像的 Web UI.
    • balenaEtcher - SD 卡刻录机应用程序，对最终用户来说很简单，对开发人员来说是可扩展的，并且可以在任何平台上运行.
    • Drago - Wireguard 网络的灵活配置管理器，非常适合建立跨越多个 Raspberry Pi 的安全覆盖.
    • Hardened Kernel Builder for RPi - 使用单个命令交叉编译 Raspberry Pi 的 Linux 内核，增强安全性.
    • HealthyPi - HAT，包括重要的健康监测，如心电图、呼吸、脉搏血氧饱和度和可选的血压和体温感应.
    • iotwifi - Wifi AP + 8MB 客户端管理 Docker Container, 使用 REST API 同时控制 wifi 客户端和站模式.
    • OpenVPN-Setup - 将 Raspberry Pi 设置为 OpenVPN 服务器的 Shell 脚本.
    • Orangetool - 用 Python 控制单板计算机的功能.
    • pi-gen - Tool used to create the raspberrypi.org Raspbian images. This can be used to create your own custom images with specific packages installed, etc.
    • PiBakery - 基于块的、易于使用的 Raspberry Pi 设置工具.
    • Pieman - 基于 Raspbian、Devuan、Ubuntu 和 Alpine Linux 创建自定义映像的脚本.
    • Pimod - 使用类似 Docker 的简单配置文件重新配置 Raspberry Pi 映像.
    • PiShrink - Bash 脚本会自动缩小 pi 图像，然后在启动时将其调整为 SD 卡的最大大小.
    • PiVPN - 使用单个 bash 命令创建功能齐全的 OpenVPN/WireGuard VPN 服务器.
    • Pi-Apps - 最流行的 Raspberry Pi 计算机应用商店，100% 开源 bash 脚本.
    • Processing - 在 Raspberry Pi 上运行的处理开发环境.
    • Pwnagotchi - Pwnagotchi 是一种基于 AI 的 Wi-Fi 破解工具，可从其周围的 WiFi 环境中学习，以最大限度地利用其捕获的可破解 WPA 密钥材料.
    • RaspAP-webgui - 一个简单、响应迅速的 Web 界面，用于控制 Raspberry Pi 上的 wifi、hostapd 和相关服务.
    • Raspbian QEMU with network - 在 Raspbian 上启动 QEMU 的简单工具，并对系统和内核进行所需的修改. 自动网络访问. 方便在您的 PC 中创建图像.
    • Resin Bridge - 简单的应用程序/存储库，用于将 Raspberry Pi 的 wlan0 接口桥接到通过以太网 (eth0) 插入它的设备.
    • TorTiPi - 用于自动将 Raspberry Pi 转换为基于 Tor 的 wifi 热点的任务的 Shell 脚本.
    • WebStation SYSMON - 用于服务器和无头 Raspberry Pi 设置的直观的远程访问系统性能监控和任务管理工具.
    • WiFi config generator - 使用wifi设置生成wpa_supplicant.conf文件的简单工具
