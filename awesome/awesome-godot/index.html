    Godot

    Awesome Godot ¶

    精心策划的名单 free/libre Godot 的游戏、插件、附加组件和脚本.

    在 Godot 中寻找第三方编程语言支持？ See Vivraan/godot-lang-support.

    Games¶

    使用 Godot 的开源游戏.

    2D¶

    Godot 3.2+¶

    • Librerama - 免费/自由的快节奏街机迷你游戏合集.
    • Portal 2D - 门户游戏的 2D 翻拍.
    • spindle of serendipity - 可自定义条目的猜词游戏.
    • SuperStarfighter - 快节奏的本地派对游戏，星舰在自上而下的竞技场中战斗.
    • TuxBuilder - 正在重新实现 SuperTux 的工作中.

    Godot 3.1¶

    • Blipshift - 简单的自上而下移动平台游戏.
    • Circle Jump - 移动街机游戏，令人上瘾且过瘾. 提供了如何制作的教程，因此您可以自己重新创建并学习 Godot.
    • Haldric - 韦诺之战的官方正在进行中的重新实施.
    • SealedBite - 2D 平台游戏，GitHub Game Off 2019 jam 的获胜者.
    • Unknown Horizons - Unknown Horizo​​ns 的官方正在进行中的重新实现.

    Godot 3.0¶

    • Breakable - 一款类似突破的游戏.
    • Memory Game - 简单的记忆游戏.

    Godot 2.1¶

    • BlockPop - 一个简单的 Breakout 克隆.
    • Bombs - 简单的手机游戏. 你需要触摸炸弹来拆除它们.
    • Captain Holetooth - 详细世界中的探索性平台游戏. 面向年轻玩家.
    • Dolphin Island 2 - 为 A Game By It's Cover 2015 Game Jam 制作的平台游戏.
    • DynaDungeons - 幻想世界中的炸弹人克隆.
    • Kraken-Rampage - Ludum Dare 33 条目在 48 小时内完成.
    • Minehunter - 一个扫雷克隆.
    • Minilens - 免费拼图平台游戏.
    • Mountain of Hope - 在 Theatre Game Jam (PGA 2015) 上制作的游戏.
    • Ninja-Training - 作为 1 小时挑战完成的小型跑步游戏.
    • No hope - Ludum Dare 33 条目在 48 小时内完成.
    • Planet Rider - 外星环境中的 2D 物理跑酷游戏.
    • Platshoot - 2D 射击游戏.
    • Ringed - 广泛使用灯光效果的自上而下的射击游戏.
    • Tanks of Freedom - 免费的回合制等距策略游戏.
    • Three Hungry Mice - 为 2016 年 3 月的 Godot Engine Jam 制作的老鼠突围游戏.

    3D¶

    Godot 3.2+¶

    • Broken Seals - 一个正在进行的第三人称动作角色扮演游戏，支持多人游戏（需要自定义引擎模块）.
    • Super Tux Party - 一款免费/自由且开源的派对游戏，旨在复制马里奥派对等游戏的感觉.
    • Thrive - 关于生命进化的免费开源游戏（Mono）.

    Godot 3.1¶

    • Rapid Rescue - Rapid Rescue 是一款受迷宫启发的本地多人策略游戏.

    Godot 3.0¶

    • Meteorite - 受银河战士 Prime 启发的低分辨率银河恶魔城 FPS，为 LOWREZJAM 2018 制作.

    Godot 2.1¶

    • Cheese Hunter - 2016 年 3 月的 Godot Engine Jam 游戏.

    Projects¶

    使用 Godot 制作的非游戏开源项目（工具/实用程序）.

    Godot 3.2+¶

    • Arrow - 一种使用节点设计游戏叙事的工具.
    • Godello - 使用 Godot 制作的类似 Trello 的看板应用程序. Godot 的复杂非游戏应用程序的概念证明，以及多个用户之间的实时在线交互. 包括后端代码.
    • Lorien - 适用于 Windows、Linux 和 macOS 的无限画布绘图/白板应用程序. 支持绘图板和压力感应.
    • Material Maker - 按程序创建 PBR 材质（类似于 Substance Designer）.
    • Pixelorama - 2D 像素艺术编辑器.
    • ProtonGraph - 用于程序内容创建的基于节点的工具. 类似于可视化脚本，但用于 3D 模型生成（需要自定义引擎模块）.

    Templates¶

    用作您自己游戏基础的项目.

    Godot 3.2+¶

    • Crystal Bit Godot Game Template - 自以为是的游戏模板. 它包括持续集成、带有图形转换的场景加载和游戏暂停处理.
    • First Person Starter - 带有第一人称控制器的模板，可从 Inspector 轻松调整.
    • Minimum Game - 模板自上而下的 2D 像素艺术游戏，具有多个房间、HUD、菜单和自动保存.
    • Multiplayer First Person Shooter - 多人第一人称射击游戏示例项目.
    • Multiplayer Third Person Shooter - 多人第三人称射击示例项目.

    Godot 3.1¶

    • Godot-GameTemplate - 模板包含所有必要的东西. 专为像素艺术游戏而设计.

    Demos¶

    用于学习 GDScript、其概念和各种游戏功能的演示.

    Godot 3.2+¶

    • Godot Demo Projects - 官方 Godot 演示项目（除 TPS 演示之外的所有项目）.
    • TPS Demo - 具有高品质图形的官方戈多第三人称射击游戏 (TPS) 演示.
    • Godot experiments - 多个 2D、3D 和 VR 实验.
    • The Tower - 一个基于体素的跑酷游戏，展示了 Voxelman 模块的功能.

    Godot 3.1¶

    • Realistic Water Shader - 逼真的 3D 水着色器，带有泡沫和焦散.
    • OpenRPG - JRPG 风格的游戏演示和模板.
    • voxelgame - 类似 Minecraft 的体素地形演示，利用 godot_voxel 模块.

    Godot 3.0¶

    • Open Source A-RPG Demo --RPG演示.

    Godot 2.1¶

    • FPS Test - 单人 FPS 测试. 物理有效，但没有敌人，也没有适当的射击.
    • Pathfinding 2d - 一个演示项目，演示使用瓷砖地图进行寻路.
    • UDP Snapshot interpolation - 让您连接到服务器并查看网络物理.
    • UDP State synchronization - 比快照插值更有效的网络演示.
    • Voxel Game - 类似 Minecraft 的体素地形演示.

    Plugins and scripts¶

    让你做新东西或增强 Godot 功能的脚本.

    Various Godot versions¶

    • 2D Day/Night Cycle - ☀️ 2D 的日/夜循环（Godot 3.x）.
    • 2D Destructible Objects - 一个脚本，它接受一个精灵，把它分成块并使它们爆炸（Godot 3.x）.
    • AgonesSDK - 插件添加 Agones Godot (Godot 3.x) 的 SDK 功能.
    • Anima - 与 Tween (Godot 3.x) 相比，以更少的代码运行顺序和并行动画.
    • Aseprite Wizard - 用于从 Aseprite 导入动画作为 SpriteFrames (Godot 3.x) 的插件.
    • BoxCam2D - 实现基于网格的 2D 相机移动的最简单方法（Godot 3.x）.
    • CRT Shader - 使用许多着色器参数模拟 CRT 显示器的 Godot 着色器（Godot 3.x）.
    • SimpleGodotCRTShader - 一个简单的 Godot 着色器，可以模拟 CRT 显示器（Godot 2.x、3.x）.
    • EffekseerForGodot3 - 进口 Effekseer 粒子效果（Godot 3.x）.
    • Escoria - 点击式冒险游戏框架（Godot 2.1、3.2+）.
    • GdUnit3 - Godot 单元测试框架（Godot 3.x）.
    • GodotPhoenixChannels - 针对 Phoenix 框架 (Godot 3.*) 的 Channels API 的 GDScript 和 Godot 引擎实现.
    • Godot Recipes - 大量的教程和技术，无论大小.
    • Godot Shaders - 用于 2D 和 3D 的大型免费和开源着色器库.
    • Godot SQLite - SQLite 的 GDNative 包装器，可以将 SQLite 数据库用作项目中的数据存储（Godot 3.2+）.
    • GodotTIE - 文本界面引擎来控制文本输出（就像在 RPG 对话中一样）（Godot 2.x、3.x）.
    • gterm - 绘制类似于 *nix 终端模拟器（Godot >= 2.1）的 GUI 控件.
    • GUT - 在 GDScript (Godot 3.x) 中编写单元测试的实用程序.
    • HTerrain - 基于高度图的地形. 支持纹理绘制、着色、孔洞、细节层次和草（Godot 3.x）.
    • InGrid - 无限滚动网格（Godot 3.x）.
    • obs-websocket-gd - 通过 obs-websocket (Godot 3.x) 从 Godot 控制 Open Broadcaster 软件的脚本和编辑器插件.
    • SMRT-Godot - 对话系统和编辑器相当可定制，非常适合故事驱动的游戏（Godot 3.x）.
    • Scrolling Background - 滚动背景节点（Godot 2.1,3.x）.
    • GodotNotificationCenter - 通知中心（Godot 3.x）.
    • Ridiculous Coding - 使您在 Godot 1000× 中的编码体验更加荒谬（Godot 3.x）.
    • StoryTeller Engine - 对话和互动小说引擎（Godot 3.x）.
    • Tiled importer - 导入地图 Tiled （戈多 3.x）.
    • TileSet Builder - 快速构建具有样式的图块集（Godot 3.x）.

    Godot 3.2+¶

    • Cartographer - 基于高度图的 3D 地形编辑器.
    • CSG Mesh Exporter (.OBJ) - 将 CSG 节点导出到 OBJ 网格以提高编辑性能.
    • Debanding Material Shader - 简单的材质着色器，添加了去带噪声，用于 GLES2 渲染器，因为它不支持全屏去带.
    • Dijkstra map for Godot - Godot 游戏引擎的 GDNative 项目，引入了 Dijktra Map 寻路节点.
    • Code Snapshot - 一个插件，可让您在编辑器中对代码进行美化截图.
    • Delaunator GDScript - 用于二维点的 Delaunay 三角剖分的 GDScript 库.
    • Dialogic - 创建对话、角色和场景以显示对话.
    • Funexpected Flash Tools - Adob​​e Animate (Flash) 插件，允许您为 Godot 引擎导出项目.
    • GDGIFExporter - 完全用 GDScript 制作的 GIF 导出器.
    • GitHub integration - 无需打开浏览器即可与 GitHub 交互.
    • GDGotm - GDScript API gotm.io.
