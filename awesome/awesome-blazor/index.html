    Introduction¶

    What is Blazor?¶

    Blazor is a .NET web framework to build client web apps with C#.

    Blazor lets you build interactive web UIs using C# instead of JavaScript. Blazor apps are composed of reusable web UI components implemented using C#, HTML, and CSS. Both client and server code is written in C#, allowing you to share code and libraries. 更多信息 official Blazor website.

    Get started¶

    要开始使用 Blazor，请按照 Blazor Get Started 文档.

    做 Build a web app with Blaz或者 Microsoft Learn 上的学习课程也是一个好主意. 另一个很好的入门资源是 Jeff Fritz 的初学者系列 Channel9 或者 YouTube.
