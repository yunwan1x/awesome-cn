    Mastodon

    Awesome Mastodon ¶

    >与Mastodon相关的内容精选清单！

    Mastodon是世界上最大的免费，开源，去中心化微博网络. 见 Official 类别以了解有关网络的更多信息.

    Official¶

    • Website
    • GitHub repository
    • Documentation & official explaination
    • API documentation

    Clients¶

    • Tusky -Android客户端.
    • Twidere -适用于Twitter，GNU Social和Mastodon的Android应用.
    • Tooty -实验性多帐户Mastodon网络客户端（Elm）.
    • toot -Mastodon CLI客户端（Python）.
    • madonctl -Mastodon CLI客户端（运行）.
    • naumanni -专为Mastodon设计的Web用户界面.
    • Tooter -SailfishOS的本地客户端.
    • Fedilab -Android客户端.
    • Pinafore -Mastodon的替代Web客户端，专注于速度和简便性.
    • Tootle -专为基本OS设计的简单Mastodon客户端.
    • Brutaldon -Brutaldon是Mastodon的Brutalist Web 1.0 Web界面.
    • Halcyon -Mastodon和Pleroma的替代Web客户端，具有类似Twitter的界面.
    • Planiverse -Mastodon的极简主义，无JS Web客户端.
    • Toot! -适用于iOS的Mastodon客户端.
    • Sengi -适用于Mastodon和Pleroma的面向桌面的Webapp，专注于多帐户使用.

    Federated servers¶

    • Mastodon -最知名的微博平台.
    • Pleroma -轻量级微博平台.
    • GnuSocial -最早的微博平台.
    • Microblog.pub -单用户轻量级微博平台.
    • Hubzilla -具有文件，联系人和事件共享的博客/社交网络平台.
    • Friendica -社交网络平台.
    • Peertube -视频共享平台.
    • FunkWhale -音频共享平台.
    • Plume -博客平台.
    • WriteFreely -博客平台.
    • Prismo -链接聚合平台.
    • PixelFed -照片共享平台.
    • NextCloud Social -云平台内部的微博.

    Tools¶

    • Mastodon instances -Mastodon实例列表.
    • unmung.com/mastoview -预览任何实例的本地或联合时间表.
    • Toot scheduler -现在安排，稍后再嘟嘟.
    • Last -在提供Atom提要和epub（Perl）的网页上汇总嘟嘟声.
    • Forget -在用户定义的时间段后删除嘟嘟声（Python source code).
    • Mastodon Toot Bookmarklet -书签以标记当前页面（source code)
    • Mastodon – Simplified Federation! -将远程关注/互动按钮上的点击重定向到您自己的实例（source code).
    • Fediverse Explorer -每小时更新一次热门话题标签和流行嘟嘟声.
    • Mastodon #nowplaying Toot Bookmarklet -书签以嘟嘟您当前正在听的音乐. 适用于YouTube. （source code)

    User styles¶

    • Variable width -使Mastodon随浏览器的宽度缩放.
    • Narrow drawer -使用可变宽度样式时，可使抽屉变窄，其他列也相应缩放.

    User scripts¶

    • NSFW Remover -自动显示NSFW图像.
    • Customizable interface -在网格上移动和调整列的大小.
    • BirdSite -用于将Mastodon嘟嘟交叉发布到Twitter的浏览器扩展.

    Bots¶

    • feed2toot -自动解析RSS提要，识别新帖子并将其发布在Mastodon（Python）上.
    • usercount -将用户统计信息发布到Mastodon（Python）的Bot.
    • autofollow -Mastodon（Python）的Autofollow机器人.
    • hnbot -发布100分以上的Hacker News故事（Python）.
    • translator -使用@ translator @ toot.works [langcode]将任何嘟嘟声翻译成所需的语言.
    • @TrendingBot@mastodon.social -向您展示Mastodon的趋势.
    • Remindr -从资源列表中自动向Mastodon和Twitter发送提醒（Python）.
    • News Bot -在Mastodon（ClojureScript）上镜像Twitter帐户，其来源可在 GitHub.
    • Welcome Bot -自动将欢迎DM发送给新用户（Python）.
