    CakePHP

    Awesome CakePHP ¶

    一个非常棒的 CakePHP 3.x+ 插件、资源和闪亮的东西的精选列表.

    带有 "" 图标的插件也有 CakePHP 4 兼容版本.

    如果您正在寻找 CakePHP 2.x 资源，请访问： - 这 CakePHP 2.x version 这个很棒的清单 - 这个维基有 list of not-yet upgraded plugins

    您可能会发现有用的其他列表： - CakePHP Plugins - Awesome PHP - Awesome Awesomeness

    > 对于那些想知道的人； 此列表与 plugins.cakephp.org 的不同之处在于支持 > 插件子部分（而不是整个插件/仓库），更细化 > 分组和主要关注任务特定的功能.

    Plugins¶

    APM¶

    用于应用程序性能监控的插件.

    • NewRelic plugin - 一个完整的插件，可为 CakePHP 应用程序启用完整的 New Relic 集成，包括 CLI 命名、异常发送、自定义计时等.
    • NewRelic plugin - 一个简单的插件，可以使用 New Relic PHP 代理实现名称事务和浏览器计时.

    Architecture¶

    • ：草莓： Burzum/CakeServiceLayer plugin - 服务层和域/业务模型实现.

    Asset Management¶

    用于管理、压缩和缩小网站资产的工具.

    • ：草莓： AssetCompress plugin - CakePHP 的完整资产管理器.
    • ：草莓： AssetMix plugin - 提供与 Laravel Mix 资产编译.
    • Assets plugin - 动态和“即时”资产文件.
    • Less plugin - 较少的 CakePHP 解析器插件.
    • MinifyHtml plugin - 压缩 HTML 输出.

    Auditing / Logging¶

    用于审计和日志记录的插件.

    • ：草莓： AuditStash plugin - 灵活且坚如磐石的审计日志跟踪.
    • ：草莓： DatabaseLog plugin - 简单和独立的日志记录到数据库而不是文件.
    • ：草莓： Muffin/Footprint plugin - 允许将当前登录的用户传递到模型层的插件.
    • Version plugin - 促进版本化数据库实体的插件.

    Authentication and Authorization¶

    用于实现身份验证和授权的插件和库.

    • ：草莓： Acl plugin - 将 ACL 作为数据库方法进行管理.
    • ：草莓： ADmad/JwtAuth plugin - 使用 JSON Web 令牌进行身份验证的插件.
    • ：草莓： ADmad/SocialAuth plugin - 一个插件，允许您使用 Facebook/Google/Twitter 等社交提供商进行身份验证. SocialConnect/auth 图书馆的社会标志.
    • ：草莓： Authentication plugin - 官方 CakePHP 身份验证中间件插件.
    • ：草莓： Authorization plugin - 官方 CakePHP 授权栈.
    • CakeDC/NavAuth plugin - 使用 SOAP 或 OData 服务对 Navision® 服务进行身份验证的插件. 它包括 NTLM 身份验证等.
    • ：草莓： CakeDC/Users plugin - 完整的用户管理（管理面板、记住我等）、社交登录（FB、Twitter、LinkedIn、Google、Instagram）、RBAC、API 等.
    • CookieAuth plugin - 一个简单的 Cake 3 插件，可自动使用 Cookie 对用户进行身份验证.
    • HierAuth plugin - 用于分层、基于角色、简单授权的 CakePHP 插件.
    • Muffin/OAuth2 plugin - 使用 OAuth2 身份验证 league/oauth2-client.
    • ：草莓： Muffin/Tokenize plugin - 事件驱动行为，可轻松生成一次性安全令牌.
    • MultiTenant plugin - 轻松构建支持 SaaS 的 Web 应用程序.
    • ：草莓： TinyAuth plugin - 基于身份验证和角色的（单/多）授权作为非常轻量级的方法.
    • ：草莓： Tools:Passwordable - 包含 Passwordable behavior 对于密码散列的 DRY 方法.
    • ：草莓： TwoFactorAuth plugin - 允许使用 Google Authenticator 或类似应用程序进行双重身份验证以生成一次性代码. 基于 RobThree/TwoFactorAuth 图书馆.
    • UserPermissions plugin - 允许用户组或单个用户查看特定页面.

    Caching¶

    用于缓存数据的插件.

    • ：草莓： Cache plugin - 用于将视图（HTML、CSV、JSON、XML 等）缓存为静态缓存文件.

    Code Analysis¶

    用于分析、解析和操作代码库的插件.

    • ：草莓： CakeDC/PHPStan - 一个 PHPStan 扩展，用于解决静态分析器的 getter 返回类型周围的 CakePHP 魔法.
    • ：草莓： IdeHelper plugin - 通过向现有代码添加注释，类似于烘焙对新代码的作用，有助于更好地支持 IDE.
    • ：草莓： TestHelper plugin - 提供测试增强和 TDD 支持作为浏览器后端.

    Debugging¶

    用于调试和本地开发的插件.

    • Airbrake plugin 一个插件，用于将 Airbrake 与 CakePHP 无缝集成以处理错误和异常.
    • AssociationsDebugger plugin - 将您的模型关联绘制为图表的插件.
    • ：草莓： CakephpWhoops plugin - 酷孩子的 PHP 错误和异常 filp/whoops.
    • ：草莓： DebugKit plugin - 调试的事实上的标准.
    • ErrorEmail plugin - 将异常/错误信息通过电子邮件发送给您的开发团队的插件.
    • ：草莓： Execution order - 一个演示应用程序，用于显示文件、方法和回调的执行顺序.
    • Psa/FixtureCheck plugin - 一个插件，用于帮助检测实时数据库和装置中的不匹配，以使基于装置的测试更可靠和部署更安全.
    • ：草莓： Sentry plugin 一个插件，用于将 Sentry 与 CakePHP 无缝集成以处理错误和异常.
    • ：草莓： Setup plugin - 包含调试和维护工具的轻量级设置插件.

    Dependency Injection¶

    实现依赖注入设计模式的插件.

    • PimpleDi plugin 允许基于 Pimple 库的依赖注入.
    • PipingBag plugin - 依赖注入容器插件，增加了在使用对象实例及其依赖项之前配置它们的能力，并将它们存储到容器类中以便于访问.

    E-commerce¶

    用于付款和建立在线电子商务商店的插件和应用程序.

    • PaypalWPP plugin - 用于与 Paypal Web Payments Pro 通信以获取有关您帐户的交易和信息.

    Email¶

    用于发送和解析电子邮件的插件.

    • Elastic Email plugin - 用于通过 Elastic Email API 发送电子邮件的电子邮件传输插件.
    • ：草莓： EmailQueue plugin - 带有预览和发件人外壳的电子邮件队列插件.
    • Gourmet/Email plugin - 电子邮件助手、布局等.
    • ：草莓： Mailgun plugin - 用于通过 Mailgun 发送电子邮件的电子邮件传输插件.
    • SendGrid plugin - 用于通过 SendGrid API 发送电子邮件的电子邮件传输插件.

    Environment¶

    环境插件.

    • Environments plugin - 处理环境的插件.
    • Gourmet/Aroma plugin - 基于数据库的配置.
    • Settings plugin - 通过数据库管理设置的插件.

    File Manipulation¶

    用于文件操作的插件.

    • ：草莓： FileStorage plugin - 抽象文件存储和上传插件.
    • FlyPie plugin - 使用 Flysystem 抽象文件系统访问.
    • Image plugin - 与 Cake 内置的 TranslateBehavior 非常相似的图像行为.
    • ：草莓： Josbeir/Filesystem plugin - 抽象 Flysystem + 基于文件实体的上传插件.
    • ：草莓： Josegonzalez/Upload plugin - 一个可定制的插件，使用 Flysystem 写入多个后端（Dropbox、FTP、S3、本地等）.
    • Proffer plugin - 带有缩略图生成功能的可定制上传插件.
    • Xety/Cake3Upload plugin - 一个上传文件的小插件.

    Filtering and Validation¶

    用于过滤和验证数据的插件.

    • Gourmet/Filters plugin - 额外的调度过滤器（维护、机器人、IP 等）.
    • Gourmet/Validation plugin - 额外的验证提供程序（Respect、IsoCodes 等）和规则.
    • HtmlPurifier plugin - 净化器插件具有特征、行为和帮助功能，可让您在需要的地方进行净化和过滤. 您也可以配置多组过滤规则.
    • HtmlPurifier plugin - 净化器插件行为，在数据被编组到实体和/或保存之前清理数据.

    Geolocation¶

    用于地理编码地址和处理纬度和经度的插件.

    • ：草莓： Geo plugin - 包含 Geocoder behavior 和 GoogleMaps helper.

    HTTP¶

    用于 HTTP 和客户端抽象的插件

    • ：草莓： Http/Adapter/Cake library - 适配器 HTTPlug HTTP 客户端抽象.

    I18n¶

    I18n（国际化）和 L10n（本地化）插件.

    • ：草莓： ADmad/I18n plugin - 带有 I18n 相关工具的插件.
    • ：草莓： Cake/Localized plugin - 本地化验证和随时可用的翻译 PO 文件.
    • ：草莓： ShadowTranslate plugin - 一个基于影子表替换核心翻译行为的插件.
    • Transifex plugin - 通过 Transifex API 管理 i18n PO 文件和翻译.
    • Translate plugin - 通过网络后端以简单的方式管理静态内容的翻译，包括. 从 POT 文件导入，通过 API 自动建议和自动翻译.
    • Translation plugin - 提取pot文件，翻译字符串（手动，谷歌，社区），将翻译导出到pot文件.

    Imagery¶

    用于操作图像的插件.

    • ：草莓： ADmad/Glide plugin - 一个插件使用 Glide 图像处理库.
    • HtmlToImageView plugin - 将 HTML 视图渲染为图像（jpg 或 png）使用 wkhtmltoimage.
    • Imagine plugin - 一个图像处理插件和包装器 Imagine.
    • Thumber plugin - 使用创建缩略图的插件 intervention/image.

    Libs¶

    不属于任何其他类别的有用库或工具.

    • Capcake - 使用 Capistrano 部署 CakePHP 应用程序.
    • Chronos - 一个简单的独立 DateTime API 扩展（Carbon 的继承者）.
    • Composer Installers - 多框架 Composer 库安装程序.
    • Composer/Packagist - 包和依赖项管理器.
    • Graphviz - Graphviz 库.
    • Jenkins - 私有 (GitHub) 存储库的免费替代方案.
    • Rocketeer - PHP 任务运行器和部署包.
    • Travis CI - 持续集成平台 - 测试 (GitHub) 存储库的事实标准.
