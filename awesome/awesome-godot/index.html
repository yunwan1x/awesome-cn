    Godot

    Awesome Godot ¶

    精选清单 free/libre 游戏，插件，附加组件和脚本 Godot.

    在Godot中寻求第三方编程语言支持吗？ See Vivraan/godot-lang-support.

    Games¶

    使用Godot的游戏.

    2D¶

    • Blipshift -简单的自上而下的移动平台游戏（Godot 3.1）.
    • BlockPop -一个简单的Breakout克隆（Godot 2.1）.
    • Bombs -简单的手机游戏. 您需要触摸炸弹来化解它们（Godot 2.1）.
    • Breakable -突破性游戏（Godot 3.0）.
    • Captain Holetooth -详细世界中的探索性平台游戏. 面向年轻玩家（Godot 2.1）.
    • Circle Jump -移动街机游戏，令人上瘾并且令人愉快. 提供了制作方法教程，因此您可以自己重新创建它并学习Godot（Godot 3.1）.
    • Dolphin Island 2 -为A Game By It's Cover 2015 Game Jam（Godot 2.1）开发的平台游戏.
    • DynaDungeons -幻想世界中的炸弹人克隆（Godot 2.1）.
    • Haldric -韦诺之战（Godot 3.1）的正式进行中重新实现.
    • Kraken-Rampage -48小时内完成了Ludum Dare 33的参赛作品（Godot 2.1）.
    • Librerama -免费/免费的快节奏街机迷你游戏集合（Godot 3.2）.
    • Memory Game -简单记忆游戏（Godot 3.0）.
    • Minehunter -扫雷副本（Godot 2.1）.
    • Minilens -免费的益智平台游戏（Godot 2.1）.
    • Mountain of Hope -使用Theater Game Jam（PGA 2015）（Godot 2.1）制作的游戏.
    • Ninja-Training -进行1小时挑战的小型跑步游戏（Godot 2.1）.
    • No hope -48小时内完成了Ludum Dare 33的参赛作品（Godot 2.1）.
    • Planet Rider -外星环境中的2D物理跑酷游戏（Godot 2.1）.
    • Platshoot -2D射击游戏（Godot 2.1）.
    • Ringed -自上而下的射击游戏，广泛使用光效（Godot 2.1）.
    • SealedBite -2D平台游戏，是GitHub Game Off 2019 jam（Godot 3.1）的赢家.
    • spindle of serendipity -带有可自定义条目的猜词游戏（Godot 3.2）.
    • SuperStarfighter -快节奏的本地派对游戏，带有在自上而下的竞技场中战斗的飞船（Godot 3.1）.
    • Tanks of Freedom -免费的回合制等距策略游戏（Godot 2.1）.
    • Three Hungry Mice -为2016年3月的Godot Engine Jam（Godot 2.1）制造的老鼠突围游戏.
    • TuxBuilder -SuperTux（Godot 3.1）的进行中重新实现.
    • Unknown Horizons -未知地平线的正式进行中重新实现（Godot 3.1）.

    3D¶

    • Broken Seals -具有多人游戏支持的正在进行中的第三人称动作RPG（Godot 3.2-需要自定义引擎模块）.
    • Cheese Hunter -2016年3月（Godot 2.1）的Godot Engine Jam游戏.
    • Meteorite -受银河战士Prime启发的低分辨率银河战士FPS，为LOWREZJAM 2018（Godot 3.0）设计.
    • Rapid Rescue -Rapid Rescue是一款受迷宫（Godot 3.1）启发的本地多人战略游戏.
    • Super Tux Party -免费/自由和开放源代码的派对游戏，旨在复制Mario Party（Godot 3.2）等游戏的感觉.
    • Thrive -关于生活演变的免费开放源代码游戏.

    Projects¶

    使用Godot制作的非游戏项目（工具/实用程序）.

    • Godello -用Godot制作的类似Trello的看板应用程序. 使用Godot进行复杂的非游戏应用程序以及多个用户之间的实时在线交互的概念证明. 包括后端代码（Godot 3.2）.
    • Material Maker -按程序创建PBR材料（类似于Substance Designer）（Godot 3.2）.
    • Pixelorama -2D像素艺术编辑器（Godot 3.2）.

    Templates¶

    用作您自己游戏基础的项目.

    • First Person Starter -带有第一人称控制器的模板，可从检查器轻松调整（Godot 3.2）.
    • Godot-GameTemplate -包含所有必要材料的模板. 专为像素艺术游戏（Godot 3.1）设计.
    • Multiplayer First Person Shooter -多人第一人称射击游戏示例项目（Godot 3.2）.
    • Multiplayer Third Person Shooter -多人第三人称射击游戏示例项目（Godot 3.2）.

    Demos¶

    • Demos学习GDScript，其概念和各种游戏功能.*

    • FPS Test -单人FPS测试. 物理可以工作，但是没有敌人，也没有适当的射击（Godot 2.1）.

    • Open Source A-RPG Demo -RPG演示（Godot 3.0）.
    • OpenRPG -JRPG风格的游戏演示和模板（Godot 3.1）.
    • Pathfinding 2d -一个演示项目，演示使用tilemap进行寻路（Godot 2.1）.
    • Realistic Water Shader -具有泡沫和腐蚀性的逼真的3D水着色器（Godot 3.1）.
    • The Tower -基于体素的跑酷游戏，用于显示Voxelman模块的功能（Godot 3.2）.
    • UDP Snapshot interpolation -可让您连接到服务器并查看网络物理（Godot 2.1）.
    • UDP State synchronization -比快照插值（Godot 2.1）更有效的网络演示.
    • Voxel Game -类似Minecraft的体素地形演示（Godot 2.1）.
    • voxelgame -利用godot_voxel模块（Godot 3.1），类似于Minecraft的体素地形演示.

    Modules¶

    将东西集成到Godot内核（C ++）中.

    • Entity Spell System -用于复杂（可选的多人游戏）RPG（Godot 3.2、4.0）的实体和咒语系统.
    • FMOD Integration -将FMOD音频引擎集成到Godot中的模块.
    • godot-anl -的包装 Accidental Noise Library 具有视觉噪音编辑支持.
    • godot-apple-id -在Godot中使用Apple登录的模块（Godot 3.2）.
    • godot-python -对Godot 3的Python支持.
    • godot-enet-better -更好的ENet模块，用于Godot的高性能多人游戏.
    • godot-voxel -在Godot Engine中创建体积世界的模块.
    • Gdnet for Godot 3 -Godot 3的ENet包装器.
    • GodotSteam -Godot的Steam API库绑定. 支持Windows，macOS和Linux（Godot 3.1）.
    • Head tracking using OpenCV
    • Open Adaptive Music Library
    • NativeDialogs -在Godot中使用本机对话框.
    • PortAudio -包装方法和Godot友好类型 PortAudio （Godot 3.2）.
    • RawPacker -RawArray的简单二进制打包/解包.
    • spine - Spine 动画支持模块.
    • Texture Packer -在运行时分层，打包和合并纹理的工具（Godot 3.2、4.0）.
    • Voxelman -更加关注编辑器集成，游戏相关功能和可扩展性的体素引擎（Godot 3.2、4.0）.
    • WaterSplash 2D node

    • godotdetour -绕行/绕行人群库（Godot 3.2）的GDNative实现.
    • GodotPhoenixChannels -Phoenix框架（Godot 3.1）的Channels API的GDScript和Godot引擎实现.
    • Godot Game Tools -Blender插件可轻松将资产导入Godot.
    • Godot Mixing Desk -用几个节点和几行代码制作程序声音和自适​​应/程序音乐（Godot 3.1）.

    • GodotRx - Reactive extensions for Godot C# (Godot 3.2 Mono).
    • GodotTIE -文本接口引擎，用于控制文本输出（例如在RPG对话框中）（Godot 2.x，3.x）.
    • Godot-Trail-System -先进的2D / 3D追踪系统（Godot 3.2）.
    • Grass Plugin -处理大量草，树叶和其他植被的插件（Godot 2.1）.
    • gterm -GUI控件，其绘制类似于 * nix终端模拟器（Godot 2.1）.
    • GUT -用于在GDScript（Godot 3.x）中编写单元测试的实用程序.
    • InGrid -无限滚动网格（Godot 3.x）.

    • Simplex 2D/3D -单工确定性噪声函数.

    • SMRT-Godot -对话框系统和编辑器可自定义，非常适合故事驱动的游戏（Godot 3.0）.
