    Awesome nix

    Awesome Nix ¶

    > Nix 社区中最佳资源的精选列表.

    Nix 是适用于 Linux 和其他 Unix 系统的强大包管理器，它使包管理可靠且可重复.

    请阅读 contribution guidelines 在贡献之前.

    Resources¶

    Learning¶

    • How to Learn Nix - 这就像 Let's Play，但用于晦涩的软件文档.
    • Nix Notes - 关于 Nix 的一系列简短说明，每个说明都对同一个虚拟机映像有贡献.
    • Nix Pills - 最好的学习方式，有例子.
    • Nix Shorts - 关于 Nix 的简短笔记集.
    • nix.dev - 为开发人员提供关于使用 Nix 生态系统完成工作的自以为是的指南.
    • Tour of Nix - 关于 Nix 语言结构的在线互动教程.

    Discovery¶

    • Hound - 轻松搜索所有或选定的 Nix 相关存储库.
    • Nix Package Versions - 查找频道中可用的软件包的所有版本以及可以从中下载的版本.
    • Pkgs on Nix - 包含来自所有渠道的所有版本的 Nix 包的数据库.

    Newsletters¶

    • NixOS Weekly - The 时事通讯随时了解社区更新.

    Installation Media¶

    • nixos-generators - 采用 NixOS 配置并构建多种不同的映像类型，包括 VirtualBox VM、Azure 映像和安装 ISO.
    • nixos-up - 超级简单的 NixOS 安装程序，可以从安装 ISO 中使用.

    Channel History¶

    • Channel History - 获取 Nix 频道的历史 git 提交.
    • Nix Infra Status - 获取每个 Nix 频道的年龄和当前 git 提交.
    • nixpkgs PR tracker - 一个 PR 是否已进入频道的跟踪器.

    Deployment Tools¶

    • Colmena - 一个简单的、无状态的 NixOS 部署工具，以 NixOps 和 morph 为蓝本.
    • deploy-rs - 一个简单的多配置文件 Nix-flake 部署工具.
    • krops - 用于远程或本地部署 NixOS 系统的轻量级工具包.
    • KubeNix - 使用 Nix 的 Kubernetes 资源构建器.
    • KuberNix - 通过 Nix 包的单依赖 Kubernetes 集群.
    • morph - 用于管理现有 NixOS 主机的工具.
    • Nixery - 一个与 Docker 兼容的容器注册表，它通过 Nix 临时构建图像.
    • NixOps - 官方 Nix 部署工具，兼容 AWS、Hetzner 等.
    • nixos-shell - 使用 Nix 的简单无头 VM 配置（类似于 Vagrant）.
    • pushnix - 推送 NixOS 配置并使用 ssh 触发重建的简单 cli 实用程序.
    • terraform-nixos - 一组用于部署 NixOS 的 Terraform 模块.
    • terranix - 使用 Nix 和 NixOS 模块系统编写您的 Terraform 代码.

    Command-Line Tools¶

    • comma - 快速运行任何二进制文件； 将 nix run 和 nix-index 包装在一起.
    • nixfmt - Nix 代码的格式化程序，旨在轻松应用统一的样式.
    • nixpkgs-fmt - 用于 nixpkgs 的 Nix 代码格式化程序.
    • nixpkgs-hammering - 将您的包装表情打成形状.
    • nix-alien - 在 Nix/NixOS 上轻松运行未打补丁的二进制文件.
    • nix-diff - 解释为什么两个 Nix 派生不同的工具.
    • nix-index - 快速定位具有特定文件的 Nix 包.
    • nix-prefetch - 用于更新源校验和的通用工具.
    • nix-tree - 交互式浏览 Nix 派生的依赖关系图.
    • statix - 用于检查和修复 Nix 代码中的反模式的 linter/fixer.

    Development¶

    • Arion - 在 Nix/NixOS 的帮助下运行 docker-compose.
    • cached-nix-shell - 使用缓存快速打开后续 shell 的“nix-shell”替代品.
    • Cachix - 托管二进制缓存服务； 免费用于开源项目.
    • devshell - mkShell 带有额外的位和 toml 配置选项，以便能够加入非 nix 用户.
    • flake-utils - 纯 Nix 薄片实用功能，以帮助编写薄片.
    • gitignore.nix - 功能最完整且易于使用的 .gitignore 集成.
    • lorri - 一个更好的“nix-shell”，用于增强 direnv 的开发.
    • niv - 使用包固定对 Nix 项目进行简单的依赖管理.
    • nix-direnv - 用于 direnv 环境自动加载器的快速加载器和符合 flake 的配置.
    • nixpkgs-review - 验证 Nixpkgs 中的拉取请求是否正确构建的最佳工具.
    • pre-commit-hooks.nix - 在提交时和你的 CI 上运行 linter/formatters.

    Programming Languages¶

    Arduino¶

    • nixduino - 基于 Nix 的工具，可帮助构建 Arduino 草图.

    Crystal¶

    • crystal2nix - 将 shard.lock 转换为 Nix 表达式.

    Elm¶

    • elm2nix - 将 elm.json 转换为 Nix 表达式.

    Haskell¶

    • cabal2nix - 将 Cabal 文件转换为 Nix 构建表达式.
    • haskell.nix - Nixpkgs 的替代 Haskell 基础设施.
    • nix-haskell-mode - Emacs 中的自动 Haskell 设置.
    • nixkell - 使用 Nix 和 direnv 的 Haskell 项目模板.
    • stack2nix - Generate nix expressions for Haskell projects.

    Node.js¶

    • Napalm - 支持在 Nix 中使用轻量级 npm 注册表构建 npm 包.
    • node2nix - 从“package.json”（或“package-lock.json”）生成 Nix 表达式（以文件形式存储）.
    • npmlock2nix - 从“package-lock.json”（内存中）生成 Nix 表达式，主要用于 Web 项目.
    • yarn2nix - 从 yarn.lock 文件生成 Nix 表达式.

    PHP¶

    • composer2nix - 生成 Nix 表达式以构建作曲家包.
    • composition-c4 - 支持从 composer.lock 构建作曲家包（使用 IFD）.
    • nix-phps - Flake 包含旧的和未维护的 PHP 版本（用于 CI 使用）.

    PureScript¶

    • Easy PureScript Nix - 通过 Nix 轻松使用 PureScript 和其他工具的项目.

    Python¶

    • mach-nix - 创建高度可重现的 python 环境的工具.
    • poetry2nix - 直接从构建 Python 包 Poetry's poetry.lock. 无需转换步骤.
    • pypi2nix - 为 Python 包生成 Nix 表达式.

    Ruby¶

    • Bundix - 为您的 Bundler 管理的应用程序生成 Nix 表达式.

    Rust¶

    • fenix - 适用于 nix 的 Rust 工具链和 Rust 分析器.
    • naersk - 直接从 Cargo.lock 构建 Rust 包. 无需转换步骤.
    • nix-cargo-integration - 一个允许轻松轻松地集成 Cargo 项目的库.
    • nixpkgs-mozilla - Mozilla 与 Rust 工具链和 Firefox 的叠加.
    • rust-nix-templater - 为 Rust 项目生成 Nix 构建和开发文件.
    • rust-overlay - 二进制分布式 Rust 工具链的纯且可重现的 nix 覆盖.

    Scala¶

    • sbt-derivation - sbt 的 mkDerivation，类似于 buildGoModule.
    • sbt-nix.g8 - 带有 Nix 支持的新 Scala 项目的 giter8 模板.

    NixOS Modules¶

    • musnix - 在 NixOS 中进行实时音频工作.
    • nixcloud-webservices - Nixpkgs 扩展，重点是易于部署网络相关技术.
    • Simple Nixos Mailserver - 一个完整的邮件服务器，使用 NixOS 模块进行管理.

    Overlays¶

    • awesome-nix-hpc - 高性能计算套件.
    • Home Manager - 像 NixOS 一样管理您的用户配置.
    • nix-bitcoin - 用于比特币节点的 Nix 包和 NixOS 模块，具有更高层协议，强调安全性.
    • nix-darwin - 像在 NixOS 上一样管理 macOS 配置.
    • nixpkgs-firefox-darwin - 自动更新 macOS 的 Firefox 二进制包.
    • nixpkgs-wayland - 前沿的 Wayland 套餐.
    • NUR - Nix 用户存储库. 所有覆盖之母，允许访问用户存储库并通过属性安装包.
