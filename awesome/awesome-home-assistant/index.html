    Home Assistant

    Awesome Home Assistant ¶


    https://awesome-ha.com

    Home Assistant 是一个开源的家庭自动化，可以将本地控制和 隐私第一. 由全球修补匠和 DIY 社区提供支持 爱好者. 非常适合在 Raspberry Pi 或本地服务器上运行.

    如果您想获得外观和感觉的印象， 你应该看看 Home Assistant online demo.

    令人敬畏的家庭助理是一个令人敬畏的策划列表 Home Assistant 资源. 附加软件、教程、自定义集成、附加组件、 自定义 Lovelace 卡片和插件、食谱、示例设置等等.

    该列表分为几类. 这些类别中的链接没有 预先确定的顺序； 订单是为了贡献. 如果你想贡献， 请阅读 guide.

    How to use¶

    Awesome Home Assistant 是一个很棒的列表，适合尝试自动化每个 他们家的一面. 自动化您的家庭是一项漫长、艰难且永无止境的过程 通常需要大量修补的任务.

    您可以通过以下方式浏览列表：

    • 只需按command/ctrl + F即可搜索关键字
    • 或者，使用我们网站上的搜索：https://www.awesome-ha.com

    Installing¶

    Home Assistant 有几种安装/运行方法. 很多人都有 不同的意见和他们的个人喜好. 每种方法都有其 的优点和缺点. 重要的是要知道，没有错，或者就在这里， 每种技术都安装 SAME 家庭助理.

    当前家庭助理_recommends_家庭助理操作系统安装方法.

    • Home Assistant OS - 使用托管环境进行安装（推荐方法）.
    • Home Assistant Container - 在 Docker 上安装.
    • Home Assistant Supervised - 为专家安装半托管环境.
    • Home Assistant Core - 使用 Python 虚拟环境手动安装.

    In case you need help¶

    有多种方式可以与家庭助理社区取得联系. 如果您有问题、需要帮助、想要请求功能，都没有关系， 或者只是说“嗨”.

    Official Channels¶

    • Home Assistant Discord - 加入聊天，我们大多数人都在那里.
    • Home Assistant Community - 论坛，也用于功能请求.
    • Home Assistant Subreddit - 如果您喜欢 Reddit，请订阅.
    • Home Assistant Facebook Group - Facebook 爱好者群.

    Other Channels¶

    • Dr. ZZs - Zzs 博士的 Facebook 群组.
    • Home Assistant Community Add-ons Discord - 获得对家庭助理社区附加组件的支持.
    • ESPHome Discord - 获得对您的 DIY ESPHome 项目的支持.
    • Dutch Domotics Discord - 荷兰家庭自动化爱好者的 Discord 服务器.

    Public Configurations¶

    有些人将他们完整的家庭助理配置存储在 GitHub 上. 他们是 一个很棒的学习来源和一个很好的灵感来源.

    • Vasiley - 运行两个一起工作的实例.
    • Alok Saboo - 也称为 arsaboo. 定期更新.
    • Aaron Bach - 也称为bachya. 定期更新并包含大量 Dockerized 服务.

    • Franck Nijhof - 基于 Hass.io，与其他配置结构非常不同.

    Add-ons¶

    附加组件是附加的应用程序和服务，可以同时运行 家庭助理. 家庭助理操作系统和监督安装类型， 提供能够运行和管理这些附加组件的主管.

    Official Add-ons¶

    由家庭助理团队创建和维护.

    • DuckDNS - 更新您的 Duck DNS IP 地址并使用 Let's Encrypt 生成 SSL.
    • File editor - 基于浏览器的配置文件编辑器.
    • Mosquitto - 快速可靠的 MQTT 代理.
    • Terminal & SSH - 允许使用 Web 终端或 SSH 客户端远程登录.
    • Samba - 使用 Windows 网络共享访问您的配置文件.
    • NGINX SSL proxy - 具有 SSL 终止的反向代理.
    • deCONZ - 使用 Dresden Elektronik 的 ConBee 或 RaspBee 硬件控制 ZigBee 网络.
    • TellStick - 运行 TellStick 和 TellStick Duo 服务.
    • Ada - Ada 是由 Almond 提供支持的语音助手，它是开放和隐私保护的.
    • Almond - 开放、保护隐私的虚拟助手.
    • HomeMatic - 基于 OCCU 的 HomeMatic 中心.
    • Let's Encrypt - 从 Let's Encrypt 获取免费 SSL 证书； 一个开放和自动化的证书颁发机构 (CA).
    • MariaDB - 一个开源关系数据库（MySQL 的分支）.
    • OpenZWave - 将 Z-Wave U 盘与 QT OpenZWave 守护程序一起使用.

    Custom Lovelace UI Cards¶

    Lovelace 允许人们在其上构建自定义卡片，您可以 轻松添加到您的实例.

    • Auto-Entities Card - 动态添加实体：魔法.
    • Canvas Gauge Card - 使用来自 canvas-gauges.com 的很棒的仪表.
    • Big Number Card - 显示传感器的大数字，包括作为背景的严重性级别.
    • Animated Weather Card - 漂亮的卡片显示天气，带有微妙的动画.
    • Thermostat Card - 恒温器控制卡，看起来像 Nest Thermostat.
    • Mini Media Player - 简约的媒体播放器卡.
    • Mini Graph Card - 简约的传感器图形卡.
    • Button card - 您的实体的按钮卡.
    • Slideshow card - 图像或卡片的动态幻灯片.
    • Swiper card - 轻弹/滑动多张卡片.
    • Slider Entity Row - 添加一个滑块来调整，例如，lovelace 实体卡中的灯光亮度.
    • Power Wheel Card - 一种直观的方式来表示您家正在消耗或生产的电力.
