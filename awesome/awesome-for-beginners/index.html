    新手友好项目

    Awesome First PR Opportunities ¶

    很棒的对初学者友好的项目列表.

    受启发 First Timers Only 博客文章.

    如果您是开源项目的维护者，请在您的项目中添加标签“仅限初学者”（或类似名称）并在此处列出，以便人们可以找到它.

    如果您想贡献自己的力量，请浏览此列表，然后查看 first-timers-only labelled open issues on Github，然后按照 @first_tmrs_only 在Twitter上创建新的“仅限初学者”问题时会收到通知.

    如果您不是程序员，但想贡献自己的力量，请查看 Awesome for non-programmers 列表.

    .NET¶

    • MvvmCross （标签：仅限初学者）
      用于跨平台解决方案的.NET MVVM框架，包括Xamarin.iOS，Xamarin.Android，Windows和Mac.
    • RawCMS （标签：第一期好）
      RawCMS是用ASP.NET Core编写的无头CMS，专为使用API​​优先技术的开发人员而设计.
    • Shouldly （标签：跳入）
      应该测试.NET-声明应该的方式！

    C¶

    • Neovim （标签：第一期好）
      Vim-fork专注于可扩展性和敏捷性.

    C¶

    • grok.net （标签：第一期好）
      跨平台.NET grok实施
    • OpenRA （标签：简单）
      一个实时战略游戏引擎，支持早期的Westwood经典游戏，例如Command＆Conquer：Red Alert.
    • osu! （标签：优先事项）
      音乐游戏. 节奏仅需点击一下即可！
    • Uno Platform (label: good first issue)
      OSS project for creating pixel-perfect, single-source C# and XAML apps which run natively on iOS, Android, macOS, Linux and Web via WebAssembly.
    • Cake (label: Good first issue)
      Cake (C# Make) is a free and open source cross-platform build automation system with a C# DSL for tasks such as compiling code, copying files and folders, running unit tests, compressing files and building NuGet packages.

    C++¶

    • electron （标签：第一期好）
      使用JavaScript，HTML和CSS构建跨平台的桌面应用程序
    • Godot Engine (label: junior job)
      2D and 3D cross-platform game engine. Also has C# and Python code.
    • projectM （标签：第一期好）
      使用OpenGL和GLSL的音乐可视化程序库. 拥有使用Qt5，SDL，emscripten，iTunes，Kodi的应用程序.
    • Roc Toolkit （标签：需要帮助）
      用于通过网络进行实时音频流传输的工具包.
    • tensorflow （标签：统计信息：欢迎捐款）
      使用数据流图进行计算以进行可扩展的机器学习
    • Yugabyte DB （标签：第一期好）
      分布式SQL数据库.
    • MoveIt （标签：第一期好）
      易于使用的开源机器人操纵平台，用于开发商业应用，原型设计和基准测试算法.

    Clojure¶

    • Alda （标签：低挂水果）
      一种针对音乐家的音乐编程语言. ：笔记：

    ClojureScript¶

    • LightTable （标签：第一期好）
      下一代代码编辑器！ KickStarter上资金最高的项目之一.

    CSS¶

    • sButtons （标签：第一期好）
      简单的按钮，您可以轻松地用于下一个项目.

    Dart¶

    • dart.dev （标签：初学者）
      一个涵盖Dart语言和公共库的网站，面向Dart库，Web应用程序，服务器端代码和移动（Flutter）应用程序的开发人员.

    Elixir¶

    • Ecto （标签：级别：入门者）
      Ecto是Elixir的数据库包装程序和语言集成查询
    • Elixir （标签：级别：入门者）
      Elixir是一种动态的功能语言，旨在用于构建可扩展和可维护的应用程序

    Go¶

    • CockroachDB （标签：第一期好）
      可扩展，可生存，高度一致的SQL数据库
    • Docker/CLI （标签：exp /初学者）
      Docker CLI
    • Helm （标签：第一期好）
      Kubernetes软件包管理器
    • Hugo （标签：exp /初学者）
      在GoLang中用爱构建的快速灵活的静态网站生成器
    • Kubernetes （标签：第一期好）
      生产级集装箱调度与管理系统
    • Mattermost （标签：抢劫，难度/ 1：容易，技术/使用）
      Golang和React中的开源Slack替代方案
    • Moby （标签：exp /初学者）
      开源应用容器引擎
    • Terraform （标签：第一期好）
      安全有效地构建，更改和版本化基础结构的工具.
    • TiDB （标签：新贡献者）
      分布式可伸缩混合事务和分析处理（HTAP）数据库
    • script （标签：第一期好）
      一个Go库，用于执行shell脚本擅长的那种任务：读取文件，执行子进程，计算行数，匹配字符串等. 初学者非常欢迎，他们会在PR过程中获得详细的代码审查和帮助.
    • httpexpect （标签：需要帮助）
      Go的端到端HTTP和REST API测试.
    • Killgrave （标签：第一期好）
      在Go中生成模拟服务器的简单方法.
    • lxd （标签：简单）
      系统容器和虚拟机管理器.
    • PureLB （标签：GoodFirstIssue）
      Kubernetes的负载均衡器协调器，使用标准的Linux网络和路由协议.
    • containerd （标签：exp /初学者）
      行业标准的容器运行时，着重于简单性，健壮性和可移植性.
    • Meshery （标签：第一期好）
      网格，服务网格管理平面.

    Java¶

    • Strongbox （标签：第一期好）
      Strongbox是用Java编写的工件存储库管理器.
    • TEAMMATES （标签：第一期好）
      TEAMMATES是一个免费的在线工具，用于管理学生的同伴评估和其他反馈路径.
    • elasticsearch （标签：第一期好）
      开源，分布式，RESTful搜索引擎.
    • JabRef （标签：第一期好）
      使用包括JavaFX在内的现代Java功能管理文献参考的桌面应用程序. Dedicated to code quality and constructive feedback：两个开发人员都会审核每个请求请求，以提供高质量的反馈并确保高质量的新贡献.
    • Wikimedia Commons Android App （标签：第一期好）
      允许用户将其Android手机/平板电脑中的图片上传到Wikimedia Commons.
    • XWiki （标签：入职）
      XWiki 是一个用Java编写的免费Wiki软件平台，其设计着重于可扩展性. 初学者应遵循 onboarding wiki.
    • zerocode （标签：第一期好）
      API自动化，无需编码，简单的JSON响应断言，测试REST，SOAP，Kafka和Java / DB API，CI / Jenkins Friendly.
    • SirixDB （标签：第一期好）
      SirixDB是一种进化的，版本化的NoSQL文档存储（XML和JSON）（主要是用Java编写）. 它使用从ZFS和Git借来的许多概念在提交期间存储压缩快照. 每个修订均已建立索引，并且可以使用临时查询来查询文档存储. 它特别适合现代硬件.
    • Images-to-PDF （标签：第一期好）
      一个将图像转换为PDF文件的Android应用.
    • Trino (formerly Presto SQL) （标签：第一期好）
      用于大数据的分布式SQL查询引擎. 寻求指导 project's Slack.
    • appsmith （标签：第一期好）
      拖放内部工具生成器
