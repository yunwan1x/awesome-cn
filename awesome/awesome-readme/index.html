    • Hall-of-fame - 帮助在 README 上显示对 repo 贡献者的认可. 具有新的/趋势/顶级贡献者. 每小时更新一次.
    • Make a README - 编写自述文件的指南. 包括带有实时 Markdown 渲染的可编辑模板.
    • README best practices - 一个可以复制粘贴 README.md 的地方
    • readme-md-generator - 生成漂亮的 README.md 文件的 CLI
    • Readme.so - 一个简单的编辑器，可让您快速添加和自定义项目自述文件所需的所有部分.
    • READMINE - 用于软件项目的全面、清晰和自我描述的 README 文件模板； 复制它并根据需要进行编辑.
    • Standard Readme - 标准的 README 样式规范. 也有一个生成器来帮助创建符合规范的自述文件.
    • Zalando's README Template - 帮助您涵盖所有基础知识的简单模板.

    Creating GIFs¶

    在自述文件中嵌入动画 gif 可以快速展示您的项目的作用并吸引读者的眼球. 以下是一些可以帮助您快速为您的项目创建 gif 的程序：

    • Gifox - $4.99 - 最干净的用户界面、热键、许多高级功能
    • Gifski - 免费 - 比其他颜色更鲜艳，但仍然保持较小的尺寸
    • Giphy Capture - 免费 - 易于上传到 giphy.com，有点烦人的用户体验.
    • LICEcap - 免费 - 不那么直观，功能更多
    • Peek - 免费 - Linux 用户简单易用.
    • Recordit - 免费 - 简单、干净的用户界面，但会自动上传到 recordit.co
    • ScreenToGif - 免费 - 开源、可定制的用户界面和易于编辑的 GIF，易于上手.
    • terminalizer - 免费 - 记录您的终端并生成动画 gif 图像或共享网络播放器
    • ttystudio - 免费 - 对于命令行工具，一个终端到 gif 的记录器减少了麻烦.
