    Awesome quantum computing

    Awesome Quantum Computing ¶

    Quantum computing 利用纠缠和叠加等量子力学现象来操纵量子位在量子计算机上进行计算. 目前可用的是在公共可用的量子计算机上创建和运行程序的工具以及了解它们的资源.

    这是有关学习和开发量子计算机的最新资源的精选列表. 目标是建立一个分类的社区驱动的最新、高质量资源集合.

    随时欢迎分享、建议和贡献！ 请先看一下贡献指南和质量标准. 感谢所有贡献者，你们太棒了，没有你们就不可能！

    有关开源量子软件项目的更多资源，请查看 qosf repo.

    Learning¶

    • An Interactive Introduction to Quantum Computing - David Kemp 用于量子门计算的交互式学习.
    • CNOT - 易于理解，逐步介绍量子计算概念.
    • Chris Ferrie - 澳大利亚悉尼大学教授，《婴儿量子计算》（以及更多）的作者，以及关于 Medium 的优秀量子计算讲座.
    • Documentation for Forest and pyQuil - Rigetti Computing 的 SDK 教程.
    • Documentation for Strawberry Fields - 有关量子计算的光子连续变量方法的背景信息，以及 Xanadu 的 SDK 教程.
    • D-Wave Leap - 演示和教育资源以及访问 D-Wave 的量子计算机.
    • IBM Q Full User Guide - 简要介绍量子计算和 IBM Q 的简短教程.
    • Introduction to Quantum Computing - CERN（欧洲核研究组织）关于量子计算的在线介绍性讲座.
    • John Preskill's Notes on Quantum Computation - 加州理工学院量子计算课程物理 219/计算机科学 219 的笔记，作者是 John Preskill.
    • Nielsen and Chuang - 关于量子算法问题的工作示例.
    • Qiskit Tutorials - Jupyter 笔记本简单地演示了如何使用 Qiskit.
    • Quantum Algorithm Implementations for Beginners - 使用 qiskit 对 20 种不同量子算法的简要调查.
    • Quantum Algorithm Zoo - 量子算法的综合目录.
    • Quantum Algorithms - 关于量子算法的开源讲义集，重点是机器学习和数据分析.
    • Quantum Computer Programming - 为没有量子力学背景的人教授量子计算的斯坦福课程.
    • Quantum Computing Foundations - 涵盖 Azure Quantum 和量子计算元素的 Microsoft Learn 学习路径.
    • Quantum Computing Playground - 能够模拟多达 22 个量子位的 3D 量子状态可视化工具.
    • Quantum Computing UK - 免费的 Qiskit 教程和代码库.
    • Quantum Computing for the Very Curious - Michael Nielsen 和 Andy Matuschak 介绍量子计算的论文.
    • Quantum Computing with Microsoft QDK - Manning liveProject 系列，指导学习者使用 Microsoft QDK 实现各种协议和算法.
    • Quantum Inspire Knowledge Base - 易于阅读的知识库，丰富的基本量子计算概念.
    • Quantum in the Cloud - 四量子位光子量子模拟器和计算机.
    • Quantum Katas - Programming exercises for learning quantum computing and Q#.
    • Quantum Machine Learning for Data Scientists - 量子机器学习算法的解释.
    • Quirk - 基于浏览器的拖放式量子电路模拟器，可实时做出反应、模拟和动画.
    • QWorld's Bronze-Qiskit - 非常易于访问的面向计算/编程的教程. 它从矩阵的基础开始到格罗弗算法. 还有随附的视频解释了这些教程.
    • The Quantum Quest - 为高中生设计的关于量子计算原理的介绍性网络课程.

    MOOCs¶

    • Brilliant.org Quantum Computing - 通过问题的解释. 与微软和 Alphabet X 一起策划.
    • Introduction to Quantum Information Science - 牛津大学 Artur Ekert 教授的“量子信息导论”.
    • MIT Quantum Information Sciences - 麻省理工学院量子信息科学课程系列讲义.
    • Programming a Quantum Computer with Qiskit - 2 小时的指导课程，重点是学习如何利用 Qiskit 为量子计算机编码.
    • QC101 Quantum Computing & Quantum Physics for Beginners - 关于量子密码学和如何运行量子程序的入门课程.
    • Quantum Computing: Theory to Simulation and Programming - 了解 D-Wave 量子退火器架构以及一些实际任务.
    • Quantum Cryptography - 了解量子通信如何提供由自然法则保证的安全性.
    • Quantum Information Science I, Part I - 量子信息和计算基础课程.
    • Quantum Machine Learning - 了解量子技术可以为机器学习带来的好处.
    • Quantum Mechanics and Quantum Computation - 对量子力学基本原理的概念介绍.
    • The Introduction To Quantum Computing - 对计算、其背后的数学及其量子对应物的微妙介绍，然后深入讨论了一些量子算法.
      • Quantum Computing. Less Formulas - More Understanding - 上一课的同一位教授，这次更侧重于量子概念而不是数学.
    • The Quantum Internet and Quantum Computers: How Will They Change the World? - 了解量子计算机和量子互联网发展背后的原理和承诺，以及它们将如何影响我们的未来.
    • Understanding Quantum Computers - 用最少的数学介绍量子计算的关键概念.

    Development Tools¶

    • Amazon Braket - 完全托管的服务，提供在量子模拟器和计算机上运行量子电路的开发环境.
    • Blueqat - 用于量子门计算的 Python 软件开发工具包.
    • Cirq - 用于编写、操作和优化 NISQ 电路以在量子计算机上运行的 Python 库.
    • IBM Quantum Experience - 在线量子作曲家在真实的量子计算硬件上运行实验.
    • Mitiq - 用于在量子计算机上实施错误缓解技术的 Python 工具包.
    • NISQAI - 用于在近期量子计算机上执行量子人工智能的库.
    • Ocean - D-Wave 的 SDK，用于使用 Python 在他们的量子计算机上进行开发.
    • Orquestra - Zapata Computing 的统一量子操作环境，允许启用量子的工作流.
    • Paddle Quantum - 百度用于量子机器学习的python工具包.
    • PennyLane - 用于量子计算和量子机器学习的开源框架，集成了各种其他平台.
    • Project Q - 在 Python 中实现量子计算的框架.
    • pyQuil - 使用 Rigetti 的 Quil 进行量子编程的 Python 库.
    • pytket - 用于与剑桥量子计算的 t|ket> 接口的 Python 模块； 用于电路优化和量子位分配的工具.
    • QCL - 用于量子计算机的较旧的 C 语言. 只有一个模拟器和调试器.
    • Qiskit SDK - IBM 的软件开发工具包，用于在模拟器和真实硬件上编写和运行量子算法.
    • Qiskit.js - IBM 制作的用于 JavaScript 的 Qiskit.
    • Qrack - 高性能 LGPL 许可的 C++ 量子模拟器库、文档和测试代码.
    • Quantum++ - 高性能现代 C++11 量子计算库.
    • Quantum Inspire - 在模拟器或量子硬件上运行量子算法的平台 - QuTech.
    • Quantum Programming Studio - 基于 Web 的量子编程 IDE 和模拟器.
    • Quipper - 用于量子计算的嵌入式、可扩展、函数式编程语言.
    • Qurry - 基于函数和概率范式的量子概率编程语言.
    • QuTiP - Python 中的量子工具箱，用于模拟开放量子系统的动力学.
    • Q# - Microsoft quantum development kit and Q# programming language.
    • Strangeworks Platform - 一个与硬件无关的平台和接口，允许专注于开发而不是特定的硬件解决方案构建.
    • TensorFlow Quantum - 一个量子机器学习库，它集成了 Cirq 和 TensorFlow，用于经典或量子数据的混合量子经典模型的原型设计.
    • Tequila - 由 Alan Aspuru-Guzik 小组（多伦多大学）开发的可扩展量子信息和学习架构.

    Books¶

    • An Introduction to Quantum Computing - 在可访问性和数学严谨性之间取得了极好的平衡. 它适合本科生.

    • Dancing with Qubits - 量子计算的工作原理以及它如何改变世界.
    • Learn Quantum Computation using Qiskit - 一本涵盖量子算法并展示如何使用 Qiskit 在真实硬件上运行它们的开源教科书. 还包括先决条件.
    • Learn Quantum Computing with Python and Q# - Introduces quantum computing using Python and Q#, Microsoft's new language for quantum programming.
    • Problems and Solutions in Quantum Computing - 易于高级量子计算和信息问题的详细解决方案.
    • Programming Quantum Computers: Essential Algorithms and Code Samples - 重点介绍概念和编程示例（多种语言）的量子计算实践介绍.
    • Quantum Computation and Quantum Information - 为那些在数学、计算机科学和物理方面有一些先验知识的人准备的综合教科书.
    • Quantum Computing: An Applied Approach - 量子计算的实践介绍，解释了量子计算的基础到量子系统背后的数学.
    • Quantum Computing: A Gentle Introduction - 只需要基本的大学数学知识来解释量子计算.
    • Quantum Computing Explained - 使用有效解决方案解释量子计算的会话方法.
    • Quantum Computing for Computer Scientists - 使用计算机科学专业本科生可以使用的方法来解释量子计算.
    • Quantum Computing for Everyone - 为一般读者介绍量子位、纠缠和量子隐形传态等主题.
    • Quantum Computing for Java Developers - 通过其实际实现的镜头解释量子计算.
    • Quantum Computing for the Quantum Curious - 面向高中生、本科生和普通大众的免费量子计算教科书.
    • Quantum Computing in Action - 适用于希望尽早开始量子计算的各级 Java 开发人员.
    • Understanding Quantum Technologies - 优秀的书籍，提供了涵盖所有维度的 360 度量子技术方法.

    Popular Science¶

    • Quantum Computing Since Democritus - 对量子计算和计算复杂性理论的可爱介绍. 它面向尽可能广泛的目标受众，并包含一些与哲学相关的主题.
    • Seth Lloyd. Programming the Universe_ A Quantum Computer Scientist Takes on the Cosmos - 如果宇宙是一台巨大的量子计算机呢？ 它带领读者探索宇宙计算模型及其对物理学的影响.
    • The Fabric of Reality: The Science of Parallel Universes and Its Implications - 它具有哲学精神，关于揭示现实解释的统一结构.
