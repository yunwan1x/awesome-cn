    Specifications from W3C for Enhancing Mobile Web Experience¶

    • Battery API -关于电池信息的访问.
    • Connection API -关于访问有关连接的信息.
    • Generic Sensor API -公开传感器数据的规范.
    • Geolocation API -关于地理位置的访问.
    • HTML Media Capture -用户访问相机或麦克风时.
    • Payment Request API -最小集成度的付款方式.
    • Presentation API -访问演示文稿显示.
    • Remote Playback API -从网页控制媒体的远程播放.
    • Vibration API -访问主机设备的振动机制.
    • Visual Viewport API -访问屏幕的可视部分（屏幕上的键盘除外），捏缩放区域之外的区域等.
    • Wake Lock Api -在访问唤醒锁定请求时（防止设备进入节能状态）.
    • Web Bluetooth -在API上通过蓝牙4发现设备并与之通信.
    • Web NFC API -访问设备的近场通信功能.
    • Web Notifications -在用于最终用户通知的API上.

    • Adaptive Loading - Improving web performance on slow devices -Nate Schloss的Addy Osmani概述了使站点适应硬件和网络限制的实践.
    • Fundamentals of Mobile Web Development -Matt Gaunt撰写的有关移动Web开发的最佳做法的概述.
    • Future App Model: Advanced Service Worker -杰克·阿奇博尔德（Jake Archibald）简要介绍了如何使用流和服务工作者以及其他新的API优化Web应用程序和网站.
    • Instant Loading: Building offline-first Progressive Web Apps -杰克·阿奇博尔德（Jake Archibald）介绍服务人员.
    • Mobile Web Performance -Estelle Weyl在您的网站上如何实现最佳移动体验.
    • Mobile Web Performance Auditing -如何在网站上审核性能问题，作者：保罗·刘易斯（Paul Lewis）.

    Auditing Tools¶

    • Lighthouse -用于Web应用程序和网页的分析工具.
    • Mobile-Friendly Test -用于测试您的网站是否适合移动设备的工具.
    • PageSpeed Insights -该工具可以分析您的页面并提供有关提高下载速度的建议.
    • PageSpeed -自动优化您的站点的Web服务器模块.
    • Site Speed Checker -将您的移动网站与其他网站的速度进行比较.
    • Test Your Mobile Speed -Google提供的工具，可对移动网站进行全面的速度检查.
    • Webpagetest -使用真实的浏览器并以真实的消费者连接速度从多个位置进行网站速度测试.

    Platform Monitoring¶

    • Can I Use -浏览器中各种Web功能的支持状态.
    • Mobile Browser Market Share -不同平台和国家/地区之间的市场份额.
    • Standards for Web Applications on Mobile -与移动相关的W3C规范的当前状态.
    • What Web Can Do Today -您的浏览器具备的功能.

    Other Useful Tools and Libraries¶

    • Browsersync -节省时间的同步浏览器测试.
    • Eruda -用于移动网络浏览器的控制台.
    • Interact.js -借助惯性和捕捉功能，可拖放，调整大小和多点触摸手势，适用于现代浏览器.
    • Fastclick -消除了旧版浏览器中300ms的延迟.
    • Lozad.js -使用IntersectionObserver API的高性能，轻便和可配置的惰性加载器
    • React Dates -适用于网络的易于国际化，适合移动设备使用的datepicker React库.
    • Screen Sizes -所有流行智能手机的屏幕尺寸列表.
    • Shiny -移动网站的闪亮思考.
    • Slideout -适用于您的移动网络应用程序的独立且轻巧的导航菜单.
    • Swiper -具有硬件加速过渡功能的最现代的移动触摸滑块.
    • Web Inspector Remote (Weinre) -用于在非主流浏览器中调试网站的Web检查器.
    • Web Starter Kit -多设备网站的工作流程.
    • Workbox -用于向网络应用添加离线支持的JavaScript库.
    • WorkerDOM -提供Web Workers内部DOM完整表示的库.
