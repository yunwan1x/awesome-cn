    Awesome neovim

    Awesome Neovim ¶

    > 很棒的 Neovim 插件集合. 主要针对 Neovim 的特定功能.

    Neovim is a Vim-based text editor engineered for extensibility and usability, to encourage new applications and contributions.

    Wishlist¶

    Have a problem a plugin can solve? Add it to the nvim-lua wishlist.

    UI¶

    Neovim supports a wide variety of UI's.

    • CosmicNvim/cosmic-ui - Cosmic-UI 是对特定 vim 功能的简单封装. 旨在提供一种快速简便的方法来使用 Neovim 创建 Cosmic UI 体验！
    • yatli/fvim - Cross platform Neovim front-end UI, built with F# + Avalonia.
    • smolck/uivonim - 目标：利用最新 Neovim 功能的功能丰富的跨平台 GUI.
    • qvacua/vimr - Swift 中用于 macOS 的 Neovim GUI.
    • vhakulinen/gnvim - Neovim 的 GUI，没有任何网络膨胀.
    • Kethku/neovide - Rust 中没有废话 Neovim 客户端.
    • beeender/glrnvim - A terminal wrapper that launches Neovim inside of alacritty.
    • akiyosi/goneovim - Neovim GUI written in Golang, using a Golang qt backend.
    • DinVim - DinVim for macOS 是一个安全可靠的工作 macOS 沙箱环境，为 Vim 用户提供真正的 macOS 体验.
    • RMichelsen/Nvy - C++ 中的 Neovim 客户端.
    • asvetliakov/vscode-neovim - Neovim integration for Visual Studio Code.
    • equalsraf/neovim-qt - Neovim client library and GUI, in Qt5.
    • lunixbochs/actualvim - Sublime Text 3 input mode using Neovim.
    • vv-vim/vv - 适用于 macOS 的 Neovim 客户端. 具有良好 macOS 集成的纯粹、快速、简约的 Vim 体验. 针对速度和漂亮的字体渲染进行了优化.
    • jebberjeb/javafx-neovimpane - 由 Neovim 支持的 JavaFX 文本窗格，使用 Clojure 创建.
    • rohit-px2/nvui - Neovim 的现代前端.
    • Lyude/neovim-gtk - 一个用 Rust 编写的 GTK3+ 客户端，与原始 vim-gtk3 非常相似，支持连字.

    Plugin¶

    Plugin Manager¶

    • wbthomason/packer.nvim - A use-package inspired plugin manager for Neovim. Uses native packages, supports Luarocks dependencies, written in Lua, allows for expressive config.
    • savq/paq-nvim - 用 Lua 编写的 Neovim 包管理器.
    • NTBBloodbath/cheovim - Neovim configuration switcher written in Lua. Inspired by chemacs.

    LSP¶

    (requires Neovim 0.5)¶

    • neovim/nvim-lspconfig - Quickstart configurations for the Neovim LSP client.
    • nvim-lua/lsp-status.nvim - This is a Neovim plugin/library for generating statusline components from the built-in LSP client.
    • nvim-lua/lsp_extensions.nvim - Repo to hold a bunch of info & extension callbacks for built-in LSP.
    • RishabhRD/nvim-lsputils - nvim-lsp 操作的更好的默认值.
    • tami5/lspsaga.nvim - 基于 Neovim 内置 lsp 的轻量级 lsp 插件，具有高性能 UI.
    • onsails/diaglist.nvim - quickfix 中的实时渲染工作区诊断，loclist 中的缓冲区诊断.
    • folke/trouble.nvim - 一个漂亮的诊断列表，可帮助您解决代码造成的所有问题.
    • folke/lsp-colors.nvim - A plugin that adds missing LSP diagnostics highlight groups for color schemes that don't yet support the builtin LSP client.
    • kosayoda/nvim-lightbulb - 只要在当前光标位置有 textDocument/codeAction 可用，插件就会在符号列中显示一个灯泡.
    • onsails/lspkind-nvim - The plugin adds vscode-like icons to Neovim lsp completions.
    • ojroques/nvim-lspfuzzy - 一个让 LSP 客户端使用 FZF 的小插件.
    • gfanto/fzf-lsp.nvim - 为 lsp 内置的 Neovim 启用 fzf 模糊搜索的功能.
    • ray-x/lsp_signature.nvim - 键入时的 Lsp 签名提示.
    • rmagatti/goto-preview - 在浮动窗口中预览本地 LSP 的 goto 定义调用.
    • nanotee/sqls.nvim - Neovim 的 Sqls（sql 数据库连接插件 + LSP 客户端）插件.
    • jubnzv/virtual-types.nvim - Neovim 插件，将类型注释显示为虚拟文本.
    • jose-elias-alvarez/nvim-lsp-ts-utils - 用于改进 Neovim 内置 LSP 客户端的 TypeScript 开发体验的实用程序.
    • ray-x/navigator.lua - Learn existing code quickly and navigate code like a breeze. A swiss army knife makes exploring LSP and 🌲Treesitter symbols a piece of 🍰.
    • liuchengxu/vista.vim - 在当前文件的窗口中列出所有函数和变量.
    • simrat39/symbols-outline.nvim - A tree like view for symbols in Neovim using the Language Server Protocol. Supports all your favourite languages.
    • tamago324/nlsp-settings.nvim - 使用 JSON 文件设置 Neovim LSP 的插件.
    • simrat39/rust-tools.nvim - 使用 Neovim 的内置 LSP 更好地开发 rust 的工具.
    • stevearc/aerial.nvim - 用于浏览和快速导航的代码大纲窗口.
    • jose-elias-alvarez/null-ls - 使用 Neovim 作为语言服务器，通过 Lua 注入 LSP 诊断、代码操作等.
    • jakewvincent/texmagic.nvim - 通过定义任意数量的自定义 LaTeX 构建引擎并使用魔术注释选择它们来增强 Texlab 的 lspconfig 设置.
    • nanotee/nvim-lsp-basics - Basic wrappers for LSP features.
    • weilbith/nvim-code-action-menu - A floating pop-up menu for code actions to show code action information and a diff preview.
    • mfussenegger/nvim-lint - Neovim (>= 0.5) 的异步 linter 插件补充内置的语言服务器协议支持.
    • b0o/SchemaStore.nvim - A Neovim Lua plugin providing access to the SchemaStore 目录.
    • ldelossa/litee.nvim - Neovim 缺少的 IDE 功能.
    LSP Installer¶
    • anott03/nvim-lspinstall - 易于安装语言服务器.
    • alexaandru/nvim-lspupdate - 更新安装（或自动安装，如果缺少）LSP 服务器.
    • williamboman/nvim-lsp-installer - 轻松安装超过 50 台 LSP 服务器 - 完全支持 Windows.

    Markdown¶

    • oberblastmeister/termwrapper.nvim - Neovim 终端功能的包装器，使它们更加用户友好.
    • pianocomposer321/consolation.nvim - A general-purpose terminal wrapper and management plugin for Neovim, written in Lua.
    • jghauser/kitty-runner.nvim - 穷人的REPL. 轻松将缓冲线和命令发送到 kitty 终端.
    • jlesquembre/nterm.nvim - A Neovim plugin to interact with the terminal, with notifications.

    Marks¶

    Fuzzy Finder¶

    • nvim-telescope/telescope.nvim - Telescope.nvim is a highly extendable fuzzy finder over lists. Built on the latest awesome features from Neovim core. Telescope is centered around modularity, allowing for easy customization.
    • vijaymarupudi/nvim-fzf - 在 Neovim (>= 0.5) 中使用 fzf 的 Lua API. 允许 UI 速度和可用性的完全异步.
