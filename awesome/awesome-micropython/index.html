    Awesome micropython

    精选的 MicroPython 库、框架、软件和资源的精选列表.

    MicroPython 是 Python 3 编程语言的精简高效实现，其中包含 Python 标准库的一小部分，并针对在微控制器和受限环境中运行进行了优化.

    Libraries¶

    您可以在其他地方寻找 MicroPython 库：

    • PyPi - 此过滤器仅显示 PyPi 上的 MicroPython 库. 注意：您不能 pip install micropython 库. 看 MicroPython docs 有关 upip 的更多信息.
    • GitHub Search - 在 GitHub 上搜索包含 MicroPython 的存储库.
    • GitHub Topic - MicroPython - 浏览带有 MicroPython 标记的项目的 GitHub 主题.
    • Libraries.io - 用于 MicroPython 的 Libraries.io 查询.
    • GitLab Explore - 探索 GitLab 上的存储库.

    AI¶

    • MicroMLP - 用于 MicroPython 的微神经网络多层感知器（用于 ESP32 和 Pycom 模块）.

    Analytics¶

    • uMath - 微控制器的计算机代数.
    • micropython-ulab - 用于 MicroPython 的类似 numpy 的快速矢量模块.
    • micropython-fourier - MicroPython 的内联 ARM 汇编程序中的快速傅立叶变换.
    • ulinalg - 小尺寸矩阵处理模块，带有一些专门用于 MicroPython (Python3) 的线性代数运算.
    • micropython-mtx - MicroPython 上的快速矩阵乘法和线性求解器.
    • micropython-vec - MicroPython 上的矢量操作.

    Audio¶

    • JQ6500 - JQ6500 UART MP3 模块的驱动程序.
    • KT403A-MP3 - KT403A 驱动程序，由 DFPlayer Mini 和 Grove MP3 v2.0 使用.
    • micropython-buzzer - 在蜂鸣器上播放 nokia compose 和 mid 文件.
    • micropython-dfplayer - 使用 UART 的 DFPlayer Mini 驱动程序.
    • micropython-longwave - MicroPython 板的 WAV 播放器.
    • micropython-vs1053 - VS1053b MP3 播放器的异步驱动程序.
    • micropython-midi - MicroPython 的 MIDI 实现示例.
    • upy-rtttl - 用于铃声文本传输语言 (RTTTL) 的 Python 解析器.

    Communications¶

    Bluetooth¶

    • PyBoard-HC05-Android - Pyboard HC05 蓝牙适配器示例应用程序.
    • uble - 用纯 python 为 micropython 编写的轻量级蓝牙低功耗驱动程序.

    CAN¶

    • micropython-spacecan - Spacecan 是用于嵌入式系统的 SpaceCAN 协议的 MicroPython 实现.

    Cryptography¶

    • mpyaes - 用于 AES 加密的 MicroPython 模块.
    • micropython-aes - AES 算法与纯 python 实现.
    • ucrypto - 用于进行快速椭圆曲线加密的 Micropython 包，特别是数字签名. API 设计灵感来自 fastecdsa 和基于 tomsfastmath 的实现.
    • ucryptoauthlib - Microchip Crypto Authentication 安全元素的轻量级驱动程序，用纯 python 编写，用于 micropython.

    DNS¶

    • ICantBelieveItsNotDNS - “我不敢相信这不是 DNS！” (ICBIND) 是用 MicroPython 编写的 ESP8266 的权威 DNS 服务器.
    • MicroDNSSrv - MicroPython 的微型 DNS 服务器，可以简单地响应带有或不带有通配符的多域上的 A 查询（用于 Pycom 模块和 ESP32）.

    Ethernet¶

    • Official wiznet5k - WIZnet5x00 系列以太网控制器的官方驱动程序.

    FTP¶

    • micropython-ftplib - MicroPython 的 FTP 客户端库.
    • FTP-Server-for-ESP8266-ESP32-and-PYBD - MicroPython 平台上用于 ​​ESP8266/ESP32/PYBD 的小型 FTP 服务器.
    • MicroFTPServer - 可以在带有 MicroPython 的 ESP8266 上运行的最小 FTP 服务器.

    GPS¶

    • micropyGPS - 全功能 GPS NMEA 句子解析器.
    • micropython-gnssl76l - 移远通信 GNSS L76-L (GPS) 的 MicroPython I2C 驱动程序.
    • mpy-agps - 辅助定位服务 (AGPS) 的 MicroPython 实现.

    GSM¶

    • micropython-upyphone - 使用 pyboard 和 sim800l 的 GSM 手机.

    IoT¶

    • microhomie - 适用于 IoT 的 Homie MQTT 约定的 MicroPython 实现.
    • uPyEcho - 模拟 Belkin WeMo 设备，在 ESP32 上使用 MicroPython 与 Amazon Echo (Alexa) 配合使用.
    • SonosRemote - 在 ESP8266 上运行并使用 Sonos HTTP API 的 Sonos 安装遥控器.
    • micropython-home-assistant - 基于 MicroPython 的脚本可扩展您的家庭助理驱动的家庭自动化项目.

    IR¶

    • micropython-necir - TL1838 红外接收器 LED 的 NEC 红外捕获.
    • Micropython-IR - Pyboard 红外远程嗅探和重播.
    • micropython_ir - 从红外遥控器和红外“冲击波”应用程序接收的非阻塞设备驱动程序.
    • micropython-amg88xx - Grid-EYE 热红外阵列传感器 (Adafruit 3538) 的驱动程序.
    • micropython-ys-irtm - YS-IRTM 5V NEC 红外 UART 收发器的 MicroPython 示例.
    • esp8266_ir - 通过 websocket 控制红外信号.

    LoRaWAN¶

    • uPyLoRaWAN - 使用 MicroPython 的 ESP32 满足 LoRa 和 LoRaWAN.
    • DeLoReAN - DeLoReAN 是一个（De）专用（Lo）RaWAN 节点（重新）记录（A）ny（N）FC 标签.

    MDNS¶

    • micropython-mdns - MDNS 的纯 Python 实现，支持服务发现.

    MQTT¶

    • micropython-mqtt - “弹性”异步 MQTT 驱动程序. 加上一种使用 ESP8266 将 MQTT 带到非联网目标的方法.
    • MQBoard - 用于在 MicroPython 板上使用 MQTT 和 asyncio 的微框架，主要在 ESP32 上.
    • pysmartnode - MicroPython 智能家居框架.
    • umqtt_aws_iot - 使用 MicroPython 将 UMQTT 消息发布到 AWS IoT.
    • sonoff-mqtt by davea - 使用 MQTT 控制 Sonoff/ESP8266 的 MicroPython 脚本.
    • micropython-sonoff-switch - 使用 MicroPython 为 iTead Sonoff Switch 实现了 MQTT 可控开关.
    • micropython-thingspeak-mqtt-esp8266 - 使用 MQTT 和在 ESP8266/ESP32 平台上运行的 MicroPython 发布和订阅 Thingspeak.
    • uMQTT - 在 WiPy 板上为 MicroPython 发布 MQTT.

    NTP¶

    • esp8266_ntp_webserver - MicroPython + esp8266 + ntp + 网络服务器.
    • micropython-ntpd - MicroPython 中 ntpd 的实现.
    • micropython_ntpserver - 为 MicroPython 编写的 NTP 服务器.
    • micropython-ntpclient - 使用 uasyncio 的 MicroPython NTP 客户端.

    OneWire¶

    • Official OneWire - 对于使用 OneWire 总线的设备，例如 Dallas ds18x20.

    Onkyo EISCP¶

    • eiscp-micropython - 用于 Onkyo-EISCP 协议的 Micropython 端口，其中包括 Pioneer.

    Radio¶

    • micropython-radio - nRF24L01 2.4Ghz 无线电模块的协议.
    • micropython-rfsocket - 流行的基于 433MHzn 的 RFSocket 的 MicroPython 实现.
    • Official nRF24L01 - nRF24L01 2.4Ghz 无线电模块的官方驱动程序.
    • micropython_remote - 捕获和重放 433MHz 遥控代码. 控制远程开关电源适配器.
    • micropython-ys-rf34t - 使用 YS-RF34T 433MHz ASK/OOK UART 收发器的 MicroPython 示例.

    REPL¶

    • webrepl - MicroPython WebREPL.
    • zepl - 使用 ZeroMQ 的 MicroPython WebREPL 控制台应用程序.
    • jupyter_micropython_remote - Jupyter 内核可通过串行/网络 REPL 在 MicroPython 板上直接执行代码.

    RFID¶

    • micropython-mfrc522 - NXP MFRC522 RFID 读写器的驱动程序.
    • micropython-wiegand - 韦根协议阅读器.

    RTC¶

    • micropython-tinyrtc-i2c - DS1307 RTC 和 AT24C32N EEPROM 的驱动程序.
    • Micropython_TinyRTC - DS1307 RTC 驱动程序.
    • micropython-mcp7940 - Microchip MCP7940 RTC 的驱动程序.

    Serial¶

    • mpy-miniterm - 通过串行 REPL 与 MicroPython 设备进行无缝串行调试和文件同步的工具.
    • micropython-modbus - modbus-tk 的 MicroPython 端口.

    SMTP¶

    • uMail - 一个轻量级、可扩展的 SMTP 客户端，用于在 MicroPython 中发送电子邮件.

    Telnet¶

    • MicroTelnetServer - 用于 MicroPython 和 ESP8266 的简单 telnet 服务器，允许 telnet 客户端访问 REPL.

    WiFi¶

    • HueBridge - 飞利浦色调桥.
    • micropython-wifimanager - ESP8266 板上 MicroPython 的简单网络配置实用程序.

    Web¶

    • MicroWebSrv - 支持 WebSockets、html/python 语言模板和路由处理程序的微型 HTTP Web 服务器，用于 MicroPython（在 Pycom 模块和 ESP32 上使用）.
    • MicroWebSrv2 - 最后一个用于物联网 (MicroPython) 或大型服务器 (CPython) 的微型 Web 服务器，它支持 WebSockets、路由、模板引擎和真正优化的架构（内存分配、异步 I/O）.
    • tinyweb - 用于 MicroPython 的简单轻量级 HTTP 异步服务器.
    • upy-websocket-server - MicroPython (ESP8266) websocket 服务器实现.
    • micropython-captive-portal - MicroPython 的强制门户演示.
    • uPyPortal - 使用 ESP32 (WeMos) 的 MicroPython 强制门户.
    • ESP8266WebServer - 用于 MicroPython 的 ESP8266 网络服务器.
    • microCoAPy - 在 MicroPython 中实现 CoAP（受限应用协议）的微型客户端/服务器.
    • micropyserver - MicroPyServer 是一个用于 MicroPython 项目的简单 HTTP 服务器.
    • MicroRESTCli - 基于 MicroWebCli for MicroPython 的微型 JSON REST Web 客户端（用于 Pycom 模块和 ESP32）.
    • micropython-noggin - 一个非常简单的 MicroPython 网络服务器.
    • uwebsockets - ESP8266 的 MicroPython websockets 实现.
    • microdot - 用于 MicroPython 的不可思议的小型 Web 框架.
    • micropython-utelegram - MicroPython 的 Telegram API 包装器.

    Zigbee¶

    • ZbPy - MicroPython IEEE802.15.4 / Zigbee 解析器.

    Display¶

    E-Paper¶

    • micropython-epaper - 适用于 Embedded Artists 2.7 英寸电子纸显示器的 Pyboard 驱动程序.
    • micropython-ili9341 - SSD1606 有源矩阵电子纸显示器 128x180.
    • micropython-waveshare-epaper - 各种 Waveshare 电子纸模块的驱动程序.
    • micropython-waveshare-epd - 用于运行 Pycom 风格的 MicroPython 的设备的 Waveshare 电子纸显示驱动程序.
    • ssd1675a - 基于 SSD1675 的电子纸显示器的驱动程序.

    GUI¶

    • Micropython IDE for VSCode - 用于 VSCode README 的 MicroPython IDE.
    • Micropython-REPLink for VSCode - 与 MicroPython REPL 终端交互的便捷快捷方式.
    • Mu Editor - Code with Mu: a simple Python\MicroPythonb\CircuitPython editor for beginner programmers.
    • Thonny IDE - Thonny：适合初学者的 Python IDE.

    Shells¶

    On Device¶

    • upy-shell - 一个简单的基于命令行的 MicroPython 外壳.
    • Micropython-Editor - 用 Python 编写的用于 PyBoard、WiPy、ESP8266、ESP32、PyCom 和 Adafruit 设备的小型板载编辑器.

    On Host¶

    • rshell - 将文件复制或同步到板，从您的终端输入 REPL.
    • ampy - Adafruit MicroPython 工具 - 通过串行连接与 MicroPython 板交互的实用程序.
    • mpfshell - 一个用于 ESP8266 和 WiPy 的基于 shell 的简单文件浏览器.
    • mpsync - 一个小工具，可在检测到更改时将文件夹同步到 MicroPython 板.
