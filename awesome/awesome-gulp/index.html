    Plugins¶

    Compilation¶

    • gulp-sass - Sass→CSS with libsass.
    • gulp-ruby-sass - Sass→使用Ruby Sass的CSS.
    • gulp-compass - Sass→带有Ruby Sass和Compass的CSS.
    • gulp-less - Less →CSS.
    • gulp-stylus - Stylus →CSS.
    • gulp-postcss - 管道CSS通过 PostCSS 处理器只有一个解析.
    • gulp-coffee - Coffeescript →JavaScript.
    • gulp-typescript - TypeScript →JavaScript.
    • gulp-react - Facebook React JSX模板→JavaScript.
    • webpack-stream - 跑 webpack 作为与gulp方便地整合的流.

    Transpilation¶

    • gulp-babel - ES6→ES5 babel.
    • gulp-traceur - 使用ES6→ES5 Traceur.
    • gulp-regenerator - ES6→ES5 Regenerator.
    • gulp-es6-transpiler - [：no_entry：] ES6→ES5 with es6-transpiler.
    • gulp-myth - Myth - 用于CSS规范未来版本的polyfill.
    • gulp-cssnext - [：no_entry：]使用明天的CSS语法，今天使用 cssnext.

    Concatenation¶

    • gulp-concat - 连接文件.

    Minification¶

    • gulp-clean-css - 使用缩小CSS clean-css.
    • gulp-csso - 使用缩小CSS CSSO.
    • gulp-uglify - 使用缩小JavaScript UglifyJS2.
    • gulp-htmlmin - 使用缩小HTML html-minifier.
    • gulp-imagemin - 使用缩小PNG，JPEG，GIF和SVG图像 imagemin.
    • gulp-svgmin - 使用gulp缩小SVG文件.

    Optimization¶

    • gulp-uncss - 删除未使用的CSS选择器 UnCSS.
    • gulp-css-base64 - 将CSS文件中找到的所有资源（url（）声明中的资源）转换为base64编码的数据URI字符串.
    • gulp-svg2png - 将SVG转换为PNG.
    • gulp-responsive - 生成不同大小的图像.
    • gulp-svgstore - 使用将svg文件合并为一个 <symbol>元素.
    • gulp-iconfont - 从多个SVG图标创建图标字体.

    Injecting Assets¶

    • gulp-useref - 解析HTML文件中的构建块以替换对非优化脚本或样式表的引用.
    • gulp-inject - 将每个文件转换为字符串，并将每个转换后的字符串注入目标流文件中的占位符.
    • wiredep - Wire Bower依赖于您的源代码.

    Templating¶

    • gulp-angular-templatecache - 在$ templateCache中连接并注册AngularJS模板.
    • gulp-jade - Jade →HTML.
    • gulp-handlebars - Handlebars 模板→JavaScript.
    • gulp-hb - Handlebars 模板→HTML.
    • gulp-nunjucks - Nunjucks 模板→JavaScript.
    • gulp-dustjs - Dust 模板→JavaScript.
    • gulp-riot - Riot 模板→JavaScript.
    • gulp-markdown - Markdown→HTML.
    • gulp-template - Lodash 模板→JavaScript.
    • gulp-swig - Swig 模板→HTML.
    • gulp-remark - Gulp插件 remark - 由插件驱动的降价处理器

    Linting¶

    • gulp-csslint - 用CSS自动linting CSS CSSLint.
    • gulp-htmlhint - HTMLHint 包装器来验证您的HTML.
    • gulp-jshint - 用JavaScript检测JavaScript中的错误和潜在问题 JSHint.
    • gulp-jscs - 检查JavaScript代码样式 jscs.
    • gulp-coffeelint - 一种有助于保持的风格检查器 CoffeeScript 代码干净.
    • gulp-tslint - TypeScript gulp的linter插件.
    • gulp-eslint - 识别并报告ECMAScript / JavaScript代码中的模式.
    • gulp-w3cjs - 用HTML验证HTML w3cjs.
    • gulp-lesshint - 少量文件 lesshint.
    • gulp-check-unused-css - 检查HTML模板中是否有未使用的CSS类.

    Live Reload¶

    • browser-sync - 在构建网站时保持多个浏览器和设备同步（recipes).
    • gulp-livereload - livereload的Gulp插件.

    Caching¶

    • gulp-changed - 仅传递已更改的文件.
    • gulp-cached - 简单的内存文件缓存.
    • gulp-remember - 记住并回忆通过它的文件.
    • gulp-newer - 仅传递较新的源文件.

    Flow Control¶

    • merge-stream - 将多个流合并为一个交错流.
    • streamqueue - 逐行排队流.
    • run-sequence - 按顺序运行一系列依赖gulp任务.
    • gulp-if - 有条件地执行任务.

    Logging¶

    • gulp-notify - gulp的通知插件.
    • gulp-size - 显示项目的大小.
    • gulp-debug - 调试乙烯基文件流以查看通过gulp管道运行的文件.
    • gulp-beer - 使用交互式系统通知和自定义服务器进行错误显示的错误报告.

    Testing¶

    • gulp-mocha - 跑 Mocha 试验.
    • gulp-jasmine - 跑 Jasmine 2 Node.js中的测试
    • gulp-protractor - Gulp包装纸 Protractor 试验.
    • gulp-coverage - Node.js的覆盖率报告，独立于测试运行者.
    • gulp-karma - 用于吞咽的Karma测试运行器.
    • gulp-ava- 跑 AVA 吞咽测试.

    Miscellaneous Plugins¶

    • gulp-util - 有用的实用程序集.
    • gulp-plumber - 防止因错误导致管道破裂.
    • gulp-load-plugins - 自动加载gulp插件.
    • main-bower-files - 通过动态获取库文件简化构建过程设置.
    • autoprefixer - 解析CSS并通过Can I Use将规则添加到规则中.
    • gulp-sourcemaps - 提供源地图支持.
    • gulp-replace - gulp的字符串替换插件.
    • gulp-rename - 轻松重命名文件.
    • gulp-rev - 通过将内容哈希附加到文件名来进行静态资产修订：unicorn.css→unicorn-d41d8cd98f.css.
    • del - 使用globs删除文件/文件夹.
    • gulp-exec - 运行shell命令.
    • gulp-strip-debug - 从JavaScript代码中删除控制台，警报和调试器语句.
    • gulp-cssimport - 解析CSS文件，查找导入，抓取链接文件的内容并用它替换import语句.
    • gulp-inline-css - 将CSS属性内联到HTML文件的style属性中.
    • gulp-gh-pages - 将内容发布到Github页面.
    • gulp-ng-annotate - 添加AngularJS依赖注入注释 ng-annotate.
    • gulp-bump - Bump任何semver JSON版本.
    • gulp-file-include - 包含gulp的文件.
    • gulp-zip - ZIP压缩文件.
    • gulp-git - 用gulp运行Git命令.
    • gulp-filter - 使用globbing过滤乙烯基流中的文件.
    • gulp-preprocess - 基于自定义上下文或环境配置预处理文件.
    • gulp-eval - Eval JS表达式或需要CommonJS模块和JSON文件.
