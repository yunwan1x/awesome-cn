    Awesome micropython

    很棒的 MicroPython 库、框架、软件和资源的精选列表.

    MicroPython 是 Python 3 编程语言的精简高效实现，其中包括 Python 标准库的一小部分，并经过优化以在微控制器和受限环境中运行.

    Libraries¶

    您可以在其他地方查找 MicroPython 库：

    • PyPi - 此过滤器仅显示 PyPi 上的 MicroPython 库. 注意：您不能 pip 安装 micropython 库. 看 MicroPython docs 有关 upip 的更多信息.
    • GitHub Search - 在 GitHub 上搜索包含 MicroPython 的存储库.
    • GitHub Topic - MicroPython - 浏览带有 MicroPython 标记的项目的 GitHub 主题.
    • Libraries.io - Libraries.io 查询 MicroPython.
    • GitLab Explore - 探索 GitLab 上的存储库.

    AI¶

    • MicroMLP - 用于 MicroPython 的微神经网络多层感知器（用于 ESP32 和 Pycom 模块）.

    Analytics¶

    • uMath - 微控制器的计算机代数.
    • micropython-ulab - 用于 MicroPython 的类似 numpy 的快速向量模块.
    • micropython-fourier - MicroPython 的内联 ARM 汇编器中的快速傅立叶变换.
    • Filters 使用 ARM Thumb 汇编程序的 FIR 滤波器. 使用在线实用程序，您可以从图表中获取 滤波器实现所需的频率响应.
    • ulinalg - 小型矩阵处理模块，带有一些专门用于 MicroPython (Python3) 的线性代数运算.
    • micropython-mtx - MicroPython 上的快速矩阵乘法和线性求解器.
    • micropython-vec - MicroPython 上的向量操作.
    • MicroPython_Statistics - MicroPython 的统计模块.

    Audio¶

    • micropython-jq6500 - JQ6500 UART MP3 模块的驱动程序.
    • KT403A-MP3 - DFPlayer Mini 和 Grove MP3 v2.0 使用的 KT403A 驱动程序.
    • micropython-buzzer - 在蜂鸣器上播放诺基亚 compose 和 mid 文件.
    • micropython-dfplayer - 使用 UART 的 DFPlayer Mini 驱动程序.
    • micropython-longwave - MicroPython 板的 WAV 播放器.
    • micropython-vs1053 - VS1053b MP3 播放器的异步驱动程序.
    • micropython-midi - MicroPython 的 midi 实现示例.
    • upy-rtttl - 用于铃声文本传输语言 (RTTTL) 的 Python 解析器.
    • micropython-i2s-examples - 运行 MicroPython 的微控制器上的 I2S 支持示例.
    • micropython-osc - MicroPython 的最小 OSC 客户端和服务器库.

    Communications¶

    APIs¶

    • micropython-utelegram - MicroPython 的 Telegram API 包装器.
    • uEagle - MicroPython Rainforest EAGLE 客户端.
    • micropython-youtube-api - MicroPython 中的 YouTube API.
    • micropython_esp8266_tweetbot - MicroPython v1.8.4 (ESP8266) 的推特机器人.
    • telegram-upy - MicroPython 的 Telegram API 包装器.
    • micropython-thingspeak - 用于从运行 MicroPython 的物联网设备（例如 ESP8266）向 thingspeak.com 发送数据的库.
    • micropython_pushbullet - 如何在 ESP8266 上使用 PushBullet 和 MicroPython 的简单示例.
    • esp32-youtube-display - 使用 Google API 和 MicroPython 显示 YouTube 指标.
    • micropython-spotify-web-api - 一个库，用于通过 MicroPython 从 IoT 设备使用 Spotify 的 Web API.

    Bluetooth¶

    • PyBoard-HC05-Android - Pyboard HC05 蓝牙适配器示例应用程序.
    • uble - 用纯 Python 为 MicroPython 编写的轻量级蓝牙低功耗驱动程序.
    • MicroPythonBLEHID - 用于 MicroPython 的基于蓝牙低功耗 (BLE) GATT 库的人机接口设备 (HID).
    • upyble - 用于蓝牙低功耗 MicroPython 设备的命令行工具.
    • micropython-xiaomi-ble-adv-parse - 从一些小米蓝牙低功耗 (BLE) 传感器中被动检索传感器数据.
    • mijia-temphum-upy - 用于读取某些小米米家 BLE 温湿度传感器的 MicroPython 库.

    CAN¶

    • micropython-spacecan - Spacecan 是用于嵌入式系统的 SpaceCAN 协议的 MicroPython 实现.
    • Robomaster-Micropython - Robomaster S1 - MicroPython CAN BUS 控制器.
    • micropython-mcp2515 - MicroPython MCP2515 驱动程序，从 Arduino MCP2515 CAN 接口库移植.

    Compression¶

    • ufastlz - 用于 FastLZ 的 MicroPython 包装器，一个闪电般快速的无损压缩库.

    Cryptography¶

    • mpyaes - 用于 AES 加密的 MicroPython 模块.
    • micropython-aes - 具有纯 python 实现的 AES 算法.
    • ucrypto - 用于进行快速椭圆曲线加密的 MicroPython 包，特别是数字签名. API 设计灵感来自 fastecdsa 和基于 tomsfastmath 的实现.
    • ucryptoauthlib - Microchip Crypto Authentication安全元素的轻量级驱动程序，用纯python为MicroPython编写.
    • embit - MicroPython 和 Python3 的最小比特币库，专注于嵌入式系统.
    • microotp - ESP8266 MicroPython OTP 生成器.
    • micropython-rsa-signing - 在 MicroPython 上进行 RSA 签名.
    • micropython-cryptomsg - 使用 AES CBC 模式加密和解密消息的 MicroPython 模块.

    DNS¶

    • ICantBelieveItsNotDNS - “我不敢相信这不是 DNS！” (ICBIND) 是用 MicroPython 编写的 ESP8266 的权威 DNS 服务器.
    • MicroDNSSrv - MicroPython 的微型 DNS 服务器，用于简单地响应带有或不带有通配符的多域上的 A 查询（用于 Pycom 模块和 ESP32）.
    • tinydns - 用于 MicroPython 的非常简单的 DNS 异步服务器.
    • micropython-captiveportal - MicroPython 的最小异步强制门户（与 uasyncio v3/MicroPython 1.13+ 以及早期版本兼容）.
    • Micropython-DNSServer-Captive-Portal - 带有 DNS 和 Web 服务器的 MicroPython WiFi AP 强制门户.

    Ethernet¶

    • Official wiznet5k - WIZnet5x00 系列以太网控制器的官方驱动程序.
    • micropy-ENC28J60 - 用于 MicroPython (RP2) 的 ENC28J60 以太网芯片驱动程序.

    FTP¶

    • micropython-ftplib - MicroPython 的 FTP 客户端库.
    • FTP-Server-for-ESP8266-ESP32-and-PYBD - MicroPython 平台上用于 ​​ESP8266/ESP32/PYBD 的小型 FTP 服务器.
    • MicroFTPServer - 可以在带有 MicroPython 的 ESP8266 上运行的最小 FTP 服务器.
    • micropython-uaioftp - MicroPython 的轻量级 FTP 库.

    GPS¶

    • micropyGPS - 全功能 GPS NMEA 句子解析器.
    • micropython-gnssl76l - 适用于 Quectel GNSS L76-L (GPS) 的 MicroPython I2C 驱动程序.
    • mpy-agps - 辅助定位服务 (AGPS) 的 MicroPython 实现.
    • Asynchronous GPS driver 接收和解析 GPS 数据作为 uasyncio 任务.

    GSM¶

    • micropython-upyphone - 使用 pyboard 和 sim800l 的 gsm 手机.
    • micropython-sim800 - 用于 sim800 的 MicroPython 驱动程序.
    • sim800 - 用于与 MicroPython 中的 SIM800 模块接口的库.
    • MicroPython-AM7020 - AM7020 窄带物联网 (NBIoT) 模块的 MicroPython 驱动程序.

    IoT¶

    • uPyEcho - 在 ESP32 上使用 MicroPython 与 Amazon Echo (Alexa) 配合使用的模拟 Belkin WeMo 设备.
    • SonosRemote - 在 ESP8266 上运行并使用 Sonos HTTP API 的 Sonos 安装遥控器.
    • micropython-home-assistant - 基于 MicroPython 的脚本可扩展您的家庭助理驱动的家庭自动化项目.
    • micropython-iot - 一种使用 ESP8266、ESP32 或 Pyboard D 端点设计 IOT 应用程序的方法.
    • iot-core-micropython - 使用 MicroPython 连接到 Google Cloud IoT Core.
    • SmartUPy - 使用MicroPython控制“涂鸦式”智能电源插座.
    • aws-iot-GET-POST-loop - 使用 AWS-IOT REST API 获取/发布设备状态信息的 MicroPython 代码.
    • sensor-mqtt-homeassistant - 一个基于 esp8266/32 MicroPython 的传感器平台，用于 gpio、dht、模拟、led 等. 包括来自 Web 服务器和 MQTT/Homeassistant 集成的 .py 代码远程更新.

    IR¶

    • micropython-necir - 用于 TL1838 IR 接收器 LED 的 NEC 红外捕获.
    • Micropython-IR - Pyboard 红外遥控嗅探和回放.

    • micropython-amg88xx - Grid-EYE 热红外阵列传感器 (Adafruit 3538) 的驱动程序.

    • micropySX126X - 用于 MicroPython 和 CircuitPython 的 Semtech SX126X LoRa 驱动程序.

    MDNS¶

    • micropython-mdns - MDNS 的纯 python 实现，支持服务发现.

    Modbus¶

    • JetBrains IntelliJ/PyCharm MicroPython Plugin - IntelliJ 和 PyCharm 中的 MicroPython 设备插件.
    • Micropython IDE for VSCode - 用于 VSCode 自述文件的 MicroPython IDE.
    • Micropython-REPLink for VSCode - 与 MicroPython REPL 终端交互的便捷快捷方式.
    • Mu Editor - 带有 Mu 的代码：适用于初学者的简单 Python\MicroPythonb\CircuitPython 编辑器.
    • Thonny IDE - Thonny：适合初学者的 Python IDE.
    • Pyboard File Manager - Pyboard 文件管理器：Pyboard.py 兼容设备的 Windows GUI.

    Logging¶

    • micropython-ulogger - 为 MicroPython 定制的轻量级日志模块.

    Shells¶

    On Device¶

    • upy-shell - 一个简单的基于命令行的 MicroPython 外壳.
    • Micropython-Editor - 用 Python 编写的用于 PyBoard、WiPy、ESP8266、ESP32、PyCom 和 Adafruit 设备的小型板载编辑器.

    On Host¶

    • rshell - 将文件复制或同步到 Boards，从终端输入 REPL.
    • ampy - MicroPython Tool - 通过串行连接与 MicroPython 板交互的实用程序.
    • mpfshell - 一个用于 ESP8266 和 WiPy 的简单的基于 shell 的文件浏览器.
    • mpsync - 一个小工具，可在检测到更改时将文件夹同步到 MicroPython 板.
    • mpremote 强大的官方shell支持挂载宿主机当前目录到目标机. 在不更改目标文件系统的情况下运行代码.
