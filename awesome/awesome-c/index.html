    awesome-c ¶

    精选的C好的东西列表. 此列表仅包含* free software 代码，以及对物质资源没有恶意的卖方.

    寻找维护人员：该列表当前处于非活动状态. 我正在寻找可以使其再次变得出色的维护者. 看 #26 以获得更多信息，但过程很简单：在此列表中添加或修复几件事，您将成为850+星列表的贡献者.

    此列表以前由维护 @kozross . 他决定将清单切换为 new platform，所以我决定将其分叉，以便我们可以将其保留在GitHub上.

    Contents¶

    Build Systems¶

    这些是使C中的项目的构建和测试自动化的工具.

    • aimake -旨在避免复杂配置的构建工具. GNU GPL3 or later
    • Autoconf -可扩展的M4宏程序包，这些程序包会生成Shell脚本以自动配置软件源代码程序包. 自动工具的一部分. GNU GPL3 or later
    • Automake -自动生成符合GNU编码标准的Makefile.in文件的工具. 需要使用Autoconf. 自动工具的一部分. GNU GPL3 or later
    • CMake -用于跨平台构建，测试和打包的工具. 3-clause BSD
    • Jam -构建系统，设计起来比做起来容易. 隐含地理解C构建规则. Jam License
    • Libtool -通用库支持脚本. 自动工具的一部分. GNU GPL3 or later
    • Meson -极其快速，用户友好的构建系统. 基于忍者. Apache 2.0
    • Premake -为Visual Studio，Xcode或GNU Make等工具集构建脚本生成器. 3-clause BSD
    • Ninja -小型，简单的构建系统，注重速度. Apache 2.0
    • Bazel -为各种操作系统和目标构建系统. Apache 2.0
    • Buck -由Facebook创建和使用的构建系统. Apache 2.0
    • Qbs -用于软件项目的现代构建工具. LGPL3
    • qmake -Qt Framework附带的构建系统. GNU GPL3 with Qt Exception
    • xmake -基于lua的跨平台构建实用程序. Apache 2.0

    Compilers¶

    • Clang -LLVM的C编译器. 支持C11. NCSA
    • CompCert -完全验证的C编译器. 支持几乎所有的C89. GNU GPL2.1 or later
    • GCC -提供C编译器作为其编译器集的一部分. 支持C11和OpenMP. GNU GPL3 or later
    • PCC -杰出的C编译器. 支持C99. Various free licenses
    • TCC -Tiny C编译器； 一个小型，快速的C编译器. 支持C99（复杂类型除外）. GNU LGPL2.1
    • Intel SPMD -C语言变体的编译器，用于单个程序，多个数据编程. Various licenses
    • ccache - Compiler cache designed to speedup recompilation. GNU GPL3 or later

    Compiler libraries¶

    用于编译器，解释器和基于低级项目的库.

    • yasm -Yasm模块化组装项目. 2-clause BSD

    Compression¶

    • libzip -用于读取，创建和修改zip存档的AC库. 3-clause BSD
    • lzo -非常快速的数据压缩库. GNU GPL2.1
    • zlib -庞大而精致的压缩库. 3-clause BSD
    • bzip2 -免专利的高质量数据压缩器. 3-clause BSD
    • lz4 -快速压缩算法.
    • snappy -快速压缩库（在C ++中实现，对C的本地绑定）.
    • zstd -快速的实时压缩算法 3-clause BSD
    • quicklz -快速压缩库. GNU GPL2.1
    • pixz -Pixz（发音为pixie）是xz的并行索引版本. 2-clause BSD

    Crypto¶

    • GnuTLS -安全的通信库，实现SSL，TLS和DTLS. GNU LGPL2.1 or later
    • libgcrypt -具有多种可用密码的通用密码库. GNU LGPL2.1or later (code) 和 GNU GPL2.1 or later (manual 和 tools)
    • OpenSSL -SSL和TLS协议的实现，还包括一个加密库. Dual Licensed under the OpenSSL License and the SSLeay License
    • libsodium -现代且易于使用的加密库. Expat
    • libtomcrypt -非常全面，模块化和便携式的加密工具包. Public Domain
    • mbed TLS -C的另一种加密实现. Apache 2.0
    • libressl -OpenSSL的现代化分支. Various Licenses

    Database¶

    这列出了具有C API的数据库和数据存储.

    • BerkeleyDB -用于键值数据的高性能嵌入式数据库的库. GNU AGPLv3
    • Hiredis -Redis的简约客户端库. 3-clause BSD
    • LMDB -超快速，超紧凑的键值嵌入式数据存储. OpenLDAP License
    • MariaDB -健壮，可扩展且可靠的SQL Server，旨在替代MySQL. 3-clause BSD
    • mongo-c-driver -的高性能客户端库 MongoDB. Apache 2.0
    • PostgreSQL -强大的对象关系数据库系统. PostgreSQL licence
    • recutils -一套工具和一个C库，用于访问可编辑的纯文本数据库文件，称为recfiles. GNU GPL3 or later
    • Redis -高级键值存储. 3-clause BSD
    • sophia -现代化的可嵌入键值数据库. FreeBSD
    • SQLite -具有C接口的自包含，无服务器，零配置的事务型SQL数据库引擎. Public Domain
    • UnQLite -具有C接口的自包含，无服务器，零配置，事务型NoSQL引擎. FreeBSD

    Deep Learning¶

    • Darknet -用C和CUDA编写的开源神经网络框架. 它快速，易于安装，并支持CPU和GPU计算.

    Documentation Generation¶

    • Cxref -生成LaTeX，HTML，RTF或SGML中C程序的文档. GNU GPL2.1
    • DocOnce -标记适度的标记语言，可用于生成多种格式. 3-clause BSD
    • Doxygen -用于从带注释的源生成C文档的事实上的标准工具. 可以生成各种格式. GNU GPL2.1
    • GTK-Doc -用于从带注释的源生成C文档的工具. 有对自动工具的支持. GNU GPL2.1 (code) 或者 GNU FDL1.1

    Editors¶

    这些是特别高级的IDE类型的编辑器. 如果需要程序员的文本编辑器，请在其他地方查找. 此外，无论您做出何种选择，无论如何都支持C.

    • Anjuta DevStudio -GNOME IDE. GNU GPL2.1
    • Code::Blocks -支持C的可扩展，可配置的IDE. GNU GPL3
    • CodeLite -跨平台IDE. GNU GPL2.1
    • Eclipse -用Java编写的IDE. EPL
    • Geany -非常小而快速的IDE. GNU GPL2.1 or later
    • KDevelop -KDE IDE. GNU GPL2.1
    • Qt Creator -用C ++和Qt（Qt SDK的一部分）编写的跨平台IDE. 支持C语代码模型. GNU GPL3 with Qt exception
    • Visual Studio Code -代码编辑，重新定义. Visual Studio代码.

    Embedded Systems¶

    这些项目使人们可以使用微控制器和其他资源受限的体系结构.

    RTOS¶

    实时操作系统（RTOS）旨在为处理延迟低延迟的数据的实时应用程序提供服务.

    • Amazon FreeRTOS -用于微控制器的RTOS，使小型，低功耗边缘设备易于编程. MIT
    • ChibiOS -具有完整开发环境（HAL，驱动程序，支持文件和工具）的RTOS. Various Licenses
    • Contiki -将低成本，低功耗的微控制器连接到Internet. 3-clause BSD
    • Zephyr Project -可扩展的实时RTOS，支持多种硬件体系结构，针对资源受限的设备进行了优化. Apache 2.0
    • RIOT OS -实时多线程操作系统，支持物联网中常见的一系列设备. LGPL-2.1

    Environments¶

    这是旨在使Windows进入21世纪以支持C的技术的列表.

    • Cygwin -旨在在Windows下广泛模拟POSIX兼容环境. Various Licenses
    • MinGW-w64 -在Windows上具有64位支持的C开发的极简环境. Various Licenses

    Frameworks¶

    本节包含大型库，这些库提供了数据结构和您期望的“现代”标准库的其他内容.

    • APR -Apache可移植运行时； 另一个跨平台实用程序函数库. Apache 2.0
    • C Algorithms -收集C的通用算法和数据结构. ISC
    • C Macro Collections -仅标头，在C中由宏生成，泛型和类型安全的Collections. MIT
    • CPL -通用管道库； 一组旨在成为一个全面，高效且强大的软件工具包的库. GNU GPL2.1
    • EFL -收集了大量有用的数据结构和功能.
    • GLib -实用程序功能和结构库，旨在实现便携式，高效和强大. GNU LGPL2.1
    • GIO -易于使用的现代VFS API. GNU LGPL2.1
    • GObject -C的面向对象的系统和对象模型. GNU LGPL2.1
    • libnih -C函数和结构的轻量级库. GNU GPL2.1
    • libU -基本实用程序的小型库，包括内存分配，字符串操作和日志记录. 3-clause BSD
    • PBL -大型实用程序库，其中包括数据结构. GNU LGPL2.1 或者 later (library) 或者 GNU GPL2.1 或者 later (test code)
    • qlibc -简单而强大的C库，旨在替代GLib，同时注重体积小巧. qLib license （如同 FreeBSD)
    • stb -C的单文件库范围. Public Domain
    • libsrt -C的安全实时库. 3-clause BSD

    • ioquake3 -Quake3引擎，终于释放了. GNU GPL2.1

    • Quake -雷神之锤引擎. GNU GPL2.1
    • Quake2 -Quake2引擎. GNU GPL2.1
    • Spearmint -专为FPS游戏设计的引擎. GNU GPL3 or later

    Resources¶

    These are libraries of all sorts that are useful for game programming.

    • Allegro -跨平台，视频游戏开发和多媒体库. zlib
    • Chipmunk2D -快速，轻量的2D游戏物理库. Expat
    • CSFML -绑定 SFML 在C中 zlib
    • FreeGLUT -替代OpenGL Utility Toolkit. 允许使用OpenGL上下文创建和管理窗口. X11
    • GLFW -用于使用OpenGL上下文创建窗口的多平台库. zlib
    • libao -具有多种输出的跨平台音频库. GNU GPL2.1 or later
    • RetroArch -的参考前端 libretro. GNU GPL3
    • SDL and SDL2 -跨平台开发库，旨在通过OpenGL提供对音频，键盘，鼠标，操纵杆和图形硬件的低级访问. SDL2是最新版本. zlib
    • raylib -简单易用的库，用于学习视频游戏编程. zlib

    Generic Programming¶

    • klib -通用算法和数据结构的小型轻量级实现. MIT

    Graphics¶

    • Cairo -2D图形库. GNU LGPL2.1 或者 MPLv1.1
    • Cogl -GPU图形和实用程序API. Expat
    • Clutter -基于OpenGL的UI库. GNU LGPL2.1
    • giflib -用于读取和写入gif图像的库. MIT
    • heman -图像工具的微型库，用于处理高度图，法线图，距离场等. Expat
    • libcaca -用于基于终端的接口的ASCII渲染器. WTFPLv2
    • libimagequant -小型便携式库，可将RGBA图像高质量转换为8位索引彩色图像. FreeBSD
    • libjpeg-turbo -更快的用于读取和写入JPEG文件的库. Various Licenses
    • libpng -官方的PNG参考库. libpng license
    • libxmi -用于2D矢量图形栅格化的函数库. GNU GPL3 or later
    • mozjpeg -改进的JPEG编码器. 3-clause BSD

    Graphic APIs¶

    • OpenGL -行业采用2D和3D图形API. 的更多资源 awesome-opengl.
    • OpenGL ES -工业界为移动和嵌入式设备采用了2D和3D图形API.
    • OpenGL SC -适用于需要系统安全认证的行业的图形和计算标准.
    • Vulkan -用于现代跨平台开发的显式图形和计算API. 的更多资源 awesome-vulkan.

    Graphical User Interface¶

    • XForms Toolkit -专为XWindow系统设计的小部件工具包. GNU LGPL2.1
