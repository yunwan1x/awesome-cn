    Awesome CSV ¶

    精心策划的CSV相关工具和资源列表

    CSV 仍然是远古时代最具未来主义意义的数据格式.

    XML兴衰成败. JSON只是泛滥成灾. YAML是一个中毒的圣杯. CSV的优势将胜过所有人.

    当最后的蟑螂呼吸到最后一口气时，她的垂死之举将是在后代的CSV文件中记录其死亡日期.

    以下是一些处理CSV的出色工具：

    Tools¶

    • NimbleText/Live -使用模式来操作CSV； 世界上最简单的代码生成器*.
    • PapaParse -强大的浏览器内CSV解析器.
    • d3-dsv -d3.js解析器和格式化程序模块，用于分隔符分隔的值.
    • CSVKit -包含csvsql / csvgrep / csvstat等的CSV实用程序.
    • XSV -用Rust编写的快速CSV命令行工具包.
    • sed (gnu tool) -流编辑器.
    • gawk (gnu tool) -文本处理和数据提取使用 awk.
    • awk by example -使用awk的综合示例.
    • Miller -像sed / awk / cut / join / sort等一样，用于CSV等名称索引数据.
    • ParaText -CSV解析速度为每秒2.5 GB.
    • CSVGet -以CSV格式从网站获取结构化数据.
    • CSVfix -用于处理CSV数据的工具.
    • Tad -快速免费的跨平台CSV查看器.
    • Nvd3-tags -一个用于从csv数据制作图表的小型库.
    • Powershell: Import-CSV -强大的内置处理CSV的工具（example).
    • CSV Tools -有用的CSV实用程序的集合.
    • graph-cli -灵活的命令行工具，可从CSV数据创建图形.
    • CSV to SQL -在线工具，用于从CSV数据创建插入/更新/删除等.
    • C#: kbCSV -高效，易于使用的.NET CSV解析和编写库.
    • csvprintf -UNIX命令行实用程序，用于基于CSV文件解析和格式化输出.
    • Mockaroo -CSV / JSON / SQL / Excel的随机数据生成器.
    • Ron's CSV Editor -处理大文件，做一些神奇的事情. 用于永恒格式的永恒编辑器.
    • Rainbow CSV plugins -收集用于CSV / TSV语法突出显示的文本编辑器插件. 可以用来 Vim, VS Code, Atom, Sublime Text 和其他编辑.
    • Mighty Merge -加入/联合csv文件.

    Repair or Validate CSV¶

    • Csvlint.go -用于根据RFC 4180验证CSV文件的命令行工具.
    • csvstudio -修复大型CSV文件中语法错误的智能应用程序.
    • scrubcsv -从CSV文件中删除不良记录并进行规范化（需要防锈）
    • reconcile-csv -查找一组相关CSV之间的关系

    Generate Table Schema¶

    • CSV Schema —分析CSV文件并生成数据库表架构，所有这些都在浏览器中 -想要：此类别中的更多工具.

    Treat CSV as SQL¶

    • TextQL -针对CSV或TSV执行SQL.
    • Datasette Facets -任意CSV文件或SQLite DB的多面浏览和JSON API.
    • q -直接在CSV文件上运行SQL
    • RBQL -彩虹查询语言，一种类似SQL的语言，带有JavaScript或Python后端.
    • PSKit Query — Powershell模块使您可以对对象运行简单查询，包括使用csv导入的对象

    Convert to or from CSV¶

    • CSV to Table -将CSV文件转换为可搜索和可排序的HTML表.

    CSV <-> JSON¶

    • Agnes -两种方式将Csv转换为Json **.
    • csv2json -在线工具，可将CSV或TSV格式的数据转换为JSON和 vice versa.
    • csv-to-json -简单，隐私友好且离线优先的在线csv至json转换器.

    • Fake Name Generator -批量生成带有其他身份数据的假名以进行测试.

    Conferences¶

    Standards¶

    >关于标准的妙处在于，有太多可供选择的标准.
    —（可能）格蕾丝·霍珀.

    • RFC 4180 (html version）-“ 逗号分隔值（CSV）文件的通用格式和MIME类型”.
    • Definition of the CSV Format
    • MIME Type Registration of text/csv
    • W3C: Model for Tabular Data and Metadata on the Web
    • CSV Schema Language -用于定义和验证CSV数据的语言.
    • csv,specs -逗号分隔值（CSV）格式规范（和测试），包括 CSV v1.0，CSV v1.1，CSV严格，CSV <3个数字，CSV <3 JSON，CSV <3 YAML.
    • Tabular Data Resource - 一种 Data Resource 专门用于描述表格数据，例如CSV文件或电子表格
    • CSVY -向CSV文件添加YAML标头以描述其格式的标准
