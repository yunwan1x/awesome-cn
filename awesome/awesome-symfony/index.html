    Symfony

    Awesome Symfony ¶

    很棒的名单 Symfony 捆绑包、实用程序和资源.

    目录：

    Administration¶

    • AdminCrudBundle - AdminCrudBundle - 使用 SensioGeneratorBundle. 扩展控制器，添加分页器、过滤器等.
    • AdmingeneratorGeneratorBundle - Symfony2 的 Admingenerator，解析 generator.yml 文件以构建类
    • EasyAdminBundle - Symfony 应用程序的简单管理生成器
    • SonataAdminBundle - AdminBundle - 缺少的 Symfony2 管理生成器
    • AdminLTEBundle - 基于 AdminLTE 模板的管理主题
    • Umbrella framework - 用于创建管理后端的管理组件和主题.

    Development¶

    • AccessibleBundle - 使用强大的注释定义类的 getter、setter 和构造函数.
    • ApiExceptionBundle - 异常 API 包.
    • AvAjaxBundle - 这个包提供了一个简单的结构来运行 ajax 操作.
    • BeelabTestBundle - 这个包只包含 Symfony WebTestCase 的一个自以为是的扩展.
    • CacheAdministrationBundle - 此捆绑包包含一个控制器，该控制器具有清除各种缓存的操作.
    • ControllerExtraBundle - 一组有用的控制器注释.
    • DunglasActionBundle - 基于 Action-Domain-Responder 模式的 Symfony 控制器系统的替代品.
    • http-bundle - 提供额外的 HTTP 相关功能.
    • JMSDebuggingBundle - 提供高级调试工具.
    • LadybugBundle - 简单且可扩展的 PHP Dumper.
    • LiipCodeBundle - 一组 Symfony2 控制台命令，可帮助开发人员处理识别类、模板、包、服务等的各种方法.
    • LiipFunctionalTestBundle - 一些用于在 Symfony 2 中编写功能测试的辅助类.
    • ListenersDebugCommandBundle - 用于调试侦听器的控制台命令.
    • ParamConverterBundle - 此捆绑包为 Symfony 提供了额外的参数转换器.
    • PhpMetricsCollectorBundle - 在 Symfony2 调试工具栏中集成 PhpMetrics.
    • PsyshBundle - 提供一个加载了 Symfony 上下文的增强型 PHP 交互式 shell.
    • PUGXGeneratorBundle - SensioGeneratorBundle 的增强.
    • redaktilo-bundle - 将编辑器集成到 Symfony 2 中.
    • SandboxBundle - 在沙盒环境中覆盖控制器逻辑和响应.
    • StatsDClientBundle - 使用现成的统计数据轻松监控您的 symfony2 生产环境.
    • TagDebugCommandBundle - 集成用于检查和调试标签的 TagDebug 库.
    • TwigReflectionBundle - 显示 Twig 中的内容.
    • WebfactoryExceptionsBundle - 轻松开发自定义的、用户友好的错误页面.
    • WebProfilerExtraBundle - 在 Web Profiler 中添加路由、容器、资产和树枝信息.
    • XhprofBundle - XHProf 包.

    Forms¶

    • CraueFormFlowBundle - 多步骤形式.
    • InfiniteFormBundle - 一系列有用的表单类型和扩展.
    • IvoryCKEditorBundle - Symfony 中的 CKEditor 集成.
    • IvoryOrderedFormBundle - 提供表单订购支持.
    • KarserRecaptcha3Bundle - 将 Google ReCAPTCHA v3 集成到 Symfony 中.
    • LexikFormFilterBundle - Lexik 表单过滤器包.

    Internationalization¶

    • BazingaJsTranslationBundle - 将您的 Symfony2 翻译消息公开给您的客户端应用程序的一种非常好的方式.
    • JMSI18nRoutingBundle - 捆绑多语言网站以支持国际航线.
    • JMSTranslationBundle - 轻松翻译您的网站 - 提取消息并通过基于 Web 的用户界面翻译它们.
    • LuneticsLocaleBundle - 从不同的参数猜测访问者的语言环境.
    • TimezoneBundle - 服务器端时区检测.
    • TranslationFormBundle - 翻译字段，以方便使用 Translatable Doctrine 扩展.
