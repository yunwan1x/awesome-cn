    Awesome jax

    JAX 带来自动微分和 XLA compiler 一起通过一个 NumPy类似 API，用于对 GPU 和 TPU 等加速器进行高性能机器学习研究.

    这是精选的 JAX 库、项目和其他资源的精选列表. 欢迎投稿！

    Libraries¶

    • 神经网络库
      • Flax - 以灵活性和清晰度为中心.
      • Haiku - 专注于简单性，由 DeepMind 的 Sonnet 的作者创建.
      • Objax - 具有类似于 PyTorch 的面向对象设计.
      • Elegy - 用于 Jax 生态系统的与框架无关的 Trainer 界面. 支持 Flax、Haiku 和 Optax.
      • Trax - “含电池”深度学习库，专注于为常见工作负载提供解决方案.
      • Jraph - 轻量级图神经网络库.
      • Neural Tangents - 用于指定有限和_无限_宽度的神经网络的高级 API.
      • HuggingFace - 用于各种自然语言任务 (Flax) 的预训练 Transformer 生态系统.
      • Equinox - 可调用的 PyTrees 和过滤的 JIT/grad 转换 => JAX 中的神经网络.
    • NumPyro - 基于 Pyro 库的概率编程.
    • Chex - 用于编写和测试可靠 JAX 代码的实用程序.
    • Optax - 梯度处理和优化库.
    • RLax - 用于实现强化学习代理的库.
    • JAX, M.D. - 加速的微分分子动力学.
    • Coax - 将 RL 论文转化为代码，这是一种简单的方法.
    • SymJAX - 符号 CPU/GPU/TPU 编程.
    • mcx - 表达和编译用于高性能推理的概率程序.
    • Distrax - 重新实现 TensorFlow Probability，包含概率分布和双射器.
    • cvxpylayers - 构建可微凸优化层.
    • TensorLy - 张量学习变得简单.
    • NetKet - 用于量子物理学的机器学习工具箱.

    New Libraries¶

    本节包含制作精良且有用的库，但尚未经过大量用户群的实战测试.

    • 神经网络库
      • FedJAX - JAX 中的联合学习，建立在 Optax 和 Haiku 之上.
      • Equivariant MLP - 构建等变神经网络层.
      • jax-resnet - Implementations and checkpoints for ResNet variants in Flax.
    • jax-unirep - 图书馆实施 UniRep model 用于蛋白质机器学习应用.
    • jax-flows - 在 JAX 中规范化流程.
    • sklearn-jax-kernels - 使用 JAX 的 scikit-learn 内核矩阵.
    • jax-cosmo - 可微的宇宙学图书馆.
    • efax - JAX 中的指数族.
    • mpi4jax - 将 MPI 操作与 CPU 和 GPU 上的 Jax 代码相结合.
    • imax - 图像增加和转换.
    • FlaxVision - 亚麻版 TorchVision.
    • Oryx - 基于程序转换的概率编程语言.
    • Optimal Transport Tools - 捆绑实用程序以解决最佳运输问题的工具箱.
    • delta PV - 具有自动微分功能的光伏模拟器.
    • jaxlie - 用于刚体变换和优化的李理论库.
    • BRAX - 可微分物理引擎来模拟环境以及学习算法来为这些环境训练代理.
    • flaxmodels - Jax/Flax 的预训练模型.
    • CR.Sparse - 用于稀疏表示和压缩感知的 XLA 加速算法.
    • exojax - 与 JAX 兼容的系外行星/棕矮星的自动可微谱建模.
    • JAXopt - JAX 中的硬件加速 (GPU/TPU)、可批处理和可微优化器.
    • PIX - PIX 是 JAX 中的图像处理库，用于 JAX.

    Models and Projects¶

    JAX¶

    • Fourier Feature Networks - 正式实施 Fourier Features Let Networks Learn High Frequency Functions in Low Dimensional Domains.
    • kalman-jax - 使用迭代卡尔曼滤波和平滑对马尔可夫（即时间）高斯过程进行近似推断.
    • GPJax - JAX 中的高斯过程.
    • jaxns - JAX 中的嵌套采样.
    • Amortized Bayesian Optimization - 相关代码 Amortized Bayesian Optimization over Discrete Spaces.
    • Accurate Quantized Training - 用于在 JAX 和 Flax 中运行和分析神经网络量化实验的工具和库.
    • BNN-HMC - 论文的实现 What Are Bayesian Neural Network Posteriors Really Like?.
    • JAX-DFT - JAX 中的一维密度泛函理论 (DFT)，实现了 Kohn-Sham equations as regularizer: building prior knowledge into machine-learned physics.
    • Robust Loss - 论文参考代码 A General and Adaptive Robust Loss Function.

    Flax¶

    • Performer - 执行者（通过 FAVOR+ 的线性变换器）架构的亚麻实现.
    • JaxNeRF - 实施 NeRF: Representing Scenes as Neural Radiance Fields for View Synthesis 具有多设备 GPU/TPU 支持.
    • Big Transfer (BiT) - 实施 Big Transfer (BiT): General Visual Representation Learning.
    • JAX RL - 强化学习算法的实现.
    • gMLP - 实施 Pay Attention to MLPs.
    • MLP Mixer - 最小的实现 MLP-Mixer: An all-MLP Architecture for Vision.
    • Distributed Shampoo - 实施 Second Order Optimization Made Practical.
    • NesT - 正式实施 Aggregating Nested Transformers.
    • XMC-GAN - 正式实施 Cross-Modal Contrastive Learning for Text-to-Image Generation.
    • FNet - 正式实施 FNet: Mixing Tokens with Fourier Transforms.
    • GFSA - 正式实施 Learning Graph Structure With A Finite-State Automaton Layer.
    • IPA-GNN - 正式实施 Learning to Execute Programs with Instruction Pointer Attention Graph Neural Networks.
    • Flax Models - 在 Flax 中实现的模型和方法的集合.
    • Protein LM - 为蛋白质实施 BERT 和自回归模型，如中所述 Biological Structure 和 Function Emerge from Scaling Unsupervised Learning to 250 Million Protein Sequences 和 ProGen: Language Modeling for Protein Generation.
    • Slot Attention - 参考实现 Differentiable Patch Selection for Image Recognition.
    • Vision Transformer - 正式实施 An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale.
    • FID computation - 港口 mseitzer/pytorch-fid 到亚麻.

    Haiku¶

    • AlphaFold - AlphaFold v2.0 推理管道的实现，在 Highly accurate protein structure prediction with AlphaFold.
    • Adversarial Robustness - 参考代码 Uncovering the Limits of Adversarial Training against Norm-Bounded Adversarial Examples 和 Fixing Data Augmentation to Improve Adversarial Robustness.
    • Bootstrap Your Own Latent - 论文的实现 Bootstrap your own latent: A new approach to self-supervised Learning.
    • Gated Linear Networks - GLN 是一系列无反向传播的神经网络.
    • Glassy Dynamics - 论文的开源实现 Unveiling the predictive power of static structure in glassy systems.
    • MMV - 模型代码 Self-Supervised MultiModal Versatile Networks.
    • Normalizer-Free Networks - 官方俳句实现 NFNets.
    • NuX - 使用 JAX 规范化流程.
    • OGB-LSC - 此存储库包含 DeepMind 对 PCQM4M-LSC （量子化学）和 MAG240M-LSC （学术图） 的轨迹 OGB Large-Scale Challenge (OGB-LSC).
    • Persistent Evolution Strategies - 用于论文的代码 Unbiased Gradient Estimation in Unrolled Computation Graphs with Persistent Evolution Strategies.
    • WikiGraphs - 重现结果的基线代码 WikiGraphs: A Wikipedia Text - Knowledge Graph Paired Datase.

    Trax¶
