    Awesome plotters

    Awesome Plotters ¶

    计算机控制绘图机和其他视觉艺术机器人的代码和资源的精选列表.

    Hardware¶

    Plotters¶

    • AxiDraw - 笔式绘图仪来自 Evil Mad Scientist, 很受欢迎 #plottertwitter.
    • Line-us - 一个可爱的小机器人绘图臂.
    • Makeblock XY Plotter - 可破解的 XY 绘图仪套件（已停产？）.
    • Drawing Robot - 带 3D 打印的 AxiDraw 克隆 Arduino CNC Shield 控制器运行 grbl 固件.
    • 4xiDraw - 另一个可 3d 打印的 AxiDraw 克隆带 Arduino CNC Shield 控制器运行 grbl 固件.
    • WaterColorBot - XY 艺术机器人和软件，用于绘制水彩颜料.
    • EggBot - Pen plotter for egg-shaped and spherical objects.
    • HP Pen Plotters - 来自 HPGL 标准创建者的老式桌面和落地式笔式绘图仪. 7475A 型很常见，通常可以在 eBay 上找到.
    • Roland Pen Plotters - 老式平板 HPGL 笔式绘图仪. 在 eBay 上搜索“roland dxy”.
    • Polargraph - 原始极谱硬件和软件项目.
    • Makelangelo - 开源极坐标图艺术机器人.
    • BrachioGraph - 由木棒、伺服系统和运行 Python 的 Raspberry Pi 制成的廉价而简单的绘图仪.
    • Arduino CNC Drawing Machine - 一个相当简单的 3d 打印 AxiDraw 风格的绘图仪，具有良好的视频文档.

    Motor Controllers¶

    • grblShield - 转动一个所需的所有步进电机控制硬件 Arduino 使用 grbl 固件转换为基于 G 代码的运动控制器. (adafruit)
    • TinyG - 功能更强大、功能更强大的基于 G 代码的 6 轴运动控制硬件. (adafruit)
    • Arduino CNC Shield - Grbl 兼容的 Arduino 步进电机控制屏蔽，类似于 grblShield.
    • Raspberry Pi CNC Hat - 带有步进控制器和运行 grbl 的微控制器的 Raspberry Pi 附加板. 与 Pi 的串行引脚接口.

    Accessories¶

    • WiFi232 - 通过 DB25 插头 Wifi 到 RS-232 串行. 以无线方式控制您的串行绘图仪.
    • Plotter Cable Pinout - 适用于大多数 HP 和 Roland 绘图仪的绘图仪电缆示意图. 在 eBay 或 Amazon 上搜索“DB9 到 DB25 串行空调制解调器电缆”或类似产品，以找到出售的产品.

    Pens¶

    • Sharpie Fine Point Plotter Adapter - 适合 HP-GL 绘图仪中的标准 Sharpie 的 3D 打印适配器.
    • Parametric 3d-Printable Plotter Pen Adapter - 可调节型号以打印各种笔的适配器.
    • Plotter Pen STL Models - 短和长标准绘图笔的准确 STL 模型.
    • Pens for AxiDraw - 适合一般绘图仪滥用的笔列表.
    • Pens for EggBot - 以鸡蛋和玻璃为重点的笔建议，但仍然普遍适用的信息.
    • JetPens – The Best White Ink Pens - 对许多白色墨水笔的全面回顾，并附有其覆盖特性的图片.

    Software¶

    HPGL¶

    HPGL 是一种基于串行/文本的协议，被大多数旧的笔式绘图仪甚至许多新的乙烯基切割机使用.

    • Chiplotle - 用于生成 HPGL 并与串行绘图仪接口的 Python 库.
    • Chiplotle on Parallel Port - 调整 Chiplotle 以使用并行端口连接.
    • HPGL Reference Guide - 基于 HTML 的 HPGL 参考.
    • HP 7475A Interfacing and Programming Manual - 包含完整 HPGL 参考的扫描版 PDF 手册.
    • djipco/hpgl - 与 HPGL 兼容的绘图仪和打印机通信的 Node.js 库.
    • hp2xx - 将 HPGL 转换为其他矢量和光栅格式的 GNU 工具. 也可用作 X11 中的预览.
    • vec - 使用海龟图形界面生成 HPGL 的示例 C 代码.
    • d3-hpgl - HTML Canvas API 的适配器，因此您可以使用流行的输出 HPGL D3 图书馆.
    • HPGL Viewer - 使用 JavaScript 和 HTML5 画布的 HPGL 查看器.
    • HPGL Sender - 用于预览 HPGL 并将其发送到绘图仪的 Web 界面.
    • HPGLGraphics - 用于编写 HPGL 文件的处理库.

    G-code¶

    G 代码是用于控制 CNC 机器的基于文本的标准. 虽然它是为工业机器设计的，但它在许多业余爱好者 3d 打印机固件中的使用使其在小型 DIY 项目中也无处不在.

    • grbl - 用于 Atmega 328 微控制器和 Arduino 的高性能 G 代码解释固件.
    • cncjs - 基于 Web 的界面控制运行 grbl、TinyG 或其他基于 G 代码的固件的 CNC 机器.
    • node-gcode - 基于 Node.js 的 G 代码解释器和模拟器.
    • svg2gcode - 基于 Node.js 的命令行实用程序，用于将 SVG 转换为 G 代码.
    • svg2gcode - 基于 Python 的实用程序，用于快速 SVG 到 G 代码的转换.
    • jscut - 用于将 SVG 转换为 G 代码的基于 Web 的实用程序.
    • Universal-G-Code-Sender - 基于 Java 的 grbl 兼容的跨平台 G 代码发送器.
    • ChiliPeppr Hardware Fiddle - 基于 Web 的模块化工作区，用于可视化 G 代码和控制硬件.
    • gcode-generative-for-processing - 处理库，旨在从简单的形状创建 gcode. （专为与 Creality CR10 一起使用而设计）

    Plotter-Specific¶

    特定于特定绘图仪或控制器的软件.

    • axidraw - Inkscape 的官方 AxiDraw 扩展.
    • axi - AxiDraw v3 的非官方 Python 库.
    • xy - Makeblock XY 绘图仪机器人套件的​​实用程序.
    • LaserGRBL - 用于 grbl 控制器的激光优化 Windows GUI. 可以重新用于 DIY 笔式绘图仪，使用螺线管进行笔向上/向下移动.
    • Line-us Inkscape Plugin - 将图纸发送到 Line-us plotter 直接来自 Inkscape.
    • Line-us API Examples - 示例代码 Line-us 绘图仪的基于 G 代码的 API.
    • @beardicus/line-us - 用于控制的 JavaScript 库 Line-us 来自 Node 或浏览器的机器.
    • PenPlotter - 使用重复固件的 Polargraph 控制器.
    • Makelangelo-firmware - Makelangelo polargraph 机器人的固件.
    • RoboPaint - WaterColorBot 软件.
    • AxiTurtle - 处理中 AxiDraw 的海龟图形.
    • GRBL-Plotter - 绘图仪优化的 Windows GUI，用于带有 SVG 和 DXF 导入的 grbl 控制器，以及灵活的上/下笔控制.
    • saxi - AxiDraw 的驱动程序和库. 使用恒定加速度运动规划并自动调整纸张大小.
    • MP2300-Tools - 用于将 HPGL 转换为 Graphtec 的 GPGL 格式的软件，以及用于 Graphtec 绘图笔适配器的 CAD 文件.

    Vector Creation¶

    从头开始或通过从其他格式转换来创建矢量图的工具.

    • Inkscape - 流行的跨平台开源矢量图形编辑器.
    • p5.js - “JavaScript 库，使艺术家、设计师、教育工作者和初学者都可以使用编码”.
    • Paper.js - “矢量图形脚本的瑞士军刀”.
    • ln - 用 Go 编写的基于矢量的 3D 渲染器.
    • autotrace - 将位图图像转换为矢量图形.
    • stipplegen - 从位图图像创建有趣的点画图. (blog post)
    • SquiggleDraw - “SquiggleDraw 将从图像创建一个 SVG 文件，使用亮度来改变正弦波的幅度”.
    • svgurt - 基于 Web 的 PNG 到 SVG 创意面条.
    • maptrace - 通过跟踪光栅图像生成防水多边形矢量地图.
    • Drawbot_image_to_gcode_v2 - 创建用于绘图机器人的 G 代码.
    • blackstripes - 将 PNG 图像转换为 SVG 线条图.
    • Ribbon - 用 Go 编写的蛋白质带状图.
    • penplot - JavaScript 绘图仪艺术的开发环境.
    • penkit - 用于创建基于行的 SVG 图形的 Python 库.
    • generativeExamples - 生成可打印 PDF 的示例处理代码.
    • Let's make map - 基于 Web 的工具，用于从 Mapzen 切片导出 SVG 地图.
    • SuperformulaSVG for web - 一个生成的线条艺术网络应用程序.
    • scribbleplot - 处理中的涂鸦图像转换.
    • Maker.js - 用于为 CNC 和激光切割机创建 2D 矢量绘图的库.
    • Turtletoy - 带有 SVG 导出的基于浏览器的 JavaScript 海龟图形 API.
    • cozyvec - 绘图仪艺术和推文图的 Web/独立终端环境.
    • makio135/plotter - 一个 Observable 笔记本系列充满了面向绘图员的工作.
    • PlotterFun - 基于浏览器的图像到 SVG 转换器，类似于 SquiggleDraw.

    Vector Utilities¶

    操作和优化基于矢量的文件格式的工具.

    • svgsort - 用于绘制 SVG 文件的路径规划，减少了用笔移动所花费的时间.
    • svgo - 用于优化 SVG 文件的基于 Node.js 的工具.
    • Polargraph Optimizer - 优化极坐标图的绘制计划.
    • penkit-optimize - 一个 SVG 优化器，它使用车辆路径求解器来最小化绘图时间.
    • svg-crowbar - 仅用于从 HTML 文档中提取 SVG 的 Chrome 书签.
    • vpype - 以绘图仪为中心的基于 Python 的 CLI 实用程序，用于生成和操作 SVG，包括缩放和优化路径.

    Fonts¶

    单行矢量字体或“雕刻字体”.

    • Summary of single line fonts - 良好的信息和其他资源和字体的链接.
    • Hershey Vector Font - 60 年代矢量字体的.fnt 格式. 包括对字体原始数据格式的很好的概述.
    • hershey-fonts - Hershey 字体的 C 库和原始字体数据.
    • OneLineFonts.com - 有一些单行字体可供购买的商业网站.
    • svg-fonts - SVG 格式的单行字体，主要用于 Hershey Text Inkscape 插件.
    • CNC Text Tool - 基于浏览器的 Hershey 文本工具，可导出为 SVG.
