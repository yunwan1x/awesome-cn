    JMeter

    Awesome JMeter ¶

    精心策划的资源集合，涵盖 Apache JMeter 以及相关的东西和闪亮的东西：插件、集成、测试技术、devops 实践等.

    Apache JMeter 是开源的纯 Java 应用程序，旨在负载测试功能行为和衡量性能.

    这份名单是从 an occasional answer 在 Stack Exchange 和个人 JMeter 相关链接集合上，从 awesome 项目并通过这些改进 amazing contributors.
