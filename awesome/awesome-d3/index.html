    D3

    Awesome D3 ¶

    这个列表记录了有趣 D3js 库、插件和实用程序.
    我们决定不在这里列出教程、资源或概念，因为已经有 very good list of readings 在那里，您可以查看并做出贡献.

    Curators: Moritz Klack 和 Christopher Möller webkid.io

    您还可以通过我们的互动浏览列表 D3 Discovery 工具.

    Charts¶

    • billboard.js - 可重复使用的图表库 [条形、线条、面积、甜甜圈、饼图、阶梯、样条]
    • britecharts - 客户端可重复使用的图表库 [bar, line, donut, sparkline, step]
    • chart-parts - 用于数据可视化的灵活、反应友好的图形语法
    • cirrusjs - 一个多渲染图表库[区域、线、堆积]
    • cubism - 时间序列可视化
    • c3 - 可重复使用的图表库[线、样条、步长、面积、堆积、条形、饼、甜甜圈]
    • dagre-d3 - 在客户端布局有向图
    • dc.js - 对于大量数据
    • dimple - 面向对象的业务分析 API [气泡、区域、堆叠]
    • d2b - 用于轴、饼图、桑基图、旭日图的图表库
    • d3.chart - 构建可重用图表的框架
    • d3.chart.sankey - 可重复使用的 D3 Sankey 图
    • d3-bar - 条形图
    • d3-beeswarm - 计算蜂群排列的插件
    • d3-boxplot - d3js 箱线图插件
    • d3-bumps-chart - 用于可视化颠簸图表的插件
    • d3-circle - 圆图 [甜甜圈]
    • d3-dag - 用于可视化有向 acylic 图的布局算法
    • d3-dot - 点图
    • d3-ez - 易于重复使用的 D3 图表和组件 [条形、甜甜圈、馅饼、气泡、玫瑰、热图]
    • d3-flame-graph - 来自分层数据的火焰图
    • d3-funnel - 漏斗和金字塔图表库
    • d3-gridding - 使用网格的图表模型
    • d3-heatmap - 热图
    • d3-horizon-chart - 地平线图表模块
    • d3-line - 折线图
    • d3-message-sequence - 动态/静态消息序列图
    • d3-timeseries - 时间序列图表库
    • d3-upset - 绘制 UpSet 图的函数 [沮丧、条形、交叉点、维恩、关系]
    • d3-x3d - 带有 D3 和 X3D 的数据驱动 3D 图表 [气泡、条形、表面积、散点图、面积]
    • d3fc - 交互式图表组件的集合 [线、条、堆积、散点、烛台、ohlc]
    • d3plus - 用于轻松创建可视化的扩展库 [分散、堆叠、线、条、饼、网络、气泡、框、地图]
    • d3panels - 带有链接刷牙的交互式图表 [点、散点、线、热图、直方图]
    • d3pie - 一个可配置的饼图库和生成器
    • D3xter - 直接绘图 [情节、时间线、条形图、直方图、饼图]
    • D4 - 可重复使用的图表 DSL [条形图、甜甜圈图、线条图、散点图、堆积图、瀑布图]
    • dTree - 家谱图书馆
    • epoch - 一个通用的实时可视化库[区域、条形、线条、饼图、散点图]
    • EventDrops - 基于时间/事件系列的交互式可视化
    • Fancycharts.js - 可视化百分比值的库 [条形、圆形、甜甜圈]
    • firespray - 流式图表 [条形、折线、堆积]
    • forest-d3 - 时间序列图表库 [线、面积、堆积、条形、散点、饼图 ohlc]
    • Gantt-Chart - 甘特图库
    • insights - 用于可视化和导航图形的库
    • kotojs - 基于 ES6 的框架
    • metrics-graphics - 针对时间序列数据的可视化进行了优化 [线、散点、区域]
    • micropolar - 极坐标图库
    • mpld3 - 导出 matplotlib 图形以在浏览器中工作
    • neo4jd3 - Neo4j 图形可视化
    • nvd3 - 可重复使用的图表和图表组件 [箱线图、柱状图、烛台、线条、条形图、饼图、散点图、迷你图]
    • oecd-simple-charts - 简单的图表库【箱线图、堆积条形图、珍珠图】
    • peek - 面向对象的图表库【散点图、折线图、面积图、条形图、饼图、甜甜圈】
    • plotly.js - 高级图表库[散点图、折线图、条形图、饼图、箱线图、直方图、热图]
    • plottable - 灵活的交互式网络图表 [面积、条形、折线、饼图、散点图、堆积图]
    • radar-chart-d3 - 雷达图模块
    • rickshaw - 用于创建交互式实时图形的工具包 [线、散点、条形]
    • taucharts - 注重设计和灵活性的图表 [线、条、面积、堆积]
    • techan.js - 视觉、股票图表和技术分析 [烛台，ohlc]
    • uvCharts - 支持多种不同的图表类型 [条形图、面积图、饼图、堆积图、折线图、极坐标图、甜甜圈图]
    • vega - 可视化语法
    • vega-lite - 交互式图形的高级语法
    • vega-lite-api - Vega-Lite 的 JavaScript API.
    • venn.js - Area proportional Venn and Euler diagrams
    • visavail - 时间数据可用性可视化
    • vizabi - 构建可视化数据探索工具的框架[气泡、地图、线、条、桑基、甜甜圈]
    • WebCola - 图形可视化和探索的布局
    • xkcdgraphs - Xkcd 样式图 [线]
