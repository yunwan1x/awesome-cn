    Awesome-Selfhosted¶

    Self-hosting is the practice of locally hosting and managing applications instead of renting from SaaSS 提供者.

    这是一个列表 Free 软件 network services 和 web applications which can be hosted locally. Non-Free software is listed on the Non-Free 页.

    See Contributing.
