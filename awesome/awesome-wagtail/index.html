    真棒鹡鸰 ¶

    > 来自 Wagtail 社区的精选软件包、文章和其他很酷的资源的精选列表.

    Wagtail 是一个由 Django 提供支持的 Python CMS，专注于灵活性和用户体验.

    您可能还喜欢 Awesome Django 和 Awesome Python . ：蛇：

    General resources¶

    • Official site
    • GitHub repository
    • Twitter account
    • Roadmap
    • PyPI classifiers for Wagtail

    Apps¶

    Blogging/news¶

    • Puput - Puput 是一个强大而简单的 Django 应用程序，用于管理博客. 它使用了不起的 Wagtail CMS 作为内容管理系统.
    • wagtail_blog - 在 Wagtail 中实现的类似 WordPress 的博客应用程序.
    • wagtailnews - 提供新闻/博客功能的 Wagtail 插件.
    • wagtail-blog-app - Wagtail Django CMS 的博客应用程序.
    • Django Wagtail Feeds - Add support for RSS Feeds, Facebook Instant Articles and Apple News Publisher to your Wagtail CMS Projects.
    • Snotra_RSS - Snotra_RSS 是 Wagtail 的 Atom 和 RSS 新闻聚合器应用程序.

    Rich text editor extensions¶

    • wagtail-readability - 测试您输入 Wagtail 的内容的可读性.
    • wagtailembedder - Wagtail 富文本字段的片段嵌入器.
    • Wagtail TinyMCE - Wagtail 的 TinyMCE 编辑器集成.
    • Wagtail Froala - 扩展 Wagtail 以在 RichTextField/RichTextBlock 中使用 Froala WYSIWYG 编辑器.
    • Wagtail Medium Editor - 一个可定制的 Wagtail 媒体编辑器，支持链接锚点.
    • WagtailDraftail – 用于 Wagtail 的 Draft.js 编辑器，建立在 Draftail 和 draftjs_exporter.
    • wagtail-readinglevel – Wagtail 插件，用于确定文本输入到富文本字段的阅读级别.

    Widgets¶

    • wagtailgmaps - 用于 Wagtail 字段的简单 Google 地图地址格式化程序.
    • Wagtail-Geo-Widget - Wagtail 中 GeoDjango PointField 字段的 Google 地图小部件.
    • wagtail-leaflet-widget - 一个 Leaflet JS - 基于 OSM 的 wagtail 地理位置小部件.
    • wagtail-markdown - Wagtail 的 Markdown 字段和块.
    • wagtail-autocomplete - 自动完成“ForeignKey”、“ParentalKey”和“ManyToMany”字段的选择器.
    • wagtail-instance-selector - 一个用于创建和选择相关项目的“外键”小部件. 类似于 Django 的 raw_id_fields.
    • wagtail-generic-chooser - 提供用于为 Wagtail 管理员构建选择器弹出窗口和表单小部件的基类，与 Wagtail 的页面、文档、片段和图像的内置选择器的外观和感觉相匹配.
    • wagtail-multi-upload - 允许为一个页面上传多个相关图像.
    • wagtail-color-panel - 引入了用于在 Wagtail 中选择颜色的面板.

    StreamField¶

    • Wagtail FontAwesome - 将 FontAwesome 图标添加到 StreamField.
    • Wagtail Commonblocks - 用于 Wagtail 的通用 StreamField 块.
    • Wagtail SVGmap - 通过内联 SVG 为 Wagtail 提供 ImageMap 功能.
    • Wagtail ClearStream - 使 Wagtail 的 StreamField 更加模块化的应用程序.
    • UWKM Streamfields – 一组基本的 Wagtail StreamField 块，用于娱乐和获利.
    • wagtail-inventory - 通过它们包含的 StreamField 块搜索 Wagtail 页面.
    • Wagtail Code Block - 带有实时 PrismJS 语法高亮显示的 Wagtail CMS 的 StreamField 代码块.
    • Wagtail Blocks - 一组很棒的 Wagtail CMS 流场块和图表.
    • Wagtail UI Plus - StreamFields 和 StreamBlocks 的 Wagtail 编辑器界面的多项 UI 改进.
    • Wagtail Cache Block - 将 HTML 片段缓存添加到 StreamField 块的模板标签

    Static site generation¶

    • Wagtail-bakery - 一组用于将 Django Wagtail 站点烘焙为平面文件的助手.
    • Wagtail-Netlify - 轻松将静态呈现的 Wagtail 站点发布到 Netlify.

    Settings management¶

    • Wagtail-Constance - Wagtail CMS 的 django-constance 集成.
    • Wagtail-Flags - Wagtail 网站的功能标志.

    E-commerce¶

    • wagtailinvoices - 用于创建发票的 Wagtail 模块.
    • longclaw - Wagtail CMS 的商店模板.
    • django-oscar-wagtail - Oscar Commerce 的 Wagtail 集成（或 Wagtail 的 Oscar Commerce 集成？）.

    SEO and SMO¶

    • wagtail-metadata - 一种帮助社交媒体和搜索引擎元数据的工具.
    • wagtail-metadata-mixin - 用于 Wagtail CMS 页面的 OpenGraph、Twitter Card 和 Google+ 代码段标签.
    • wagtail-schema.org - 用于 Wagtail 站点的 Schema.org JSON-LD 标签.
    • wagtail-opengraph-image-generator - 帮助您为 Wagtail 页面自动创建 Open Graph 图像.
    • wagtail-redirect-importer - 您友好的社区导入器，可让您从不同的表格数据格式（例如 .csv 和 .xls）导入重定向
    • wagtail-meta-preview - 添加了在 Wagtail 管理中获取 Facebook、Twitter 和 Google 的共享预览的功能.
    • Wagtail Yoast - 一种通过 SEO 建议提高文本可读性的工具.

    Analytics¶

    • Wagtail Analytics - Wagtail 管理员中的 Google Analytics 仪表板.

    Customer experience¶

    • Wagtail Experiments – Wagtail 的 A/B 测试.
    • Wagtail Personalisation - 个性化模块，使编辑人员能够根据直接在管理界面中配置规则的段来创建自定义页面或部分页面.

    Security¶

    • wagtailenforcer - 如果您需要在 Wagtail 站点上实施安全协议，那么您来对地方了.
    • wagtail-yubikey - 在 Wagtail 管理面板上启用 YubiKey 两因素身份验证.
    • wagtail-2fa - 通过将 Wagtail 与 django-otp 集成，向 Wagtail 添加两因素身份验证.

    Media¶

    • wagtailmedia - 用于在管理员内管理视频和音频文件的 Wagtail 模块.
    • Wagtail Alt Generator - 基于计算机视觉生成图像描述和标签的模块.
    • Wagtail FilePreviews - 使用来自 FilePreviews.io 的图像预览和元数据扩展 Wagtail 的文档.
    • Wagtail-Textract - 制作 Wagtail 搜索文档内容（PDF、Excel 和 Word 等）.
    • Wagtail-Lazyimages - 一个插件，可生成微小的模糊占位符图像，用于延迟加载 Wagtail 图像 medium.com 样式.
    • Wagtail Image Import - 用于从 Google Drive 导入图像的插件.
    • Wagtail SVG - 用于在管理员中管理 SVG 文件的 Wagtail 模块.
    • Wagtail Makeup - 一个插件，用它替换你的所有图像 Unsplash 图像.

    Translations¶

    • Wagtail Modeltranslation - 包含集成混合模型的简单应用程序 django-modeltranslation 进入Wagtail面板系统.
    • wagtailtrans - 用于支持多语言站点的 Wagtail 附加组件.

    Forms¶

    • wagtailpolls - 用于向 Wagtail CMS 添加轮询功能的插件.
    • Wagtailsurveys - Wagtail 模块，提供构建民意调查和调查的能力.
    • Wagtail ReCaptcha - wagtail-django-captcha 提供了一种简单的方法来集成 django-recaptcha 使用 Wagtail 表单生成器时的字段.
    • Wagtail Simple Captcha - Wagtail ReCaptcha 的自托管替代方案，可轻松集成 django-simple-captcha 字段进入 Wagtail 表单生成器.
    • wagtailstreamforms - 在 Wagtail 的管理员中构建表单以用于流场.
    • wagtail-contact-reply - 直接回复来自 Wagtail 管理员的表单提交
    • Wagtail JotForm - 用于 Wagtail 页面的可嵌入 Jotform 表单.

    Testing¶

    • wagtail-linkchecker - 一种帮助在您的 Wagtail 站点上查找损坏链接的工具.
    • Wagtail Accessibility – 在 Wagtail 中开发时辅助可访问性的插件.
    • Wagtail Factories - Wagtail 的工厂男孩课程.
    • Wagtail Foliage - 在 Wagtail 中以编程方式构建页面树的实用程序.

    Modeladmin¶

    • wagtail-admin-list-controls - 向 Wagtail 的模型管理员列表视图添加高级搜索、排序和布局控件.

    Misc¶

    • wagtailmenus - Torchbox 的 Wagtail CMS 的扩展，可帮助您以一致、灵活的方式管理和呈现多级导航和简单的平面菜单.
    • Wagtail Error Pages - Wagtail 的漂亮、智能、可定制的错误页面.
    • Wagtail Themes - Wagtail 的站点特定主题加载器.
    • Wagtail Sharing – 更轻松地共享 Wagtail 草稿.
    • Wagtail Gridder - Grid card layout similar to Google image search results, with an expanded area for card details.
    • Wagtail Condensed Inline Panel - 适用于大量内联的 Wagtail 的 InlinePanel 的直接替代品（可折叠，支持拖放）.
    • Joyous - Wagtail 的日历应用程序.
    • Wagtail App Pages - Extend Wagtail pages using an actual URL config and django views.
    • Wagtail Transfer - Wagtail 的官方扩展，允许在 Wagtail 项目的多个实例之间传输内容
    • Wagtail Import Export - 在 Wagtail 实例之间导入/导出页面.
    • Wagtail Import/Export Tool - 重构 Wagtail Import Export . 此工具支持导入/导出在导入/导出页面上使用的图像、文档和片段.
    • Wagtail Tag Manager - 一个 Wagtail 插件，允许更轻松且符合 GDPR 标准的脚本和标签管理.
    • Wagtail Cache - 使用 Django 缓存中间件的 Wagtail 的简单页面缓存.
    • Wagtail GraphQL - 自动将 GraphQL 支持添加到 Wagtail 网站的应用程序.
    • Wagtail Orderable - Mixin 支持在管理面板中进行拖放排序.
    • Wagtail Live Preview - 内容旁边的实时页面预览.
    • Wagtail Resume – 一个 Wagtail 项目，旨在简化为开发人员创建简历.
    • Wagtail Content Import - 使用可定制的映射系统将 Google Docs 或 Docx 中的内容导入 StreamFields.
    • Wagtail Trash - 将页面放在垃圾桶中，从那里可以恢复它们而不是被永久删除.
    • Wagtail PDF View - 使用 Wea​​syprint 或 LaTeX 将 Wagtail 页面和模型渲染为 PDF 文档.

    Tools¶

    Templates & Starter Kits¶

    • Wagtail Cookiecutter Foundation - 使用 Zurb Foundation 6 的 Wagtail CMS 的 Cookiecutter 模板.
    • Beginner Wagtail Cookiecutter – Wagtail CMS 的超级简单实现.
    • Wagtail Starter Kit – 一个带有 wagtail、django 布局、vagrant、配置文件、前端构建系统等的 cookiecutter！
    • Wagtail Pipit – Pipit 是一个 Wagtail 样板，旨在通过 React 渲染的前端提供简单而现代的开发人员工作流程.
    • Django Cookiecutter Wagtail – 带有 Wagtail 的 Django Cookiecutter 模板. 基于原始的“Django Cookiecutter”. 特性：Docker 支持使用 docker-compose 进行开发和生产（使用 Traefik 和 LetsEncrypt 支持）、可定制的 PostgreSQL 版本、Bootstrap 4、使用 Amazon S3 或 Google Cloud Storage 的媒体存储等等.
