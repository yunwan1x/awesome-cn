    Phalcon

    Awesome Phalcon¶

    精选的 Phalcon 库和资源列表. 灵感来自 awesome-go.

    Contributing¶

    请快速浏览 contribution guidelines 第一的. 谢谢大家 contributors ; 你摇滚！

    加入我们 Discord 与其他很棒的 phalcon 维护者聊天！

    ACL¶

    访问控制列表.

    • PhalconUserPlugin - 基于 Vökuró ACL 思想的插件

    Application Skeleton¶

    各种应用骨架.

    • Album O'Rama - Phalcon 框架的示例模块化应用程序
    • Base App - Phalcon 框架中的基础应用程序
    • INVO Application - Phalcon 框架的示例应用程序
    • MVC - Phalcon MVC 文件结构示例
    • Phalcon Composer - Phalcon with Composer 支持 MySql MongoDb Redis，干净+美观
    • Vökuró - Phalcon 框架的示例应用程序（Acl、Auth、安全性）
    • Webird - 创建 Webird 是为了将最新的 PHP 和 Node.js 创新合并到一个应用程序堆栈中
    • NovaMOOC - 带有 API/BackEnd 和 FrontEnd + Authentication with JWT + Cypress 测试和 GitHub Actions + Docker compose 的示例应用程序.
    • Sakura Panel - 一个 Phalcon 面板，无需编辑面板源代码即可安装插件（Auth、ACL、Lite Bootstrap 模板等）.

    Authentication & OAuth¶

    用于实现身份验证方案的库.

    • Padlock - 基于 docker 的 phalcon 身份验证服务器构建在 PHP OAuth 2.0 服务器之上
    • phalcon-authmiddleware - 向调度程序添加中间件事件. 与 ACL 或自定义身份验证库兼容的通用设计.
    • Phalcon Auth - 基于守卫和提供者的开箱即用的身份验证组件.

    CMS & Blogs¶

    • giada-www - 官方网站 Giada Loop Machine
    • KikCMS - 基于 Phalcon 框架的 CMS
    • Skopy Blog Engine - 一个简单的博客引擎，适合那些想要开始学习 Phalcon 的人
    • Yona CMS - 基于 Phalcon 框架的模块化结构的 CMS
    • PhalconCMS - 基于 Phalcon 框架的博客
    • Hummingbird CMS - 基于 Phalcon 4 的 CMS，具有许多强大的功能

    Command Line¶

    命令行应用程序和工具.

    • phalcon-console - 使用出色的 Phalcon 框架的命令行应用程序示例引导应用程序
    • phalcon-cron - Phalcon 的 Cron 组件

    Config¶

    • Phalcon Config Loader for Yaml - 加载 app/config 目录中的所有 yml.

    Dashboard¶

    管理面板和仪表板.

    • PhalconTime - TimeKeeping tool / Dashboard skeleton

    Debug¶

    调试和分析工具.

    • dd - 此软件包将向您的 Phalcon 应用程序添加 dd 和 dump 助手
    • Phalcon BB Debugger - Phalcon BB Debugger 强大且易于安装.
    • Phalcon Debugbar - 集成 PHP Debug Bar 使用 Phalcon 框架
    • Prophiler - PHP Profiler & Developer Toolbar（为 Phalcon 构建）

    i18n¶

    i18n 和 l10n 库和服务列表.

    • xgettext-template - 使用与 Volt 模板相同的命令行程序从 Volt 模板中提取 gettext 消息 xgettext invocation.

    Integration¶

    与第三方服务集成

    • phalcon-logentries - 将日志消息发送到 Logentries 日志管理服务

    ## 这里 IDE 扩展列表

    • volt-phalcon-language - VS Code 的扩展，提供对 Phalcon Volt 语法和自动完成的支持

    Miscellaneous¶

    这些库被放在这里是因为其他类别似乎都不适合

    • Breadcrumbs - 强大而灵活的组件，用于在 Phalcon 2+ 中构建站点面包屑.
    • Feedback - 旨在替代 Phalcon 的内置 Flash 和消息功能
    • Incubator - 用于发布/共享/实验新适配器、原型或功能的存储库，这些新适配器、原型或功能可能会被纳入 Phalcon 框架
    • yarak - Laravel 启发了 Phalcon 开发工具
    • phalcon-data-table - 允许您简化与 Phalcon 中引导表的交互

    ORM¶

    实现对象关系映射或数据映射技术的库.

    • phalcon-boundmodels - 根据 Phalcon 框架内的调度程序参数自动获取模型
    • phalcon-repositories - Phalcon 的简易存储库模式
    • phalcon-seeder - Phalcon 的数据库播种器组件
    • phalcon-redis-model - 基于redis的ORM和Easy Criteria（MySQL分片的对应关系）

    ODM¶

    实现对象文档映射器技术的库.

    • phalcon-collection-paginator - 一种 pagination adapter 对于扩展Phalcon\Mvc\Collection 的类

    Provisioning¶

    用于为 Phalcon 应用程序配置系统的工具.

    • ansible-phalcon - 在 Debian 中安装 Phalcon 框架的 Ansible 角色（提供 PHP 5.6 和 PHP 7.0 包）
    • setupify - 一组 bash 脚本，用于为部署或开发提供基于 Zephir 和 Phalcon 的系统

    RESTful¶

    代表性状态转移.

    • phalcon-json-api-package - 旨在帮助您在 Phalcon 中创建 JSON:API 的 Composer 包
    • PhREST API - Phalcon 框架 REST API 包
    • REST API - 使用 Phalcon 实现 API 应用程序

    Routing¶

    各种路由库和扩展.

    • Phalcon-autorouter - 无需复杂路由定义即可自动加载模块的简单方法
    • Phalcon Routing for Yaml - 可以使用 yaml 轻松配置路由

    Searching¶

    搜索工具和库.

    • ElasticsearchIndexer - Phalcon 的 Elasticsearch 索引器组件

    SEO¶
