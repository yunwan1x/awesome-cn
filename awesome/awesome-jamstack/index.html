    JAMstack









    Jamstack : 名词 \ 'jam-stak' \
    基于客户端 JavaScript、可重用 API 和预构建标记的现代 Web 开发架构.



    这是一个精心策划的很棒的Jamstack资源列表.



    您可能还喜欢真棒静态生成器.

    想贡献一些资源吗？ 伟大的！ 请在贡献之前阅读贡献指南.



    General¶

    • Jamstack
    • Jamstack resources - 关于 Jamstack 的视频和文章.
    • the New Dynamic - 在我们的目录中挑选工具来使用 Jamstack. 浏览我们的展示并获得灵感.

    Chats¶

    • Appernetic Gitter Chat
    • Jamstack Gitter Chat
    • Netlify Gitter Chat
    • NetlifyCMS Gitter Chat
    • the New Dynamic Slack

    Platforms¶

    • Appernetic - 静态网站的可视化内容管理，使用 Hugo 和 GitHub Pages.
    • Netlify - 用于自动化现代 Web 项目的多合一平台.
    • Vercel - 适用于现代 Web 应用程序的多合一无服务器平台，具有免配置工具和工作流.
    • Azure Static Web Apps - 具有集成 CI/CD 工作流程、身份验证、CDN 等的全栈无服务器托管.

    有关静态 Web 应用程序的更多资源，请参阅（很棒的静态 Web 应用程序）[https://github.com/staticwebdev/awesome-azure-static-web-apps].

    No-Code Platforms¶

    • Jamstack.new - 一键创建一个新的 Jamstack 网站，由 Stackbit 提供
    • Draftbox - 快速、安全的 WordPress 或 Ghost 博客前端，无需编码.
    • Plasmic - 强大的设计工具，用于可视化构建 React 组件和 Jamstack 网站.

    Jamstack Sites Showcase¶

    • React - 建立在盖茨比之上.
    • Squoosh.app - 托管在 Netlify 上，展示了现代 Web 应用程序的高级功能.
    • Hopper - 基于 Gatsby 构建并托管在 Netlify 上.
    • VSCode Power User Course - 基于 Gatsby 构建并托管在 Netlify 上的 PWA.
    • CloudyCam - PWA 构建在 Next.js 上并托管在 Zeit Now v2 Serverless 平台上.
    • FbaMonthly - 托管在 Netlify 上，使用 Jekyll、Netlify Functions 和 Netlify Forms 构建.
    • Jarod Peachey - 使用 Gatsby + MaterialUI 构建并托管在 Netlify 上
    • ProtonDB - 自 2018 年 8 月起在 Netlify 上托管 React/MaterialUI/Firebase.在 Linux 游戏社区中表现突出.
    • Creative Designs Guru - 基于 Netlify 上托管的 Eleventy.js 构建，并使用 Tailwind CSS 设计样式
    • HandleDroid - 使用 Next.js、MongoDB、Auth0、AWS CloudWatch、SendGrid、Stripe 构建并托管在 Netlify 上.
    • Backlinko - 使用 Next.js、无头 WordPress 和 Netlify 构建.

    Static Site Generators¶

    • Gatsby - 超快的 React 静态站点生成器.
    • Next.js - 用于静态和服务器渲染应用程序的轻量级框架.
    • Hugo - Hugo 是一个用 Go 编写的静态站点生成器.
    • Jekyll - Jekyll 是一个简单的、支持博客的静态站点生成器，非常适合个人、项目或组织站点.
    • Hexo - 一个快速、简单且强大的博客框架，由 Node.js 提供支持.
    • Metalsmith - 一个极其简单、可插拔的静态站点生成器.
    • Phenomic - 模块化网站编译器.
    • Gridsome - Vue 驱动的静态站点生成器，用于为任何 Headless CMS、API 或 Markdown 文件构建 CDN 就绪的网站.
    • Nuxt.js - 用于服务器端渲染、静态和传统单页应用程序的基于 vue 的多功能框架.
    • eleventy - 一个更简单的静态站点生成器，可将各种模板文件转换为 HTML.
    • Nift - 一个用 C++ 开发的跨平台开源网站生成器，速度快如闪电，功能极其强大.
    • Astro - 构建更快的网站，同时交付更少到几乎没有 Javascript.

    有关更完整的列表，请参阅 StaticGen.

    CMS¶

    • Cosmic - 具有 REST 和 GraphQL API 选项的无头 CMS.
    • Kentico Kontent - 可扩展的云原生无头 CMS.
    • Contentful - 数字团队的内容基础设施.
    • NetlifyCMS - 开源基于 Git 的 CMS.
    • ButterCMS - 无头 CMS 和内容 API.
    • Scrivito - 为数字机构和大中型企业构建的基于云的 JavaScript CMS.
    • GraphCMS - GraphQL 无头 CMS.
    • Prismic - 适用于开发人员和营销人员的无头 API CMS.
    • Siteleaf - 适用于静态站点的友好 CMS.
    • DatoCMS - 您的编辑会喜欢的基于 API 的 CMS.
    • Prose - Prose 是用于管理网站的 GitHub 内容编辑器.
    • Tipe - 下一代 API 优先 CMS.
    • Forestry - Headless CMS for sites built with static site generators.
    • Coisas - 用于编辑 GitHub Markdown（和其他）文件的客户端 CMS.
    • Cockpit - 自托管的无头和 api 驱动的 CMS.
    • HeadlessCMS - Jamstack 站点的内容管理系统列表.
    • TakeShape - 无头 GraphQL API CMS，一键部署到 Netlify.
    • Ghost - 用于在线出版物的无头 CMS
    • Strapi - 开源 Headless CMS 前端开发人员喜爱.
    • Flotiq - 一个轻松的无头 CMS.
    • TreacleWP - WordPress API 托管.
    • Agility CMS - 开发者和内容编辑者两全其美.

    API¶

    Authentication¶

    • Auth0 - 单点登录和基于令牌的身份验证.
    • Netlify Identity - 带来一整套身份验证功能，由 GoTrue API.
    • Jexia UMS - REST API 用户管理系统，用于您的 web 端，带有 JWT + 额外功能的身份提供程序.

    Comments¶

    • Disqus - 全球评论系统，可改善网站上的讨论并连接整个网络的对话.
    • Facebook Comments - 评论插件允许人们使用他们的 Facebook 帐户评论您网站上的内容.
    • Utterances - 基于 GitHub 问题构建的轻量级评论小部件. 将 GitHub 问题用于博客评论、维基页面等.

    Forms¶

    • Formcarry - 用于表单的无障碍 HTML 表单端点、强大的仪表板、可靠的垃圾邮件拦截、附件上传和 Zapier 集成.
    • Formcake - 为开发人员构建的表单后端：Zapier 集成、零依赖、简单的 API 和无限的表单.
    • Getform - 为设计人员和开发人员形成后端平台. 在几分钟内为您的静态站点设置表单端点，并通过 Zapier 集成和 Webhooks 支持扩展您的数据.
    • Netlify Forms - 通过在部署时直接解析 HTML 文件，在构建时内置表单处理.
    • StaticKit - 静态站点的现代表单，带有用于 React 和 HTML 的客户端库.
    • Arengu - 使用 Arengu 构建注册和登录表单、启用社交登录、添加 2FA 或使用无密码流程通过您的 API 或身份提供者对用户进行身份验证.
    • Static Forms - 无需任何服务器端代码即可轻松集成 HTML 表单. 用户提交表单后，我们会将表单内容发送到您注册的电子邮件中.

    E-commerce¶

    • Flatmarket - Flatmarket 是一个免费、开源的静态网站电子商务平台.
    • GoCommerce - Jamstack 网站的无头电子商务.
    • Snipcart - 一个强大的开发者购物车平台.
    • Moltin - 面向开发人员的电子商务 API.
    • Trolley - 专为 Jamstack 设计的购物车.
    • Shopify - Shopify 无头电子商务解决方案.
    • Commerce.js - 面向开发人员的 API 优先电子商务平台.
    • CartQL - 为 Jamstack 构建的 GraphQL 购物车和结账 API，可与任何框架和库存配合使用.

    Search¶

    • Algolia - 在您的业务中构建搜索的最可靠平台.
    • Lunr - 搜索变得简单（在前端）.
    • CloudSh - 使用几行 JavaScript 进行强大的搜索.

    Database¶

    • GraphQL - 用于 API 的查询语言和用于使用现有数据完成这些查询的运行时.
    • Graphcool - 开源和自托管后端即服务，用于开发无服务器 GraphQL 后端.
    • FaunaDB - 关系 NoSQL 数据库，在云中或本地具有连接、索引和多区域 ACID 事务.
    • Jexia DataSet - REST API Schema 和 Shemaless 云数据存储，具有内置验证器、关系、聚合功能.

    File management¶

    • Jexia FileSet - 具有索引和额外功能的应用程序的 REST API 文件管理器.

    Automation¶

    • Zapier - 触发将 1000 多个应用程序连接在一起的操作.
    • DataFire - DataFire 是一个用于构建和集成 API 的开源框架.

    Serverless¶

    • Netlify Functions - Netlify 允许您在没有 AWS 帐户的情况下部署 Lambda 函数，并在 Netlify 中直接处理函数管理.
    • Amazon Lambda - 让您无需配置或管理服务器即可运行代码.
    • 微软 Azure
    • Azure Functions - 无服务器计算服务，使您能够按需运行代码，而无需显式配置或管理基础设施.
    • Azure Logic Apps - 简化构建可跨云服务和本地系统集成应用程序和数据的自动化可扩展工作流.
    • 谷歌云
    • App Engine - 完全抽象出基础设施的无服务器应用程序，因此您只需专注于代码.
    • Cloud Functions - 用于构建和连接云服务的无服务器环境.
    • Cloud Datastore - 具有自动分片和复制功能的高度可扩展的 NoSQL 数据库.
    • Cloud Storage - 满足高 QPS 需求的地理冗余对象存储.
    • Cloud Pub/Sub - 适用于所有消息大小和速度的地理冗余实时消息传递.
    • Apigee - 多云环境的企业 API 管理.
    • Endpoints - 基于 Google Cloud 构建的 API 管理应用.
    • Cloud Dataflow - 无服务器流和批处理数据处理服务.
    • BigQuery - 无服务器数据仓库服务，可帮助您为企业部署高级云数据仓库解决方案.
    • Cloud ML Engine - 基于自定义 Google 硬件（张量处理单元）自动扩展的无服务器机器学习服务.
    • Serverless - 用于部署和操作无服务器架构的工具包.
    • Cloudinary - 无服务器媒体（图像/视频）管理平台. 为 Jamstack 提供每种流行语言的 SDK 和媒体小部件，以便轻松管理媒体、CDN、存储、转换等.
    • imgix - 无服务器图像交付和管理服务. imgix 连接到您的图像存储位置（例如 S3、GCS、Web 文件夹），并使用简单而强大的 URL 参数转换、优化和智能地交付您的图像.
    • Elastictable - 后端即服务解决方案建立在 AWS 上，可提高在线业务的成功率.
    • Stormkit - 强大的无服务器基础设施，适用于由 AWS 提供支持的现代 javascript 应用程序. 它与您的 git 流程集成并无缝构建、部署和扩展您的应用程序.

    有关更完整的列表，请参阅 Awesome Serverless.
