    Streaming

    Awesome Streaming ¶

    一个很棒的精选清单 streaming (stream processing) 框架、应用程序、阅读材料和其他资源. 灵感来自 other awesome projects.

    Website¶

    https://manuzhang.github.io/awesome-streaming/ 是一个更有活力的网站，您可以在这里找到很棒的项目的更新.

    Streaming Engine¶

    • Apache Apex [Java] - 大数据流和批处理的统一平台.
    • Apache Ballista [Rust] - 由 Apache Arrow 提供支持的分布式计算平台.
    • Apache Flink [Java] - 支持有状态计算、数据驱动窗口语义和迭代流处理的高吞吐量、低延迟数据流处理系统.
    • Apache Heron (incubating) [Java] - 来自 Twitter 的实时、分布式、容错流处理引擎.
    • Apache Samza [Scala/Java] - 基于 Kafka（消息传递、存储）和 YARN（容错、处理器隔离、安全和资源管理）构建的分布式流处理框架.
    • Apache Spark Streaming [Scala] - 可以轻松构建可扩展的容错流应用程序.
    • Apache Storm [Clojure/Java] - 分布式实时计算系统. Storm 之于流处理就像 Hadoop 之于批处理一样.
    • AthenaX [Java] - 生产中使用的 Uber 流分析框架
    • Faust [Python] - 流处理库，将思想从 Kafka Streams 移植到 Python
    • Gearpump [Scala] - 基于 Akka 构建的轻量级实时分布式流引擎.
    • Hazelcast Jet [Java] - 基于 Hazelcast 构建的通用分布式数据处理引擎.
    • hailstorm [Haskell] - 基于 Storm 的具有恰好一次语义的分布式流处理.
    • Maki Nage [Python] - 面向数据科学家的流处理框架，基于 Kafka 和 ReactiveX.
    • mantis [Java] - Netflix 构建实时流处理应用生态系统的平台
    • mupd8(muppet) [Scala/Java] - 用于处理快速/流数据的 mapReduce 风格的框架.
    • Onyx [Clojure] - 分布式、无主、高性能、容错数据处理.
    • s4 [Java] - 通用、分布式、可扩展、容错、可插拔的平台，允许程序员轻松开发用于处理连续无界数据流的应用程序.
    • SABER [Java/C] - 基于窗口的混合 CPU/GPU 流处理引擎.
    • Scramjet Transform Hub [JavaScript/Node.js] - 用于运行用 JavaScript 或 TypeScript 编写的多个数据处理应用程序（序列）的数据处理引擎
    • SPQR [Java] - 通过管道处理大量数据流的动态框架.
    • tigon [C++/Java] - 基于 Hadoop 和 HBase 的高吞吐量实时流处理框架.
    • Teknek [Java] - 使用交互式原型 shell SOL（流操作员语言）进行简单优雅的流处理 Mesos，专为需要灵活性和控制的高性能数据处理作业而设计.
    • Trill [.NET/C#] - Trill is a high-performance one-pass in-memory streaming analytics engine from Microsoft Research.
    • Wallaroo [Python] - 一个快速的流处理框架. Wallaroo 可以轻松实时地对数据做出反应. 通过消除基础设施的复杂性，从原型到生产从未如此简单.
    • LightSaber [C++] - 基于窗口的多核流处理引擎. LightSaber 使用代码生成来实现高效的窗口聚合.
    • HStreamDB [Haskell] - 为物联网数据存储和实时处理而构建的流数据库.
    • Kuiper [Golang] - 由 Golang 实现的边缘轻量级物联网数据分析/流媒体软件，可以在各种资源受限的边缘设备上运行.

    Streaming Library¶

    • Apache Kafka Streams [Java] - Apache Kafka 中包含的轻量级流处理库（从 0.10 版本开始）.
    • Akka Streams [Scala] - Akka Actors 上的流处理库.
    • Benthos [Go] - Benthos 是一种高性能且有弹性的消息流服务，能够连接各种源和接收器，并对有效负载执行任意操作、转换和过滤器
    • FS2(prev. 'Scalaz-Stream') [Scala] - Scala 的组合式、流式 I/O 库.
    • monix [Scala] - 用于编写异步和基于事件的程序的高性能 Scala / Scala.js 库.
    • Scramjet Framework - 在 Node.js 对象流之上编写的函数式反应流编程框架.
    • Streamline [Java] - Hortonworks 的流分析框架，设计为围绕现有流解决方案（如 Storm）的包装器. 旨在允许用户拖放流式组件以专注于业务逻辑.
    • StreamAlert [Python] - Airbnb 的实时数据分析和警报.
    • Swave [Scala] - 用于 Scala 的轻量级反应流基础设施工具包.
    • Streamz [Python] - 用于构建管道以管理连续数据流的轻量级库； 支持涉及分支、连接、流量控制、反馈、背压等的复杂管道.
    • Stream Ops [Java] - A fully embeddable data streaming engine and stream processing API for Java.
    • Tributary [Python] - 用于构建数据流图的 Python 库. 支持使用模拟复杂事件处理器的 python 生成器构建的同步、反应性数据流，以及延迟评估的非循环图和功能性柯里化流.
    • YoMo [Go] - 用于构建低延迟地理分布式系统的开源流无服务器框架. YoMo 建立在上面 QUIC Transport Protocol 和函数式反应式编程接口.

    Streaming Application¶

    • straw [Python/Java] - 实时流搜索平​​台.
    • storm-crawler [Java] - 基于 Apache Storm 的网络爬虫 SDK.

    IoT¶

    • sensorbee [Go] - 用于物联网的轻量级流处理引擎.
    • Apache Edgent [Java] - 一种编程模型和运行时，支持在网关和边缘设备上进行连续流分析，这些设备可以与集中式系统一起工作，以在整个物联网生态系统中提供高效、及时的分析：从中心到边缘，由 IBM 开源.
    • Apache StreamPipes [Java] - 自助（工业）物联网工具箱，使非技术用户能够连接、分析和探索物联网数据流.

    DSL¶

    • Apache Beam [Java、Python、SQL、Scala、Go] - 用于定义和执行数据处理工作流以及数据摄取和集成流的统一模型和特定语言 SDK 集，支持企业集成模式 (EIP) 和领域特定语言 (DSL) )，由谷歌开源.
    • coast [Scala] - 一种在 Samza 之上构建 DAG 并提供仅一次语义的 DSL.
    • Esper [Java] - 用于复杂事件处理 (CEP) 和事件序列分析的组件.
    • Streamparse [Python] - 允许您通过 Apache Storm 针对实时数据流运行 Python 代码.
    • summingbird [Scala] - 允许您编写看起来像原生 Scala 或 Java 集合转换的 MapReduce 程序的库，并在许多著名的分布式 MapReduce 平台上执行它们，包括 Storm 和 Scalding.

    Data Pipeline¶

    • Apache Kafka [Scala/Java] - 分布式、分区、复制提交日志服务，提供消息传递系统的功能，但具有独特的设计.
    • Apache Pulsar [Java] - 分布式发布-订阅消息平台，具有非常灵活的消息模型和直观的客户端 API.
    • brooklin [Java] - 一个分布式系统，用于在各种异构源和目标系统之间流式传输数据，具有高可靠性和来自 Linkedin（取代数据总线）的大规模吞吐量.
    • camus [Java] - Linkedin 的 Kafka -> HDFS 管道.
    • databus [Java] - Linkedin 的源不可知分布式变更数据捕获系统.
    • flume [Java] - 分布式、可靠且可用的服务，用于高效地收集、聚合和移动大量日志数据.
    • fluvio [Rust/WASM] - 具有在线计算能力的实时可编程数据流平台.
    • Gazette [golang] - 基于云存储构建的分布式流媒体基础设施，可以轻松混合和匹配批处理和流媒体范式.
    • LogDevice [C++] - Facebook 的高性能分布式系统，用于流式传输和存储顺序数据，使用日志结构.
    • metaq [Java] - 淘宝的高可用、高性能分布式消息系统
    • NATS streaming [Go] - 快速磁盘支持的消息传递解决方案
    • nsq [Go] - 实时分布式消息传递平台，旨在大规模运行，每天处理数十亿条消息.
    • RudderStack [Go] - 一个开源的客户数据基础设施（segment，mparticle 替代方案）.
    • suro [Java] - 用于收集、聚合和分派大量应用程序事件（包括日志数据）的数据管道服务.
    • StreamSets Data Collector [Java] - continuous big data ingestion infrastructure that reads from and writes to a large number of end-points, including S3, JDBC, Hadoop, Kafka, Cassandra and many others.

    Online Machine Learning¶

    • Apache Samoa [Java] - 分布式流式机器学习 (ML) 框架，其中包含分布式流式 ML 算法的编程抽象.
    • DataSketches [Java] - 来自 Yahoo! 的草图库.
    • streamDM [Scala] - 使用来自华为的 Spark Streaming 挖掘大数据流.
    • StreamingBandit [Python] - 提供一个网络服务器来快速设置和评估上下文多臂老虎机 (cMAB) 问题的可能解决方案.
    • StormCV [Java] - 通过添加计算机视觉 (CV) 特定操作和数据模型，支持使用 Apache Storm 进行视频处理.
    • trident-ml [Java] - 基于 Trident 的实时在线机器学习库.
    • yurita [Scala] - 基于 Paypal 的 Spark Structured Streaming 构建的异常检测框架.

    Streaming SQL¶

    • pipelinedb [C] - 一个开源关系数据库，它在流上连续运行 SQL 查询，以增量方式将结果存储在表中.
    • squall [Java] - Squall 在 Storm 之上执行 SQL 查询以进行在线处理.
    • StreamCQL [Java] - 实时计算系统上的连续查询语言.
    • ksqlDB [Java] - 云原生、源代码可用 database 专为流处理应用程序而构建
    • Materialize [Rust] - 一个源可用的流式 SQL 引擎，用于维护来自消息代理和数据库的数据的物化视图.
    • Siddhi [Java] - 云原生流和复杂事件处理引擎，了解流 SQL 查询，以便从不同数据源捕获事件、处理它们、检测复杂条件并将输出实时发布到各种端点.

    Benchmark¶

    • storm-benchmark [Java] - 一组测试 Storm 性能的基准.
    • storm-perf-test [Java] - 一个简单的风暴性能/压力测试.
    • streaming-benchmarks [Java] - 低延迟（流）解决方案的基准测试，包括 Apache Storm、Apache Spark、Apache Flink 等.
    • flotilla [Go] - 用于扩展基准测试的自动化消息队列编排.

    Toolkit¶

    • akka [Scala] - 用于在 JVM 上构建高度并发、分布式和弹性消息驱动应用程序的工具包和运行时.
    • pulsar [Python] - 基于 Actor 的事件驱动的 Python 并发框架.
    • aeron [Java/C++] - 高效可靠的单播和多播消息传输.
    • StreamFlow [Java] - 旨在帮助构建和监控处理工作流的流处理工具.
    • samza-luwak [Java] - 使用 Luwak，一个建立在 Lucene 上的存储查询引擎，实现对流的全文搜索.
    • Turbine [Java] - 用于将服务器发送事件 (SSE) JSON 数据流聚合为单个流的工具.

    Closed Source¶

    • Amazon Kinesis Streams [Java] - AWS 提供的实时、完全托管和可扩展的数据流引擎.
    • Azure Stream Analytics [.NET] 由 Microsoft Azure 提供的可大规模扩展、完全托管、实时的数据流引擎.
    • Cloud Dataflow [Java, Python, SQL, Scala] - Google 的托管流和批处理数据处理引擎. 支持运行 Beam 管道.
    • concord [C++] - 在 Apache 之上用 C++ 构建的分布式流处理框架.
    • IBM Streams [Python/Java/Scala] - 分布式处理和实时分析平台. 提供开箱即用的高级分析工具包，如地理空间、时间序列等.
    • jubatus [C++] - 分布式处理框架和流式机器学习库.
    • millwheel - 用于构建谷歌广泛使用的低延迟数据处理应用程序的框架.
