    Honeypots

    Awesome Honeypots ¶

    精选的令人敬畏的蜜罐列表，以及相关组件等，分为 Web、服务等类别，重点关注免费和开源项目.

    每个类别中的项目没有预先确定的顺序，顺序是为了贡献. 如果你想贡献，请阅读 guide.

    发现更多很棒的列表 sindresorhus/awesome.

    Related Lists¶

    • awesome-pcaptools - 在网络流量分析中很有用.
    • awesome-malware-analysis - 这里有一些重叠用于工件分析.

    Honeypots¶

    • 数据库蜜罐

      • Delilah - 用 Python 编写的 Elasticsearch Honeypot（最初来自 Novetta）.
      • ESPot - 用 NodeJS 编写的 Elasticsearch 蜜罐，捕捉每一次利用 CVE-2014-3120 的尝试.
      • Elastic honey - 简单的 Elasticsearch 蜜罐.
      • MongoDB-HoneyProxy - MongoDB蜜罐代理.
      • NoSQLpot - 基于 NoSQL 风格数据库构建的蜜罐框架.
      • mysql-honeypotd - 用 C 编写的低交互 MySQL 蜜罐.
      • MysqlPot - MySQL蜜罐，仍处于早期阶段.
      • pghoney - 低交互 Postgres 蜜罐.
      • sticky_elephant - 中等交互 postgresql 蜜罐.

    • 网络蜜罐

      • EoHoneypotBundle - Symfony2 表单的蜜罐类型.
      • Glastopf - 网络应用蜜罐.
      • Google Hack Honeypot - 旨在提供对使用搜索引擎作为攻击您资源的黑客工具的攻击者的侦察.
      • HellPot - 试图使访问其位置的机器人和客户端崩溃的蜜罐.
      • Laravel Application Honeypot - Laravel 应用程序的简单垃圾邮件预防包.
      • Nodepot - NodeJS Web 应用蜜罐.
      • Servletpot - 网络应用蜜罐.
      • Shadow Daemon - 适用于 PHP、Perl 和 Python 应用程序的模块化 Web 应用程序防火墙/高交互蜜罐.
      • StrutsHoneypot - 基于 Struts Apache 2 的蜜罐以及用于 Apache 2 服务器的检测模块.
      • WebTrap - 旨在创建欺骗性网页，以欺骗和重定向攻击者远离真实网站.
      • basic-auth-pot (bap) - HTTP 基本身份验证蜜罐.
      • bwpot - 易碎的网络应用程序honeyPot.
      • django-admin-honeypot - 假 Django 管理员登录屏幕，以通知管理员尝试未经授权的访问.
      • drupo - Drupal 蜜罐.
      • honeyhttpd - 基于 Python 的 Web 服务器蜜罐构建器.
      • phpmyadmin_honeypot - 简单有效的 phpMyAdmin 蜜罐.
      • shockpot - 用于检测 Shell Shock 漏洞利用尝试的 WebApp Honeypot.
      • smart-honeypot - 演示智能蜜罐的 PHP 脚本.
      • Snare/Tanner - Glastopf 的继任者
        • Snare - 超级下一代高级反应式蜜罐.
        • Tanner - 评估 SNARE 事件.
      • stack-honeypot - 在回复中插入垃圾邮件机器人陷阱.
      • tomcat-manager-honeypot - 模仿 Tomcat 管理器端点的蜜罐. 记录请求并保存攻击者的 WAR 文件以供日后研究
      • WordPress 蜜罐
        • HonnyPotter - WordPress 登录蜜罐，用于收集和分析失败的登录尝试.
        • HoneyPress - Docker 容器中基于 Python 的 WordPress 蜜罐.
        • wp-smart-honeypot - WordPress 插件，通过更智能的蜜罐减少垃圾评论.
        • wordpot - WordPress 蜜罐.

    • 服务蜜罐

      • ADBHoney - 模拟运行 Android 调试桥 (ADB) 服务器进程的 Android 设备的低交互蜜罐.
      • AMTHoneypot - 针对英特尔 AMT 固件漏洞 CVE-2017-5689 的蜜罐. X - Ensnare - 易于部署的 Ruby 蜜罐.
      • HoneyPy - 低交互蜜罐.
      • Honeygrove - 基于 Twisted 的多用途模块化蜜罐.
      • Honeyport - 用 Bash 和 Python 编写的简单蜜端口.
      • Honeyprint - 打印机蜜罐.
      • Lyrebird - 现代高交互蜜罐框架.
      • MICROS honeypot - 在 Oracle Hospitality Applications (MICROS) 的 Oracle Hospitality Simphony 组件中检测 CVE-2018-2636 的低交互蜜罐.
      • RDPy - 用 Python 实现的 Microsoft 远程桌面协议 (RDP) 蜜罐.
      • SMB Honeypot - 高交互SMB服务蜜罐，能够捕获类似wannacry的恶意软件.
      • Tom's Honeypot - 低交互 Python 蜜罐.
      • WebLogic honeypot - 用于检测 Oracle 融合中间件的 Oracle WebLogic Server 组件中的 CVE-2017-10271 的低交互蜜罐.
      • WhiteFace Honeypot - 用于 WhiteFace 的基于扭曲的蜜罐.
      • dhp - 模拟 Docker HTTP API 小片段的简单 Docker 蜜罐服务器.
      • honeycomb_plugins - Honeycomb 的插件库，Cymmetria 的蜜罐框架.
      • honeyntp - NTP 记录器/蜜罐.
      • honeypot-camera - 观察相机蜜罐.
      • honeypot-ftp - FTP蜜罐.
      • honeytrap - 用 Go 编写的高级蜜罐框架，可以与其他蜜罐软件连接.
      • pyrdp - RDP 中间人和 Python 3 库，能够实时或事后观看连接.
      • troje - 在单独的 LXC 容器中运行与服务的每个连接的蜜罐.

    • 分布式蜜罐

      • DemonHunter - 低交互蜜罐服务器.

    • 反蜜罐的东西

      • kippo_detect - 检测 kippo 蜜罐存在的攻击性组件.

    • ICS / SCADA 蜜罐

      • Conpot - ICS/SCADA 蜜罐.
      • GasPot - Veeder Root Gaurdian AST，在石油和天然气行业中很常见.
      • SCADA honeynet - 为工业网络构建蜜罐.
      • gridpot - 用于真实行为电网蜜网的开源工具.
      • scada-honeynet - 模仿流行 PLC 的许多服务，更好地帮助 SCADA 研究人员了解暴露的控制系统设备的潜在风险.

    • 其他/随机

      • Damn Simple Honeypot (DSHP) - 带有可插拔处理程序的蜜罐框架.
      • NOVA - 使用蜜罐作为检测器，看起来像一个完整的系统.
      • OpenFlow Honeypot (OFPot) - 将未使用 IP 的流量重定向到基于 POX 的蜜罐.
      • OpenCanary - 模块化和分散的蜜罐守护进程，运行多个金丝雀版本的服务，当服务被（滥用）使用时发出警报.
      • ciscoasa_honeypot Cisco ASA 组件的低交互蜜罐，能够检测 CVE-2018-0101、DoS 和远程代码执行漏洞.
      • miniprint - 中等交互打印机蜜罐.

    • 僵尸网络 C2 工具

      • Hale - 僵尸网络命令和控制监视器.
      • dnsMole - 分析 DNS 流量并潜在地检测僵尸网络命令和控制服务器活动以及受感染的主机.

    • IPv6 攻击检测工具

      • ipv6-attack-detector - 由蜜网项目组织支持的 Google Summer of Code 2012 项目.

    • 动态代码检测工具包

      • Frida - 注入 JavaScript 以探索 Windows、Mac、Linux、iOS 和 Android 上的原生应用程序.

    • 将网站转换为服务器蜜罐的工具

      • HIHAT - 将任意 PHP 应用程序转换为基于 Web 的高交互蜜罐.

    • 恶意软件收集器

      • Kippo-Malware - Python 脚本将下载所有以 URL 形式存储在 Kippo SSH 蜜罐数据库中的恶意文件.

    • 分布式传感器部署

      • Community Honey Network - CHN 旨在使部署蜜罐和蜜罐管理工具变得简单和灵活. 默认部署方法使用 Docker Compose 和 Docker，通过几个简单的命令进行部署.
      • Modern Honey Network - 多 snort 和蜜罐传感器管理，使用虚拟机网络、小规模 SNORT 安装、隐蔽 dionaeas 和集中式服务器进行管理.

    • 网络分析工具

      • Tracexploit - 重播网络数据包.

    • 日志匿名器

      • LogAnon - 日志匿名化库，有助于使日志和网络捕获之间的匿名日志保持一致.

    • 低交互蜜罐（路由器后门）

      • Honeypot-32764 - 路由器后门蜜罐 (TCP 32764).
      • WAPot - 可用于观察指向家庭路由器的流量的蜜罐.

    • 蜜网农场流量重定向器

      • Honeymole - 部署多个传感器，将流量重定向到蜜罐的集中集合.

    • HTTPS 代理

      • mitmproxy - 允许拦截、检查、修改和重放流量.

    • 系统仪表

      • Sysdig - 开源的系统级探索允许从正在运行的 GNU/Linux 实例中捕获系统状态和活动，然后保存、过滤和分析结果.
      • Fibratus - 用于探索和跟踪 Windows 内核的工具.

    • 用于 USB 传播恶意软件的蜜罐

      • Ghost-usb - 用于通过 USB 存储设备传播的恶意软件的蜜罐.

    • 数据采集

      • Kippo2MySQL - 从 Kippo 的基于文本的日志文件中提取一些非常基本的统计信息并将它们插入到 MySQL 数据库中.
      • Kippo2ElasticSearch - 将数据从 Kippo SSH 蜜罐 MySQL 数据库传输到 ElasticSearch 实例（服务器或集群）的 Python 脚本.

    • 被动网络审计框架解析器

      • Passive Network Audit Framework (pnaf) - 结合多种被动和自动分析技术以提供网络平台安全评估的框架.

    • 虚拟机监控和工具

      • Antivmdetect - 创建与 VirtualBox 一起使用的模板的脚本，使 VM 检测更加困难.
      • VMCloak - Cuckoo Sandbox 的自动虚拟机生成和伪装.
      • vmitools - 带有 Python 绑定的 C 库，可以轻松监控正在运行的虚拟机的低级细节.

    • 二进制调试器

      • Hexgolems - Pint Debugger Backend - 用于 PIN 的调试器后端和 LUA 包装器.
      • Hexgolems - Schem Debugger Frontend - 调试器前端.

    • 移动分析工具

      • Androguard - Android 应用程序的逆向工程、恶意软件和软件分析等.
      • APKinspector - 强大的 GUI 工具，供分析师分析 Android 应用程序.

    • 低交互蜜罐

      • Honeyperl - 基于 Perl 的蜜罐软件，带有为许多功能开发的插件，例如：wingates、telnet、squid、smtp 等.
      • T-Pot - 来自电信提供商 T-Mobile 的多合一蜜罐设备

    • 蜜网数据融合

      • HFlow2 - 用于蜜网/网络分析的数据合并工具.

    • 服务器

      • Amun - 漏洞仿真蜜罐.
      • Artillery - 开源蓝队工具，旨在通过多种方法保护 Linux 和 Windows 操作系统.
      • Bait and Switch - 将所有恶意流量重定向到部分镜像您的生产系统的蜜罐.
      • Bifrozt - Automatic deploy bifrozt with ansible.
      • Conpot - 低交互服务器端工业控制系统蜜罐.
      • Heralding - 证书捕获蜜罐.
      • HoneyWRT - 低交互 Python 蜜罐，旨在模仿可能成为攻击者目标的服务或端口.
      • Honeyd - 看 honeyd tools.
      • Honeysink - 开源网络天坑，提供检测和预防给定网络上的恶意流量的机制.
