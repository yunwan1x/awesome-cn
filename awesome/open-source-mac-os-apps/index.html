    Awesome macOS open source applications¶

    Applications¶

    Audio¶

    • AUHost - 使用 AVFoundation API 托管 AudioUnits v3 的应用程序. ![swift_icon]
    • Audacity - 免费、开源、跨平台的音频软件！[c_icon]
    • Aural Player - Aural Player 是 macOS 平台的音频播放器应用程序. 受 Windows 经典 Winamp 播放器的启发，它的设计直截了当且易于使用. ![swift_icon]
    • AutoMute - 当耳机断开连接/Mac 从睡眠中唤醒时自动静音. ![objective_c_icon]
    • Background Music - 背景音乐，一种 macOS 音频实用程序：自动暂停您的音乐、设置单个应用程序的音量并录制系统音频. ![cpp_icon]
    • BlackHole - BlackHole 是一种现代 macOS 虚拟音频驱动程序，它允许应用程序以零额外延迟将音频传递给其他应用程序. ![c_icon]
    • CAM - 使用 ffmpeg 的 macOS 摄像头录制！[javascript_icon]
    • Clementine - Clementine 是适用于 Windows、Linux 和 macOS 的现代音乐播放器和库管理器. ![cpp_icon]
    • Cog - Cog 是 macOS 的开源音频播放器. 基本布局是带有两个可伸缩抽屉的单窗格播放列表界面，一个用于浏览用户的音乐文件夹，另一个用于查看音频文件属性，如比特率. ![objective_c_icon]
    • Karaoke Forever - 举办精彩的卡拉 OK 派对，每个人都可以在手机浏览器中排队播放歌曲. 使用 WebGL 可视化播放 MP3+G 和 MP4. ![javascript_icon]
    • LocalRadio - LocalRadio 是用于在 Mac 和移动设备上收听“软件定义无线电”的软件. ![objective_c_icon]
    • LosslessCut - 无需重新编码的无损视频/音频编辑的瑞士军刀. ![javascript_icon]
    • Lyricism - macOS 应用程序向您显示当前 iTunes 或 Spotify 正在播放的歌词. ![objective_c_icon] ![swift_icon]
    • LyricsX - iTunes、Spotify 和 Vox 的歌词. ![swift_icon]
    • MPV - 轻量级、高度可配置的媒体播放器. ![c_icon]
    • MonitorControl - 直接从菜单或使用键盘本机键控制外部显示器的亮度、对比度或音量. ![swift_icon] ![objective_c_icon]
    • Mous Player - 适用于 BSD/Linux/macOS 的简单而强大的音频播放器. ![cpp_icon]
    • Music Bar - 音乐栏是 macOS 应用程序，可将音乐控件直接放置在您的菜单栏中. ![swift_icon]
    • NoiseBuddy - 在触控栏或菜单栏中控制 AirPods Pro 上的聆听模式. ![swift_icon]
    • PlayStatus - PlayStatus 是一个 macOS 应用程序，允许从菜单栏控制 Spotify 和 iTunes 音乐播放. ![swift_icon]
    • Plug - 从 Hype Machine 发现和聆听音乐. ![swift_icon]
    • Scope - 音频示波器！[swift_icon]
    • ShazamScrobbler - 将黑胶唱片、收音机、电影拼凑到 Last.fm. ![objective_c_icon]
    • Sonora - 适用于 macOS 的最小、设计精美的音乐播放器. ![objective_c_icon]
    • SpotMenu - 菜单栏中的 Spotify 和 iTunes. ![objective_c_icon] ![swift_icon]
    • SpotSpot - 适用于 macOS 的 Spotify 迷你播放器. ![javascript_icon]
    • Spotify4BigSur - 用于通知中心的 Spotify 小部件. ![swift_icon]
    • Suohai - macOS 的音频输入/输出源锁定. ![swift_icon]
    • Tickeys - 打字的即时音频反馈. macOS 版本. ![rust_icon]
    • Tuner - 乐器调音器！[swift_icon]
    • [Un]MuteMic - 用于静音和取消静音麦克风输入音量的 macOS 应用程序. 非常适合播客. ![objective_c_icon] ![c_icon]
    • eqMac2 - 适用于 Mac 的全系统均衡器. ![cpp_icon]
    • fre:ac - fre:ac 音频转换器项目. ![cpp_icon]
    • iTunes-Volume-Control - 此应用程序允许您使用音量增大和音量减小热键来控制 iTunes 音量. ![objective_c_icon]
    • jmc - jmc 是新的 macOS 媒体管理器. ![swift_icon]
    • shairport-sync - macOS/Linux/FreeBSD/OpenBSD Airplay 音频接收器. ![c_icon] ![cpp_icon]
    • waveSDR - 使用 RTL-SDR USB 设备的 macOS 本机桌面软件定义无线电应用程序. ![swift_icon]
