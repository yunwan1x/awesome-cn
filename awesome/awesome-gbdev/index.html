    Disambiguation¶

    Game Boy Advance¶

    > Game Boy Advance 开发由另一个项目涵盖 awesome-gbadev 列表. > 但是，GBA 可以运行 GB/GBC 游戏. 与本机硬件相比，它以略有不同的方式执行此操作：此列表的仿真器开发部分对此进行了介绍.

    Game Boy Color and Super Game Boy¶

    > 此列表重点关注最初的 Game Boy（GB 或 DMG，1989）、Game Boy Color（GBC 或 CGB）和 Super Game Boy (SGB) 是非常相似的系统，但有一些重要的系统区别，例如：

    >- 不同的硬件规格； >- 特定的硬件和软件功能； >- 特定寄存器； >- 特定的错误、怪癖和可利用的行为.

    >如果您打算为 SGB 或 GBC 开发您的软件，或者您想知道它如何在其他系统上运行，您可能想利用并适应这些差异，请检查 Game Boy Color 类别并查找对 GBC/CGB 和 SGB 的具体参考.

    Software Development¶

    The Choosing tools for Game Boy development 文章概述了 Game Boy 的可用开发工具.

    Assemblers¶

    • RGBDS - 汇编程序和链接程序包. Documentation.
    • ASMotor - 针对 Game Boy 和其他 CPU 的汇编引擎和开发系统. 由原始RGBDS作者撰写. Documentation.
    • wla-dx - 另一个 GB-Z80/Z80/... 多平台交叉汇编程序包. Documentation.

    Compilers¶

    • GBDK - 维护和现代化的 GBDK（Game Boy 开发工具包）由 SDCC 工具链的更新版本提供支持. 提供 C 编译器、汇编器、链接器和一组库.
    • API docs: Getting Started
    • Examples
    • Documentation, links and tools
    • Turbo Rascal Syntax Error - 完整套件（IDE、编译器、编程语言、资源编辑器），用于为 8 / 16 位计算机系列开发游戏/演示，包括 Game Boy 和 Game Boy Color.

    Experimental/Proof of Concepts¶

    • RGBDS-Live - 浏览器内编码环境以试用 RGBDS.
    • Wiz - 一种用于在复古控制台平台（Game Boy、NES、Atari 2600 等）上编写自制软件的高级汇编语言.
    • gbforth - 基于 Forth 的 Game Boy 开发工具包.
    • gbasm-rs - 适用于 Game Boy z80 汇编代码的固执的基于 Rust 的编译器.
    • gbasm - 基于 JavaScript 的 Game Boy z80 汇编代码编译器.
    • tniASM - 宏汇编程序.
    • Assembler - 用 Swift 编写的汇编程序.
    • llvm-gbz80 / clang-gbz80 - 到 GBZ80 CPU 的 Clang/LLVM 端口（类似于已弃用 euclio/llvm-gbz80).
