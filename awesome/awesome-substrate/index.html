    Awesome substrate

    Awesome Substrate ¶

    > 一个很棒的列表是一个由社区策划的很棒的东西的列表.

    Substrate 是一个构建可升级、模块化和高效的框架 区块链. Substrate 是一个开源库 Rust 代码 由维护 Parity Technologies.

    Official Resources¶

    • Developer Hub - 开发者门户.
    • Tutorials - 引导练习让您入门.
    • Knowledge Base - 相关的深入文档 基板开发.
    • Recipes - 解决常见问题的工作代码示例.
    • Reference Docs - Rustdoc API 文档.
    • Seminar - 协作学习 Crowdcast.
    • Technical Chat - 问问题 ＆ 与其他 Substrate 开发人员聊天.
    • Builders Program - 为有远见的人提供白手套支持 团队.
    • Web3 Foundation Grants - 为生态系统发展提供资金.
    • Users - 使用 Substrate 构建团队.
    • GitHub - 基板源代码.
    • Developer Hub GitHub - 开发者中心存储库.
    • Polkadot GitHub - 多个的基板 monorepo Parity 赞助的连锁店.
    • subport - 支持存储库.
    • Parity Blog - 了解 Substrate 生态系统的更新.
    • Parity Events - 即将举行的活动.
    • Twitter - 关注我们以了解最新信息.
    • Polkassembly - 讨论和投票 Kusama 和 Polkadot 的地方 治理.

    Community Resources - Guides¶

    Frameworks¶

    • FRAME - 一个惯用的系统 构建 Substrate 运行时.
    • ink! - Substrate 链的智能合约语言.
    • Cumulus - 简化的 Polkadot 兼容平行链.
    • Bridges - 工具的集合 跨链通信.
    • Frontier - Substrate 的端到端以太坊仿真 链.
    • Polkadot-JS - 丰富的前端开发框架.
    • SubstraTEE - 可信的链下执行框架，使用 Intel SGX 可信执行 环境.
    • FRAME Benchmarking - 这 用于运行时开发的 FRAME 系统的官方基准测试框架.

    Templates¶

    • Node - 一个新鲜的基于 FRAME 节点，准备黑客攻击.
    • Pallet - 轻松创建 可插拔的区块链功能.
    • Parachain - 一个新 基于 Cumulus 的 Substrate 节点，准备好进行黑客攻击（有关更多信息，请参阅框架下的“Cumulus”）.
    • Front-End - 使用 Polkadot-JS API 和 React 为基于 Substrate 的链构建前端.
    • substrate-stencil - 一个模板 包含质押和治理功能的基板节点.

    Pallets¶

    • Open Runtime Module Library (ORML) - 社区 维护 Substrate 运行时模块的集合.
    • Sunshine Bounty - 分散式 用于管理赏金计划的自治组织 (DAO).
    • Sunshine Identity - 受 Keybase 启发 身份管理.
    • Sunshine Faucet - 分配 开发链的资源.
    • Commodities - 定义和实现一个不可替代 令牌接口.
    • Chainlink Feed Pallet - Chainlink 饲料托盘和示例节点 展示如何集成到 Substrate 链.

    Client Libraries¶

    • subxt - 官方 Rust 客户端.
    • Substrate API Client - Rust 客户端维护 Supercomputing Systems AG.
    • Polkadot-JS API - Semi-official JavaScript library for 基于底物的链.
    • Python Substrate Interface - 维护者 Polkascan Foundation.
    • Go Substrate RPC Client - 维护者 Centrifuge.
    • Substrate C++ API - 维护者 Usetech.
    • Substrate .Net API - 由 Usetech 维护.
    • Substrate .NET Standard API 在 nuget， 和 Unity 3D integration example - 维护者 DOTMog.
    • Kotlin Substrate Client - 维护者 Nodle.io

    Ecosystem Tools¶

    • Polkadot-JS Apps UI - 半官方区块浏览器和前端 用于基于基板的链.
    • Polkadot-JS Extension - 浏览器扩展 与基于 Substrate 的链交互.
    • Polkadot Tool Index - 这是 一个非常有用的工具列表，可用于您使用 Polkadot 和 任何基础链，包括区块浏览器、钱包、网络监控和报告， 客户端、基准测试、模糊测试、分叉、SCALE 编解码器、CLI 工具等等.
    • Sidecar - 一起运行的 REST 服务 基板节点.
    • Archive - Substrate 链的索引引擎.
    • TxWrapper - 离线交易的有用库 创建.
    • Subkey - 命令行实用程序 使用加密密钥.
    • Polkascan - Polkascan 维护的多链区块浏览器 基础.
    • VSCode Substrate - 官方 Visual Studio Code 的插件.
    • Parity Signer - 将未使用的手机升级为气隙 硬件钱包.
    • Fork Off Substrate - 帮助引导程序的脚本 具有运行链状态的新链.
    • Substrate debug-kit - 工具集合 和用于调试基于 Substrate 链的库，包括 offline-election, 这是一种用于预测提名的股权证明选举的工具.
    • sup - 用于生成或升级一个命令行工具 基板节点.
    • Substrate Dev Hub Utils - 非官方 用于使用官方 Substrate Developer Hub 资源的实用程序.
    • Subscan - 基于 Substrate 的链的多网络浏览器.
    • Substrate Docker Builders - 一套 Dockerfiles 和 GitHub Actions 为基于 Substrate 的链自动构建和推送 Docker 镜像.
    • Halva - 用于改善体验的工具链 在基板上开发.
    • offchain::ipfs - 基板注入 IPFS.
    • Gantree Library - 一套技术 通过快速启动和拆除来管理由 Substrate 驱动的平行链网络.
    • Redspot - 一种 Truffle- 类似于 FRAME 智能合约的工具包 合同托盘.
    • Proxy Hot Wallet Demo - 一个示范 基于 Substrate 原语的安全、方便和灵活的热钱包架构.
    • Substrate Faucet Bot - 基于 Python 的水龙头 发展目的.
    • Hydra - 一个用于 Substrate 节点的 GraphQL 框架 hosted playground . 看看伟大 docs.
    • Substrate Graph - GraphQL 索引器 基于底物的链.
    • SubQuery - GraphQL 索引器和查询服务，允许用户 轻松创建索引数据源并免费在线托管它们.
    • Megaclite - Polkadot 的零知识工具 生态系统.
    • Europa - 一个用于 Substrate 运行时执行的沙箱 环境.
    • Jupiter - 为智能合约编写的测试网 FRAME 合同托盘.
    • Staking Rewards Collector - 要解析的脚本 并为给定的 Kusama 或 Polkadot 地址输出质押奖励并交叉引用它们 与每日价格数据.
    • polkadot-launch - 简单的 CLI 工具来启动一个 本地 Polkadot 测试网络.
    • substrate-js-utils - 一套有用 用于使用 Polkadot{JS} API 的 Substrate 的 JavaScript 实用程序. 还 deployed as a website.
    • substrate-graph-benchmarks - 图表 FRAME 托盘的基准输出.
    • substrate-balance-calculator - 分解 您的 Substrate 账户的余额.
    • substrate-balance-graph - 创建一个 Substrate 地址的代币余额随时间变化的图表.
    • polkadot-js-bundle - 一个独立的 JS 包含 Polkadot{JS} 库的包.
    • substrate-society - 一个基本的前端 FRAME Society 托盘.
    • Aleph.im - 可扩展、分散的数据库、文件存储和计算 为 Substrate 链等提供服务.
    • subsee - CLI 以 JSON 形式检查基板节点的元数据.
    • polkadot-runtime-prom-exporter - 一种 Prometheus Polkadot 运行时指标的导出器（可修改用于基板使用）.
    • subwasm - 用于离线检查运行时 WASM blob 的 CLI. 它显示信息、元数据并可以比较运行时. 它还可以帮助您直接从节点获取运行时.
    • srtool - Docker 映像以确定性地构建运行时.
    • srtool-cli -srtool Docker 镜像的 CLI 前端.
    • srtool-actions - Github 操作可轻松使用 srtool Docker 映像构建您的 onw 运行时.
