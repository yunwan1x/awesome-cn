    2020¶

    • VSCode - vscode-R + vscode-r-lsp VSCode R 语言支持
    • gt - 从 R 轻松生成信息丰富、具有出版质量的表格
    • lightgbm - 光梯度提升机.
    • torch - 具有“GPU”加速功能的张量和神经网络.

    2019¶

    • ggforce - ggplot2 扩展框架
    • rayshader - 通过 rgl 进行 2D 和 3D 数据可视化
    • vroom - 快速读取分隔文件

    Integrated Development Environments¶

    集成开发环境

    • VSCode - vscode-R + vscode-r-lsp VSCode R 语言支持
    • RStudio - 适用于 R 的强大且高效的用户界面.在 Windows、Mac 和 Linux 上运行良好.
    • Emacs + ESS - Emacs Speaks Statistics 是一个用于 emacs 文本编辑器的附加包.
    • Sublime Text + R-Box - Sublime Text ⅔ 的附加包.
    • TextMate + r.tmblundle - TextMate ½ 的附加包.
    • StatET - 一个基于 Eclipse 的 R IDE.
    • Microsoft R - Revolution R 将免费提供给学术用户，商业软件将专注于大数据、大规模多处理器功能.
    • R Commander - 提供基本图形用户界面的包.
    • Deducer - 菜单驱动的数据分析 GUI，带有类似数据编辑器的电子表格.
    • Radiant - 一个独立于平台的基于浏览器的 R 业务分析界面，基于 Shiny.
    • Vim-R - R 的 Vim 插件.
    • Nvim-R - 用于 R 的 Neovim 插件.
    • Jamovi 和 JASP - Desktop software for both Bayesian 和 Frequentist methods, using a UI familiar to SPSS users.
    • Bio7 - IDE 包含用于生态建模的模型创建、科学图像分析和统计分析工具.
    • RTVS - 用于 Visual Studio 的 R 工具.
    • RKWard - 用于 R 的可扩展 IDE/GUI.

    Syntax¶

    软件包改变了你使用 R 的方式.

    • pipeR - 多范式管道实现.
    • lambda.r - R 中的函数式编程和简单的模式匹配.
    • purrr - 本着 underscore.js 精神的 R FP 包.

    Data Manipulation¶

    烹饪数据包.

    • tidyr - 使用传播和收集功能轻松整理数据.
    • rlist - A toolbox for non-tabular data manipulation with lists.
    • ff - 旨在存储大型数据集的数据结构.
    • lubridate - 一组处理日期和时间的函数.
    • bigmemory - 共享内存和内存映射矩阵. big* 包提供了额外的工具，包括线性模型（biglm) 和随机森林 (bigrf).
    • fuzzyjoin - 在不精确匹配时将表连接在一起.
    • tidyverse - 从 tidyverse 轻松安装和加载软件包.
    • snakecase - 自动解析字符串并将其转换为蛇或骆驼等案例.
    • DataExplorer - 用最少的代码快速进行探索性数据分析.

    Data Formats¶

    用于读取和写入不同格式数据的包.

    • arrow - Arrow C++ 库的接口.
    • fst - 闪电般快速的 R 数据帧序列化.
    • haven - 改进了在 R 中导入 SPSS、Stata 和 SAS 文件的方法.
    • jsonlite - 在 R 中解析 JSON 文件的强大而快速的方法.
    • qs - R 对象的快速序列化.
    • readxl - 将 excel 文件（.xls 和 .xlsx）读入 R.
    • rio - 用于数据 I/O 的瑞士军刀.
    • readODS - 将 OpenDocument 电子表格作为 data.frames 读入 R.
    • RcppTOML - Rcpp 绑定到 TOML 文件的 C++ 解析器.
    • vroom - 快速读取分隔文件.
    • writexl - 用于 R 的 xlsx 导出器的便携式、轻量级数据框.
    • yaml - R package for converting objects to and from YAML.

    Graphic Displays¶

    用于显示数据的包.

    • ggfortify - 使用一行代码实现 ggplot2 流行统计包的统一接口.
    • ggrepel - 相互排斥重叠的文本标签.
    • ggalt - ggplot2 的额外坐标系、几何图形和统计变换.
    • ggstatsplot - 带有统计详细信息的基于 ggplot2 的绘图
    • ggtree - 系统发育树的可视化和注释.
    • ggtech - ggplot2 技术主题和比例
    • ggplot2 Extensions - ggplot2 扩展的展示.
    • lattice - 强大而优雅的高级数据可视化系统.
    • corrplot - 相关矩阵或通用矩阵的图形显示. 它还包含一些算法来进行矩阵重新排序.
    • rgl - R 的 3D 可视化设备系统.
    • Cairo - R 图形设备使用 cairo 图形库创建高质量的显示输出.
    • extrafont - 在 R 图形中使用字体的工具.
    • showtext - 启用 R 图形设备以使用系统字体显示文本.
    • animation - 在 R 中生成动画图形的简单方法，使用 ImageMagick.
    • gganimate - 使用 ggplot2 创建简单的动画.
    • misc3d - 处理 3d 绘图、等值面等的强大功能.
    • xkcd - 在图表中使用 xkcd 样式.
    • imager - 基于 CImg 库的图像处理包，用于处理图像并显示它们.
    • hrbrthemes - 有意见、以排版为中心的 ggplot2 主题和主题组件.
    • waffle - 在 R 中制作华夫饼（方形饼图）图表.
    • dendextend - 可视化、调整和比较层次聚类树.
    • idendro - 树状图（层次聚类树）的交互式探索.
    • r2d3 - D3 可视化的 R 接口
    • Patchwork - 将单独的 ggplots 组合成同一个图形.
    • plot3D - 绘制多维数据
    • plot3Drgl - 绘制多维数据 - 使用 'rgl'
    • httpgd - R 的异步 http 服务器图形设备.

    HTML Widgets¶

    交互式可视化包.

    • heatmaply - 与 D3 的交互式热图.
    • d3heatmap - 与 D3 的交互式热图（不再维护）.
    • DataTables - 将 R 矩阵或数据框显示为交互式 HTML 表格.
    • dygraphs - 在 R 中绘制时间序列数据.
    • Leaflet - 最流行的 JavaScript 库交互式地图之一.
    • MetricsGraphics - 可以轻松创建 D3 散点图、折线图和直方图.
    • networkD3 - 来自 R 的 D3 JavaScript 网络图.
    • scatterD3 - 带有 D3 的交互式散点图.
    • rbokeh - R 接口 Bokeh.
    • threejs - 交互式 3D 散点图和地球仪.
    • timevis - 创建完全交互式的时间线可视化.
    • visNetwork - 使用 vis.js 库进行网络可视化.
    • wordcloud2 - wordcloud2.js 的 R 接口.
    • highcharter - 基于 htmlwidgets 的 highcharts 的 R 包装器
    • echarts4r - Echarts 第 4 版的 R 包装器

    Reproducible Research¶

    用于文学编程和可重复工作流程的软件包.

    • redoc - 可逆可复制文件
    • tinytex - 轻量级且易于维护的 LaTeX 发行版
    • xtable - 将表格导出为 LaTeX 或 HTML.
    • rapport - R 模板系统.
    • rmarkdown - R 的动态文档.
    • Sweave - 设计用于使用 R 编写 LaTeX 报告的包.
    • texreg - 在 LaTex 和 HTML 中格式化统计模型.
    • checkpoint - 从检查点服务器上的快照安装包.
    • brew - 预先计算数据以增强您的报告模板. 可以与knitr结合使用.
    • officer - 用于生成 Microsoft Word、Microsoft PowerPoint 和 HTML 报告的 R 包.
    • flextable - 用于在 Microsoft Word、Microsoft PowerPoint 和 HTML 报告中嵌入复杂表格（合并单元格、多级页眉和页脚、条件格式）的 R 包. 它与 [officer] 包配合并与 [rmarkdown] 报告集成.
    • bookdown - 使用 R Markdown 创作书籍.
    • ezknitr - 使用“knitr”时避免典型的工作目录痛苦
    • drake - 一个 rOpenSci 可重复数据科学工作流的包太大了 knitr.
    • R Suite - 用于为 R 设计灵活且可重复的部署工作流的包.
    • kable - 使用 'knitr' 中的 'kable()' 构建精美的 HTML 或 'LaTeX' 表格.

    Web Technologies and Services¶

    网上冲浪套餐.

    • Web Technologies List - 有关如何一起使用 R 和万维网的信息.
    • shinyjs - 在几秒钟内轻松改善您的 Shiny 应用程序中的用户交互和用户体验.
    • RCurl - R 的通用网络 (HTTP/FTP/...) 客户端接口.
    • curl - 一个现代灵活的 R 网络客户端.
    • httpuv - HTTP 和 WebSocket 服务器库.
    • XML - 在 R 中解析和生成 XML 的工具.
    • xml2 - 用于在 R 中解析和生成 XML 的优化工具.
    • OpenCPU - 用于 R 处理并发调用的 HTTP API，基于 Apache2 Web 服务器，将 R 代码公开为 REST Web 服务并创建全尺寸、多页 Web 应用程序.
    • Rfacebook - 通过 R 访问 Facebook API.
    • RSiteCatalyst - Adob​​e Analytics 的 R 客户端库.
    • plumber - 将现有 R 代码公开为 Web API 的库.
    • golem - 用于构建生产级 Shiny 应用程序的框架.

    Parallel Computing¶

    并行计算包.

    • parallel - R 从 2.14.0 版开始，其中包括一个新的包并行合并（稍微修改）包的副本 multicore 和 snow.
    • Rmpi - Rmpi​​ 为 MPI API 提供了一个接口（包装器）. 它还提供交互式 R 从属环境.
    • foreach - 并行执行循环.
