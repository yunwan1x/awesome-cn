    Awesome deno

    Awesome Deno ¶

    Deno 是一个简单、现代且安全的 JavaScript 和 TypeScript 运行时，它使用 V8 并使用 Rust 构建.

    这个列表是最好的 Deno 模块和资源的集合.

    Docs¶

    Official Docs¶

    • Official Site
    • Deno API Reference
    • Deno Unstable API Reference
    • Deno Manual

    External Docs¶

    • V8 Docs for Deno
    • DenoBeginner.com - 针对初学者的完全免费的 deno 速成课程.

    Online Playgrounds¶

    • deno.town
    • Deno Playground
    • maman/deno-playground
    • DenoBR Playground
    • myCompiler.io
    • Repl.it

    Modules¶

    注意：Deno 有一些官方模块可以在 deno_std. 考虑提交给 deno.land/x 注册表.

    Automation¶

    • autopilot - Autopilot - 使用 Deno 实现跨平台 Web 自动化.

    CLI utils¶

    • cac - 用于构建命令行应用程序的简单而强大的框架.
    • charmd - 一个简单的、可扩展的 Markdown 渲染器，适用于您的终端.
    • chart - 没有依赖性的控制台 ASCII 折线图.
    • cli-spinner - 在运行长任务时在终端中显示微调器.
    • cliffy - 构建交互式命令行工具的完整解决方案.
    • kia - Deno 的简单终端旋转器
    • terminal_images - 用于在终端中显示图像的 Deno 模块和 CLI 工具.

    Database¶

    • deno_mysql - MySQL 数据库驱动程序.
    • deno_mongo - MongoDB 数据库驱动程序.
    • redis - deno 的 redis 客户端的实验性实现.
    • denodb - 用于 Deno 的 MySQL、SQLite、MariaDB、PostgreSQL 和 MongoDB ORM.
    • dndb - 为 Deno 编写的持久且可嵌入的 NoSQL 数据库引擎.
    • dsddb - 一个使用 JSON 格式的简单持久键值数据库.
    • dso - 基于mysql的简单ORM库.
    • postgres - PostgreSQL 数据库驱动程序.
    • sql-builder - 一个 sql 查询生成器.
    • maxminddb - 一个允许使用 MaxmindDB geoIP 数据库文件的库
    • cotton - 用于 deno 的 SQL 数据库工具包
    • yongo - deno 中的猫鼬 api 子集（如填充）但不会完全复制猫鼬

    Editor framework¶

    • Denops - 使用 Deno 编写 Vim/Neovim 插件的生态系统.

    Frontend development¶

    • postcss - 使用 JS 插件转换样式的工具.

    Frontend framework¶

    • aleph.js - Deno 中的 React 框架，灵感来自 Next.js.

    Image¶

    • ImageScript - JavaScript 中的图像处理，利用 WebAssembly 来提高性能.

    Logging¶

    • gardens - 适用于任何地方的 JavaScript 的有用日志记录实用程序.

    Mail¶

    • deno-smtp - deno 的 smtp 邮件发件人.

    Markdown¶

    • marked - Markdown 到 HTML 转换器.

    String utils¶

    • camelcase - 将破折号/点/下划线/空格分隔的字符串转换为驼峰式：foo-bar → fooBar.
    • deno_case_style - 不同大小写样式的字符串验证器和格式器. 例如：camelCase 等.
    • deno-prettystring - 格式化、修剪和删除字符串中字符之间的多余空格.
    • deno-slugify - deno 的字符串 slugifier.
    • normalize_diacritics - 删除字符串中的重音/变音符号.

    Template engine¶

    • dejs - 用于 deno 的 Ejs 模板引擎.
    • deno_tiny_templates - deno 的模板渲染器.
    • eta - 快速、轻量且可配置的嵌入式模板引擎.
    • handlebars - deno 的把手模板引擎
    • mustache - deno 的 Mustache 模板引擎

    Testing¶

    • deno-pretty-assert - deno 的彩色 assertEqual.
    • deno-puppeteer - 提供高级 API 以通过 DevTools 协议控制 Chromium 或 Chrome 的库.
    • expect - 在 deno 中编写像 expect 测试这样的笑话的助手.
    • merlin - deno 的测试和基准测试框架 ‍♂️
    • pretty_benching - 一个小型库，可让您的 Deno 基准测试进度和结果看起来很漂亮.
    • rhum - Deno 的轻量级测试框架.
    • superdeno - 用于测试 Deno HTTP 服务器的超级代理驱动库.
    • superoak - 通过 SuperDeno 使 Oak 的 HTTP 断言变得容易.
    • tincan - 一个类似 Jest 的 Deno 测试库.
    • unexpected - 可扩展的 BDD 断言工具包.

    Utils¶

    • buckets - 将资产和脚本捆绑在一个可执行文件中.
    • bytes_formater - 格式化字节（Uint8Array、ArrayBufferView 等）输出，在调试 IO 函数时很有用.
    • coffee - Deno 配置 - 一种类型安全、易于使用的 Deno 配置管理器.
    • computed_types - Joi 喜欢 Typescript 和 Deno 的验证器.
    • dcc - Deno Cache Clean，下次运行时重新加载 deps.
    • denon - 带有等待生成器的文件观察器.
    • deno-config - 通过 cli、.env 和 json 文件简化 deno 应用程序配置管理的实用程序
    • deno-context - 将截止日期、取消和其他请求范围的值传播到多个承诺. 这种行为就像 Go 的上下文.
    • deno_cron - 一个 cron 作业调度程序，允许您编写具有大量灵活性的人类可读的 cron 语法
    • deno-deamon - 使 Deno 程序在后台运行. --deno-dotenv
    • pietvanzoen/deno-dotenv - deno 的 Dotenv 操作.
    • cardosomarcos/deno-dotenv - 从 .env 为 Deno 项目加载环境变量.
    • deno-fnparse - 一个非常简单的 JavaScript 解析器组合器.
    • deno-globrex - globrex 到 deno，glob 到正则表达式的端口.
    • deno_notify - 在所有平台上发送桌面通知.
    • deno-opn - 打开网站、文件、可执行文件等内容. 跨平台.
    • deno-plugin-prepare - 用于管理 Deno 原生插件依赖项的库.
    • deno_random_interval - 生成随机间隔的助手.
    • deno_tokenizer - deno 的简单标记器.
    • deno-using - 带有 deno 语句的 Python 风格.
    • dinoenv - 使用 deno 管理环境变量的小库.
    • ensure - 确保您运行的是最低版本的 Deno、Typescript 或 V8.
    • evt - 为 EventEmitter 键入安全替换.
    • fossil - 值类型验证套件.
    • garn-yaml - 读取或写入 yaml 内插环境变量.
    • garn-validator - 轻松创建验证.
    • invert-kv - 在 Deno 中反转键值对.
    • lazy - 类似 linq 的惰性评估迭代模块.
    • maze_generator - 用于生成、解决、分析和显示迷宫的 Javascript 模块.
    • ms - 轻松将各种时间格式转换为毫秒.
    • online - 检查您当前是否在 Deno 在线.
    • optionals - 类似 Rust 的错误处理和具有详尽模式匹配的选项.
    • qrcode - Deno 的二维码图像生成器.
    • recursive-readdir - 在 Deno 中递归读取目录.
    • rubico - [a]同步函数组合； 它只是有效.
    • solc - Deno 的 Solidity 绑定.
    • type-fest - 基本 TypeScript 类型的集合（sindresorhus/type-fest 的端口）.
    • wasm-gzip - 为 Deno 加密和解密 gzip.
    • watch - 文件观察者.
    • wu-diff-js - 使用 wu(O(NP)) 算法计算两个切片之间差异的差异库.

    Web framework¶

    • abc - 一个更好的 Deno 框架来创建 Web 应用程序.
    • alosaur - Alosaur - 具有许多装饰器的 Deno Web 框架.
    • attain - Deno 的中间件 Web 框架，它使用受 express 和 Oak 启发的 http 标准库. 通过适当的内存使用快速而稳定.
    • aqua - Deno 的最小且快速的 Web 框架.
    • deno-express - Node Express 移植到 Deno.
    • denotrain - 多合一的 Web 框架，如具有中间件支持的 Node.js 的 express 或 fastify.
    • dinatra - Sinatra 喜欢 deno 的轻量级 Web 应用程序框架.
    • doa - Deno 的 http 服务的中间件框架. 用❤️从Koa移植过来
    • drash - Deno 的 HTTP 服务器的 REST 微框架，具有零依赖性.
    • dragon - ⚡强大的 HTTP 路由器和 URL 匹配器，用于使用 Dragon 构建 Deno Web 服务器.
    • microraptor - 轻量级框架，可通过验证轻松实现网络路由.
    • oak - Deno 网络服务器的中间件框架.
    • oak-http-proxy - Deno Oak HTTP 服务器的代理中间件.
    • opine - 从 ExpressJS 移植的快速、简约的 Web 框架.
    • opine-http-proxy - Deno Opine HTTP 服务器的代理中间件.
    • pogo - Deno 的服务器框架.
    • servest - 渐进式 HTTP 服务器/路由器.

    WebSocket¶

    • deno-websocket - 一个简单的 WebSocket 库，如 node.js 库的 ws.
    • dropper - 自定义基于事件的 WebSockets 框架，用于在 Deno 上构建实时应用程序
    • websocket_server - 一个 WebSocket 服务器库.
    • sockets - Deno 的 WebSocket 库.

    Web utils¶

    • compression - Deno HTTP 压缩中间件.
    • djwt - 基于 JWT 和 JWS 规范在 Deno 上制作 JSON Web 令牌 (JWT).
    • forwarded - forwarded 库的 Deno 端口.
    • gentleRpc - 适用于 Deno 和浏览器的 JSON-RPC 2.0 TypeScript 库.
    • gql - 通用 GraphQL HTTP 中间件.
    • graphql-tag - 来自模板文字的 GraphQL 模式 AST.
    • nats - 一个 Deno 客户端 NATS messaging system.
    • obsidian - 原生 GraphQL 缓存客户端和服务器模块.
    • qs - 具有嵌套支持的查询字符串解析器.
    • router - 一个高性能的基本路由器可以在任何地方工作.
    • status - Deno 的 HTTP 代码和状态实用程序.
    • ts-prometheus - 普罗米修斯客户端.
    • up - 检查网站是否在 Deno 中启动.
    • youtube-deno - 用于与 YouTube 进行任何交互的 YouTube 数据 API 的 Deno 客户端库.

    Webview¶

    • webview - webview 的 Deno 绑定，一个用于创建基于 web 的桌面 GUI 的小库.

    XML¶

    • deno-xml-parser - 从 segmentio/xml-parser 移植的 Deno XML 解析器.
    • sax-ts - SAX 风格的 XML 解析器移植自 sax-js.

    Registries¶

    • crux.land - 用于托管小型 (<10kB) 单个 deno 脚本的免费注册服务.
    • Deno PKG - 在 Deno 项目中使用来自 GitHub 的代码的更简单方法.
    • deno.land/x/ - 官方的第 3 方模块注册表.
    • nest.land - 一个不可变的、由区块链驱动的 Deno 包注册表.

    Showcases¶

    • AuthCompanion - 基于令牌的用户管理服务器.
    • Deno Rest - deno RESTful apis 的样板.
    • Deno Seed - 用于开发的完整样板. ：幼苗：
    • GitHub Profile Trophy - 在自述文件中添加动态生成的 GitHub Trophy
    • UsingDeno - 使用 Deno 精选的 Web 应用程序和项目列表.

    Tools¶

    • commands - 为 node js 和 deno 创建命令快捷方式
    • clone - 方便克隆的简单实用程序.
    • decense - 使用一个 deno run 命令生成许可证.
