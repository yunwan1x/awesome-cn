    Awesome KDE ¶

    >精选的KDE桌面环境应用程序，扩展，模块，主题和工具列表.

    KDE 是一个免费软件社区，致力于创建开放和用户友好的计算体验，它提供高级图形桌面，用于通讯，工作，教育和娱乐的各种应用程序以及可轻松构建新应用程序的平台.

    >标有（图钉）的项目是在KDE自己的基础结构上托管的项目，由KDE社区开发.

    Applications¶

    Internet & Networking¶

    • Falkon -基于QtWebEngine的跨平台Web浏览器.
    • Konqueror -使用KHTML渲染引擎（和可选的Webkit），因此它支持最新的Web标准，例如HTML5，JavaScript，CSS3等.
    • Qt WebBrowser -Qt WebBrowser是使用Qt和Qt WebEngine功能开发的嵌入式设备的浏览器.
    • Otter Browser -Otter Browser旨在使用Qt5重新创建经典Opera（12.x）UI的最佳方面.
    • Firefox KDE -来自mozilla.org的具有OpenSUSE补丁的独立Web浏览器，可以更好地与KDE集成.
    • KGet -多功能且用户友好的下载管理器.
    • KTorrent -一个BitTorrent应用程序，可让您使用BitTorrent协议下载文件.
    • qBittorrent -基于Qt工具箱和libtorrent-rasterbar的高级C ++客户端程序.
    • KMail -KMail是Kontact（KDE的集成个人信息管理器）的电子邮件组件.
    • KDE Connect -在所有设备上进行通信的项目：在台式计算机上接收电话通知，通过电话控制桌面等.
    • GSConnect -KDE Con​​nect的完整实现，特别是与Nautilus，Chrome和Firefox集成的GNOME Shell.
    • Plasma-browser-integration -一系列新功能 Firefox, Chrome 和基于Chromium的浏览器可在您的桌面上使用.
    • KRDC -客户端应用程序，允许您查看或控制另一台运行兼容服务器（VNC＆RDP）的计算机上的桌面会话.
    • Konversation -建立在KDE平台上的用户友好型Internet中继聊天（IRC）客户端.
    • Akregator -使您能够关注新闻站点，博客和其他启用RSS / Atom的网站.
    • KNetworkManager -后端NetworkManager软件的KDE客户端.
    • Smb4K -高级网络邻居浏览器和Samba共享安装实用程序.

    Graphics & Imaging¶

    • Gwenview -KDE提供的一种快速且易于使用的图像查看器.
    • Okular -通用文档查看器.
    • Spectacle -用于捕获桌面屏幕截图的应用程序.
    • KolourPaint -一个简单的绘画程序，可以快速创建光栅图像. 它可用作修饰工具和简单的图像编辑任务.
    • Krita -为概念艺术家，插画家，遮罩和纹理艺术家以及VFX行业设计的绘画工具.
    • digiKam -具有开源功能的专业照片管理.
    • KColorChooser -一个简单的应用程序，可以从屏幕或调色板中选择颜色.

    Multimedia¶

    • Amarok -Amarok是一款功能强大的音乐播放器，适用于Linux，Unix和Windows，界面直观.
    • Kdenlive -使用MLT视频框架的Linux非线性视频编辑器.
    • SMPlayer -带有内置编解码器的媒体播放器，几乎可以播放所有视频和音频格式.
    • Elisa -由KDE社区开发的音乐播放器，力求简单易用.
    • KMix -声音通道混音器和音量控制.
    • KWave -基于KDE框架5的声音编辑器.

    Office & Productivity¶

    • Calligra -包括高效且功能强大的办公组件.
    • Kate -KDE提供的多文档，多视图的文本编辑器.
    • KWrite -KDE的文本编辑器，基于KDE Frameworks编辑器组件.
    • Kronometer KDE的秒表应用程序.
    • RSIBreak -RSIBreak可用于提醒您不时休息.

    System Administration¶

    • Dolphin -KDE桌面中包含的文件管理器.
    • Discover -适用于终端不是用于安装应用程序和附件或更新的最佳解决方案的用户和发行版.
    • Konsole -KDE桌面中包含的终端仿真器.
    • Yakuake -基于KDE的Konsole技术的下拉终端仿真器.
    • KDiskFree -显示可用的文件设备（硬盘驱动器分区，驱动器等），以及有关它们的容量，可用空间，类型和安装点的信息.
    • KInfoCenter -提供有关计算机系统的详细信息的实用程序.
    • KSystemLog -系统日志查看器工具.
    • Sweeper -清除程序有助于清除用户在系统上留下的不需要的痕迹，并重新获得磁盘空间，以删除未使用的临时文件.
    • Kup -KDE等离子桌面的备份调度程序.
    • Krusader -KDE的高级双面板（指挥官风格）文件管理器.

    Utilities & Tools¶

    • Ark -存档工具.
    • KFind -由KRunner或您的菜单启动的独立搜索工具. 它还作为“工具”菜单中的“查找文件”集成到Konqueror中.
    • KFontview -用于查看和安装不同类型字体的应用程序.
    • KDE Wallet Manager -管理KDE系统上的密码. KDE钱包子系统允许对与KDE钱包集成的每个应用程序进行访问和密码管理.
    • KeePassXC -跨平台社区驱动的Keepass密码管理器端口.
    • QtPass -的多平台GUI pass.
    • Android File Transfer -具有简约UI的可靠MTP客户端.
    • KRename -重命名多个文件的实用程序.
    • KWin-lowlatency -尝试减少在KDE中使用的流行KWin合成器中的延迟和卡顿现象.
    • KWinFT -Wayland合成器和X11窗口管理器.
    • Yin-Yang -Linux的自动夜间模式.
    • fancontrol-gui -风扇控制的GUI，它是lm_sensors的一部分.
    • Wacom GUI -KDE的Python / PyQt Wacom GUI.

    Development & Programming¶

    • KDevelop -适用于C，C ++，Python，QML / JavaScript和PHP的跨平台IDE.
    • Kompare -差异/补丁前端.
    • Qt Designer -Qt工具，用于使用Qt小部件设计和构建图形用户界面（GUI）.
    • Umbrello -UML Modeller是基于KDE技术的统一建模语言（UML）图程序.
    • Okteta -十六进制编辑器.

    Education¶

    • KTurtle -一种教育性编程环境，旨在使学习编程变得尽可能容易.
    • LabPlot -交互式图形和科学数据分析.

    Customization¶

    Plasma desktop¶

    Extensions¶

    • Latte-Dock -Latte是基于Plasma框架的扩展坞，可为您的任务和等离子体提供优雅而直观的体验.
    • Kvantum -用于Qt4 / Qt5，KDE和LXQt的基于SVG的主题引擎，重点是优雅，可用性和实用性.
    • Window Title Applet -Plasma 5小程序，可在面板中显示当前窗口标题和图标.
    • Hoppla -KDE Plasma桌面小部件，用于控制Philips Hue灯.
    • Places Widget -Plasma 5小部件，可访问用户位置.
    • Window AppMenu Applet -等离子5小程序，以显示窗口菜单.
    • Dynamic Wallpaper Engine -一个墙纸插件，可根据您所在位置的当前时间不断更新桌面背景.

    KCM Modules and Systems¶

    • kde-gtk-config -用于KDE的GTK2和GTK3配置器.
    • kcm-wacomtablet -Wacom Linux驱动程序的KDE GUI.
    • kded_rotation -小巧的KDED模块，用于处理平板电脑和混合笔记本电脑上的自动屏幕旋转.

    Themes¶

    • Adapta KDE -流行的GTK主题Adapta for Plasma 5桌面的端口，其中包含一些附加功能.
    • Arc KDE -流行的GTK主题Arc for Plasma 5桌面的端口，其中包含一些附加功能.
    • Materia KDE -流行的GTK主题Plasma 5桌面材料的端口，其中包含一些附加功能.
    • Helium -等离子5的半透明主题.
    • Breath -Manjaro Linux的呼吸主题（等离子5）.
    • Hello -KDE Plasma桌面的各种颜色方案，主题，图标及更多内容的汇编. 它不受限制，让您负责.
    • McMojave -适用于KDE Plasma桌面的MacOSX Mojave主题.
    • Canta -KDE Plasma桌面的平面设计主题.
    • Darkine KDE -适用于KDE Plasma 5桌面，SDDM等的主题.
    • Qogir -KDE Plasma桌面的平面设计主题.
    • Layan -KDE Plasma桌面的平面设计主题.
    • ChromeOS -KDE Plasma桌面的主题设计主题.

    Qt and GTK+ Applications Appearance¶

    • breeze-gtk -为配合KDE的微风而打造的GTK主题.
    • Breeze10 -KDE Plasma的Windows 10风格窗户装饰.
    • SierraBreeze -用C ++编写的类似OSX的KDE Plasma窗口装饰.
    • Akava-Kv -面向用户的KDE主题Kvantum.
    • BreezeBlurred -用Qt C ++编写的KDE Breeze窗口装饰的分支.
    • BreezeEnhanced -KDE Breeze装饰叉，带有其他选件.
    • KDE-Rounded-Corners - 圆角化您的窗户.
    • Sweet KDE -KDE等离子的黑暗现代主题.
    • XBoomer -用于KDE等离子的XP窗户装饰.
    • Chrome-KDE5BreezeDark -使用Breeze Dark窗户装饰时，在KDE5上使用Chrome的主题.

    Icons¶

    • Papirus -材质图标主题，最初基于Paper.
    • Antu -专为Plasma Desktop设计的平滑图标主题.
    • Zafiro -采用扁平设计技术制作的简约图标，使用褪色并始终伴有白色. 首要任务是简单.
    • La Capitaine -图标包，灵感来自macOS和Google的Material Design的最新版本.
    • Tela -一个扁平的彩色设计图标主题.
    • Flatery -平面样式的Linux图标主题.
    • Candy -甜蜜的渐变图标.

    Cursors¶
