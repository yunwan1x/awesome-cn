    真棒榆树


    社区驱动的有用 Elm 教程、库和软件列表. 灵感来自 awesome 列出东西. 请随时改进此列表.

    -贡献指南

    Learn¶

    了解这个很棒的东西是什么.

    • Official tutorial — 一般信息和带示例的深入指南.
    • Elm in Action — 适合 Elm 初学者的深入书籍，来自 Manning Publications.
    • Architecture Tutorial - 如何创建模块化 Elm 代码，可以很好地与您的应用程序一起扩展.
    • Exercism Elm Track - 榆树练习集.
    • Learn you an Elm - 带有详尽示例和说明的 Elm 教程.
    • Beginning Elm - 对 Elm 编程语言的温和介绍.
    • Elm patterns - Elm 中常见模式的集合.
    • Elm Koans - 练习学习榆树.
    • Learn Elm - 探索美妙的编程语言，让前端 Web 应用程序成为构建和维护的乐趣！
    • Learn Elm in Y Minutes - 语法和功能概述. 榆树页面 learnxinyminutes.com
    • Elm Maybe - Dealing with null/Nothing - 使用 Maybe 类型，带有很好的注释代码示例.
    • Programming Elm - 来自 The Pragmatic Programmers 的详尽书籍，涵盖了基础知识和高级概念.
    • Elm cheat sheet - 语法和功能概述.
    • Elm Tutorials on Codementor - 关于使用 Elm 构建 Web 应用程序的两个教程.
    • Elm programming language - Elm 作为编程语言的简要概述.
    • Elm: A Beginners' Guide to Elm and Data - Elm 和数据的初学者课程
    • Practical Elm for a Busy Developer - 一本关于开发 Elm 应用程序的实践方面的非初学者书籍.
    • Haskell to Elm - 收集 Elm 与 Haskell 不同之处的示例，针对来自 Haskell 背景的 Elm 初学者.
    • A nice app on Elm street - 榆树简介
    • The Elmish Book 构建 Web 应用程序 F# 从第一原则遵循榆树架构.
    • Elm patterns - 编码模式的集合.
    • Codings hints - Evan 的自述文件列表.
    • Ellies catalog - Ellie 中的小例子的集合.
    • Elm Cookbook - 一本关于榆树的电子书.
    • Awesome Elm PLTD - Elm 编程语言理论和开发的有用资源.

    Outdated Tutorials and books (Elm 0.18 or earlier)¶

    • Elm: Building Reactive Web Apps - 了解如何使用 Elm 构建响应式 Web 应用程序.
    • Writing native - 了解如何为 Elm 创建原生 JavaScript 模块.
    • Elm: Functional frontend development - 有关基础知识和高级主题的系列文章.
    • A Concise Introduction to Elm - 对语言特性的高度概括.
    • Elm Seeds - 简短的截屏视频，教您 Erik Person 的 Elm 编程语言.
    • Elm For Beginners - Video Course - 构建您的第一个 Elm Web 应用程序.
    • Single-Page Web Apps in Elm - Elm 的五部分教程.
    • Elm FAQ - 榆树常见问题解答 Elm Community.
    • Elm Tutorial by Auth0 - 关于在 Elm 中构建应用程序从身份验证到调用 API 的教程.

    Articles¶

    阅读要点. 查看 Elm 官方博客： elm-lang.org/blog

    Why Elm?¶

    • Side-effects of Elm in production - 来自 Bellroy 的体验报告

    • Blazing Fast HTML - Elm 中的虚拟 DOM.
    • Move fast and don’t break things. Running a startup on Elm - 瑞典初创公司对 Elm 开发的一些想法.

    Miscellaneous articles¶

    • Learning FP the hard way - Elm 语言的经验.
    • Blog of Brian Hicks - 包含有关 Elm 的各种主题的博客.

    Outdated articles (Not relevant for current Elm architecture)¶

    • Getting Started with Elm - Elm 教育教程系列.
    • Elm & Guarantees - 现实地看待 Elm 的位置，并不优于其他选项.

    Code generators¶

    • HTML to Elm - 将 HTML 转换为 Elm Html. 将应用程序移植到 Elm 时很有用.

    • Elm TypeScript Interop - 从 Elm 生成 TypeScript 定义（Elm TS Interop 是它的改进版本）.
    • Quicktype - 从 JSON 生成 JSON 解码器和编码器
    • JSON Schema to Elm - 从 JSON 模式规范生成 Elm 类型、JSON 解码器、JSON 编码器和模糊测试
    • Elm Bridge - 从 Haskell 生成 Elm 类型
    • haskell-to-elm - 从 Haskell 类型生成 Elm 类型、编码器和解码器
    • PostCSS Elm Tailwind - 在你的榆树上放一些顺风

    Libraries¶

    您可以在以下位置找到数百个高质量的软件包：

    • Elm Search - 在 Elm 文档中搜索运算符、函数签名等.

    Boilerplates¶

    新 Elm 项目的良好起点.

    • create-elm-app - 无需构建配置即可创建 Elm 应用程序.
    • elm-boil - 命令行实用程序，用于创建易于运行、构建和部署的 Elm 样板项目
    • elm-live - A flexible dev server for Elm. Live reload included.
    • elm-webpack-4-starter - Elm webpack 4 入门模板.
    • example-elm-hot-webpack - 示例展示了 Elm 0.19 和 Webpack 的热模块重新加载
    • Elm Batteries - Elm、Parcel、Cypress 和 Netlify 的项目模板和生成器
