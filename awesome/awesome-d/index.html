    Official Website¶

    官方网站网址.

    • dlang.org - D 的官方网站.
    • wiki.dlang.org - D的官方维基.
    • code.dlang.org - D 的官方库/模块注册表.
    • Github Organization - 所有官方 D 工具和代码的 D. Repo 官方 GitHub 组织.
    • forum.dlang.org - 官方论坛. 许多有趣的讨论每天都在发生.
    • blog.dlang.org - 官方博客.
    • Language Specification -D 编程语言规范.
    • Issue tracking - D 的官方问题跟踪/报告系统.如果您发现 D 编译器和/或库中的错误，请来报告它们！

    Getting Help¶

    当你被卡住时.

    • Official D Forum Learn Group - 回答 D 问题的最高流量站点.
    • D on Stack Overflow - 流量比论坛少，但可能更容易搜索.
    • D on Rosetta Code - 如何在 D 中做许多基本事情的例子.

    People¶

    使 D 成为语言的人.

    • Walter Bright - D. Walter Bright 的父亲是 D 编程语言的创建者和第一个实现者，并为其他几种语言实现了编译器.
    • Andrei Alexandrescu, PhD - C++ 大师. The D Programming Language 和 Modern C++ Design 的作者. Andrei 与 Walter Bright 共同设计了 D 的许多重要特性，并编写了 D 的大部分标准库. Andrei 是高级 C++ 编程和算法方面的培训师，现在正在组织中积极宣传 D.
    • Átila Neves - Deputy Leader of D.
    • 你 - 如果你在 D 中做过一些有趣的事情，请添加你的信息.正是你，让 D 变得很棒的人.

    Events¶

    • DConf - D 语言名人就与 D 语言及其生态系统相关的一切交流知识、见解和灵感的首要活动.

    Organizations¶

    D 相关书籍. 您可以在 Books D 维基页面. * TDPL - * D 编程语言 * 由 Andrei Alexandrescu 编写. * Programming in D - Ali Çehreli 关于 D 编程的非常详细的书，涵盖了语言的许多领域. 有免费的在线版本，适合初学者. * D Cookbook - 一个包含实际任务的配方包装参考指南，这些任务被简明地解释为开发和扩展用户使用 D 编程语言的能力. 作者：Adam D.     Package Management¶

    用于包和依赖管理的库.

    • code.dlang.org - 官方 D 库存储库. 由配音支持.
    • dub - D 的官方包和构建管理系统.

    Compilers¶

    从源代码编译软件.

    • dmd - D 编程语言的参考编译器. 稳定，构建速度非常快，非常适合学习和快速原型设计/开发. 目前前端用 D 实现，在 dmd、ldc 和 gdc 之间共享，后端用 C++ 实现.
    • ldc - 基于 LLVM 的 D 编译器. 使用 DMD 前端和 LLVM 后端. 构建速度比 dmd 慢，但生成的代码比 DMD 更优化. 它支持LLVM的所有目标平台.
    • gdc - GNU D 编译器. 使用 DMD 前端和 GCC 后端. 由于使用 GCC，目前针对大多数平台. 在大多数情况下，生成的代码运行速度比 DMD 快，与 LDC 相当. 在与官方 GCC 工具链集成的过程中.
    • sdc - 愚蠢的 D 编译器. 写在 D. 每天都变得更聪明.
    • dil - D 编程语言的编译器. 写在 D.

    用于提高 D 开发效率的工具.

    • dfmt - D 源代码的格式化程序

    从源代码管理项目和编译软件.

    • scons-d - 由于 Russel Winder，Scons 内置了对构建 D 项目的支持.
    • premake - Premake 内置了对 D 项目的支持
    • reggae - D 中的元构建系统
    • Makefile - D 项目的 Makefile 模板
    • cmake-d - CMake D 项目
    • cook2 - 用于 D 中项目的快速增量构建工具
    • button - 一个通用的构建系统，只需按一下按钮即可构建您的软件.
    • wild - 狂野的构建系统，用于构建 PowerNex 核心
