    音乐

    Awesome Music ¶

    很棒的音乐库、工具、框架和软件的分类集合.

    总是欢迎贡献和建议！ 请看一下 contribution guidelines and quality standard 第一的.

    谢谢大家 contributors，你太棒了，没有你就不可能！

    Audio Editing¶

    • Audacity - 免费的跨平台数字音频编辑器.
    • Ardour - 一个强调录音的跨平台数字音频工作站.
    • LMMS - 另一个跨平台的数字音频工作站，更倾向于制作节拍.

    Audio Libraries¶

    • audio.js - audio.js 是一个嵌入式 javascript 库，允许在任何地方使用 HTML5 的 <audio> 标签.
    • howler.js - 现代网络的 Javascript 音频库.
    • JSyn - 适用于 Java 的音频合成软件 API.
    • Librosa - 用于音乐和音频分析的 python 包，提供创建音乐信息检索系统所需的构建块.
    • midi.js - 用于创建基于 Web 的 MIDI 应用程序的框架.
    • mxml - MusicXML 解析和布局库.
    • PortAudio - 一个免费、跨平台、开源的音频 I/O 库.
    • RustAudio - 用 Rust 编写的免费且有用的音频、DSP 和音乐库的集合.
    • Soundpipe - 一个用 C 编写的轻量级 DSP 库.

    Audio Tools¶

    • Beets - 强大的命令行音乐组织器和操纵器.
    • Cecilia - 用于音乐和信号处理的基于 CSound 的图形环境.
    • cyanrip - 以最少的用户需要翻录和编码标准音频 CD. 跨平台.
    • fre:ac - 免费的音频转换器. 它支持音频 CD 翻录和标签编辑.
    • Jack - 命令行 CD 开膛手.
    • K3b - 是一个功能齐全的 CD/DVD/蓝光刻录和翻录应用程序.
    • Koel - 开源个人音频流服务器.
    • Lame - 高质量、免费和开源的 MP3 编码器.
    • lltag - 自动命令行 mp3/ogg/flac 文件标记器和重命名器.
    • matchering - 用于自动参考音频母带制作的容器化 Web 应用程序和库.
    • MusicBrainz - 打开音乐信息和元数据百科全书，为所有内容提供唯一ID
    • Musikernel - 未来派音频制作平台（硬件支持、API、主机、插件等）
    • Rubyripper - 是一种用于类 Unix 操作系统的安全数字音频提取应用程序（“cd ripper”）.
    • SoX - 一个免费的开源命令行实用程序，用于播放、转换和操作音频文件.
    • split2flac - 将 flac/ape/wv/wav + 提示表拆分为单独的轨道
    • Whipper - Python CD-DA 开膛手更喜欢准确性而不是速度.
    • X Audio Copy - GTK 和 GNOME 应用程序，用于以有损和无损音频格式翻录 CD 音频和编码.

    Free Music Scores¶

    • cellist/Lilypond-Sheet-Music - 用 lilypond 排版的音乐集（主要是室内乐和大提琴音乐）.
    • ciconia/music - 主要是巴洛克音乐的集合，JS Bach 的许多神圣作品，用 lilypond/ripple 编码.
    • CPDL - 合唱公共领域库包含超过 20K 的免费声乐乐谱.
    • Éditions Nicolas Sceaux - 巴洛克乐谱（主要是法国音乐），使用 lilypond 排版.
    • IMSLP - 国际乐谱库项目是网络上最大的公共领域乐谱和零件集合.
    • Mutopia - 基于公共领域版本的乐谱.
    • wbsoft/lilymusic - 在免费许可下，美丽的 LilyPond 得分.

    MIDI Tools & Libraries¶

    • Midifile - 用于解析标准 MIDI 文件的 C++ 库.
    • Timidity - 一个播放 MIDI 文件的开源命令行合成器.

    Music Notation¶

    • Abjad - 基于 Lilypond 的用于形式化分数控制的 Python API.
    • Denemo - 基于 Lilypond 的免费乐谱编辑器.
    • Digital Music Stand - 用于显示乐谱的免费网络应用程序.
    • Jan Angermüller's music fonts page - 不同音乐字体的列表和视觉比较.
    • Frescobaldi - 免费的 Lilypond 乐谱编辑器.
    • Guido - 用于乐谱图形渲染的通用便携式库和 API.
    • Hacklily - 基于 LilyPond 的在线乐谱编辑器（可选的 GitHub 集成）.
    • i-score - 具有 DAW 功能的交互式乐谱软件.
    • Inknote - 免费、开源、基于浏览器的乐谱和作曲软件.
    • LibMEI - 用于读取和写入 MEI 文件的 C++ 库.
    • Lilybin - 基于 Web 的开源 LilyPond 编辑器.
    • Lilypond - 免费的开源音乐雕刻程序，致力于制作最高质量的乐谱.
    • Lilypond cook book - 充分利用 lilypond 的提示和技巧.
    • Lilypond Snippet Repository - Lilypond 片段想法、功能和技巧的在线数据库.
    • Lilyvm - Lilypond 版本管理器.
    • Lydown - 基于 Lilypond 的现代音乐符号语言.
    • Lyp - Lilypond 瑞士军刀 - 为高级用户轻松安装软件包、管理 Lilypond 的多个版本和其他工具.
    • MuseScore - 免费的开源乐谱和作曲软件.
    • MusicKit - 适用于 iOS 和 OSX 的乐谱渲染.
    • NOTATIO - 一个致力于乐谱实践的论坛.
    • OpenLilyLib - LilyPond 工具 - 片段、模板和扩展.
    • OpenSheetMusicDisplay - 用于在浏览器中呈现 MusicXML 的 Typescript 库.
    • Python-ly - 用于操作 LilyPond 文件的工具和库.
    • Ripple - DRY for Lilypond - 以最小的麻烦生成分数和部分.
    • Scorelib - 用于解析 SCORE 数据文件的 C++ 库.
    • Tbon - Typographic Beat-Oriented Notation for music.
    • Tunefl - LilyPond 迷你乐谱雕刻和音乐家分享服务.
    • Verovio - 用于将 MEI 乐谱雕刻成 SVG 的库和工具包.
    • Vexflow - 用于呈现乐谱和吉他指法的 JavaScript 库.

    Music Programming¶

    • Alda - 音乐家的音乐编程语言.
    • Bach: Automated Composer's Helper - Max 的一组跨平台补丁和外部组件，旨在将丰富的计算机辅助合成带入实时世界.
    • Chuck - 强定时、并发和动态音乐编程语言.
    • CoffeeCollider - 一种用于 HTML5 中实时音频合成和算法组合的语言. “编写 CoffeeScript，并作为 SuperCollider 进行处理.”
    • Common Music - 一个开源音乐创作系统.
    • Csound - 用户可编程和用户可扩展的声音处理语言和软件合成器.
    • Faust - 一种专为实时信号处理和合成而设计的函数式编程语言.
    • IanniX - 基于 Iannis Xenakis 作品的图形开源音序器，用于数字艺术.
    • Inscore - 用于设计交互式增强音乐乐谱的环境.
    • Leipzig - Overtone 的合成库.
    • Midica - 带有反编译器和播放器的 MIDI 编程语言，支持卡拉 OK.
    • music21 - 计算音乐学工具包.
    • Music Suite - 一种基于 Haskell 的通用音乐符号和理论语言.
    • Nyquist - 一种声音合成和合成语言.
    • OpenMusic - 基于 Lisp 的可视化编程语言.
    • Orca - 音乐的现场编程环境.
    • Overtone - 用于设计合成器和与音乐合作的开源工具包.
    • Platonic Music Engine - 一个用 Lua 编写的开源音乐生成框架.
    • Pure Data - 一种用于音频和其他多媒体的可视化编程语言.
    • py-modular - 一个模块化和实验性的编程环境，具有 Python 中的基本 DSP 例程.
    • Sonic Pi - 强调教育用途的实时编码合成器.
    • Sporth - 基于小堆栈的音频语言.
    • SuperCollider - 一种用于实时音频合成和算法组合的编程语言.
    • Tidal - 一种基于 Haskell 构建的用于实时编码模式的语言.

    Music Theory Teaching¶

    • Artusi - 互动音乐理论练习.
    • Four Score and More - 根据乐谱量身定制的乐理练习.
    • musictheory.net - 课程和练习
    • OpenMusicTheory - 不断增长的在线音乐理论和听觉技能“教科书”.
    • Theorytab - 具有和弦功能的歌曲数据库.

    Standards¶

    • MEI (Music Encoding Initiative) - 一项开源工作，用于定义以机器可读结构对音乐文档进行编码的系统.
    • MusicXML - 用于交换数字乐谱的标准开放格式.
    • SMuFL (Standard Music Font Layout) - 将音乐符号映射到 Unicode 以用于音乐字体的规范.
