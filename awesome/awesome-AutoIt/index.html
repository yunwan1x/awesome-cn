    AutoIt

    Awesome AutoIt¶

    >精选的UDF精选清单，示例脚本，工具和有用的资源 AutoIt.

    由...制作 Juno_okyo . 受启发 awesome 列出的东西.

    UDF/Libraries¶

    库列表user defined functions （UDF）. 编写这些库是为了使它们易于集成到您自己的脚本中，并且对于任何程序员来说都是非常宝贵的资源.

    Automation¶

    • Active Directory (by water) -广泛的库，用于控制和操作Windows活动目录. 链接到文档页面.
    • GTK+ (by prazetto) -GTK +框架| 小部件.
    • IUIAutomation MS framework (by junkew) -IUIAutomation MS框架可自动执行chrome，FF，IE等
    • Java (by seangriffin) -在您的应用程序和Java应用程序之间创建访问桥. 允许您自动化一些Java应用程序.
    • SAP (by seangriffin) -SAP业务管理自动化.
    • SAPWizard (by ozmike) -SAPWizard UDF.
    • Service (by arcker) -使用AutoIt代码构建自己的服务.
    • WiFi (by MattyD) -对您的无线局域网进行低级控制.
    • AutoItMonkey -AutoItMonkey是一种UDF，可在您的Android模拟器或设备上运行，并生成用户事件的伪随机流，例如单击，触摸或手势，以及许多系统级事件.
    • ImperiusAutoIt -Imperius AutoIt是 ImperiusGeorge 支持本地Android应用程序的移动测试自动化客户端.
    • 3270 (by Worlor) -用于通过AutoIT自动执行3270终端的库

    Browsers¶

    • Chrome (by seangriffin) -与上述Google Chrome浏览器相同. 使用Chrome UDF自动执行Chrome中最常见的任务.
    • Firefox (by Stilgar) -对自动化的支持比IE少，但仍然非常好.
    • FireFox Profile Backup 1.0 (by careca) -备份或还原您的firefox配置文件，应用程序将在主驱动器中搜索该配置文件，这将是备份文件.
    • _FF_AutoLogin (by Stilgar) -这是FireFox和FF.au3的自动登录功能和登录功能生成器.
    • _FF_DM (by Stilgar) -UDF控制FireFox下载管理器（而不是下载窗口）.
    • _FF_FoxBox (by Stilgar) -UDF使用FF.au3控制Fox！Box（AVM FRITZ！Box的Mozilla Firefox扩展）.
    • _FF_Screengrab (by Stilgar) -UDF控制Screengrab！ （FireFox-AddOn）与FF.au3.
    • _HTML (by Stilgar) -从任何HTML（XML）来源获取信息，而无需任何浏览器.
    • HTMLDocumentEvents (by SmOke_N) -跟踪IE文档事件.
    • IEEx (by SmOke_N) -带有某些Javascript选项的IE扩展库.
    • NavInfo (by Nessie) -使用此UDF，您可以检查是否安装了指定的浏览器/软件以及所使用的版本.
    • Opera (by MrCreatoR) -与Opera相同. 使用Opera UDF自动执行Opera中最常见的任务.

    Microsoft Office¶

    • Microsoft Office Access (by randallc) -自动化Microsoft Access. -Microsoft Office Excel（由water等人编写）-该UDF包含在AutoIt中. 链接到 documentation 页面.
    • Microsoft Office Excel Charts (by water, GreenCan) -使用Microsoft Excel创建图表.
    • Microsoft Office Outlook (by water) -自动化Microsoft Outlook. 链接到 documentation 页面.
    • Microsoft Office PowerPoint (by Toady) -自动化Microsoft PowerPoint. -Microsoft Office Word（由water等人撰写）-此UDF包含在AutoIt中. 链接到 documentation 页面.
