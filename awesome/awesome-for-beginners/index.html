    新手友好项目

    Awesome First PR Opportunities ¶

    这个很棒的初学者友好项目列表是由 Doppler – 以方便和安全的方式管理所有环境变量.

    灵感来自 First Timers Only 博客文章.

    如果您是开源项目的维护者，请在您的项目中添加“仅限初学者”（或类似的）标签，并在此处列出，以便人们可以找到它.

    如果您不是程序员但想做出贡献，请查看 Awesome for non-programmers 列表.

    .NET¶

    • MvvmCross （标签：仅限初学者）
      用于跨平台解决方案的 .NET MVVM 框架，包括 Xamarin.iOS、Xamarin.Android、Windows 和 Mac.
    • RawCMS （标签：第一期好）
      RawCMS 是用 ASP.NET Core 编写的无头 CMS，专为采用 API 优先技术的开发人员构建.
    • Shouldly （标签：跳入）
      应该测试 .NET - Asserting * 应该 * 的方式！

    C¶

    • Neovim （标签：good-first-issue）
      Vim-fork 专注于可扩展性和敏捷性.

    C¶

    • grok.net （标签：第一期好）
      跨平台 .NET grok 实现
    • OpenRA （标签：简单）
      支持早期 Westwood 经典游戏的实时战略游戏引擎，例如命令与征服：红色警戒.
    • osu! （标签：good-first-issue）
      音乐游戏. 节奏只需点击即可！
    • Uno Platform (label: good first issue)
      OSS project for creating pixel-perfect, single-source C# and XAML apps which run natively on iOS, Android, macOS, Linux and Web via WebAssembly.
    • Cake (label: Good first issue)
      Cake (C# Make) is a free and open source cross-platform build automation system with a C# DSL for tasks such as compiling code, copying files and folders, running unit tests, compressing files and building NuGet packages.

    C++¶

    • electron （标签：第一期好）
      使用 JavaScript、HTML 和 CSS 构建跨平台桌面应用程序
    • Godot Engine (label: junior job)
      2D and 3D cross-platform game engine. Also has C# and Python code.
    • projectM （标签：第一期好）
      使用 OpenGL 和 GLSL 的音乐可视化库. 有使用 Qt5、SDL、emscripten、iTunes、Kodi 的应用程序.
    • Roc Toolkit （标签：需要帮助）
      用于通过网络进行实时音频流传输的工具包.
    • tensorflow （标签：统计：欢迎贡献）
      使用数据流图进行可扩展机器学习的计算
    • Yugabyte DB （标签：第一期好）
      分布式 SQL 数据库.
    • MoveIt （标签：第一期好）
      易于使用的开源机器人操作平台，用于开发商业应用程序、原型设计和基准算法.

    Clojure¶

    • Alda （标签：低垂的果实）
      音乐家的音乐编程语言. ：笔记：

    ClojureScript¶

    • LightTable （标签：第一期好）
      下一代代码编辑器！ KickStarter 上资助最多的项目之一.

    CSS¶

    • sButtons （标签：第一期好）
      您可以在下一个项目中轻松使用的简单按钮.

    Dart¶

    • dart.dev （标签：初学者）
      一个涵盖 Dart 语言和公共库的网站，面向 Dart 库、Web 应用程序、服务器端代码和移动 (Flutter) 应用程序的开发人员.

    Elixir¶

    • Ecto （标签：级别：初学者）
      Ecto 是 Elixir 的数据库包装器和语言集成查询
    • Elixir （标签：级别：初学者）
      Elixir 是一种动态的函数式语言，旨在构建可扩展和可维护的应用程序

    Go¶

    • CockroachDB （标签：第一期好）
      可扩展、可生存、高度一致的 SQL 数据库
    • Docker/CLI （标签：经验/初学者）
      Docker CLI
    • Helm （标签：第一期好）
      Kubernetes 包管理器
    • Hugo （标签：经验/初学者）
      一个快速灵活的静态站点生成器，用 GoLang 构建
    • Kubernetes （标签：第一期好）
      生产级集装箱调度管理系统
    • Mattermost （标签：抢夺，难度/1：简单，技术/去）
      Golang 和 React 中的开源 Slack-alternative
    • Moby （标签：经验/初学者）
      开源应用容器引擎
    • Terraform （标签：第一期好）
      一种用于安全有效地构建、更改和版本控制基础架构的工具.
    • TiDB （标签：新贡献者）
      分布式可扩展混合事务和分析处理 (HTAP) 数据库
    • script （标签：第一期好）
      一个 Go 库，用于执行 shell 脚本擅长的任务：读取文件、执行子进程、计算行数、匹配字符串等. 非常欢迎初学者，他们将在 PR 过程中获得详细的代码审查和帮助.
    • httpexpect （标签：需要帮助）
      Go 的端到端 HTTP 和 REST API 测试.
    • Killgrave （标签：第一期好）
      在 Go 中生成模拟服务器的简单方法.
    • lxd （标签：简单）
      系统容器和虚拟机管理器.
    • PureLB （标签：GoodFirstIssue）
      Kubernetes 的负载均衡器编排器，使用标准 Linux 网络和路由协议.
    • containerd （标签：经验/初学者）
      行业标准的容器运行时，强调简单性、健壮性和可移植性.
    • Meshery （标签：第一期好）
      Meshery，服务网格管理平面.

    Java¶

    • Strongbox （标签：第一期好）
      Strongbox 是一个用 Java 编写的工件存储库管理器.
    • TEAMMATES （标签：第一期好）
      TEAMMATES 是一个免费的在线工具，用于管理学生的同行评估和其他反馈路径.
    • elasticsearch （标签：第一期好）
      开源、分布式、RESTful 搜索引擎.
    • JabRef （标签：第一期好）
      使用现代 Java 功能（包括 JavaFX）管理参考文献的桌面应用程序. Dedicated to code quality and constructive feedback：每个 Pull Request 都由两名开发人员审查，以提供高质量的反馈并确保新贡献的高质量.
    • Wikimedia Commons Android App （标签：第一期好）
      允许用户从他们的安卓手机/平板电脑上传图片到维基共享资源.
    • XWiki （标签：入职）
      XWiki 是一个用 Java 编写的免费 wiki 软件平台，其设计重点是可扩展性. 初学者应遵循 onboarding wiki.
    • zerocode （标签：第一期好）
      无需编码的 API 自动化、简单的 JSON 响应断言、测试 REST、SOAP、Kafka 和 Java/DB API，CI/Jenkins 友好.
    • SirixDB （标签：第一期好）
      SirixDB 是一个进化的、版本化的 NoSQL 文档存储（XML 和 JSON）（主要）用 Java 编写. 它使用从 ZFS 和 Git 借用的许多概念在提交期间存储紧凑快照. 每个修订版都被编入索引，并且可以使用时态查询来查询文档存储. 它特别适合现代硬件.
    • Images-to-PDF （标签：第一期好）
      将图像转换为 PDF 文件的 Android 应用程序.
    • Trino (formerly Presto SQL) （标签：第一期好）
      用于大数据的分布式 SQL 查询引擎. 求指导 project's Slack.
    • appsmith （标签：第一期好）
      拖放内部工具构建器

    JavaScript¶

    • name-suggestion-index （标签：第一期好）
      OpenStreetMap 的规范通用品牌名称
    • iD （标签：第一期好）
      JavaScript 中易于使用的 OpenStreetMap 编辑器.
    • missioncontrol （标签：仅限初学者）
      Mission Control 是一个免费开放的市场，将自动驾驶汽车（例如无人机）与寻求包裹递送等服务的用户联系起来.
    • missions （标签：仅限初学者）
      Missions 是一个移动网络应用程序，将希望通过自主无人机运送包裹的 DAV 用户连接到在开放和自由的市场中寻找工作的人.
    • annyang （标签：仅限初学者）
      一个流行的语音识别库，用于与从网站到智能镜子再到飞行无人机的所有内容进行通信.
    • PouchDB （标签：仅限第一次）
      PouchDB 是一个袖珍型数据库.
    • Leaflet （标签：第一期好）
      用于移动友好型交互式地图的 JavaScript 库.
    • Dragula （标签：仅限初学者）
      拖放如此简单，令人痛心.
    • history （标签：适合初学者）
      一个最小的、功能性的 JavaScript 历史库.
    • TodoMVC （标签：第一次投稿）
      通过在许多 JavaScript 应用程序中实现 Todo 来帮助您选择 MV* 框架.
    • AVA （标签：适合初学者）
      未来派测试运行器.
    • Kinto.js （标签：易选）
      一个离线优先的 JavaScript 客户端，利用 Kinto API 用于远程数据同步.
    • ESLint （标签：第一期好）
      一个完全可插入的工具，用于识别和报告 JavaScript 中的模式.
    • Webpack (label: D0: My First Commit (Contrib. Difficulty))
      JavaScript 和朋友的捆绑器.
    • Tessel 2 CLI （标签：贡献者）
      Tessel 2 的命令行界面.
    • Ember.js （标签：适合新贡献者）
      用于创建雄心勃勃的 Web 应用程序的 JavaScript 框架.
    • Ember.js Data （标签：适合新贡献者）
      Ember.js 的数据持久化库.
    • freeCodeCamp （标签：仅限第一次使用）
      开源代码库和课程. 学习编码并帮助非营利组织.
    • Ghost （标签：第一期好）
      只是一个博客平台
    • eslint-plugin-unicorn （标签：适合初学者）
      很棒的 ESLint 规则.
    • Hyper （标签：第一期好）

    • pdf.js （标签：5-good-beginner-bug）

    • Moment.js （标签：抢购）
      用于解析、验证、操作和格式化日期的轻量级 JavaScript 日期库.
    • serverless （标签：第一期好）
      无服务器框架
    • Hoodie Camp （标签：仅限初学者）
      Hoodie Camp 是一个供新老贡献者使用的空间.
    • React （标签：第一期好）
      用于构建用户界面的声明式、高效且灵活的 JavaScript 库.
    • React Native （标签：第一期好）
      使用 React 构建本机应用程序的框架.
    • React server （标签：good-first-contribution）

    • Yarn （标签：第一期好）

    • keystonejs （标签：第一期好）
      用于在 Node.js 中开发数据库驱动的网站、应用程序和 API 的开源框架. 建立在 Express 和 MongoDB 之上.
