    Static analysis

    该存储库列出了适用于所有编程语言、构建工具、配置文件等的静态分析工具.
    官方网站， analysis-tools.dev 基于此存储库并为每个工具添加排名、用户评论和其他资源（如视频）.

    What is Static Analysis?¶

    > 静态程序分析是在没有实际执行程序的情况下对计算机软件进行的分析 — Wikipedia

    > 作为程序员，我这几年做的最重要的事情就是积极追求静态代码分析. 比我用它预防的数​​百个严重错误更有价值的是我对软件可靠性和代码质量的看法的改变. — John Carmack (Creator of Doom)

    • PT Application Inspector - 识别代码缺陷并检测漏洞以防止网络攻击. 通过展示可能的漏洞来演示远程代码执行.

    • tfsec - Terraform 静态分析工具，通过在构建时检查云错误配置来防止潜在的安全问题，并直接与 HCL 解析器集成以获得更好的结果. 检查是否违反 AWS、Azure 和 GCP 安全最佳实践建议.
    • Tsunami Security Scanner - 具有可扩展插件系统的通用网络安全扫描器，用于以高可信度检测高严重性的类似 RCE 的漏洞. 可以添加用于查找漏洞（例如开放 API）的自定义检测器.
