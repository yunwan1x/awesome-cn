    Advanced datastructures¶

    • specter：用于查询和转换嵌套和递归数据的优雅 API
    • Persistent AVL trees：具有日志时间排名查询的持久排序映射和集合
    • Finger Tree: 双列表, 计数双列表, 计数排序集
    • Hitchhiker Tree：创建快速、可快照、可大规模扩展的数据库
    • Hierarchical set
    • Ordered: 有序集和映射
    • Lazy Map: 仅在访问时计算其值
    • Duratom: 持久化原子
    • Durable Queue：队列保留在磁盘上
    • bifurcan: 线性映射/集合/列表（在内存中连续存储条目），; 用 java 编写，但测试套件（阅读：用法示例） in clojure
