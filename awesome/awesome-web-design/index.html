    Web设计

    Awesome Web Design ¶

    >为数字设计师精心策划的精选资源列表.

    看看我 website & blog. ✨

    Blog & News¶

    • Web Designer News：设计师最好的策划故事.
    • Marvel Blog：Marvel团队的最新消息，资源和想法.
    • The Next Web：国际科技新闻，商业和文化.
    • Medium：一个由读者和作家组成的社区，为大大小小的想法提供独特的视角.
    • Creative Bloq：艺术与设计灵感.
    • Smashing Magazine：专业的网页设计师和开发人员.
    • Sitepoint：Web开发人员和设计人员的中心，分享他们对构建令人难以置信的互联网事物的热情.
    • Designer News：设计界的会面地点.
    • Reddit Web Design：设计师提供有用的帖子，提问或参与讨论的地方.
    • Morning Reader：打破科技新闻.
    • Google design：Youtube频道为设计师提供有关Google设计的视频.

    Inspiration¶

    • Awwwards：最佳网页设计趋势.
    • One Page Love：设计图库展示最好的单页网站.
    • Inspired UI：移动应用程序设计模式.
    • Behance：展示和发现创意作品.
    • Dribbble：为设计师展示和讲述.
    • Product Hunt：发现你下一个喜欢的东西的地方.
    • UI Movement：每天最好的UI设计灵感.
    • Pttrns：最精美的设计模式，资源和灵感.

    Colors¶

    • Brandcolors：世界上最大品牌的官方颜色.
    • Adobe Color CC：使用色轮创建配色方案或浏览Kuler社区的数千种颜色组合.
    • Paletton：用于创建可以很好地协同工作的颜色组合的设计器工具.
    • Coolors：为您的设计生成无限调色板.
    • Lol Colors：精选调色板灵感.
    • Colordot：人类的颜色选择器.
    • Gradients.io：为设计师和开发人员设计的渐变色.
    • UI Gradients：美丽的色彩渐变.
    • Colorzilla：吸管，颜色选择器，梯度发生器.

    Typography¶

    • Google Web Fonts：提供直观且强大的开源设计器Web字体目录.
    • Font Face Ninja Browser Extension ：您正在浏览网站，并且想要找到字体的名称？ 忍者来了！
    • Fontpair：漂亮的Google字体组合和对.
    • Type Anything：创建出色的字体组合.
    • Type Scale：预览并为项目选择正确的类型比例.
    • WhatTheFont ：看到图像中的字体，想知道它是什么？ WhatTheFont将找到最接近的匹配.
    • Typewolf：全面的Web排版信息，字体替代方案的建议以及野外使用的链接.
    • Typ.io：来自真实网站的排版灵感.

    Icons¶

    • Flat Icon：成千上万的免费平面图标.
    • Font Awesome：标志性的字体和CSS工具包.
    • Icon Finder：100万免费和高级图标.
    • The Noun Project：一切的图标.
    • Favicon Generator：Favicon＆app图标生成器.
    • Freepik：不同格式的免费矢量.
    • Vecteezy：免费矢量艺术，矢量图形，插画背景等的集合.
    • Material Icons：免费使用Google提供的材料设计图标.
    • Ionicons：Ionic Framework的高级图标字体.

    Images¶

    • Pexels：在一个地方最好的免费照片.
    • Unsplash.it：占位符使用漂亮的图库.
    • Pixabay：高品质的照片和插图免费商业用途.
    • Placehold.it：快速简单的图像占位符服务.
    • Imgur：互联网上最棒的图像.
    • Flickr：所有照片的家.

    Guidelines¶

    • iOS Human Interface Guidelines：提供设计漂亮，引人入胜的iOS应用程序所需的一切.
    • Google Material Design：仔细研究Google品牌的设计，组件和资源.
    • Laws of UX: A collection of the maxims and principles that designers can consider when building user interfaces.
    • Ultimate Guide：移动应用程序设计原则的终极指南，让您的应用程序大放异彩.
    • GoodUI：5种方法可以获得更好的用户界面.

    Design Tools¶

    • Sketch：适用于Mac的数字设计轻量级程序.
    • Figma：协作界面设计工具.
    • Adobe XD：原型设计和线框工具.
    • inVision Studio：快速原型设计，动画，内置设计系统和协作.
    • Canva：非常简单的图形设计软件.
    • Framer X：交互设计和UX设计软件.

    Prototype Tools¶

    • Zeplin：设计人员和开发人员之间的优秀协作工具.
    • Avocode：面向所有设计人员和开发人员的工作空间.
    • inVision：世界领先的原型设计，协作和工作流平台.
    • Justinmind：为Web和移动应用程序提供更快的原型，更好的沟通.
    • Marvel App：为设计师提供免费的移动和网络原型设计.
    • Atomic.io：免费的浏览器交互式设计工具.
    • Flinto：应用程序来创建设计的交互式和动画原型.

    Tutorials¶

    • Treehouse：为世界各地的人们提供价格合理的技术教育.
    • Tuts Plus Web Design Tutorials：免费的网页设计和开发教程.
    • Springboard：个性化的方法，旨在帮助您实现目标并推进您的职业发展.
    • Learn Sketch：短视频和编辑宝贵的资源，以帮助您开始！
    • Adobe XD Guide：一本12,000字长的XD初学者和现有用户综合指南.

    Books¶

    • Web design playground：从编写第一行HTML到创建有趣，有吸引力的网页的一步一步指导.
    • Don't make me think
    • The design of everyday things
    • 100 things every designer needs to know about people

    Productivity¶

    • Pomodoro：一种很酷的时间管理技术，可以提高工作和学习效率.
    • Toggl：提高工作场所生产力的时间跟踪器.
    • Trello：以有趣，灵活和有益的方式组织和优先处理您的项目.
    • RescueTime：帮助您了解您的日常习惯，从而使您能够专注并提高工作效率.
    • Qbserve：自动时间跟踪，发票和实时生产力反馈.
    • Paste：Mac的智能剪贴板历史记录管理器.
    • Sip：适用于Mac的颜色选择器.
    • Timing：Mac的自动时间和生产力跟踪.
    • Taskade：团队的协作任务列表和大纲.

    Slack Teams¶

    • Team Sketch：由Slack提供支持的Sketch设计师社区
    • Designer Hangout：来自世界各地的UXer讨论用户体验的地方.
    • Design Talks：UI / UX设计，网页设计，自由职业，设计批评等社区.
    • TheDesignerShip：拥有超过1500名设计师和创始人的社区.
