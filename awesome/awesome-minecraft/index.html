    Awesome minecraft

    Awesome Minecraft ¶

    > 与 Minecraft 相关的精彩内容的精选列表.

    Minecraft 是由 Mojang 开发的沙盒视频游戏. 该游戏允许玩家在 3D 程序生成的世界中使用各种不同的块进行构建，这需要玩家发挥创造力. 游戏中的其他活动包括探索、资源收集、制作和战斗.

    Websites¶

    Wiki¶

    • Official Minecraft Wiki - Minecraft Wiki 是 Minecraft 的完整资源，包括游戏玩法、方块、制作、更新、事件、模组、运行服务器和资源包.
    • Chinese Minecraft Wiki - 中文Minecraft Wiki是Minecraft的官方维基，涵盖了制作、方块、生物甚至PE版本的详细教程.
    • wiki.vg - 技术 Minecraft 维基.

    Forums¶

    • Minecraft Forums - 官方社区论坛，一个谈论游戏的好地方.
    • Planet Minecraft - 创意社区粉丝网站.
    • Minecraft Chinese Forum - Minecraft 最大的中国玩家社区.

    Archives¶

    • MCArchive - 存档旧的、丢失的和被遗忘的 Minecraft 模组. 可以在他们的 Discord 上找到各种其他 mod 和东西.
    • Minecraft Archive Project - 包含 Minecraft 模组、世界、资源包和许多其他资源（甚至论坛帖子！）的巨大档案.
    • MCVersions - 存档大多数 Minecraft 罐子.
    • BetaCraft Archive - 存档一些丢失的 Minecraft 服务器 jar.
    • MCPC+ Archive - 使用某些 MCPC+ 版本存档.
    • CraftBukkit Archive - 存档许多 CraftBukkit 版本.

    Mod Loaders¶

    • Minecraft Forge - 对 Minecraft 基础文件的修改以帮助 mod 之间的兼容性.
    • Quilt - Fabric 项目的强大分支.
    • Fabric - 适用于 Minecraft 的轻量级、实验性改装工具链.
    • liteloader - 适用于 Minecraft 的轻量级模组加载器，旨在为不需要修改游戏机制的模组提供简单、高性能和可靠的加载器功能.
    • Rift - 适用于 1.13-1.13.2 的 Mod Loader（甚至是快照）.
    • Meddle - 与在一处修改 1.9/1.10 快照相关的所有内容.
    • RopeMC - 一个轻量级、基于反射的 mod 加载器，旨在用户友好且独立于更高版本.

    Mods¶

    Performance¶

    • Optifine - OptiFine 是一个 Minecraft 优化模组.
    • BetterFps - BetterFps 是一个 Minecraft mod，增加了一些性能改进.
    • Sodium (for Fabric) - 旨在提高帧速率和减少微卡顿的 Fabric 模块.
    • Iris (for Fabric) - 基于钠添加着色器支持的织物模型.

    Skins And Capes¶

    • Custom Skin Loader - 用于 Minecraft 的自定义皮肤加载器.
    • Universal Skin Mod - Minecraft Forge Mod 允许玩家加载自定义皮肤和披风纹理.

    Maps¶

    • Journey Map - 在您探索时在游戏或网络浏览器中实时映射.
    • VoxelMap - 以前称为赞的小地图，提供基本的地图和雷达功能.

    Items Management¶

    • Just Enough Items (JEI) - JEI 是 Minecraft 的物品和配方查看模块，从头开始构建以提高稳定性和性能.
    • Bibliocraft - Bibliocraft 是一个存储和组织模式，从一个单一的书柜开始，现在已经扩展到更多.
    • /dank/null - 一个没有人真正得到的虚幻比例的存储模型. max tier /dank/null 对象可容纳 54 个物品槽，每个可容纳 21.4 亿个物品！

    Technology¶

    • Tinkers Construct - Tinkers' Construct 是一个关于以多种方式将工具组合在一起，然后修改它们直到它们变成其他东西的 mod.
    • Immersive Engineering - 沉浸式工程是一个具有一定魅力的技术模型，基于想法和概念，并且拥有 Damien Hazard 创建的大部分资产.
    • Botania - Botania 本质上是一个科技模组. 主题是自然魔法.
    • Thermal Foundation + Dynamics + Expansion - 服务器友好且内容丰富的魔法与技术的融合！ 标题是三个不同的模组名称，但属于同一个系列并且通常一起玩.
    • Railcraft - 大大扩展和改进了 Minecraft 中的矿车系统.
    • Actually Additions - 实际上 Additions 是一个技术模块，它添加了各种实用程序，如多功能工具、物品磁铁、原子重建器等.
    • Applied Energistics 2 - 一个关于物质、能量并用它们来征服世界的 Minecraft Mod. 主要集中在物品存储上.
    • Draconic Evolution - 最好的技术模组之一，但在游戏后期真的很强大. 添加诸如龙之能量核心之类的东西，目前还没有人能够填充……
    • Ender IO - 一个大型的末影主题模组，只是增加了许多新的有趣的东西.
    • Galacticraft - 添加火箭和新行星以供探索的模组. 可通过插件扩展.
    • OpenComputers - OpenComputers 是一个在游戏中添加计算机和机器人的模组，可以在 Lua 5.3 中编程.
    • ComputerCraft - 将基于 Lua（5.2 和 5.3）的计算机添加到 Minecraft.
    • WarpDrive - 在游戏中添加星际迷航和其他特许经营权，如船舶、武器、运输机等. 还添加了新的行星（维度）.

    Farming And Food¶

    • Pam's HarvestCraft - Pam's HarvestCraft is a mod for Minecraft that currently adds many kinds of crops and food.
    • AgriCraft - 一个增加了更复杂的作物系统的模组，比如具有基因突变的能力，还增加了更多的作物.
    • Bonsai Trees - 基本上添加了一个街区大的迷你林场.
    • Ex Nihilo: Creatio - 一个很棒的空中方块模组，它过滤沙子和砾石等有用资源，如铁和钻石，与许多模组兼容.

    Energy¶

    • RFTools - RFTools 是一个包含与 RF (Redflux) 或一般技术相关的各种工具的 mod.
    • Extreme Reactors - 添加了带有虚构燃料“耶洛里姆”的核反应堆. 原始大反应堆的延续.
    • Zetta Industries - 添加大电池主要多块的模组.
    • Applied Energistics 2 - 一个关于物质、能量并用它们来征服世界的 Minecraft Mod.

    Magic¶

    • AbyssalCraft - AbyssalCraft 是一个以探索为中心的魔法模组.
    • Astral Sorcery - 这是一个基于星星的魔法模组，利用星光和星座的力量.
    • Avaritia - Avaritia 是一个拥有非常长的进度树的模组，但在游戏后期真的很强大. 它的灵魂目的是将残局的游戏玩法拖到极致.
    • Blood Magic - 这是一个基于血液的魔法模组. 小心，你死了很多.

    Utilities¶

    • TNTUtils - 更多地控制 Minecraft 中的爆炸.
    • Akashic Tome - 基于 Morph-o-Tool 的 mod，主要提供合并不同 mod 的文档书籍的能力.
    • AppleSkin - 这个模组不会改变任何现有的机制，只是让现有的饥饿饱和度和疲惫程度在默认 GUI 中可见.
    • Attribute Fixer - 此模组修复了 Minecraft 中处理装甲点之类的属性系统，因为许多其他模组在不经意间受到该系统限制的影响.
    • Diet Hopper - 这个模组修改了 Minecraft 计算漏斗碰撞盒的方式，这样你就可以访问它后面的任何可见方块.
    • FoamFix - 修复了 Minecraft 代码的各个方面，通常使游戏更快.
    • WTHIT - 显示当前查看的块（较新版本的 Hwyla）.
    • Hwyla - 显示当前查看的块（Waila 的较新版本）.
    • WorldInfo - Shows the block currently looked at (Waila for Rift).
    • Waila - 显示当前查看的块（原始 Waila 项目）.
    • Block Helper - 显示当前查看的块（Waila 的旧版 Minecraft 版本低于 1.6）.

    General¶

    • ArmorPlus - 这个模组真的为游戏增加了大量的盔甲，以及不同的制作方法.
    • Baubles - 此模组为玩家物品栏添加了七个新插槽，可以安装其他模组添加的各种物品，如护身符、腰带、戒指等.

    Exploration¶

    • Aroma1997's Dimensional World - 这个模组增加了一个额外的维度，其目的是采矿，这意味着主世界将不再被诸如 Buildcraft 采石场之类的东西填满.
    • Twilight Forest - 添加了一个新的暮光森林维度，有新的结构、boss 和要完成的任务.

    Building¶

    • Carpenter's Blocks - 这个模组添加了新的方块，例如倾斜和角落，可以让它们看起来像游戏中的任何其他方块！ 对艺术建筑非常有用. （对于 1.12+，请参阅 BlockCraftery）
    • Chisel - 这个模组为游戏中的许多方块添加了多个新纹理，添加了各种各样的纹理. （对于小于块的编辑单位，请参见 Chisel 和 Bits.）
    • WorldEdit - WorldEdit 是一个开源的游戏内地图编辑器，可用于 Bukkit、Forge、MinecraftEdu 和许多其他平台.
    • FAWE - 代表快速异步世界编辑. 基于 WorldEdit，它只是更快，并添加了更多工具（即画笔）来使用.

    Server Plugins¶

    • AntiCheatReloaded - AntiCheatReloaded 通过监控和分析玩家的行为，帮助服务器管理员轻松识别和阻止恶意用户.
    • ViaVersion - 允许将较新的客户端连接到 Minecraft 服务器的旧服务器版本.
    • AdvancedKitsReloaded - AdvancedKitsReloaded 是 Minecraft 服务器的套件管理插件，它运行 Bukkit、CraftBukkit、Spigot 和 Paper (>1.8). 使用此插件，您可以轻松创建/编辑/删除您的工具包，您可以根据需要对其进行自定义.
    • WorldGuard - WorldGuard 可让您和玩家保护土地区域免受恶意者和不受欢迎的人的侵害，以及调整和禁用 Minecraft 的各种游戏功能.
    • CraftBook - CraftBook 可让您创建神奇的延伸桥、紧凑的红石电路、复杂的矿车机制等等——所有这些都无需客户端 mod 并且完全由服务器自定义.

    Resource Packs¶

    • BDCraft - 各种漫画风格的资源和声音包，甚至适用于许多模组、基岩版和泰拉瑞亚！
    • LB Photo Realism Reload - LB Photo Realism 纹理包的继任者，使 Minecraft 看起来更逼真.
    • Default Dark Mode - 默认 Minecraft UI 的暗模式！
    • Cyber Optics - 科幻资源包.
    • Soartex - 带有模组支持的平滑资源包.
    • Faithful - 光滑的纹理，打造出经过翻新的默认外观.

    Shader Packs¶

    • Continuum - 逼真的着色器包.
    • Chocapic13 - 逼真且可配置的着色器包.
    • Sonic Ether - Pathracing 和其他逼真的着色器包.
    • Sildur - 着色器甚至实现了 Minecraft 1.16 的内置 API.
    • Potato - 一个非常轻量级的着色器包，适用于较旧或受限的 PC.
    • ESBE 2G - 基岩版的着色器包.

    Web Applications¶

    • Blessing Skin Server - Web 应用程序将您的自定义皮肤带回离线 Minecraft 服务器.
    • yoshino - 一个精简且快速的 Minecraft 皮肤服务器，用 php 编写. (Live Demo)
    • WorldEdit Golf - 在比赛中挑战其他人以尽可能少的命令使用 WorldEdit.

    Softwares¶

    Servers¶

    • Paper - Spigot 和 BungeeCord 的杂项补丁提供更好的性能和更多的配置选项.
    • Glowstone - 以前称为 Spout 和 BukkitContrib. 从头开始编写的 Spigot Server 替代方案，其性能类似于 Paper.
    • Spigot - CraftBukkit 的继任者为 Bukkit API 提供支持.
    • Sponge - Minecraft Server 提供对 Sponge API 的支持，并可扩展到 Forge API 和 Bukkit API.
    • Mohist - 实现 Forge 和 Bukkit API 的服务器.
    • Thermos - 为 1.7.10 实现 Forge 和 Bukkit API 的服务器.
    • KCauldron - 为 1.7.10 实现 Forge 和 Bukkit API 的服务器.
    • Cauldron - 以前称为 MCPC+. 实现 Forge 和 Bukkit API 的原始服务器.
    • PowerNukkit - Nukkit 的强大分支.
    • Cloudburst/NukkitX/Nukkit - 一个基岩版服务器，为支持 Nukkit API 的插件提供支持.
    • PocketMine - 一个提供基本插件支持的基岩版服务器.

    Libraries¶

    • CraftLib - 完全 Kotlin / Java 库，用于包装整个 Minecraft 代码库，包括协议、NBT、原理图、聊天等.
    • MCProtocolLib - 一个用 Java 编写的用于与 Minecraft 客户端/服务器通信的简单库.
    • node-minecraft-protocol - 一个用于解析和序列化 minecraft 数据包的 NodeJS 库，以及身份验证和加密.
    • go-mcproto - 用 Go 编写的 Minecraft 服务器通信协议实现，允许您对自己的机器人进行编程.

    Utilities¶

    • Minecraft Command Helper - 一个关于生成 Minecraft 命令的 WPF 程序（带有 NBT 数据）. （仅限 Windows）
    • Minecraft Command Science - 几个 Minecraft vanilla JSON 生成器.
    • Amidst - 显示 Minecraft 世界的概览，而无需实际创建它.

    Launchers¶

    • MultiMC - Minecraft 的自定义启动器，让您可以轻松地同时管理多个 Minecraft 安装.
    • HMCL - 一个强大的 Minecraft 启动器，支持许多功能.

    Development¶

    • FML Mod Development Tutorial -（中文）Minecraft 1.8.9 FML Mod 开发教程.
    • SchematicWebViewer - 用于在 Web 上渲染和查看 Minecraft 原理图文件的 JavaScript/TypeScript 库.
    • Tabula - 有助于创建 3D 模型的 Minecraft Mod.
    • Mod Coder Pack - 正确反编译 Minecraft 的工具.
    • BON 2 - 重写了 Bearded Octo Nemesis 项目，支持 ForgeGradle.
    • Bearded Octo Nemesis - 反混淆工具，支持 SRG 映射，尤其是 Mod Coder Pack.
