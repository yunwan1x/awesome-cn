    Ada is powering satellites, aircrafts, ships, power plants, surgical robots、无人机、CNC、服务器、游戏和咖啡机.
    > Ada 可以说是最{高性能∩能力∩精确∩可读∩成熟}编程语言.
    > 艾达活蹦乱跳！

    > 艾达/火花万岁.

    • simple-games - 通过编写简单的游戏来学习 Ada 2012.
    • spark-by-example - 在 SPARK 中收集经过验证的函数和数据类型.

    Online books¶

    • ada-wikibook - Wikibooks 上的 Ada 编程书.

    • ada-95-tutorial - 一本从 Ada 95 的初学者到高级概念的书，包含源文件.

    Competition¶

    Compilers¶

    Open source¶

    GPL (no linking exception)¶
    • gnat-ce - GNAT 社区版. 面向自由软件开发人员、爱好者和学生的 AdaCore GNAT 版本. GNAT 社区提供的运行时库在 GPLv3 下获得许可，没有链接异常. 它仅支持 Ada 2012.
    GPL (with linking exception)¶
    • fsf-gnat - Ada 编程语言的自由软件基金会编译器，构成 GNU 编译器集合的一部分. 它支持所有版本的语言，即 Ada 2012、Ada 2005、Ada 95 和 Ada 83.
    Other / unknown licence¶
    • hac - HAC Ada Compiler - 一个完全在 Ada 中的小型、快速的 Ada 编译器.
    • byron - Byron 项目是一个社区驱动的 Ada 编译器、工具链和 IDE 系统.

    Commercial / Open source¶

    • gnat-pro - GNAT Pro 产品线为 Ada、C 和 C++ 提供全面的工具集（IDE、调试器、库、动态和静态分析工具）.

    Commercial / Closed source¶

    • janus-ada - 编译器支持 Ada 83、Ada 95、Ada 2005 和 Ada 2012 版本语言的所有主要功能.
    • object-ada - PTC ObjectAda 是一个广泛的原生和交叉开发工具和运行时环境系列.
    • apex-ada - PTC ApexAda 是业界最受欢迎的 Ada 和混合 Ada/C/C++ 应用程序开发环境之一.
    • greenhills-ada - Green Hills Software 为实时目标提供业界首个经过充分验证的 Ada 交叉编译器系列.
    • xgc - XGC 为 MIL1750、ERC32 和 LEON 等航空航天特定目标提供基于 GCC 的裸机工具链和 Ada95 支持. 以及传统的 M68000 工具链.

    Online¶

    • tutorials-point-ada-compiler - 基于 CodingGround 引擎的在线编译器.
    • jdoodle-ada-compiler - 在线编译器，提供多个版本的 GNAT 进行测试.
    • compiler-explorer - 从您的 Web 浏览器交互地运行 GNAT FSF 编译器并与程序集交互.

    Edit¶

    Integrated Development Editor¶

    The GNAT Programming Studio - ada-visual - Visual Studio 2017 的 Ada 语言集成.

    Editors¶

    • lea - LEA 是 Ada 的轻量级编辑器，旨在为开发任何规模和级别的 Ada 项目提供简单的、类似于脚本世界的“外观和感觉”，同时允许访问 GNAT 等全面的开发工具. LEA 包括 HAC，即 HAC Ada 编译器.

    Text Modes¶

    • gnu-emacs-ada-mode - 用于编辑 Ada 源的 Gnu Emacs 主要模式.
    • vim-ada-bundle - 维护 Ada Bundle：Vim/NeoVim 的完整 Ada 模式.
    • ada-tmbundle - 对 Ada 的 TextMate 支持.
    • bbedit-ada-module - 对 Ada 的 BBEdit 支持.

    VSCode Extensions¶

    • adacore-ada - 对 Ada 的语言支持. 这包括 Language Server for Ada.
    • ada-vscode - Ada 和 GPR 的基本语法突出显示.
    • ada-utilities - VSCode extension Ada programmers which provides a few utility functions.

    Deployment¶

    Build and Package¶

    • alire - 一个即用型 Ada 库的目录以及一个命令行工具 (alr)，用于获取、编译并将它们合并到您自己的项目中. 它旨在完成与 Rust 的 cargo 或 OCaml 的 opam 类似的角色.
    • gprbuild - Adacore 多语言软件构建工具.
    • ravenadm - Ravenports 的管理工具 http://www.ravenports.com.
    • synth - 用于实时系统的 Next D/Ports 构建工具（Portmaster 和 Portupgrade 工具的替代品）.
    • ada4cmake - 用于简单 gnat 项目包含的 CMake 宏.
    • cmake-ada-offa - CMake 的 Ada 语言支持.
    • cmake-ada-cho3 - Ada 的 CMake 语言支持，PLplot 的跨平台支持代码的分支.

    Continuous Integration¶

    • continuous-verification - 使用 Travis CI 自动进行 SPARK 形式验证.
    • ada-actions - 用于 GitHub 操作的 Ada 工具链

    Runtimes¶

    • bb-runtimes - GNAT 裸金属板支持包 (BSP).
    • avr-ada - 用于 8 位 AVR 微控制器的 GNAT.
    • [ada-runtime] - 一个缩小的 Ada 运行时，可以适应不同的平台.
    • cortex-gnat-rts - 此软件包包括基于 FreeRTOS 的 GNAT Ada 运行时系统 (RTS)，并针对具有 Cortex-M0、M3、-M4、-M4F MCU 的电路板.
    • adawebpack - WASM 的 GNAT RTL 和 Web API 的绑定.

    [ada-runtime]：https://github.com/Componolit/ada-runtime

    OS and Kernels¶

    • m2os - 具有对小型微控制器的简单任务支持的 RTOS.
    • marte-os - MaRTE OS 是一个适用于嵌入式应用程序的硬实时操作系统，它遵循 Minimal Real-Time POSIX.13 子集. 它为开发多线程实时应用程序提供了一个易于使用和受控的环境.
    • muen - 用于高保证的 x86/64 分离内核.
    • ewok - EwoK 是一个针对微控制器和嵌入式系统的微内核.
    • bare-bones - Ada 港口 osdev.org 最小的 32 位 x86 内核.
    • lovelace-os - Lovelace 致力于使用 Ada 2012 语言编写类似 Unix 的操作系统.
    • ada-kalinda-os - Ada_KALINDA 是一种类似 Mac Plus 的操作系统，用 Ada95 编写.
    • cxos - Ada 操作系统开发示例.
    • havk - 使用 SPARK 创建以 x86-64 为中心的安全操作系统.
    • cubit - CuBitOS 是一个多处理器、64 位、（部分）正式验证的通用操作系统，目前用于 x86-64 架构.

    Games¶

    • steamsky - 带有蒸汽朋克背景的天空 Roguelike.
    • unity-ada-tetris - 俄罗斯方块，在 Ada 中，用于 Unity 游戏引擎.
