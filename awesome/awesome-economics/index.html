    Awesome economics

    Awesome Economics ¶

    精选的经济学家链接集. 的一部分 "Awesome X" series.

    该列表会定期使用新链接进行更新. 点击右上角的“观看”以进行后续操作.

    欢迎您的贡献. 将链接添加到“读者发送的链接” by yourself 或将新内容发送给antontarasenko@gmail.com .

    Studying¶

    Courses¶

    • MIT OCW Economics -涵盖所有主要经济学领域的100多个课程. 课程包括前提条件，推荐的教科书，演讲幻灯片和作业. 本科和研究生课程.
    • edX Economics -介绍性主题，前提条件很少.
    • Khan Academy: Economics -基本主题.

    Useful Materials¶

    • Academic Search -搜索“ .edu”和其他教育领域. 这些材料比大型Internet更可靠.
    • Foundational Equations of Economics -这些等式显示了“像经济学家一样思考”的原理. 研究生教科书将这些等式置于上下文中.
    • IGM Economic Experts Panel -顶尖经济学家思考与政策相关的问题. 一些答案包含有用的细节.
    • AEA Resources for Economists -由美国经济协会维护的有用链接列表.

    Research¶

    Portals¶

    • RePEc -经济研究人员的Web服务：书目，博客聚合器，新的工作论文，软件.
    • Federal Reserve Bank of St. Louis: Web Services -有用的服务清单.

    Articles and Working Papers¶

    • IDEAS RePEc -最大的经济学出版物数据库（2,000,000个）. 使用Google可以更轻松地搜索论文：site：ideas.repec.org<search term>. 索引来源如下.
    • NBER -主要研究人员的工作论文. 其中许多论文发表在同行评审的期刊上.
    • SSRN Economics -工作论文，无期刊出版物.
    • Google Scholar -一般搜索学术文献. 具有作者页面和引用计数器. 如果仅查找经济著作，则IDEAS会更强大.

    Data¶

    Datasets

    • FRED2 -来自80个来源的380,000个（宏观）时间序列. 支持用于将数据导入Excel，Stata，R等的插件. 有一个移动应用.
    • World Bank Data -国际宏观时间序列. 有数据导入插件.
    • IMF Data -宏数据的标准参考.
    • Quandl -汇总来自多个来源的财务和经济数据. 一些数据供应商通过此服务出售其数据. 与统计软件的良好集成.
    • MEDevEcon -与发展经济学有关的数据.
    • Monetary Economics: Data Sources -宏数据源概述.
    • OFFSTATS -按国家和主题链接到官方数据源.

    Search

    • International Open Government Dataset Search -超过1,000,000个政府数据集. 工作时，此服务看起来 like this . 否则，您会看到403错误.
    • Dataset Search Engine -基于Google的搜索超过200个数据源，包括此处提到的数据源. 您可以使用Google search operators 这里.
    • StackExchange Open Data -如果找不到所需的数据，可以在这里询问.
    • Reddit /r/datasets -还有一个地方可以请求数据集.

    Software¶

    Writing

    • LaTeX -经济学家之所以写LaTeX是因为它比Word或LibreOffice更好地处理数学和引用. 如果您定期写作，LaTeX是值得学习的.
    • Beamer -用于演示的LaTeX类.
    • TikZ -用于绘制图形的扩展. 一种 how-to 和一个 manual.
    • LyX -LaTeX的免费简单编辑器.
    • Zotero -书目管理. 还要安装（a）Zotero浏览器插件，以将论文从RePEc导入您的图书馆； （b）Zotero-LyX插件可轻松引用文献.
    • Git -版本控制系统. 如果您要还原几个月前所做的更改或与其他作者合作，则很有用. DropBox也具有版本控制，但是Git更明确. 一种 short intro . 或使用 GitHub Desktop 如果您喜欢它简单.

    Computing

    • Stata -经济学中统计计算的行业标准. 免费替代品：
    • IPython -基于Python的环境. 计量经济学分析是使用免费软件包完成的：statsmodels，SciPy，NumPy，pandas.
    • RStudio -基于R的环境. R是统计学家中的标准语言，因此R存储库通常包含其他语言无法提供的专用库.
    • Matlab -经济学建模和数值优化的行业标准. 免费替代品：
    • Octave
    • Julia -设计用于满足高性能数值分析和计算科学需求的高级动态编程语言.
    • Mathematica -符号计算. 免费替代
    • Sage

    Sharing

    • GitHub -代码和数据的存储库. 在这里发布研究不是一种常见的做法，但是使用替代方法（大学主页，DropBox等）更为方便.
    • GitHub Pages -简单的静态网站.
    • GitHub LFS -大文件存储.
    • IPython Notebooks -LaTeX和Word的交互式替代品. 查看示例笔记本的外观 data-science-ipython-notebooks 和 the gallery.

    Reviews

    • Most common programs used by Economists -社区管理的常用软件列表.
    • Software for Researchers: New Data and Applications -涵盖上述软件及其他内容.

    Discussions¶

    *博客-经济学家中最流行的自我表现形式. 主要的博客聚合器： - EconAcademics.org - Economist's View -马克·托马 - Grasping Reality - Brad DeLong - Marginal Revolution -泰勒·科恩（Tyler Cowen）和亚历克斯·塔巴罗克（Alex Tabarrok） - VOX CEPR -CEPR成员 * Economics Blog Search -针对上述博客的基于Google的搜索服务. * AEA Blog Directory -主要经济博客列表. * StackExchange Economics - A Q&A website where you can ask and answer questions. * Reddit-流行的新闻聚合器. 有许多与经济学有关的部分，例如： - /r/GoodEconomics/ -有关经济问题的部分作品. - /r/EconPapers/ -讨论经济论文. * Discord-流行的聊天平台 - Academic Economics -一个设有讨论经济问题并帮助成员进行练习的房间的社区
