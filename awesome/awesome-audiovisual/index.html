    Awesome audiovisual

    Awesome Audiovisual ¶

    > 为照明、视频和音频专业人士和业余爱好者精心设计的软件、库和资源列表.

    想要添加一些东西吗？ 只需提出一个拉取请求，但一定要遵循 contributing guidelines ！ 题？ 在问它 General Discussion issue.

    Lighting¶

    Lighting Software¶

    • Avolites Titan - 在 Avolites 桌面上运行的 Titan 软件的 PC 版本. 在 11.4 版之前免费使用，尽管它偶尔会吐出随机数据. 需要版本 12 的加密狗.⚠ 不是免费的.
    • Chamsys MagicQ - MagiQ 是一款专业的灯光软件，可免费提供多达 64 个 Artnet 宇宙. 它包括强大的功能，如 2D/3D 像素映射、效果生成器、媒体服务器和可视化器.
    • Daslight - 适用于 PC 和 Mac 的 DMX 照明软件.
    • Dot2 software - 用于 Dot2 控制台的免费离线软件. 包括 Dot2 onPC 和 Dot2 3D.
    • EOS Family - ETC EOS 系列游戏机的软件. ⚠免费增值.
    • Freestyler - 免费的防雷软件
    • GrandMA2 software - 适用于 GrandMA2 灯光控制台的免费离线软件. 包括 GrandMA2 OnPC 和 MA3D. 不免费输出 DMX. ⚠免费增值.
    • GrandMA3 software - 新 GrandMA3 灯光控制台的免费离线软件. 包括 GrandMA3 OnPC. 不免费输出 DMX. ⚠免费增值.
    • Hog 4 software - 适用于 Hog 4 灯光控制台的免费离线软件. 包括猪 4 件. 不免费输出 DMX. ⚠免费增值.
    • LightFactory - PC 照明控制软件也是 NEO 控制台的后端. ⚠不是免费的.
    • ONYX - 具有很多功能的现代照明控制系统.
    • QLC+ - QLC+ 是一款免费的跨平台软件，用于控制 DMX 或模拟照明系统，如摇头灯、调光器、扫描仪等. Source Code ✓ 开源.
    • xLights - 适用于家庭圣诞灯光秀，但它是在预算内将灯光与音乐同步的好方法. Source Code ✓ 开源.

    Visualizers¶

    • Capture - 文档和可视化软件. Capture 是一款原生 macOS 和 Windows 应用程序，可让您处理灯光、视频、激光、移动风景和水景效果. 有免费的学生版. ⚠免费增值.
    • Depence² - Depence² 是一个独特的应用程序，它将设计和可视化过程与控制真实表演的潜力相结合. ⚠不是免费的.
    • Ma3D - 免费 GrandMA 控制台/软件的照明可视化工具.
    • wysiwyg - 所见即所得是一款多合一的照明设计软件，具有完全集成的 CAD、绘图、数据、可视化和虚拟显示控制. ⚠不是免费的.

    Audio¶

    Audio Software¶

    • Audacity - 音频文件编辑器和分析工具，非常适合裁剪音效和音乐曲目. ✓ 开源.
    • Ardour - 完整的实时录音和编辑套件. Source Code ✓ 开源.
    • da-Share MultiPlay - MultiPlay 是一个基于 Windows 的程序，旨在为剧院或公司使用播放音频提示. 它可以免费使用.
    • Linux Show Player - 专为舞台制作设计的免费提示播放器. Source Code ✓ 开源.
    • QLab - QLab 是 macOS 的声音、视频和灯光控制. ⚠免费增值.
    • REAPER - REAPER 是一个完整的计算机数字音频制作应用程序，提供完整的多轨音频和 MIDI 录音、编辑、处理、混音和母带制作工具集. ⚠免费增值.

    Audio tools¶

    • ARTA - 用于音频测量和分析的程序集合.
    • REW - REW 是免费的房间声学分析软件，用于测量和分析房间和扬声器的响应. 免费的 SMAART 替代品.

    Video¶

    See ebu/awesome-broadcasting.

    Tools¶

    • Livescript - 插入来自 Google Docs 的音乐/戏剧脚本，并将其用于现场“自动提示”和与网站上的每个人一起滚动. ✓ 开源.
    • StageClock - StageClock 是一个基于 Java 的图形计时应用程序. 它从连接的监视器输出干净的提要，并具有时间/日期、计时器和显示信息. ⚠免费增值.

    CADs¶

    • AutoCAD - AutoCAD® 是一种计算机辅助设计 (CAD) 软件，建筑师、工程师和建筑专业人士依靠它来创建精确的 2D 和 3D 工程图. ⚠不是免费的.
    • Vectorworks - Vectorworks 是一个多功能的内部部署应用程序，可为您的建筑和景观设计需求提供广泛的 2D 绘图、3D 建模、BIM 和渲染功能. 还有一个特别的娱乐版： Vectorworks Spotlight . ⚠不是免费的.

    Timecode¶

    时间码是由定时同步系统定期生成的数字代码序列. 时间码用于视频制作、表演控制和其他需要时间协调或记录记录或动作的应用程序.

    • El-Tee-See - LTC wav 文件生成器.
    • TCgenerator - GrandMA2 TC Generator 的收割者时间标记. ✓ 开源.
    • Timecode - ArtNet 和 LTC 时间码生成器，用 Java 制作. ✓ 开源.
    • Time MIDI sender - 用于将时间数据发送到 DasLight（通过 MIDI）的免费工具. ✓ 开源.
    • TimelordMTC - 先进的时间码管理工具. ⚠不是免费的.
    • M2Q - Python 应用程序能够通过 MIDI 触发 Chamsys 控制台/软件以进行高级同步节目. ✓ 开源.

    Monitoring¶

    • AES67 WebMeter - 接收 AES67 音频流并显示带有仪表的网页的服务器. ✓ 开源.
    • meters.lv2 - meter.lv2 是 LV2 插件格式的带有 GUI 的音频电平表的集合. ✓ 开源.
    • Open Sound Meter - 干净、开源的声级计. 支持 RTA、幅度、相位和脉冲响应、相干性、群延迟等. ✓ 开源.

    Libraries and frameworks¶

    • Open Lighting Architecture - 照明控制信息框架. 支持一系列协议和十多种 USB 设备. Source Code
    • node-dmx - Node.js 的 DMX 控制器库. nodejs ✓ 开源.

    ArtNet Libraries¶

    • artnet - Node.js 模块，可用于将 ArtDMX 包发送到 Art-Net 节点. nodejs.
    • artnet4j - 用于 Java 和处理的 Art-Net 库. java.
