    Awesome Robotic Tooling ¶

    一份精选的 C++ 和 Python 专业机器人开发工具清单，其中包含一些 ROS、自动驾驶和航空航天

    > 要停止重新发明轮子，您需要了解轮子. 此列表试图展示软件和硬件开发中的各种开放和免费工具，这些工具在专业机器人开发中非常有用.

    你的贡献对于保持这个列表的活力、提高质量和扩展它是必要的. 您可以阅读更多关于它的起源以及如何参与 contribution guide 和相关 blog post . 所有新项目条目都将有一条来自 protontypes.

    Communication and Coordination¶

    • Agile Development - 敏捷软件开发宣言.

    • Taiga - 敏捷项目管理工具.

    • kanban - 用于 GitLab 问题的免费、开源、自托管、看板.
    • Gitlab - 带有 Docker 的简单自托管 Gitlab 服务器.
    • Gogs - 构建一个简单、稳定且可扩展的自托管 Git 服务，可以以最轻松的方式进行设置.
    • Wekan - 基于流星的看板.
    • JIRA API - 用于 Jira REST API 的 Python 库.
    • Taiga API - Taiga 的 REST API 的 Python 库.

    • Grge - Grge 是增强 GitLab 的守护程序和命令行实用程序.
    • gitlab-triage - Gitlab 的问题和合并请求分类，自动化.
    • Helpy - 现代的开源帮助台客户支持应用程序.
    • ONLYOFFICE - 一个免费的开源协作系统，用于在一个地方管理文档、项目、客户关系和电子邮件通信.
    • discourse - 社区讨论的平台. 免费、开放、简单.
    • Gerrit - 基于 Git 的项目的代码审查和项目管理工具.
    • jitsi-meet - 安全、简单且可扩展的视频会议，可用作独立应用程序或嵌入 Web 应用程序中.

    • KITTI-360 - 这个大规模数据集包含 320k 图像和 100k 激光扫描，行驶距离为 73.7 公里.
    • waymo_ros - 这是一个用于将 Waymo 开放数据集连接到 ROS 的 ROS 包.
    • waymo-open-dataset - Waymo 开放数据集由 Waymo 自动驾驶汽车在各种条件下收集的高分辨率传感器数据组成.
    • Ford Autonomous Vehicle Dataset - 福特提出了一个具有挑战性的多智能体季节性数据集，该数据集由福特自动驾驶车队在不同的日期和时间收集.
    • awesome-robotics-datasets - 一组用于机器人和计算机视觉的有用数据集.
    • nuscenes-devkit - nuScenes 数据集的 devkit.
    • dataset-api - 这是 ApolloScape 数据集、CVPR 2019 自动驾驶挑战研讨会和 ECCV 2018 挑战的工具包的回购.
    • utbm_robocar_dataset - 用于自动驾驶的具有多个传感器的欧盟长期数据集.
    • DBNet - 用于驾驶行为学习的大规模数据集.
    • argoverse-api - Argoverse 数据集的官方 GitHub 存储库.
    • DDAD - 来自 TRI（丰田研究所）的新自动驾驶基准，用于在具有挑战性和多样化的城市条件下进行长距离（高达 250m）和密集深度估计.
    • pandaset-devkit - Hesa​​i & Scale 提供的自动驾驶公共大规模数据集.
    • a2d2_to_ros - 将 A2D2 数据集转换为 ROS 包的实用程序.
    • awesome-satellite-imagery-datasets - 带有计算机视觉和深度学习注释的卫星图像训练数据集列表.
    • sentinelsat - 搜索和下载哥白尼哨兵卫星图像.
    • adas-dataset-form - 用于算法训练的热数据集.
    • h3d - H3D 是来自本田的大规模全环绕 3D 多目标检测和跟踪数据集.
    • Mapillary Vistas Dataset - 一个多样化的街道级图像数据集，具有像素精确和特定于实例的人类注释，用于理解世界各地的街景.
    • TensorFlow Datasets - TensorFlow Datasets 提供了许多公共数据集作为 tf.data.Datasets.
    • racetrack-database - 包含全球 20 多个赛道（主要是 F1 和 DTM）的中心线（x 和 y 坐标）、赛道宽度和赛道线.
    • BlenderProc - 用于生成逼真训练图像的程序化 Blender 管道.
    • Atlatec Sample Map Data - 用于自动驾驶和模拟的 3D 地图，仅由旧金山市中心的两个摄像头和 GPS 创建.
    • Lyft Level 5 Dataset - Level 5 正在为 Lyft 网络开发自动驾驶系统. 我们正在从我们的自主车队收集和处理数据，并与您分享.
    • holicity - 用于学习整体 3D 结构的城市规模数据平台.
    • UTD19 - 公开可用的最大的多城市交通数据集.
    • ASTYX HIRES2019 DATASET - 用于基于深度学习的 3D 对象检测的汽车雷达数据集.
    • Objectron - 一系列以对象为中心的简短视频剪辑，伴随着 AR 会话元数据，包括相机姿势、稀疏点云和周围环境中平面的表征.
    • ONCE dataset - 具有 2D 和 3D 对象注释的大规模自动驾驶数据集.
