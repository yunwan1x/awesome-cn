    Awesome Software Engineering for Machine Learning ¶

    机器学习软件工程是构建ML应用程序的技术和准则，这些技术和指南与核心ML问题无关-例如开发新算法-而是围绕周围的活动，例如数据提取，编码，测试，版本控制，部署，质量控制，以及团队协作. 良好的软件工程实践可使用机器学习组件来增强生产级应用程序的开发，部署和维护.

    ⭐必须阅读

    科学出版物


    基于这些文献，我们对使用机器学习组件的应用程序采用软件工程实践进行了调查.

    Tooling¶

    工具可以使您的生活更轻松.

    我们仅共享开源工具或提供大量免费研究软件包的商业平台.

    • Airflow -以编程方式创作，安排和监视工作流程.
    • Alibi Detect -专注于离群，对抗和漂移检测的Python库.
    • Archai -神经体系结构搜索.
    • Data Version Control (DVC) -DVC是数据和ML实验管理工具.
    • Facets Overview / Facets Dive -强大的可视化功能有助于理解机器学习数据集.
    • FairLearn -评估和改善机器学习模型公平性的工具包.
    • Git Large File System (LFS) -在Git中用文本指针替换大型文件（例如数据集）.
    • Great Expectations -与管道集成的数据验证和测试.
    • HParams -针对机器学习项目的配置管理的周到方法.
    • Kubeflow -为想要构建和试验ML管道的数据科学家提供的平台.
    • Label Studio -具有标准化输出格式的多类型数据标签和注释工具.
    • LiFT -Linkedin公平性工具包.
    • MLFlow -管理ML生命周期，包括实验，部署和中央模型注册表.
    • Model Card Toolkit -简化并自动生成模型卡； 用于模型文档.
    • Neptune.ai -实验跟踪工具将组织和协作带入了数据科学项目.
    • Neuraxle -类似于Sklearn的框架，用于深度学习项目中的超参数调整和AutoML.
    • OpenML -一项包容性运动，旨在建立一个开放的，有组织的在线机器学习生态系统.
    • PyTorch Lightning -轻巧的PyTorch包装器，用于高性能AI研究. 缩放模型，而不是样板.
    • Robustness Metrics -轻量级模块，用于评估分类模型的鲁棒性.
    • Seldon Core -一个MLOps框架，用于在Kubernetes上打包，部署，监视和管理数千个生产机器学习模型.
    • Spark Machine Learning -Spark的ML库，其中包含常见的学习算法和实用程序.
    • TensorBoard -TensorFlow的可视化工具包.
    • Tensorflow Extended (TFX) -用于部署生产ML管道的端到端平台.
    • Tensorflow Data Validation (TFDV) -用于探索和验证机器学习数据的库. 与《大期望》相似，但适用于Tensorflow数据.
    • Weights & Biases -实验跟踪，模型优化和数据集版本控制.
