    Xamarin

    Awesome-Xamarin ¶

    为 Xamarin Android/iOS/Windows/OSX 精心挑选的主观现代/有趣和理想的开源库/工具的书签集合

    General¶

    • AutoMapper ★5,724 - .NET 中基于约定的对象-对象映射器.
    • Fluent Validation ★3,570 - 用于构建验证规则的流畅接口和 lambda 表达式.
    • Humanizer ★3,426 - 操作和显示字符串、枚举、日期、时间、时间跨度、数字和数量.
    • NodaTime ★970 - .NET 的替代日期和时间 API.
    • Polly ★4,666 - 异常处理策略，例如重试、永远重试、等待和重试或断路器.

    Architecture¶

    • Behaviors Toolkit ★34 - 一种以最少的代码向 Xamarin 应用程序添加通用和可重用交互性的方法.
    • Conditions ★48 - 帮助开发人员以流畅的方式编写前置和后置条件验证的库.
    • Stateless ★2,294 - 状态机.

    Async¶

    • AsyncEx ★1,253 - async/await 的辅助库.
    • LinqToAwait ★99 - 一个基于任务的 LINQ，设计用于异步/等待.

    Charts¶

    • MicroCharts ★1052 - 创建跨平台（Xamarin、Windows 等）简单图表.
    • MPAndroidChart/iOSCharts ★6 - MPAndroidChart/iOSCharts 绑定支持在 Xamarin Forms 中使用.
    • OxyPlot ★1,311 - .NET 的跨平台绘图库.

    Cloud¶

    • Azure - 微软 Azure.
    • Bugfender ★10 - 将您的应用程序日志存储在云端（iOS 和 Android）.
    • FireSharp ★479 - Firebase REST API 包装器.

    Database¶

    • Akavache ★1,692 - 本机应用程序的异步键值存储.
    • Breeze ★63 - 面向富客户端应用程序开发人员的数据管理库.
    • Couchbase.Lite ★299 - 轻量级嵌入式 NoSQL 数据库.
    • Lager ★2 - 使用 Akavache 作为存储后端的跨平台设置存储.
    • Massive ★1,708 - 您的数据库表的“包装器”并广泛使用 System.Dynamic.
    • Realm ★690 - 直接在手机、平板电脑或可穿戴设备中运行的移动数据库.
    • Settings ★58 - Xamarin 和 Windows 的跨平台设置插件.
    • SQLite.Net-PCL ★340 - SQLite 3 数据库支持、PCL、异步.
    • LiteDB ★4245 - 单个数据文件中的 .NET NoSQL 文档存储.
    • DB4O-GPL ★8 - 支持面向对象的数据库、嵌入式和远程连接.

    Fody¶

    • Fody ★2,150 - 用于编织 .net 程序集的可扩展工具.
      • AutoDependencyProperty.Fody - Automatically generates DependencyProperty boilerplate from simple C# properties.
      • PropertyChanged.Fody ★820 - 在编译时将 INotifyPropertyChanged 代码注入到属性中.
      • ReactiveUI.Fody ★105 - 为属性和 ObservableAsPropertyHelper 属性生成 RaisePropertyChange 通知.

    Framework¶

    • Ammy - XAML 平台的现代 UI 语言. 免费用于非商业开发.
    • Appercode.UIFramework ★13 - 允许使用单一 XAML 布局构建跨平台移动应用程序的用户界面.
    • Invention - Develop native apps for iOS, Android and Windows with 100% code sharing using Visual Studio and C#.NET.
    • SimplyMobile ★103 - 抽象的移动功能的集合.
    • Xamu-Infrastructure ★104 - 扩展、MVVM 类、行为和其他杂项. 来自 Xamarin 大学的有用代码位.

    Game Engine¶

    • CocosSharp ★463 - Cocos2D 和 Cocos3D API 的 CSharp 实现.
    • MonoGame ★5,276 - Microsoft XNA 4.x 框架的开源实现.
    • Paradox ★1,706 - Paradox3D + Silicon Studio .NET.
    • UrhoSharp ★275 - 跨平台的高级 3D 和 2D 引擎.
    • CocosCreator ★2 - CocosCreator 引擎的 Xamarin 绑定.

    IO/Storage¶

    • IO ★6 - 使用熟悉的 api 访问系统文件夹和文件.
    • PCL Storage ★270 - 用于 .NET 的一致、可移植的本地文件 IO API 集.
    • FilePicker-Plugin-for-Xamarin ★37 - 简单的跨平台插件，允许您选择文件并使用它们.

    IOC/DI¶

    • Autofac ★2,169 - 一个令人上瘾的 .NET IoC 容器.
    • DryIoc - 用于 .NET 的快速、小巧、功能齐全的 IoC 容器.
    • Funq ★2 - 高性能 DI 框架，通过使用 lambda 和泛型函数作为工厂来消除所有运行时反射.
    • LightInject ★320 - 用于 .NET 的轻巧、简单且速度惊人的 IoC 容器.
    • Ninject ★2,034 - .net 依赖注入器的忍者.
    • Stiletto ★36 - Dagger 的 .NET 端口，Square 的轻量级 Android 依赖注入器.
    • TinyIoC ★585 - 单类简单的 IoC 容器.

    Internationalization¶

    • I18NPortable ★49 - Xamarin 和 .NET 的简单和跨平台国际化/翻译.
    • Resxible ★9 - 从单个 RESX 文件自动生成多个平台相关资源文件的工具.
    • Vernacular ★167 - 跨平台本地化，转换标准字符串格式的工具.
    • SimpleLocalize ★19 - 用于在 Xmarin 项目中管理 i18n 密钥的开源工具.

    Layout¶

    • Flex ★128 - 灵活的盒子布局系统.

    Media¶

    • EZ-Compress ★11 - Xamarin 的简单图像流压缩插件.
    • Fast & Furious Image Loading ★820 - Xamarin 库可快速轻松地加载图像.
    • Lottie ★643 - 在 Android 和 iOS 上为 Xamarin 本地渲染 After Effects 动画.
    • LibVLCSharp ★174 - libvlc 的 Xamarin 绑定，为 VideoLAN 制作的 VLC 应用程序提供支持的多媒体框架.
    • MediaManager ★269 - 跨平台 Xamarin 插件从 PCL 播放媒体.
    • NGraphics ★482 - 用于渲染矢量图形的跨平台库.
    • PDFReader ★51 - 用于在 iPad 和 iPhone 上显示 PDF 文档的 iOS（仅限）库.
    • Screenshot Plugin ★21 - 一个简单的 Xamarin 和 Windows 屏幕截图插件，用于在您的应用程序中获取和保存屏幕截图.
    • SkiaSharp ★920 - Powerful C# API for doing 2D graphics. It is powered by Google’s Skia library.
    • Splat ★590 - 跨平台图像加载、颜色和内容.
    • SimpleAudioPlayer ★14 - 用于将本地文件和音频数据作为流播放的简单插件.
    • ZXing.Net.Mobile ★570 - 适用于 MonoTouch、适用于 Android 的 Mono 和 Windows Phone 的条码扫描库.
    • SupportMediaXF ★2 - 简单的跨平台插件，用于拍照或从共享代码的图库中挑选照片

    MVVM¶

    • Bind ★158 - 绑定为您提供了简单的对象属性之间的双向数据绑定.
    • EBind - 简洁、快速且功能丰富的 .NET 数据绑定与一些 Xamarin 好东西.
    • FreshMvvm ★324 - 专为 Xamarin.Forms 设计的超轻量 Mvvm 框架.
    • Infinite Scroll Plugin ★24 - 促进分页数据源的仅向前增量滚动的插件.
    • MVVMCross ★2,657 - 跨平台 mvvm 移动开发框架.
    • Cheesebaron.MvxPlugins ★79 - 插件集合.
    • MvxAms ★1 - MVVMCross Azure 移动服务插件.
    • MvxForms ★1 - 用于使用 Xamarin.Forms 的 MVVMCross 插件.
    • MugenMvvmToolkit ★127 - 跨平台 MVVM 工具包.
    • MVVMLight - 跨平台 MVVM 开发框架.
    • MvvmNano ★46 - 使用 ❤ 为 Xamarin.Forms 制作的小型智能 MVVM 框架.
    • Prism ★2,365 - 跨平台 MVVM 开发框架.
    • ReactiveUI ★3,917 - Rx MVVM 框架.
    • Wires ★28 - Wires 是一个简单的绑定库.

    Network¶

    • Apizr ★4 - 基于改装的 web api 客户端，但具有弹性（重试、连接、缓存、身份验证、日志、优先级等...）.
    • Connectivity ★200 - 跨平台网络/连接状态.
    • CrossDownloadManager ★67 - Xamarin 的跨平台下载管理器.
    • Flurl ★1,295 - Flurl 是一个现代的、流畅的、异步的、可测试的、可移植的、充满流行语的 URL 构建器和 HTTP 客户端库.
    • Fusillade ★216 - 一组 HttpMessageHandlers，使您的移动应用程序更加高效和响应.
    • Messaging ★1 - 使用默认消息应用程序拨打电话、发送短信或发送电子邮件.
    • ModernHttpClient - 通过使用移动优化的库 (NSURLSession / OkHttp) 加速 HTTP 请求.
    • NFC ★14 - 用于读取 NFC 标签的 Xamarin 插件.
    • Push Notification ★1 - 简单的跨平台插件来处理推送通知事件，例如 Android 和 iOS 上的注册、注销和消息到达.
    • Reachability ★25 - 在线/离线连接检查.
    • Refit ★2,762 - Xamarin 和 .NET 的自动类型安全 REST 库.
    • RestEase ★339 - 改装类固醇，更简单的身份验证，解析...
    • RestLess ★57 - .Net Standard 的自动类型安全无反射 REST API 客户端库.
    • RestSharp ★6,994 - 用于 .NET 的简单 REST 和 HTTP API 客户端.
    • Sockets ★185 - 对 .NET 和 WinRT 的套接字助手类的抽象.
    • Tiny.RestClient ★31 - 描述 Xamarin 和 .NET 的最简单的 Fluent REST 客户端.

    Reactive¶

    • Akavache ★1,692 - 本机应用程序的异步键值存储.
    • ReactiveUI ★3,917 - Rx MVVM 框架.
    • Refit ★2,762 - Refit 是一个深受 Square 的 Retrofit 库启发的库，它将您的 REST API 变成了一个实时界面.
    • ReactiveProperty ★361 - 在 Reactive Extensions 下提供 MVVM 和异步支持功能.
    • RxFlow ★18 - 带有 Rx（反应式扩展）的简单流控制库.
    • Sensors - 适用于 Xamarin 和 Windows 的 ACR 反应式传感器插件.
    • CrossPlatformLiveData - Android LiveData 启发了 .NET 实现 - 生命周期感知 rx 流.

    Security¶

    • Portable.BouncyCastle ★136 - Portable version of Bouncy Castle with support for .NET 4, .NET Standard 2.0, MonoAndroid, Xamarin.iOS, .NET Core.
    • BreachDetector ★11 - 检测 Xamarin 应用程序中的根、模拟、调试模式和其他安全问题.
    • Cryoprison - Xamarin 的越狱/根检测
    • Fingerprint Plugin ★165 - 用于访问指纹传感器的 Xamarin 和 MvvMCross 插件.
    • PCLCrypto ★184 - 可移植类库的加密（MD5，...）.
    • Permissions ★255 - 简单的跨平台插件，用于检查移动设备的连接状态、收集连接类型、带宽等.

    Serialization¶

    • Newtonsoft.Json ★5,812 - 流行的 .NET 高性能 JSON 框架.
    • Quicktype ★886 - 立即从 JSON 生成类和 JSON 序列化代码.

    TDD/BDD¶

    • FluentAssertions - TDD/BDD 流畅断言.
