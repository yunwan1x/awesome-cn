    Web Components

    Web Components the Right Way¶

    > 精选的 Web 组件资源精选列表.

    Web Components — 一套不同的技术，允许您创建可重用的自定义元素 — 将它们的功能封装在您的代码的其余部分之外 — 并在您的 Web 应用程序中使用它们.

    > Web Components the Right Way 是由 Mateus Ortiz 并由 Serhii Kulykov
