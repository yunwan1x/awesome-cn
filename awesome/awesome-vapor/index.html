    Vapor

    Awesome Vapor ¶

    Vapor 是目前最流行的服务器端Swift框架之一. 如果您曾经开发过iOS应用程序，并且使用它以全新的方式使用它，则可以采用已知的语言来开发快速，可扩展且可靠的后端系统，该系统可以轻松地与各种第三方服务集成. 这是一个精选的清单：

    -易于与Vapor集成并遵循Vapor提供简单，干净但功能强大的API的哲学的现代库； -精心编写的教程，书籍，视频和教育材料； -使开发过程更简单，更有趣的工具； - 和更多！

    How to use¶

    只需按Command + F即可搜索关键字. 如果您仅对与以下内容有关的条目感兴趣 Vapor 3 或仅 Vapor 4 ，您可以通过访问此句子中的链接来使用在filtered分支上可用的自动生成的过滤列表. 您也可以在“旧版”文件夹中找到旧的存档内容.

    Libraries¶

    • API Error Middleware –蒸气中间件，用于将引发的错误转换为JSON响应.
    • APNS –适用于iOS的蒸气APNS.
    • Bugsnag –报告错误与Bugsnag.
    • CouchDB Client –适用于Vapor的简单CouchDB客户端.
    • CrudRouter –为任何Fluent模型自动生成RESTful CRUD路由器.
    • CSRF –一个软件包，用于增加针对蒸气的保护以抵抗CSRF攻击.
    • CSV Framework –读取和写入CSV文件的简单框架.
    • Ferno – Vapor Firebase Realtime数据库提供程序.
    • Flash –在视图之间闪烁消息.
    • FluentQuery –在仍然使用Swift键路径的同时构建复杂的原始SQL查询.
    • Gatekeeper –蒸气的速率限制中间件.
    • Google Cloud Provider –与您的Vapor项目中的Google Cloud Platform API进行交互.
    • Guardian –现代限速中间件.
    • Imperial –使用OAuth提供程序的联合身份验证.
    • JWT Keychain –使用JWT for Vapor轻松搭建钥匙扣.
    • JWT Middleware –用于验证和授权蒸气中的请求的中间件.
    • Leaf Error Middleware –为蒸气应用程序服务自定义404和服务器错误页面.
    • Leaf Markdown -蒸气的Markdown渲染器.
    • Lingo Vapor – Lingo的蒸气提供商– Swift本地化库.
    • Local Storage –使用本地文件系统的存储驱动程序.
    • MailCore –通过SMTP，MailGun和SendGrid发送电子邮件.
    • Meow – MongoDB的另一种可编码ORM.
    • MongoKitten – Swift中的MongoDB驱动程序.
    • Pagination –简单的蒸气3分页.
    • Paginator –查询蒸气和流利的分页.
    • S3 –用于访问Amazon S3服务（和兼容）并支持大多数常用操作的库.
    • S3 Storage –用于轻松访问Amazon S3服务的库.
    • Sanitize –从Vapor JSON请求中强大的模型提取.
    • SendGrid Provider –由SendGrid支持的Vapor邮件后端.
    • SimpleFileLogger –蒸气的简单文件记录提供程序.
    • Slugify –方便拍打琴弦.
    • Storage –轻松使用多个存储和CDN服务.
    • Stripe Provider –蒸气的条带提供程序.
    • Submissions –创建表单和验证（表单）提交的便利.
    • Sugar –一包用于蒸气的糖.
    • SwifQL –使用纯Swift轻松构建灵活且类型安全的SQL.
    • SwiftyBeaver Provider – Vapor的SwiftyBeaver日志记录提供程序，服务器端Swift Web框架.
    • Telesign Provider –蒸气的Telesign提供商.
    • Vapor Mailgun Service –与Vapor一起使用以发送电子邮件的服务.
    • Vapor reCAPTCHA –使用Vapor验证Google reCAPTCHA.
    • Vapor Request Storage –蒸气1和2中可用的“ request.storage”的替代品.
    • Vapor Security Headers –加强蒸汽安全性标题.
    • Vapor Test Tools –帮助程序旨在使您在Vapor 3中轻松测试端点.
    • VaporExt –适用于各种Vapor数据类型和类的Swift扩展的集合.
    • WKHTMLTOPDF –通过wkhtmltopdf CLI工具从Leaf模板或网页构建PDF文件.
    • XMLCoding – XML编码器和解码器.
