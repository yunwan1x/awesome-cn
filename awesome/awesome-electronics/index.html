    Electronics

    Awesome Electronics ¶

    > 为电子工程师和业余爱好者精心挑选的精彩资源列表

    电子工程 (EE) 是理解、设计和构建电子电路的实践. 它通常与电气工程不同，因为它主要处理低功率直流电子电路而不是高功率交流系统，但电子工程和电气工程之间有很多重叠.

    试验和构建电子电路也是一种流行的爱好，许多专业资源通常同样适用于爱好者，反之亦然.

    此列表适用于网站、服务、软件、工具等：您认为在电子工程领域中一切都很棒. 如果您有任何要添加的内容，请按照以下说明进行操作 https://github.com/kitspace/awesome-electronics/blob/master/contributing.md.

    Learning¶

    Technical Tutorials¶

    • "skill" tag on learn.sparkfun.com - 有关各种 EE 相关技能的各种技术教程.
    • Soldering is Easy - 介绍焊接基础知识的漫画书，已被翻译成多种语言.
    • Uses of Different Soldering Iron Tips - 涵盖了所有这些不同的烙铁头的用途.
    • How to design a motherboard for your electronics project - 原理图和PCB设计入门教程

    Theory and Courses¶

    • Lessons In Electric Circuits - 免费的高质量教科书和工作表，重点是理论、模拟和苏格拉底方法.
    • Ultimate Electronics: Practical Circuit Design and Analysis - 由 CircuitLab（正在开发）提供的免费在线书籍，其中包含交互式原理图和模拟.
    • Khan Academy - Electrical Engineering - 提供电气工程及相关主题完整课程的非营利学习平台.
    • NEETS (Navy Electricity and Electronics Training Series) (alternative link)- 美国海军非居民培训课程材料.

    University Course Archives¶

    • Berkeley EECS - 全面的 EE 和 CS 课程网站档案.
    • Dr. Jacob Baker - 课程和教程，拉斯维加斯内华达大学教授.
    • Dr. Abraham, Dr. McDermot， 和 Dr. Valvano - Courses materials, professors at UT Austin

    Simulators¶

    Analog and Mixed Signal Circuit Simulators¶

    • LTspice - 来自 Linear Technologies 的行业标准免费 SPICE 电路模拟器. 另见非官方 LTwiki 和 Group.
    • ngspice - 开源 SPICE 电路模拟器.
    • Circuit JS/Falstad - 具有电子流可视化功能的免费开源在线模拟器（由 Paul Falstad 重写原始 Java 小程序）.
    • EveryCircuit - 免费试用在线、可视化、交互式电路模拟器，用于更简单的电路.
    • Qucs - 开源、跨平台、非基于 SPICE 的电路模拟器，具有 S 参数和谐波平衡功能.
    • QucsStudio - 免费、闭源、仅限 Windows 的 Qucs 分支，具有类似的界面、新引擎和更多功能.
    • Open Circuit Design Software - 开源、完整的 EDA 套件芯片设计套件，专注于跟上商业工具的步伐.
    • TINA-TI - 独家 DesignSoft-TINA 为德州仪器构建，与德州仪器模型捆绑在一起.
    • CppSim - 免费的开源电路模拟器，它利用 C++ 语言来实现非常快的模拟时间.
    • Scilab with Xcos - MATLAB 的免费开源数值计算替代品. Xcos 提供类似于 Simulink 的电气系统建模功能.
    • iCircuit - 易于使用的电子电路模拟器，其先进的模拟引擎可以处理模拟和数字电路，并具有实时不间断分析功能.
    • Micro-Cap - 具有多种交互式模拟类型的专业级混合信号模拟器.

    Verilog HDL Simulators¶

    • Verilator - 免费、开源的 Verilog 编译器. 测试台使用 C++ 或 SystemC. 非常快，但仅限于 2 态、基于循环的仿真，并且仅限于可综合代码.
    • Icarus Verilog - 免费的开源 verilog 解释器. 测试台采用行为 verilog. 模拟是 4 状态的，并且是基于事件的.

    Gerber Viewers¶

    Online¶

    • Tracespace Viewer - Gerber 查看器，可让您检查各个层以及电路板预览.
    • Gerblook - 由 Gerbv 提供支持的在线 Gerber 查看器.
    • Mayhew Labs 3dpcb - 3D Gerber 查看器.
    • CircuitPeople - Gerbers 没有多余的 2D 图层查看器，没有过多的处理.
    • Stackrate Viewer - 易于使用的在线 gerber 查看器，带有跟踪悬停和测量工具.

    Installable¶

    • Gerbv - 适用于 Linux 和 BSD 的优秀 Gerber 查看器.
    • KiCad GerbView - KiCad gerber 查看器.

    Free EDA Packages¶

    • KiCad - 带有推推式路由器、差分对等的开源 EDA 包.
    • Eagle - 最受欢迎的 EDA 软件包之一，因为它是（板尺寸受限）免费版本.
    • DesignSpark PCB - 无限制的免费 EDA 包，由 RS Components 赞助.
    • Altium CircuitMaker - 来自 go to pro 软件制造商的免费软件包.
    • gEDA - 另一个开源包，适合只喜欢脚本和 makefile、Linux 和 BSD 的人.
    • DipTrace - 带有（引脚和信号层受限）免费版本的质量原理图捕获和 PCB 设计软件.
    • LibrePCB - 适用于所有人、跨平台和 GNU GPLv3 的全新、强大且直观的 EDA 工具.
    • Horizon EDA - 一个免费开源的EDA工具，专注于快捷操作.

    CAD Specific¶

    KiCad¶

    • Xesscorp's list of KiCad 3rd party tools
    • Contextual Electronics' Shine on You Crazy KiCad - 初学者视频教程，可让您尽快了解制造的电路板.
    • Contextual Electronics' Getting to Blinky Tutorial - 更全面的初学者到中级视频教程.
    • KiCad.info Forums - 用户讨论和帮助论坛.
    • Keyboard PCB Guide - 全面的书面教程，带您创建键盘 PCB.
    • Cheatsheet （还 in landscape) - 简短的 PDF，涵盖最常见操作的菜单和键盘快捷键.
    • Footprint Collection - 收集所有在线可用的 KiCad 足迹和一些用于管理它们的脚本.
    • InteractiveHtmlBom - 用于手动拾取和放置的 html BOM 生成工具.
    • kicad-exports - 为任何 KiCAD 项目自动生成文件（原理图、gerbers、BoM、绘图、3D 模型）. 使用 Github Actions 之类的 CI 在本地或每个 git push 上运行它.

    Eagle¶

    • List of ULPs everyone should know
    • Adafruit Eagle Library
    • SparkFun Electronics Eagle Libraries

    PCB Syntax Highlighting¶

    • Language PCB - 各种 PCB 格式的语法高亮显示.

    PCB Batching Services¶

    • PCBShopper - 相当多的不同PCB批量和组装服务的比较服务.
    • OSH Park - 低成本 PCB 批量服务，提供带有标志性紫色丝网印刷的高质量电路板.
    • Aisler - 在欧洲（德国）制造和发货的经济实惠的优质电路板.
    • Dirty PCBs - 低成本的 PCB 批处理服务，以其“肮脏”的质量而自豪.
    • JLCPCB - 低成本 PCB 批量服务和内部低成本 SMT 服务.

    Part Search Engines¶

    • Octopart - 可能是最著名的零件搜索引擎.
    • Findchips - 从 Supply Frame 搜索零件.
    • Parts.io - Supply Frame 的另一个搜索引擎，旨在发现新零件.

    Project Sharing Platforms¶

    • Kitspace - 项目共享网站，可帮助您购买零件和重建项目. 开源并真正由您开发.
    • Hackaday.io - 用于分享热门博客项目的社交网站.
    • Hackster.io - 另一个用于共享项目的社交网站. 按平台、主题和产品组织良好.
    • InventHub - 基于 Git 的项目托管和协作平台，用于硬件开发.
    • CADLAB - 另一个用于硬件开发的基于 Git 的项目托管和协作平台.

    Inventory Management and Purchasing¶

    • 1-click BOM - 自动购买和零件搜索的浏览器扩展.
    • PartsBox - Web 服务通过漂亮的用户界面和 Octopart 集成来管理您的零件库存.
    • PartKeepr - 用于通过参数搜索和自动数据表导入管理零件库存的开源 Web 服务.
    • Part-DB - 另一个开源 Web 服务，用于使用许可系统和良好的条码生成器管理零件库存.

    Miscellaneous Web Services¶

    • Eyrie - 用于在线查看 Eagle 和 KiCad 设计.
    • SnapEDA - 带有免费符号和足迹的零件库. （兼容 Eagle、KiCad、Altium、OrCad、Allegro 等）

    Development Board Retailers¶

    • Sparkfun - 开源电子开发板和其他设备和材料的零售商和设计师，提供出色的随附教程.
    • Adafruit - 另一家拥有出色选择和教程的零售商和设计师.
    • Tindie - 电子产品制造商销售小批量自己设计的市场.

    Blogs¶

    • Hackaday - 可能是最流行的博客，涵盖了全体作家的电子和硬件黑客攻击.
    • bunniestudios.com - Andrew 'Bunnie' Huang 涵盖硬件黑客、开放硬件、制造等.
    • Bald Engineer - James Lewis 撰写的有关电子和嵌入式软件的项目日志、教程和文章.
    • Rheingold Heavy - 更多关于电子和嵌入式软件的项目日志、教程和文章，这些是由 Dan Hienzsch 撰写的.
    • Hackster.io - 另一个涵盖电子产品的博客.
    • Dangerous Prototypes - 关于开源硬件项目和有趣的应用笔记的博客.
    • N-O-D-E - 关于 DIY 电子产品、硬件和技术的博客.

    Forums¶

    Discussion¶

    • EEVBlog forum - 可能是讨论电子工程主题的最大和最活跃的论坛.
    • /r/electronics 和 /r/ECE 是 EE 主题的两个最活跃的 sub-reddit.

    Help¶

    • /r/askelectronics - 子 reddit 致力于帮助电子主题.
    • Electronics Stack Exchange - 在流行的 Stack Overflow 服务上运行的电子产品的问答网站.
    • EEVBlog beginners forum - 初学者提问的好地方，EEVblog上的其他子论坛应该适合更高级主题的问题.

    Podcasts¶

    • The Amp Hour - 与 Chris Gammel 和 Dave Jones (EEVBlog) 关于电子产品的即兴聊天，经常与客人
    • Embedded.fm - Christopher 和 Elecia White 经常与客人讨论嵌入式系统开发等等.
    • The Spark Gap Podcast - 每集涵盖一个特定的 EE 主题，有时还有嘉宾.
    • MacroFab Engineering Podcast - 每周播客，MacroFab 的 Parker 和 Stephen 讨论 EE 主题和行业新闻.
    • The Engineering Commons Podcast - 涵盖从机械到电气的一般工程主题.

    Videos¶

    • EEVblog - 最早和最成功的 YouTube 频道之一，Dave Jones 在这里进行拆解、教程等.
    • BigClive - YouTube channel 关于拆解（包括危险产品）、电路逆向工程和教程.
    • ElectroBOOM - YouTube 频道，通过大量喜剧来揭穿和解释 EE 主题.
