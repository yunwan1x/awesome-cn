    Awesome db tools

    Awesome Database Tools ¶

    > 社区驱动的数据库工具列表

    在这里，我们将收集有关可简化 DBA、DevOps、开发人员和普通人使用数据库的出色有用和出色实验工具的信息.

    随意添加有关您自己的 db-tools 或您喜欢的第三方 db-tools 的信息.

    IDE¶

    • AnySQL Maestro - 用于数据库管理、控制和开发的首要多用途管理工具.
    • Aqua Data Studio - Aqua Data Studio 是面向数据库开发人员、DBA 和分析师的生产力软件.
    • Database .net - 多数据库管理工具，支持 20 多个数据库.
    • DataGrip - JetBrains 用于数据库和 SQL 的跨平台 IDE.
    • DBeaver - 免费的通用数据库管理器和 SQL 客户端.
    • dbForge Studio for MySQL - 用于 MySQL 和 MariaDB 数据库开发、管理和管理的通用 IDE.
    • dbForge Studio for Oracle - 用于 Oracle 管理、管理和开发的强大 IDE.
    • dbForge Studio for PostgreSQL - 用于管理和开发数据库和对象的 GUI 工具.
    • dbForge Studio for SQL Server - 强大的集成开发环境，用于 SQL Server 开发、管理、管理、数据分析和报告.
    • dbKoda - 现代（JavaScript/Electron 框架），MongoDB 的开源 IDE. 它具有支持对 MongoDB 数据库进行开发、管理和性能调整的功能.
    • IBExpert - Firebird 和 InterBase 的综合 GUI 工具.
    • HeidiSQL - 用于管理 MySQL、MSSQL 和 PostgreSQL 的轻量级客户端，用 Delphi 编写.
    • MySQL Workbench - MySQL Workbench 是面向数据库架构师、开发人员和 DBA 的统一可视化工具.
    • Navicat - 一种数据库开发工具，允许您从单个应用程序同时连接到 MySQL、MariaDB、SQL Server、Oracle、PostgreSQL 和 SQLite 数据库.
    • Oracle SQL Developer - Oracle SQL Developer 是一个免费的集成开发环境，可简化传统和云部署中 Oracle 数据库的开发和管理.
    • pgAdmin - The most popular and feature rich Open Source administration and development platform for PostgreSQL, the most advanced Open Source database in the world.
    • pgAdmin3 - 长期支持 pgAdmin3.
    • PL/SQL Developer - 专门针对 Oracle 数据库存储程序单元开发的 IDE.
    • PostgreSQL Maestro - 完整而强大的 PostgreSQL 数据库管理、管理和开发工具.
    • Toad - Toad 是面向开发人员、管理员和数据分析师的首要数据库解决方案. 使用单个数据库管理工具管理复杂的数据库更改.
    • Toad Edge - 简化的 MySQL 和 Postgres 数据库开发工具.
    • TOra - TOra 是用于 Oracle、MySQL 和 PostgreSQL dbs 的开源 SQL IDE.
    • Valentina Studio - 免费创建、管理、查询和探索 Valentina DB、MySQL、MariaDB、PostgreSQL 和 SQLite 数据库.

    GUI Managers/Clients¶

    • Adminer - 单个 PHP 文件中的数据库管理.
    • Beekeeper Studio - 开放源代码 SQL 编辑器和数据库管理器，在其使命声明中承诺隐私.
    • DbVisualizer - 面向开发人员、DBA 和分析师的通用数据库工具.
    • HouseOps - 企业 ClickHouse Ops UI 可让您运行查询、监控 ClickHouse 运行状况并让很多其他人思考.
    • JackDB - 直接 SQL 访问您的所有数据，无论数据位于何处.
    • Jailer - 数据库子集和关系数据浏览工具/客户端.
    • OmniDB - 用于数据库管理的 Web 工具.
    • Pgweb - 基于 Web 的 PostgreSQL 数据库浏览器，用 Go 编写，适用于 macOS、Linux 和 Windows 机器.
    • phpLiteAdmin - 基于 Web 的 SQLite 数据库管理工具，用 PHP 编写，支持 SQLite3 和 SQLite2.
    • phpMyAdmin - MySQL 和 MariaDB 的 Web 界面.
    • psequel - PSquel 提供了一个干净简单的界面，让您可以快速执行常见的 PostgreSQL 任务.
    • PopSQL - 适合您团队的现代协作 SQL 编辑器.
    • Postico - 适用于 Mac 的现代 PostgreSQL 客户端.
    • Robo 3T - Robo 3T（原Robomongo）是一个以shell为中心的跨平台MongoDB管理工具.
    • Sequel Pro - Sequel Pro 是一款快速、易于使用的 Mac 数据库管理应用程序，用于处理 MySQL 和 MariaDB 数据库.
    • SQL Operations Studio - 一种数据管理工具，支持从 Windows、macOS 和 Linux 使用 SQL Server、Azure SQL DB 和 SQL DW.
    • SQLite Expert - 图形界面支持所有 SQLite 功能.
    • sqlpad - 基于 Web 的 SQL 编辑器在您自己的私有云中运行.
    • SQLPro - 适用于 macOS 的简单、强大的 Postgres 管理器.
    • SQuirreL - 用 Ja​​va 编写的图形 SQL 客户端，可让您查看 JDBC 兼容数据库的结构、浏览表中的数据、发出 SQL 命令等.
    • SQLTools - VSCode 的数据库管理.
    • SQLyog - 最完整且易于使用的 MySQL GUI.
    • Tabix - SQL 编辑器和 Clickhouse 的开源简单商业智能.
    • TablePlus - 适用于关系数据库的现代、原生且友好的 GUI 工具：MySQL、PostgreSQL、SQLite 等.
    • TeamPostgreSQL - PostgreSQL Web 管理 GUI - 从任何地方使用您的 PostgreSQL 数据库，具有丰富、闪电般快速的 AJAX Web 界面.

    CLI tools¶

    • ipython-sql - 连接到数据库以在 IPython 或 IPython Notebook 中发出 SQL 命令.
    • iredis - 具有自动完成和语法高亮显示的 Redis Cli.
    • pgcenter - 顶级的 PostgreSQL 管理工具.
    • pg_activity - PostgreSQL 服务器活动监控的顶级应用程序.
    • pg_top - 'top' 用于 PostgreSQL.
    • pspg - Postgres 寻呼机.
    • SQLcl - Oracle SQL Developer 命令行 (SQLcl) 是用于 Oracle 数据库的免费命令行界面.
    • usql - PostgreSQL、MySQL、Oracle 数据库、SQLite3、Microsoft SQL Server 的通用命令行界面， and many other databases 包括 NoSQL 和非关系型数据库！

    dbcli¶

    • athenacli - AthenaCLI 是 AWS Athena 服务的 CLI 工具，可以执行自动完成和语法突出显示.
    • litecli - 用于 SQLite 数据库的 CLI，具有自动完成和语法突出显示功能.
    • mssql-cli - 具有自动完成和语法突出显示功能的 SQL Server 命令行客户端.
    • mycli - 具有自动完成和语法突出显示功能的 MySQL 终端客户端.
    • pgcli - 具有自动补全和语法高亮功能的 Postgres CLI.
    • vcli - 具有自动完成和语法突出显示功能的 Vertica CLI.

    DB-schema navigation and visualization¶

    • dbdiagram.io - 快速简单的工具，可帮助您使用简单的 DSL 语言绘制数据库关系图并快速流动.
    • ERAlchemy - 实体关系图生成工具.
    • SchemaCrawler - 免费的数据库模式发现和理解工具.
    • Schema Spy - 将您的数据库生成为 HTML 文档，包括实体关系图.
    • tbls - 用于记录数据库的 CI 友好工具，用 Go 编写.

    Modelers¶

    • Navicat Data Modeler - 强大且经济高效的数据库设计工具，可帮助您构建高质量的概念、逻辑和物理数据模型.
    • Oracle SQL Developer Data Modeler - Oracle SQL Developer Data Modeler 是一个免费的图形工具，可提高生产力并简化数据建模任务.
    • pgmodeler - 专为 PostgreSQL 设计的数据建模工具.

    Migration tools¶

    • 2bass - 利用幂等 DDL 脚本概念的数据库配置即代码工具.
    • flyway - 数据库迁移工具.
    • gh-ost - MySQL 的在线架构迁移.
    • liquibase - 用于跟踪、管理和应用数据库架构更改的独立于数据库的库.
    • migra - 与 diff 类似，但用于 PostgreSQL 模式.
    • node-pg-migrate - 专为 postgres 构建的 Node.js 数据库迁移管理. （但也可以用于其他符合 SQL 标准的数据库 - 例如 CockroachDB.）
    • Prisma Migrate - 声明式数据库架构迁移工具，使用声明式数据建模语法来描述您的数据库架构.
    • Pyrseas - 提供将 PostgreSQL 数据库架构描述为 YAML 的实用程序.
    • SchemaHero - 用于声明式数据库模式管理的 Kubernetes 操作符（用于数据库模式的 gitops）.
    • Sqitch - 明智的数据库原生变更管理，用于无框架开发和可靠部署.
    • yuniql - 另一个使用原生 .NET Core 3.0+ 制作的架构版本控制和迁移工具，希望能更好.

    Code generation tools¶

    • ddl-generator - 从表数据推断 SQL DDL（数据定义语言）.
    • scheme2ddl - 用于将 Oracle 模式导出到一组 ddl init 脚本的命令行实用程序，能够过滤不需要的信息、在不同文件中分离 DDL、漂亮的格式输出.

    Wrappers¶

    • DreamFactory - 用于移动、Web 和 IoT 应用程序的开源 REST API 后端.
    • FBSQL Frontend Backend SQL - 在 HTML 中使用您的后端数据库（安全）.
    • Hasura GraphQL Engine - 具有细粒度访问控制的 Postgres 上快速、即时的实时 GraphQL API，还可以触发数据库事件的 webhook.
    • jl-sql - 用于 JSON 和 CSV 流的 SQL.
    • mysql_fdw - 用于 MySQL 的 PostgreSQL 外部数据包装器.
    • Oracle REST Data Services - 作为中层 Java 应用程序，ORDS 将 HTTP(S) 动词（GET、POST、PUT、DELETE 等）映射到数据库事务并返回使用 JSON 格式化的任何结果.
    • Prisma - Prisma 将您的数据库变成实时 GraphQL API.
    • PostgREST - 适用于任何 Postgres 数据库的 REST API.
    • prest - 是一种从任何用 Go 编写的数据库提供 RESTful API 的方法.
    • restSQL - 带有 Java 和 HTTP API 的 SQL 生成器，使用带有 XML 或 JSON 序列化的简单 RESTful HTTP API.
    • resquel - 轻松将您的 SQL 数据库转换为 REST API.
    • Saltcorn - 用于 Web 数据库应用程序的开源无代码构建器. 服务器和拖放式 UI 构建器，数据存储在 PostgreSQL 或 SQLite 中.
    • sandman2 - 为您的旧数据库自动生成 RESTful API 服务.
    • sql-boot - 用于 SQL 查询的高级 REST 和 UI 包装器.

    Backup tools¶

    • pgbackrest - 可靠的 PostgreSQL 备份和恢复.
    • BaRMan - PostgreSQL 的备份和恢复管理器.

    Replication/Data operation¶

    • Datasette - 用于探索和发布数据的工具.
    • dtle - MySQL 的分布式数据传输服务.
    • pgsync - 在数据库之间同步 Postgres 数据.
    • pg_chameleon - 用 Python 3 编写的 MySQL 到 PostgreSQL 副本系统.该系统使用库 mysql-replication 从 MySQL 中提取行图像，这些图像作为 JSONB 存储到 PostgreSQL 中.
    • PGDeltaStream - 一个 Golang 网络服务器，使用 Postgres 逻辑解码功能，通过 websockets 至少一次流式传输 Postgres 更改.
    • repmgr - 最流行的 PostgreSQL 复制管理器.

    Scripts¶

    • pgx_scripts - 一组有用的数据库分析和管理小脚本，由我们的 PostgreSQL Experts 团队创建.
    • pgsql-bloat-estimation - 用于测量 PostgreSQL 索引和表中统计膨胀的查询.
    • pgWikiDont - 检查您的数据库是否遵循以下规则的 SQL 测试https://wiki.postgresql.org/wiki/Don't_Do_This .
    • pg-utils - 有用的 PostgreSQL 实用程序.
    • Postgres cheat sheet - 有用的 SQL 脚本和命令 .
    • postgres_dba - Postgres DBA 和所有工程师缺少的一组有用工具.
    • postgres_queries_and_commands.sql - Useful PostgreSQL Queries and Commands.
    • TPT - 这些 sqlplus 脚本用于 Oracle 数据库性能优化和故障排除.

    Monitoring/Statistics/Perfomance¶

    • ASH Viewer - 提供 Oracle 和 PostgreSQL DB 中活动会话历史数据的图形视图.
    • Monyog - 无代理且经济高效的 MySQL 监控工具.
