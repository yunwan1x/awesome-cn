    Awesome deno

    Awesome Deno ¶

    Deno是使用V8并内置于Rust的JavaScript和TypeScript的简单，现代且安全的运行时.

    该列表是最好的Deno模块和资源的集合.

    Docs¶

    Official Docs¶

    • Official Site
    • Deno API Reference
    • Deno Unstable API Reference
    • Deno Manual

    External Docs¶

    • V8 Docs for Deno
    • DenoBeginner.com -面向初学者的deno完全免费的速成课程.

    Online Playgrounds¶

    • deno.town
    • Deno Playground
    • maman/deno-playground
    • DenoBR Playground
    • myCompiler.io
    • Repl.it

    Modules¶

    NOTICE：Deno有一些官方模块，可以在以下位置找到 deno_std. 考虑提交给 deno.land/x 注册表.

    Automation¶

    • autopilot -自动驾驶-使用Deno进行跨平台的Web自动化.

    CLI utils¶

    • cac -用于构建命令行应用程序的简单但功能强大的框架.
    • charmd -用于终端的简单，可扩展的markdown渲染器.
    • chart -控制台ASCII折线图，无依赖性.
    • cli-spinner -在执行长任务时，在终端中显示微调框.
    • cliffy -用于构建交互式命令行工具的完整解决方案.
    • kia -用于Deno的简单终端旋转器
    • terminal_images -用于在终端中显示图像的Deno模块和CLI工具.

    Database¶

    • deno_mysql -MySQL数据库驱动程序.
    • deno_mongo -MongoDB数据库驱动程序.
    • redis -Redis客户端的Deno的实验实现.
    • denodb -用于Deno的MySQL，SQLite，MariaDB，PostgreSQL和MongoDB ORM.
    • dndb -为Deno编写的持久性和可嵌入的NoSQL数据库引擎.
    • dsddb -使用JSON格式的简单持久性键值数据库.
    • dso -一个基于mysql的简单ORM库.
    • postgres -PostgreSQL数据库驱动程序.
    • sql-builder -一个SQL查询生成器.
    • maxminddb -支持使用MaxmindDB geoIP数据库文件的库

    Frontend development¶

    • postcss -使用JS插件转换样式的工具.

    Frontend framework¶

    • aleph.js -Deno的React框架，灵感来自 Next.js.

    Logging¶

    • gardens -到处都是JavaScript的有用日志记录实用程序.

    Mail¶

    • deno-smtp -deno的smtp邮件发件人.

    Markdown¶

    • marked -Markdown到HTML转换器.

    String utils¶

    • camelcase -将破折号/点号/下划线/空格分隔的字符串转换为camelCase：foo-bar→fooBar.
    • deno_case_style -用于不同大小写样式的字符串验证器和格式化程序. 例如：camelCase等.
    • deno-prettystring -格式化，修剪和删除字符串中字符之间的多余空格.
    • deno-slugify -deno的字符串节流器.
    • normalize_diacritics -删除字符串中的重音符号/变音符号.

    Template engine¶

    • dejs -用于Edeno的Ejs模板引擎.
    • deno_tiny_templates -deno的模板渲染器.
    • eta -快速，轻巧和可配置的嵌入式模板引擎.

    Testing¶

    • deno-pretty-assert -色彩艳丽的assert等于deno.
    • deno-puppeteer -提供高级API的库，可通过DevTools协议控制Chromium或Chrome.
    • expect -编写笑话的助手，如deno中的期望测试.
    • merlin -deno️的测试和基准测试框架
    • pretty_benching -一个小型图书馆，可让您的Deno基准测试进度和结果看起来很漂亮.
    • rhum -用于Deno的轻量级测试框架.
    • superdeno -由超级代理驱动的库，用于测试Deno HTTP服务器.
    • superoak -通过SuperDeno简化了Oak的HTTP断言.
    • tincan -用于Deno的类似于Jest的测试库.
    • unexpected -可扩展的BDD断言工具包.

    Utils¶

    • buckets -将资产和脚本捆绑在一个可执行文件中.
    • bytes_formater -格式化字节（Uint8Array，ArrayBufferView等）输出，在调试IO功能时很有用.
    • coffee -Deno配置-一种类型安全，易于使用的Deno配置管理器.
    • computed_types -Joi喜欢Typescript和Deno的验证器.
    • dcc -Deno Cache Clean，在下次运行时重新加载dep.
    • denon -具有等待生成器的文件监视程序.
    • deno-context -将截止日期，取消订单和其他请求范围的值传播给多个承诺. 行为就像Go的上下文.
    • deno_cron -cron Job调度程序，可让您编写具有大量灵活性的可读cron语法
    • deno-deamon -使Deno程序在后台运行. --deno-dotenv
    • pietvanzoen/deno-dotenv -Dotenv行动为deno.
    • cardosomarcos/deno-dotenv -从.env加载Deno项目的环境变量.
    • deno-fnparse -一个非常简单的JavaScript解析器组合器.
    • deno-globrex -globrex的端口为deno，globex的端口为正则表达式.
    • deno_notify -在所有平台上发送桌面通知.
    • deno-opn -打开网站，文件，可执行文件之类的东西. 跨平台.
    • deno-plugin-prepare -用于管理Deno本机插件依赖项的库.
    • deno_random_interval -帮助程序生成随机间隔.
    • deno_tokenizer -deno的简单标记器.
    • deno-using -带有deno语句的python样式.
    • dinoenv -小型库，用于使用deno管理环境变量.
    • ensure -确保您运行的是Deno，Typescript或V8的最低版本.
    • evt -输入EventEmitter的安全替代品.
    • fossil -值类型验证套件.
    • garn-yaml -读取或写入yaml内插env变量.
    • garn-validator -轻松创建验证.
    • invert-kv -在Deno中反转键/值对.
    • lazy -类似于linq的惰性求值迭代模块.
    • maze_generator -用于生成，求解，分析和显示迷宫的Javascript模块.
    • ms -轻松地将各种时间格式转换为毫秒.
    • online -检查您当前是否在Deno在线.
    • qrcode -Deno的QR码图像生成器.
    • recursive-readdir -在Deno中递归读取目录.
    • rubico -[a]同步功能组成； 它只是工作.
    • type-fest -基本TypeScript类型的集合（sindresorhus端口/ type-fest）.
    • wasm-gzip -为Deno加密和解密gzip.
    • watch -文件查看器.
    • wu-diff-js -差异库，使用wu（O（NP））算法计算两个切片之间的差异.

    Web framework¶

    • abc -更好的Deno框架来创建Web应用程序.
    • alosaur -Alosaur-具有许多装饰器的Deno Web框架.
    • attain -用于Deno的中间件Web框架，该框架使用受Express和Oak启发的http标准库. 快速稳定地使用适当的内存.
    • aqua -用于Deno的最小和快速的Web框架.
    • deno-express -将Node Express移植到Deno.
    • denotrain -带有中间件支持的多合一Web框架，例如Express或Fastify forNode.js.
    • dinatra -Sinatra像deno的轻量级Web应用程序框架.
    • doa -用于Deno的http服务的中间件框架. 用❤️从Koa移植
    • drash -零依赖性的Deno HTTP服务器的REST微框架.
    • dragon -⚡强大的HTTP路由器和URL匹配器，用于使用Dragon构建Deno Web服务器.
    • microraptor -轻量级框架，可通过验证轻松进行网络路由.
    • oak -用于Deno的网络服务器的中间件框架.
    • oak-http-proxy -用于Deno Oak HTTP服务器的代理中间件.
    • opine -从ExpressJS移植的快速，简约的Web框架.
    • opine-http-proxy -Deno Opine HTTP服务器的代理中间件.
    • pogo -Deno的服务器框架.
    • servest -渐进式HTTP服务器/路由器.

    WebSocket¶

    • deno-websocket -一个简单的WebSocket库，例如node.js库的ws.
    • dropper -基于事件的自定义WebSockets框架，用于在Deno上构建实时应用程序
    • websocket_server -WebSocket服务器库.
    • sockets -用于Deno的WebSocket库.

    Web utils¶

    • compression -Deno HTTP压缩中间件.
    • djwt -根据JWT和JWS规范在Deno上制作JSON Web令牌（JWT）.
    • forwarded -“转发的”图书馆的Deno端口.
    • gentleRpc -用于Deno和浏览器的JSON-RPC 2.0 TypeScript库.
    • gql -通用GraphQL HTTP中间件.
    • graphql-tag -来自模板文字的GraphQL模式AST.
    • nats -Deno客户 NATS messaging system.
    • obsidian -本地GraphQL缓存客户端和服务器模块.
    • qs -具有嵌套支持的查询字符串解析器.
    • router -高性能的基本路由器可在任何地方使用.
    • status -用于Deno的HTTP代码和状态实用程序.
    • ts-prometheus -普罗米修斯客户.
    • up -检查是否在Deno建立了网站.
    • youtube-deno -用于YouTube数据API的Deno客户端库，用于与YouTube进行任何交互.

    Webview¶

    • webview - Deno bindings for webview, a tiny library for creating web-based desktop GUIs.

    XML¶

    • deno-xml-parser -从segmentio / xml-parser移植的Deno XML解析器.
    • sax-ts -从移植的SAX样式的XML解析器 sax-js.

    Registries¶

    • crux.land -一种免费的注册表服务，用于托管小型（<10kB）单面脚本.
    • Deno PKG -在您的Deno项目中使用来自GitHub的代码的更简单方法.
    • deno.land/x/ -官方的第三方模块注册表.
    • nest.land -一个由区块链驱动的不可变的Deno软件包注册中心.

    Showcases¶

    • AuthCompanion -基于令牌的用户管理服务器.
    • Deno Rest -教义的RESTful API的样板.
    • Deno Seed -完整的样板可供开发. ：幼苗：
    • GitHub Profile Trophy -在自述文件中添加动态生成的GitHub Trophy
    • UsingDeno -使用Deno策划Web应用程序和项目的列表.

    Tools¶

    • commands -为节点js和deno创建命令快捷方式
    • clone -用于方便克隆的简单实用程序.
    • decense -使用一个“ deno run”命令生成许可证.
    • dedep -管理依赖版本.
    • denoget -Denoget安装可执行的Deno脚本.
    • denoify -对于希望支持Deno但又不想编写和维护端口的NPM模块作者.
    • denoinit -Denoinit为Deno项目生成有用的文件.
    • denoliver -具有实时重新加载功能的简单，无依赖的文件服务器.
    • denomander -Deno命令行界面灵感来自commander.js.
    • denon -守护程序脚本运行程序，例如nodemon. 内置并用于Deno.
    • denopkg -在您的Deno项目中使用来自GitHub的代码的更简单方法.
