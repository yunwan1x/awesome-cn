    Awesome IBM Cloud¶

    精选的IBM Cloud SDK，开源存储库，工具，博客和其他资源的精选列表.

    Carbon Design System¶

    附加组件，组件，图标和其他资源，以使用由IBM构建的设计系统.

    • carbon -由IBM构建的设计系统.
    • carbon-addons-angular -碳角项目的通用插件.
    • carbon-addons-ics -用于IBM协作解决方案的Carbon Add-on.
    • carbon-addons-iot-react -该库包含所有IBM IoT产品之间共享的React组件，以统一我们的外观. 这些小部件通常基于碳组分反应小部件.
    • carbon-angular-starter -一个简单的启动器应用程序，用于使用Carbon组件自举应用程序.
    • carbon-badges -申请Carbon徽章的网站.
    • carbon-charts -使用D3和打字稿实现的可重用图表.
    • carbon-components-angular -IBM Carbon Design System的Angular实施.
    • carbon-components-react -碳设计系统的反应组件.
    • carbon-components-svelte -切实执行碳设计系统.
    • carbon-components-vue -碳设计系统的Vue实施.
    • carbon-design-kit -Carbon Design System视觉资产（组件，图标，调色板，网格，模板）的版本化，全面的工具包.
    • carbon-icons -碳设计系统的SVG图标库.
    • carbon-icons-angular -碳图标，在Angular中！
    • carbon-icons-svelte -使用Carbon Design System精简数字和软件产品中图标的组件.
    • carbon-language-website -IBM设计语言网站.
    • carbon-sandbox -碳组件的主题乐园.
    • carbon-tutorial -碳设计系统教程的应用程序.

    • Cloud Foundry CLI -Cloud Foundry的官方命令行客户端.
    • The Kui Framework for Graphical Terminals -用于云原生开发的混合命令行/ UI开发经验.
    • IBM Cloud CLI Release -这是IBM Cloud的命令行客户端.
