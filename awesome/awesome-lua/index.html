    Lua

    Awesome Lua ¶

    >精选的Lua精选清单 packages 和 resources.

    灵感来自清单 awesome, awesome-awesomeness，和 awesome-nodejs.

    Packages¶

    Resources¶

    Implementations, Interpreters, and Bindings¶

    • Lua - Lua最初的ANSI C解释器.
    • Lua Repo - Lua团队看到的官方Lua回购镜像反映给GitHub.
    • LuaJIT - Lua的高性能Just-In-Time编译器.
    • LLVM-Lua - 将Lua编译为LLVM.
    • lua.vm.js - 网络上的Lua VM; 通过LLVM，emscripten和asm.js直接移植C解释器.
    • Moonshine - 用JavaScript实现的Lua VM. 比lua.vm.js慢，但有更好的文档，示例和JS接口.
    • Fengari - 使用无缝的JS和DOM互操作性在Javascript中重写Lua VM.
    • MoonSharp - A Lua interpreter written entirely in C# for the .NET, Mono and Unity platforms.
    • UniLua - A pure C# implementation of Lua 5.2, focused on compatibility with the Unity game engine.
    • lupa - 与LuaJIT2的Python绑定.
    • golua - Golang绑定到Lua C API.
    • GopherLua - Lua 5.1 VM和编译器在Go with Go API中实现.
    • LuaBridge - 一个轻量级库，用于在C ++和Lua之间来回映射数据，函数和类.

    注意：从LuaJIT到Lua再到lua.vm.js到Moonshine，一个基本的基准测试表明每次跳跃性能下降大约6倍.

    Package Managers¶

    • LuaRocks - 事实上的工具，用于将Lua模块安装为名为“rocks”的软件包，以及公共岩石存储库和网站. 很像npm或pip.

    Build Tools and Standalone Makers¶

    • Lake - 用Lua编写的构建引擎，类似于Ruby的rake.
    • Luabuild - 高度可定制的Lua 5.2构建系统.
    • luastatic - 将Lua程序转换为独立可执行文件的简单工具.
    • omnia - 包含电池的独立可执行文件的创建者，建立在luastatic之上.

    Debugging and Profiling¶

    • ProFi - 简单的分析器，与LuaJIT一起使用并生成报告文件.
    • luatrace - 用于跟踪/分析/分析脚本执行和生成详细报告的工具集.
    • StackTracePlus - 直接升级到Lua的堆栈跟踪，增加了本地环境并提高了可读性.
    • MobDebug - 具有断点和堆栈检查功能的强大远程调试器. 由ZeroBraneStudio使用.
    • lovebird - 基于浏览器的调试控制台. 最初是为LÖVE制作的，但适用于任何支持LuaSocket的项目.

    IDEs and Plugins¶

    • Lua Development Tools - Eclipse插件，提供代码完成，调试等. 建立在Metalua上.
    • Lua for IDEA - IntelliJ IDEA插件，其中包括代码完成，智能突出显示和实验调试.
    • ZeroBraneStudio - 轻量级，可定制，跨平台的Lua专用IDE，带有代码完成和分析，用Lua编写. 为众多Lua引擎提供广泛的调试支持.
    • BabeLua - VS2012-13的Lua编辑器/调试器扩展，具有突出显示，自动完成，linting和格式化功能.
    • lua-mode - 用于编辑Lua的Emacs主要模式.
    • vscode-lua - VSCode intellisense和linting.

    Utility Belts¶

    • Lua Fun - 为LuaJIT设计的高性能函数编程库.
    • Moses - 功能性编程实用带，灵感来自Underscore.js.
    • Penlight - 广泛，重量级的实用程序库，灵感来自Python的标准库. 提供Lua没有的电池.
    • lua-stdlib - 中等重量标准库扩展; 添加了一些有用的数据结构，实用程序功能和基本功能.
    • Microlight - 一个有用的Lua函数库; Penlight的'extra light'版本.
    • compat53 - 兼容性模块为Lua 5.2和5.1提供Lua-5.3样式的API.
    • RxLua - 反应性扩展，可观察性等

    Game Engines¶

    • LÖVE 2D - 桌面游戏开发平台. 跨平台，功能齐全，采用良好.
    • Corona SDK - 适用于iOS和Android的开发平台. 专有，但被众多顶级游戏和应用程序使用，总下载量超过1.5亿次.
    • MOAI - 开源，跨平台，移动游戏开发框架. 由Lua脚本编写的简约C ++引擎.
    • Drystal - 开源，游戏可以在Linux或任何具有最新Web浏览器的平台上运行.
    • Amulet - 适用于小型游戏和实验的开源，音频/视频工具包. 它可以在Windows，Mac，Linux，HTML5和iOS上运行.
    • LÖVR - 受LÖVE2D启发的用于创建虚拟现实体验的3D框架.

    Game Development¶

    • 皇冠
    • Coronium - 支持分析，数据对象，用户管理等的简单云平台.
    • 爱
    • awesome-love2d - 像这样的列表，但专注于游戏开发和LÖVE平台.
    • lurker - 通过在正在运行的LÖVE项目中自动交换已更改的Lua文件来缩短迭代周期.
    • HUMP - LÖVE的一套轻量级助手; 面向游戏的实用腰带.
    • MOAI
    • moaifiddle - 编辑和共享MOAI游戏引擎的短脚本，并使用WebGL在浏览器中运行它们.
    • Jumper - 用于基于网格的游戏的快速，轻量且易于使用的寻路库.
    • lume - 适用于游戏开发的实用带库.
    • NoobHub - 遵循简单的pub-sub模型，为Corona，LÖVE等网络多人游戏.
    • 碰撞检测
    • bump.lua - 最小的基于矩形的碰撞检测，可处理隧道和基本的碰撞分辨率.
    • HardonCollider - 检测任意类型的任意定位和旋转形状之间的碰撞.
    • 补间
    • flux - 一个快速，轻量级的Lua补间库，具有缓动功能和将补间组合在一起的功能.
    • tween.lua - 用于补间的小型库，具有多种缓动功能.
    • 例子
    • termtris - 一个俄罗斯方块克隆，以文化风格编写，“强调学习能力”.
    • PacPac - 用LÖVE制作的吃豆人克隆.
    • Mari0 - 用LÖVE制作的Mario和Portal的融合. 另见它 wikipedia entry.
    • Journey to the Center of Hawkthorne - 基于社区的2D平台游戏 Digital Estate Planning 插曲，用LÖVE制作.

    Logging¶

    • lua-log - 具有用于文件系统，网络，ZeroMQ等的可插拔编写器的异步日志记录库.
    • LuaLogging - 支持各种appender的Log4j启发式日志库.
    • luasyslog - 根据LuaLogging登录到syslog.

    Web/Networking Platforms¶

    • OpenResty - 通过使用Lua扩展Nginx创建的快速且可扩展的Web应用程序平台. 今天是事实上的Lua网络平台，大量使用Cloudflare，淘宝，腾讯等.
    • turbo - 受Tornado启发的事件驱动，非阻塞，基于LuaJIT的网络套件和框架.
    • Kepler Project - 使用一组通用标准和组件的面向Web的项目的集合.
    • Pegasus.lua - Pegasus.lua是一个http服务器，用于处理用Lua语言编写的Web应用程序.

    OpenResty¶

    • awesome-resty - 像这样的列表，但专注于OpenResty.
    • 核心平台
    • ngx_lua - OpenResty的核心部分. 在Nginx中嵌入Lua并公开了用于非阻塞套接字的cosocket API（与LuaSocket的API兼容）.
    • OpenResty GitHub Organization - ngx_lua，ngx_openresty和许多相关模块的存储库的主页.
    • 第三方模块
    • lua-resty-http - Lua HTTP客户端驱动程序，构建在cosocket API上.
    • 框架和工具
    • Lapis - Lua和OpenResty的全栈框架. 就像Lua的Django或Rails一样. 支持Moonscript.
    • ledge - Lua模块提供可编写脚本，符合RFC的HTTP缓存功能.
    • Sailor - 与OpenResty，Apache和其他Web服务器兼容的MVC Web框架.
    • Kong - 微服务和API管理层.

    在此页面中搜索“OpenResty”以查找其他类别（特别是数据存储）下的相关包.

    Command-line Utilities¶

    • ansicolors - 用于以彩色打印到控制台的简单功能.
    • cliargs - 一个简单的命令行参数解析模块.
    • lua-term - Terminal operations and manipulations.
    • argparse - 一个功能丰富的命令行解析器，受到argparse for Python的启发.

    Concurrency and Multithreading¶

    • 基于协同程序的多任务处理：
    • Lumen - 简单的并发任务调度.
    • ConcurrentLua - 实现Erlang风格的消息传递并发模型.
    • cqueues - 用于管理基于带协同程序的事件循环的套接字，信号和线程的库.
    • 多线程：
    • llthreads - 用于低级pthreads和WIN32线程的简单包装器.
    • llthreads2 - 更新的llthreads重写.
    • lanes - 库实现消息传递模型，每个Lua线程有一个OS线程.
    • luaproc - 消息传递模型，允许每个OS线程有多个线程，并且可以轻松地在网络中进行通用化. 也可以看看 the paper 它起源于哪里.

    有关差异的更多信息（特别是lanes和luaproc之间），请参阅此内容 comparison 期权; 有点过时，但涵盖了每个人的工作方式和重大差异.

    Templating¶

    • lustache - Mustache模板实现.
    • etlua - 嵌入式Lua模板，ERB风格.
    • lua-resty-template - 面向Lua的OpenResty模板引擎，有点像Jinja.

    Documentation¶

    • LDoc - 文档生成器，现代化和扩展 LuaDoc.
    • Locco - 两个端口 Docco，“快速，肮脏，百行长，文化编程风格的文档生成器”.
    • docroc - 将注释解析到Lua表中以生成文档.

    Object-oriented Programming¶

    • 30log - 极简主义的OOP库，包含30行的基本类，继承和mixin.
    • middleclass - 简单但强大的OOP库，包含继承，方法，元方法，类变量和mixins.

    File system and OS¶

    • LuaFileSystem - 扩展和补充Lua内置的文件系统功能集.
    • luaposix - POSIX API的绑定，包括curses.
    • lunix - 绑定到常见的Unix系统API，努力实现线程安全.
    • lua-path - 文件系统路径操作库.

    Time and Date¶

    • LuaDate - 具有解析，格式化，加法/减法，本地化和ISO 8601支持的日期和时间模块.
    • cron.lua - Lua的时间相关函数，受JavaScript的setTimeout和setInterval启发.
    • luatx - 时间，日期和时区库.

    Image Manipulation¶

    • magick - 使用FFI将Lua绑定到ImageMagick for LuaJIT.

    Digital Signal Processing¶

    • LuaFFT - 纯Lua中易于使用的快速傅里叶变换包.
    • Worp - 为LuaJIT编写的声音/音乐/ DSP引擎.

    Hardware and Embedded Systems¶

    • eLua - Lua，通过优化和特定功能进行扩展，实现高效便携的嵌入式软件开发.

    Math and Scientific Computing¶

    • SciLua - 基于LuaJIT构建的数字/科学计算框架，具有到R的接口.
    • Torch7 - 广泛支持机器学习算法的科学计算框架，由Facebook，Google等使用.
    • lhf's Lua Tools - 各种图书馆和工具，许多数学或数据相关.

    Parsing and Serialization¶

    • JSON
    • lua-cjson - 在C中实现快速JSON编码/解码并暴露给Lua.
    • luajson - 在Lue上实现的JSON编码器/解码器在LPeg之上.
    • dkjson - 以纯Lua实现的JSON编码器/解码器.
    • json.lua - 纯Lua中快速而小巧的JSON库.
    • XML
    • LuaExpat - 通过绑定到Expat库的SAX XML解析器.
    • SLAXML - 纯Lua SAX类似流XML解析器.
    • MessagePack
    • lua-MessagePack - Pure Lua实现MessagePack.
    • lua-cmsgpack - Redis使用的具有Lua绑定的MessagePack C实现.=
    • LPeg
    • LPeg - Lua的模式匹配库，基于解析表达式语法.
    • lpeg_patterns - 一系列LPeg模式.
    • LuLPeg - LPeg v0.12的纯Lua实现.
    • LPegLJ - LPeg v1.0的纯LuaJIT实现.
    • LPegLabel - LPeg的扩展，增加了对标记失败的支持.
    • lyaml - 通过绑定到LibYAML进行YAML编码/解码.
    • lunamark - 将Markdown转换为其他文本格式，包括HTML和LaTeX. 使用LPeg进行快速解析.
    • LXSH - 用LPeg编写的词法分析器和语法荧光笔的集合.
    • lua-pb - Protocol Buffers实现.

    Humanize¶

    • i18n.lua - 具有区域设置，格式和复数的国际化库.
    • inspect.lua - Lua表的人类可读表示.
    • serpent - 串行器和漂亮的打印机.
    • Ser - 具有良好性能的死简单序列化器.
    • say - i18n的简单字符串键值存储.

    Compression¶

    • lua-zlib - zlib for gzip / gunzip的简单流媒体接口.
    • lua-zip - Lua绑定到libzip. 读取和写入zip文件.

    Cryptography¶

    • LuaCrypto - Lua绑定到OpenSSL.
    • lua-lockbox - 用纯Lua编写的加密原语的集合.
    • luatweetnacl - 绑定到tweetnacl，现代高安全性加密库.
    • luaossl - “Lua宇宙中最全面的OpenSSL模块” - 由lapis，kong和lua-http使用.

    Network¶

    • LuaSocket - 网络扩展，为TCP和UDP提供套接字API，并实现HTTP，FTP和SMTP.
    • lua-websockets - WebSocket客户端和服务器模块. 与Web服务器无关，在Lua的LuaSocket上实现.
    • lua-cURLv3 - Lua绑定到libcurl.
    • lua-http - 具有客户端和服务器API，TLS和HTTP / 2的异步HTTP和WebSocket库; 基于cqueues.

    Data Stores¶

    • LuaSQL - 用于连接ODBC，ADO，Oracle，MySQL，SQLite和PostgreSQL的简单界面.
    • pgmoon - 用于OpenResty，LuaSocket和cqueues的Lua PostgreSQL驱动程序.
    • lua-resty-mysql - 用于OpenResty的Lua MySQL驱动程序.
    • lua-resty-cassandra - Lua Cassandra OpenResty和其他人的客户端驱动程序.
    • Redis
    • redis-lua - Redis的Pure Lua客户端库.
    • lua-resty-redis - Lua Redis OpenResty的客户端驱动程序.
    • lredis - 具有流水线和Pub / Sub支持的异步Redis客户端; 基于cqueues.

    Message Brokers¶

    • lua-zmq - Lua绑定到ZeroMQ.
    • lzmq - 更新的Lua绑定到ZeroMQ.
    • lua-resty-kafka - 基于OpenResty cosockets的Kafka客户端驱动程序.
    • lua-resty-rabbitmqstomp - 基于OpenResty cosockets的RabbitMQ客户端库.

    Testing¶

    • busted - BDD风格的单元测试框架，具有出色的文档和Moonscript支持.
    • telescope - 灵活且高度可定制的测试库.
    • luassert - 断言库扩展了Lua的内置断言.
    • lust - 最小的测试框架.

    Foreign Function Interfaces¶

    • LuaJIT FFI - LuaJIT调用外部C函数和使用纯Lua代码中的C数据结构的机制.
    • luaffi - 独立FFI库，与LuaJIT FFI接口兼容.

    Analysis Tools and ASTs¶

    • luadec51 - Lua版本5.1的Lua Decompiler.
    • luacov - 简单的覆盖分析仪，被破坏和望远镜用于检查测试覆盖范围.
    • luacov-coveralls - LuaCov记者为coveralls.io.
    • luacheck - 简单的静态分析仪，可以检测偶然的全局变量和未定义或阴影的局部变量.
    • Metalua - 纯Lua解析器和编译器，用于生成AST. 许多其他工具以这种方式使用Metalua解析器.
    • LuaInspect - Lua最强大的代码分析和linting工具，基于Metalua构建. 由ZeroBraneStudio等人使用.
    • LuaMinify - Minifier，它还带有自己的静态分析工具，词法分析器和解析器.
    • Typed Lua - 一个Lua的打字超集，编译成普通的Lua.
    • lua-parser - 使用LPegLabel编写的Lua 5.3解析器，具有改进的错误消息.

    Experimental, etc¶

    • punchdrunk.js - Moonshine +LÖVEAPI重新实现=在浏览器中运行LÖVE游戏.
    • luvit - Node.js的底层架构（libUV），顶部是Lua而不是JavaScript.
    • graphql-lua - Lua的实现 GraphQL.

    Scriptable by Lua¶

    • luakit - Lua可扩展的基于webkit的快速，小型，基于Webkit的浏览器框架.
    • Hammerspoon - 功能强大，可扩展的OS X自动化工具. 社区维护的分支 Mjolnir.
    • kpie - 用于处理窗口的脚本实用程序.
    • lumail - 基于控制台的邮件客户端，具有广泛的脚本功能.
    • AwesomeWM - 一个高度可配置和可扩展的X窗口管理器，由Lua编写和配置.
    • Textadept - 非常轻量级，可定制的跨平台编辑器，（主要）编写（由Lua编写）.
    • KoReader - 电子书阅读器应用程序支持PDF，DJVU，EPUB，FB2等，可在Kindle，Kobo，PocketBook和Android设备上运行.

    Miscellaneous¶

    • MoonScript - Moonscript是一种动态脚本语言，可编译为Lua. 它减少了冗长，并提供了丰富的功能，如理解和类. 它的作者称它为'CoffeeScript for Lua'.
    • sitegen - 一个静态站点生成器，它使用MoonScript并支持HTML和Markdown，页面分组和插件.
