    Preact

    Awesome Preact ¶

    >精选的有关以下方面的令人敬畏的事情 Preact 生态系统：star2：

    Preact 是具有相同ES6 API的3kb快速React替代方案. 组件和虚拟DOM.

    • Real Apps

    Community¶

    • Slack （讨论论坛）
    • Stack Overflow
    • Github
    • Twitter

    Toolkits¶

    • Preact CLI -在几秒钟内构建一个Preact Progressive Web App.
    • Vite -用于Preact，Vue或React的快速本机ESM驱动的Web开发构建工具.
    • nwb -使用React，Inferno或Preact进行快速开发.
    • React App Rewire Preact -结合使用Preact和create-react-app而不弹出.
    • Preact CLI PostCSS -它会删除Preact CLI上的默认postcss配置，因此您可以使用postcss.config.js.
    • Create Preact App -创建没有构建配置的Preact应用.
    • Storybook Preact -Storybook for Preact是用于Preact组件的UI开发环境.

    Boilerplates¶

    • Official Boilerplate -由Webpack支持的准备就绪的Preact入门项目.
    • Preact Simple Starter -具有Preact，Preact-mdl和Webpack2的PWA Simple Starter.
    • Preact Offline Starter -Webpack2样板，用于使用Preact构建SPA / PWA /离线前端应用程序.
    • TypeScript Preact Starter -使用TypeScript进行Preact的准系统入门项目.
    • TypeScript PWA Preact Starter -具有TypeScript和SASS的PWA Starter（131kb）
    • Preact Redux SSR Example -使用Redux示例的服务器端渲染.
    • Preact PWA -PWA专注于原始性能，服务器端渲染，预渲染，redux，express，汇总.
    • Preact Boilerplate -由Parcel提供支持的绝对最小的Preact入门项目.
    • Preact Chrome Extension -功能齐全的Preact Chrome扩展入门套件

    Routing¶

    • Preact Router -Preact的URL路由器.
    • Preact Route Async -（440b gzip）路由组件，可异步加载页面组件.
    • Wouter -微型（1KB gzip）路由器，用于使用类似React Router的API进行Preact / React.

    Components¶

    • Preact Material Components -“网络材料组件”的预先包装器.
    • Preact Scroll Header -一个（800b gzip）标头，将在滚动Preact时显示/隐藏.
    • Preact Progress -Preact的简单轻量级（约590字节gzip）进度条组件.
    • Preact Compat -使用带有Preact （的任何React库full example).
    • Preact Render To String -通用渲染.
    • Preact Markup -将HTML和自定义元素呈现为JSX和组件.
    • Preact Portal -将Preact组件渲染到（a）SPACE中.
    • Preact Richtextarea -简单的HTML编辑器组件.
    • Preact Token Input -用于标记输入的文本字段，用于标记之类的东西.
    • Preact Virtual List -轻松呈现具有数百万行的列表（demo).
    • Preact Cycle -Preact的功能反应范例.
    • Preact Layout -小而简单的布局库.
    • Preact Socrates -的Preact插件 Socrates.
    • Preact Flyd - 用 flyd Preact + JSX中的FRP流.
    • Preact I18nline -整合周围的生态系统 i18n-js 通过Preact通过 i18nline.
    • Preact MUI -MUI CSS Preact库.
    • Preact MDL - 用 MDL 作为Preact组件.
    • Preact Photon -使用构建漂亮的桌面用户界面 photon.
    • Preact Classless Component -创建没有class关键字的preact组件.
    • Preact Hyperscript -用于创建元素的类似于超脚本的语法.
    • Shallow Compare -简化的shouldComponentUpdate帮助器.
    • Preact Codemod -将您的React代码转换为Preact.
    • Preact Helmet -Preact的文档主管.
    • Preact Delegate -委托DOM事件.
    • Preact No SSR -跳过组件的服务器端渲染.
    • Preact Head -独立的声明式\<Head /> 对于Preact.
    • Preact Side Effect -创建其嵌套属性更改会映射到全局副作用的组件.
    • Preact Tiny Atom -预先整合 Tiny Atom.
    • Preact Level List -实时更新Preact的leveldb列表组件.
    • Preact Country Picker -基于为Preact制作的引导程序3的国家/地区选择器.
    • Preact Fluid -Preact的最小UI套件.
    • Preact Feather Icons -Preact的羽毛图标.
    • Preact Animate On Change -当属性更改时添加CSS3动画.
    • Preact Async Route -preact-router的异步路由组件.
    • MU Forms -（P）React的失效简单表单库.
    • Pimg -渐进式图像组件； 用于延迟加载图像.
    • Preact Component Console -控制台模拟器. 通过动态延迟模拟打字.
    • Preact Translate -Preact的简约翻译（i18n）库.
    • Preact Dock -适用于Preact应用程序的简单DnD和启用触摸功能的Dock.
    • Preact Particles -轻巧的组件，可轻松将酷炫的粒子动画添加到网站.
    • Pant -基于Preact （docs and demos ）. 从Vue的出色Vue组件移植而来 Vant.
    • Preact Transitioning -公开Preact组件，可轻松实现基本的CSS动画和过渡.

    Libraries¶

    • Redux Zero -基于Redux的轻量级状态容器，具有单个存储且没有reducer.
    • Unistore -350b / 650b状态容器，具有用于Preact＆React的组件操作.
    • FPreact -提供了一个替代的api，用于创建preact组件，受到elm的启发.
    • ProppyJS - A tiny library for functional props composition
    • ClearX -零学习曲线的React，Preact和Inferno的快速轻松状态管理.
    • Preact-urql - 用 urql 带有Preact核心+挂钩.
    • hooked-head -用来操纵的钩子<head> DOM部分. 它有一个带有核心preact支持的子包（使用preact / hooks`）.

    Testing Utils¶

    • Preact JSX Chai -JSX断言测试_（没有DOM，就在Node中）_.
    • Preact Render Spy -使Preact组件可以访问产生的虚拟dom进行测试.
    • Preact Test Utils -预先模拟反应-测试-利用酶.
    • Preact Testing Library -简单而完整的Preact DOM测试实用程序，可鼓励良好的测试实践.
