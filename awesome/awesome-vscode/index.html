    Lint and IntelliSense¶

    如果尚未释放出令人敬畏的 linting 必杀技： > lint 是最初赋予特定程序的名称，该程序标记了 C 语言源代码中的一些可疑和不可移植的结构（可能是错误）. 该术语现在普遍应用于标记以任何计算机语言编写的软件中的可疑用法的工具.

    与其他一些编辑器不同，VS Code 支持 IntelliSense、linting、out-of-the-box 并且不需要任何单独的扩展来运行 linter 包. VS Code 中已经集成了一些 linter，您可以在官方文档中找到完整列表， Languages 部分.
